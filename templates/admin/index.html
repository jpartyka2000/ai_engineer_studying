Management Commands Reference

Question Generation

# Analyze coverage (see what's missing)

python manage.py generate_interview_questions <subject> --analyze-only

# Generate 10 questions to fill gaps

python manage.py generate_interview_questions <subject> --num 10

# Generate until full coverage

python manage.py generate_interview_questions <subject> --complete
Export / Import Questions

# Export all subjects to questions/

python manage.py export_questions --all

# Export single subject

python manage.py export_questions <subject>

# Import from questions/ directory

python manage.py import_question_export --input-dir questions
Quick Actions (Admin)

From the Subjects list, select subjects and use the action dropdown:

  • Analyze question coverage - See covered/missing topics
  • Generate 10 interview questions - Quick generation
  • Generate until full coverage - LLM fills all gaps
  • Export questions to JSON - Export to questions/ directory

Or click "Generate Questions" on any subject for detailed options.

Full documentation: COMMANDS.md in project root
