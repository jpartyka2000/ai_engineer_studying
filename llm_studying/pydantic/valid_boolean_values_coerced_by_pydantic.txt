PROMPT:
What does this code print?

from pydantic import BaseModel

class User(BaseModel):
    age: int
    active: bool

user = User(age="25", active="yes")
print(user.age, type(user.age))
print(user.active, type(user.active))

OPTIONS:
A) 25 <class 'int'> / True <class 'bool'>
B) "25" <class 'str'> / "yes" <class 'str'>
C) 25 <class 'int'> / ValidationError
D) ValidationError (cannot convert "25" to int)

YOUR ANSWER: A
CORRECT ANSWER: C

EXPLANATION:
- "25" → 25 works (string to int coercion is supported)
- "yes" → ValidationError (not a valid bool input)

Pydantic v2 bool coercion only accepts:
- True, False (actual booleans)
- 1, 0 (integers)
- "true", "false", "1", "0" (strings, case-insensitive)

"yes", "no", "on", "off" are NOT valid boolean inputs.
