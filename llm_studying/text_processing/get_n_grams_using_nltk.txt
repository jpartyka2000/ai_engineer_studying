To get all possible trigrams from a list of strings in Python, you can use the `nltk` library's `ngrams` function, specifying `n=3` for trigrams. Here's how you can do it:

### Code Example:
```python
from nltk import ngrams

# Sample list of strings (replace with your actual list)
text_list = [
    "This is a sample sentence",
    "Here is another example",
    "Trigrams are useful for text analysis"
]

# Create a list to store all trigrams
all_trigrams = []

# Loop through each string in the list
for text in text_list:
    # Split the string into words
    words = text.split()
    
    # Generate trigrams from the words
    trigrams_in_text = list(ngrams(words, 3))
    
    # Add the trigrams to the all_trigrams list
    all_trigrams.extend(trigrams_in_text)

# Output: all possible trigrams across all strings in text_list
print(all_trigrams)
```

### Explanation:
1. **`ngrams(words, 3)`:** This function generates trigrams (three consecutive words) from a list of words.
2. **`split()`:** Each string is split into individual words.
3. **`extend(trigrams_in_text)`:** Collects all trigrams from each string and stores them in `all_trigrams`.

### Output:
The code will output all possible trigrams across your list of strings, for example:
```
[('This', 'is', 'a'), ('is', 'a', 'sample'), ('a', 'sample', 'sentence'),
 ('Here', 'is', 'another'), ('is', 'another', 'example'),
 ('Trigrams', 'are', 'useful'), ('are', 'useful', 'for'),
 ('useful', 'for', 'text'), ('for', 'text', 'analysis')]
```

This approach ensures that you extract every possible trigram (sequences of three consecutive words) from each string in your list.
