The easiest way to do a word count across a list of strings in Python is by using the `collections.Counter` class, which can tally up the occurrences of each word quickly and efficiently. Here’s a step-by-step guide:

### Step-by-Step Example

```python
from collections import Counter

# Sample list of strings
string_list = [
    "Hello world",
    "Hello Python",
    "Python is great",
    "Hello world Python"
]

# Step 1: Combine all strings into a single string
all_text = " ".join(string_list)

# Step 2: Split the combined string into words (based on spaces)
words = all_text.split()

# Step 3: Use Counter to count occurrences of each word
word_count = Counter(words)

# Print the result
print(word_count)
```

### Output:
```
Counter({'Hello': 3, 'Python': 3, 'world': 2, 'is': 1, 'great': 1})
```

### Explanation:
1. **Combine All Strings**: By joining all strings in the list with `" ".join(string_list)`, you create a single string that can be split into words.
2. **Split Into Words**: Use `.split()` to get a list of individual words.
3. **Count Words**: Use `Counter` to get the frequency of each word.

### Benefits:
- **Efficient**: `Counter` is optimized for counting items, making it faster than manually iterating through the list.
- **Simple**: Easy to read and implement, even for large datasets.
- **Flexible**: You can customize it to clean the words (e.g., lowercase, remove punctuation) before counting.

### Advanced: Cleaning Words Before Counting
If your strings contain punctuation or different cases, you might want to clean them before counting:

```python
import string

# Function to clean and count words
def clean_and_count(strings):
    # Combine all strings into one
    all_text = " ".join(strings)
    # Remove punctuation and make lowercase
    cleaned_text = all_text.translate(str.maketrans("", "", string.punctuation)).lower()
    # Split and count
    return Counter(cleaned_text.split())

# Example usage
word_count = clean_and_count(string_list)
print(word_count)
```

### Output:
```
Counter({'hello': 3, 'python': 3, 'world': 2, 'is': 1, 'great': 1})
```

This method helps ensure that "Hello," "hello," and "hello!" are treated as the same word.

To create a sorted list of the counts from a `Counter` object, you can use the `.most_common()` method. This method returns a list of tuples, where each tuple contains a word and its corresponding count, sorted in descending order of frequency. If you want them sorted in ascending order instead, you can use Python's `sorted()` function.

### Example: Using `.most_common()`
```python
from collections import Counter

# Sample data
words = ["apple", "banana", "apple", "orange", "banana", "apple"]

# Create a Counter object
word_count = Counter(words)

# Get the counts sorted in descending order
sorted_counts = word_count.most_common()

# Print the sorted counts
print(sorted_counts)
```

### Output:
```
[('apple', 3), ('banana', 2), ('orange', 1)]
```

### Example: Custom Sorting in Ascending Order
If you want to sort the counts in ascending order, you can do the following:
```python
# Sort the Counter object by count in ascending order
sorted_counts_asc = sorted(word_count.items(), key=lambda x: x[1])

# Print the sorted counts
print(sorted_counts_asc)
```

### Output:
```
[('orange', 1), ('banana', 2), ('apple', 3)]
```

### Explanation:
1. **`most_common()`**: Returns a list of `(word, count)` tuples sorted in descending order of frequency.
2. **`sorted()`**: You can specify a custom sorting using the `key` argument, where `lambda x: x[1]` sorts by the count value.

This approach is efficient, simple, and flexible, allowing you to customize the sorting as needed.

You can use the `.most_common()` method from the `Counter` object to get the top 100 most common words. The `.most_common()` method returns a list of tuples, where each tuple contains a word and its frequency count, sorted in descending order of frequency.

Here's how you can do it:

```python
from collections import Counter

# Sample data: list of words
words = ["apple", "banana", "apple", "orange", "banana", "apple", "grape", "orange", "grape", "kiwi", "banana", "kiwi"]

# Create a Counter object
word_count = Counter(words)

# Get the top 100 most common words
top_100_words = word_count.most_common(100)

# Print the result
print(top_100_words)
```

### Explanation:
1. **`Counter(words)`**: Creates a `Counter` object that counts the occurrences of each word in the list.
2. **`word_count.most_common(100)`**: Retrieves the 100 most common words from the `Counter` object, sorted by frequency.

### Output:
```
[('apple', 3), ('banana', 3), ('orange', 2), ('grape', 2), ('kiwi', 2)]
```

This method is efficient, and you can adjust the argument passed to `.most_common()` to get as many top entries as you need.

To get all of the unique words (keys) from a `Counter` object, you can simply use the `.keys()` method. If you want them as a list, you can convert the result to a list directly.

Here's how you can do it:

```python
from collections import Counter

# Sample data: list of words
words = ["apple", "banana", "apple", "orange", "banana", "apple", "grape", "orange", "grape", "kiwi", "banana", "kiwi"]

# Create a Counter object
word_count = Counter(words)

# Get all unique words
all_words = list(word_count.keys())

# Print the result
print(all_words)
```

### Output:
```
['apple', 'banana', 'orange', 'grape', 'kiwi']
```

### Explanation:
1. **`word_count.keys()`**: Returns all the unique keys (words) from the `Counter` object.
2. **`list(word_count.keys())`**: Converts the keys into a list, providing all the unique words.

This will give you all the words that were counted by the `Counter`, without their frequency counts.
