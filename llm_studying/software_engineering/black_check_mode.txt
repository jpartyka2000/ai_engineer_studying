The command:

```bash
black --check /path/to/file
```

runs the **Black** code formatter in **"check" mode** on the specified file. Instead of modifying the file, Black will check whether the file is already formatted according to its rules.

- If the file is **properly formatted**, Black will exit with a status code **0**.
- If the file is **not properly formatted**, Black will exit with a status code **1** and print a message indicating which files need reformatting.

This is useful for **CI/CD pipelines**, **pre-commit hooks**, or just verifying code style without making changes.
