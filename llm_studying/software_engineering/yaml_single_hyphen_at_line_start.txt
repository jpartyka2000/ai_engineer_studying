A single hyphen (`-`) at the start of a line in YAML indicates a **list item**.  

### **Example:**
```yaml
fruits:
  - Apple
  - Banana
  - Cherry
```
This represents a list of strings:
```python
{"fruits": ["Apple", "Banana", "Cherry"]}
```

### **How It Works in Your YAML:**
```yaml
prompts:
  - |-
    Hi there LLM,
    Please translate the following text to French:
    "{{name}}: {{text}}"
  - |-
    Translate the following text to German:
    "{{name}}: {{text}}"
```
Here, `prompts` is a **list**, where each `-` introduces a new item. Each item is a **multi-line string** (denoted by `|-`).

---

### **Nested Lists Example:**
```yaml
shopping:
  - Fruits:
      - Apple
      - Banana
  - Vegetables:
      - Carrot
      - Broccoli
```
This results in:
```python
{
  "shopping": [
    {"Fruits": ["Apple", "Banana"]},
    {"Vegetables": ["Carrot", "Broccoli"]}
  ]
}
```
