Yes ✅ — if your Python script uses **argparse**, you automatically get a command-line help option.

By default, argparse adds two standard flags:

* `-h`
* `--help`

So from the terminal you can run:

```bash
python your_script.py -h
```

or

```bash
python your_script.py --help
```

and argparse will print a usage message and descriptions of the arguments you defined.

---

### Example

Suppose your script has:

```python
import argparse

parser = argparse.ArgumentParser(description="Demo program")
parser.add_argument("--verbose", action="store_true", help="Increase output detail")
parser.add_argument("filename", help="Input file")
args = parser.parse_args()
```

If you run:

```bash
python script.py -h
```

You’d see output like:

```
usage: script.py [-h] [--verbose] filename

Demo program

positional arguments:
  filename     Input file

optional arguments:
  -h, --help   show this help message and exit
  --verbose    Increase output detail
```

---

✅ **Bottom line:** Yes — just run your script with `-h` or `--help` from the command line, and argparse will show you how to use it.
