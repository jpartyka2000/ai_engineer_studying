```bash
git add '*.py'
```

Or if that doesn't catch files in subdirectories:

```bash
git add '**/*.py'
```

---

## Quick Comparison

| Command | What It Adds |
|---------|--------------|
| `git add *.py` | .py files in current directory only |
| `git add '*.py'` | Same (quotes prevent shell expansion) |
| `git add '**/*.py'` | .py files in all subdirectories |
| `git add .` | Everything |

The quotes matter — they let Git handle the glob instead of your shell.
