GIT TUTOR — QUESTION RECORD

Skill Level: Medium
Category: Comparing Changes
Type: Coding / Command
------------------------------------------------------------
QUESTION:
You want to see all the differences between your current 
branch (`feature`) and the `main` branch.

Write the command to compare these two branches.
------------------------------------------------------------
YOUR ANSWER: Something with ... operator
CORRECT ANSWER: git diff main feature
  (also valid: git diff main..feature)
STATUS: ⚠️ Partially Correct (right concept!)
------------------------------------------------------------
EXPLANATION:
There are multiple ways to compare branches, and the dot 
operators do slightly different things:

SIMPLE COMPARISON (Two Branch Tips):
------------------------------------------------------------
git diff main feature
git diff main..feature    (two dots, same as above)

Shows the complete difference between the current state 
of main and the current state of feature.

THREE DOT OPERATOR (...):
------------------------------------------------------------
git diff main...feature

Shows changes made on feature SINCE it diverged from main.
Ignores changes made on main after the branch point.

VISUALIZING THE DIFFERENCE:
------------------------------------------------------------
         C---D---E  feature
        /
   A---B---F---G  main

git diff main..feature (two dots):
  Compares G to E
  Shows ALL differences between the two tips

git diff main...feature (three dots):
  Compares B to E
  Shows only what's NEW on feature (C, D, E)
  Ignores F and G on main

WHICH TO USE:
------------------------------------------------------------
"What does feature look like compared to main right now?"
  → git diff main..feature (or git diff main feature)

"What changes did feature introduce since branching off?"
  → git diff main...feature

For PR/merge preview, three dots (...) is often more useful 
since it shows just the new work on the feature branch.

OTHER USEFUL COMPARISON COMMANDS:
------------------------------------------------------------
git diff main feature --stat    # Summary only
git diff main feature -- file.js  # Specific file
git log main..feature           # Commits, not diff
