Trees (and all Git objects) are stored in the `.git/objects` directory. They're stored as compressed files named by their SHA-1 hash.

For example, a tree with hash `8a3f2b9c...` would be at:
```
.git/objects/8a/3f2b9c...
```

(Git uses the first two characters as a subdirectory to avoid having too many files in one folder.)

**To inspect trees directly:**
```bash
# See the tree of your current commit
git cat-file -p HEAD^{tree}

# See a specific tree by hash
git cat-file -p 8a3f2b9c

# List all objects (commits, trees, blobs)
git rev-list --objects --all
```

**Note:** The files in `.git/objects` are compressed and not human-readable if you open them directly — use `git cat-file` to view them.
