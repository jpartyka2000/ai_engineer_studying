To list commits from a remote branch, first fetch the latest data, then use `git log`:

```bash
git fetch origin
git log origin/<branch-name>
```

**Useful options:**

```bash
# Limit to last 10 commits, one line each
git log origin/<branch-name> --oneline -n 10

# Show author and relative time
git log origin/<branch-name> --format="%h %an %ar - %s" -n 10

# Show with dates
git log origin/<branch-name> --format="%h %ad %s" --date=short -n 10

# Show commits since a specific date
git log origin/<branch-name> --since="2025-01-20" --oneline

# Show commits from the last 3 days
git log origin/<branch-name> --since="3 days ago" --oneline
```

**Comparing branches:**

```bash
# Commits on feature branch that aren't in main
git log origin/main..origin/<branch-name> --oneline

# Commits in main that aren't in feature branch
git log origin/<branch-name>..origin/main --oneline
```

**Finding the branch:**

```bash
# List all remote branches
git branch -r
```

**Note:** The `origin/` prefix refers to the remote tracking branch. If your remote is named something other than `origin`, use that name instead (e.g., `upstream/<branch-name>`).
