To push your new local branch to the remote repo:

```bash
git push -u origin <branch-name>
```

**Example:**
```bash
git push -u origin feature/new-feature
```

**What this does:**
- `git push` - pushes your commits to remote
- `-u` (or `--set-upstream`) - sets up tracking so future `git push` and `git pull` commands know where to push/pull from
- `origin` - the name of your remote (usually "origin" by default)
- `<branch-name>` - your local branch name

**After this first push with `-u`, you can just use:**
```bash
git push
```

**To verify it worked:**
```bash
git branch -vv
```
This shows your local branches and what remote branches they're tracking.

**Quick shorthand if you're already on the branch:**
```bash
git push -u origin HEAD
```
`HEAD` automatically uses your current branch name.
