**`git diff --stat`** gives you a high-level summary of what changed:

```bash
git diff --stat
```

Example output:
```
src/main.py          | 12 ++++++------
src/utils.py         |  5 +++++
tests/test_main.py   | 28 ++++++++++++++--------------
3 files changed, 25 insertions(+), 20 deletions(-)
```

It shows:
- Which files changed
- How many lines were added/removed per file (with a visual bar)
- Total summary at the bottom

Useful variations:
- `git diff --stat HEAD~3` - summary of last 3 commits
- `git diff --stat main` - summary of changes vs main branch
- `git diff --staged --stat` - summary of what you're about to commit
