## Location of .gitignore

```
your-repo/
├── .git/            ← Git's internal folder
├── .gitignore       ← Right here, at the repo root
├── src/
├── README.md
└── ...
```

**.gitignore lives at the root of your repository**, alongside (not inside) the `.git/` folder.

---

## A Few Notes

| Situation | Location |
|-----------|----------|
| **Main .gitignore** | Repo root (most common) |
| **Subdirectory .gitignore** | Can also exist in subfolders — applies to that folder and below |
| **Global .gitignore** | `~/.gitignore_global` (for all your repos, like .DS_Store) |
