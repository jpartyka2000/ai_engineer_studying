**`git diff --name-only`** lists just the filenames that changed, nothing else:

```bash
git diff --name-only
```

Example output:
```
src/main.py
src/utils.py
tests/test_main.py
```

Great for:
- Piping to other commands: `git diff --name-only | xargs grep "TODO"`
- Quick scan of what's touched
- Scripting

Useful variations:
- `git diff --name-only HEAD~1` - files changed in last commit
- `git diff --name-only main` - files changed vs main branch
- `git diff --name-only --staged` - files staged for commit
- `git diff --name-only --diff-filter=A` - only added files (`M` for modified, `D` for deleted)
