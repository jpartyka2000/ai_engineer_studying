To create a local tracking branch for the remote branch `remotes/origin/library-format`, use the following command:

```bash
git checkout --track origin/library-format
```

or, if you're using newer Git versions:

```bash
git switch --track origin/library-format
```

### Alternative Explicit Method
If you prefer to create the branch explicitly:

```bash
git branch library-format origin/library-format
git checkout library-format
```

or using `switch`:

```bash
git branch library-format origin/library-format
git switch library-format
```

This will create a local branch named `library-format` that tracks the remote branch `origin/library-format`. Now, when you run `git branch`, it will appear in the list.
