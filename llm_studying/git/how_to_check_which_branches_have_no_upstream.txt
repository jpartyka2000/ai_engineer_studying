**Check which local branches have no upstream:**

```bash
git branch -vv
```

This shows all local branches with their tracking status. Branches with a remote show `[origin/branch-name]`, while branches without an upstream show nothing.

Example output:

```
  feature-x     a1b2c3d [origin/feature-x] Add new feature
  experiment    e4f5g6h Some local experiment
* main          h7i8j9k [origin/main] Latest commit
```

In this example, `experiment` has no upstream (no `[origin/...]`).

---

**List only branches with no upstream:**

```bash
git for-each-ref --format='%(refname:short) %(upstream:short)' refs/heads | awk '$2 == "" {print $1}'
```

---

**Simpler alternative using `git branch`:**

```bash
git branch --format='%(refname:short) %(upstream:short)' | grep -E '^\S+\s*$'
```

---

**Check a specific branch:**

```bash
git config --get branch.<branch-name>.remote
```

If this returns nothing, the branch has no upstream.

Example:

```bash
git config --get branch.experiment.remote
# (no output = no upstream)

git config --get branch.main.remote
# origin
```
