============================================================
DELETING A LOCAL BRANCH IN GIT — OVERVIEW
============================================================

BASIC COMMANDS
------------------------------------------------------------
# Safe delete (only if merged)
git branch -d branch-name

# Force delete (even if not merged)
git branch -D branch-name

-d vs -D EXPLAINED
------------------------------------------------------------
-d (lowercase) — SAFE DELETE
  Only deletes if the branch has been fully merged
  Protects you from accidentally losing work

  $ git branch -d feature-login
  Deleted branch feature-login (was f4e5d6c).

  If NOT merged, you get an error:
  $ git branch -d unmerged-feature
  error: The branch 'unmerged-feature' is not fully merged.
  If you are sure you want to delete it, run 'git branch -D unmerged-feature'.

-D (uppercase) — FORCE DELETE
  Deletes the branch no matter what
  Use when you're sure you don't need it

  $ git branch -D abandoned-experiment
  Deleted branch abandoned-experiment (was a1b2c3d).

COMMON SCENARIOS
------------------------------------------------------------
SCENARIO 1: Delete a merged feature branch
  $ git branch -d feature-login
  ✅ Safe — work is preserved in main

SCENARIO 2: Abandon an experimental branch
  $ git branch -D experiment-failed
  ⚠️ Force delete — commits may be lost

SCENARIO 3: Delete multiple branches
  $ git branch -d branch1 branch2 branch3

SCENARIO 4: Delete branches matching a pattern
  $ git branch | grep "feature-" | xargs git branch -d

YOU CAN'T DELETE THE CURRENT BRANCH
------------------------------------------------------------
$ git branch -d main
error: Cannot delete branch 'main' checked out at '/repo'

Switch to another branch first:
$ git checkout develop
$ git branch -d main

RECOVERING A DELETED BRANCH
------------------------------------------------------------
Accidentally deleted? Use reflog to find it:

$ git reflog
a1b2c3d HEAD@{5}: commit: Last commit on deleted branch

$ git checkout -b recovered-branch a1b2c3d

As long as Git hasn't garbage collected (~30 days), 
you can recover it!
