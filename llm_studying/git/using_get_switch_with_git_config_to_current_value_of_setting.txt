**Check the value of a Git config setting:**

```bash
git config --get <key>
```

**Examples:**

```bash
# Check your configured username
git config --get user.name

# Check your email
git config --get user.email

# Check if fetch.prune is enabled
git config --get fetch.prune

# Check the default branch name
git config --get init.defaultBranch

# Check the remote URL for origin
git config --get remote.origin.url
```

If the setting exists, it prints the value. If it doesn't exist, it prints nothing and returns exit code 1.

---

**Related options:**

```bash
# Show where the value comes from (system, global, local)
git config --show-origin --get user.name

# List all config settings
git config --list

# List all with their source files
git config --list --show-origin

# Get value from a specific scope
git config --global --get user.name
git config --local --get user.name
git config --system --get user.name
```

---

**Check if a setting exists in a script:**

```bash
if git config --get fetch.prune > /dev/null; then
    echo "fetch.prune is set"
else
    echo "fetch.prune is not set"
fi
```
