In GitHub, you can monitor the CI/CD process in real time by following these steps:

1. **Go to Your Repository**: Navigate to the GitHub repository where you committed the code.

2. **Access GitHub Actions**:  
   - Click on the **"Actions"** tab at the top of the repository.

3. **Find the Workflow Run**:  
   - You'll see a list of recent workflow runs.  
   - Click on the workflow corresponding to your latest commit.

4. **Monitor the Running Jobs**:  
   - Inside the workflow, you will see different jobs running.  
   - Click on an individual job to see its logs in real time.

5. **Check Status Indicators**:  
   - ✅ Green check = Successful job  
   - ❌ Red cross = Failed job  
   - ⏳ Yellow dot = Running job  

6. **Use Logs for Debugging**:  
   - You can expand different steps in the job to view logs and error messages.

If your CI/CD process uses an external service (e.g., Jenkins, CircleCI, GitHub-hosted runners), you may need to check their respective dashboards for real-time logs.
