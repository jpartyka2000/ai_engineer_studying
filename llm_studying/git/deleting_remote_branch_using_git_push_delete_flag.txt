============================================================
DELETING A REMOTE BRANCH IN GIT — OVERVIEW
============================================================

BASIC COMMAND
------------------------------------------------------------
git push origin --delete branch-name

This removes the branch from the remote server (GitHub, 
GitLab, Bitbucket, etc.).

EXAMPLE
------------------------------------------------------------
$ git push origin --delete feature-login
To github.com:username/repo.git
 - [deleted]         feature-login

ALTERNATIVE SYNTAX (OLDER)
------------------------------------------------------------
git push origin :branch-name

The colon with nothing before it means "push nothing to 
this branch" — which deletes it.

$ git push origin :feature-login

Both commands do the same thing. The --delete syntax is 
clearer and preferred.

LOCAL VS REMOTE DELETION
------------------------------------------------------------
Deleting remote does NOT delete local (and vice versa).

For full cleanup, do both:

# Delete from remote
$ git push origin --delete feature-login

# Delete locally
$ git branch -d feature-login

WHAT ABOUT STALE REMOTE REFERENCES?
------------------------------------------------------------
After someone else deletes a remote branch, your local 
Git still "remembers" it as origin/branch-name.

Clean up stale references with:
$ git fetch --prune

Or set auto-prune:
$ git config --global fetch.prune true

COMMON ERRORS
------------------------------------------------------------
ERROR: Remote branch not found
  $ git push origin --delete old-branch
  error: unable to delete 'old-branch': remote ref does not exist

  The branch doesn't exist on remote. Check spelling or 
  it may already be deleted.

ERROR: Permission denied
  You may not have push access to the repository or 
  the branch may be protected.

PROTECTED BRANCHES
------------------------------------------------------------
Many teams protect important branches (main, develop) 
from deletion. If you get a permission error:
- Check repository settings
- Ask an admin to delete it
- Or remove branch protection first
