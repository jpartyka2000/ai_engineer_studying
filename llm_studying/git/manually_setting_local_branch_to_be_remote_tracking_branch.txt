**Set a local branch to track a remote branch:**

```bash
git branch --set-upstream-to=origin/feature-x feature-x
```

Or if you're already on the branch:

```bash
git branch --set-upstream-to=origin/feature-x
```

Short form:

```bash
git branch -u origin/feature-x
```

---

**Set tracking when pushing a new branch:**

```bash
git push -u origin feature-x
```

The `-u` (or `--set-upstream`) flag sets up tracking automatically.

---

**Set tracking when creating a branch:**

```bash
git checkout -b feature-x origin/feature-x
```

Or with newer Git:

```bash
git switch -c feature-x origin/feature-x
```

Git automatically sets up tracking when you create a branch from a remote-tracking branch.

---

**Manual method using git config:**

```bash
git config branch.feature-x.remote origin
git config branch.feature-x.merge refs/heads/feature-x
```

---

**Verify it worked:**

```bash
git branch -vv
```

Output:

```
* feature-x a1b2c3d [origin/feature-x] Latest commit
```

---

**Remove tracking:**

```bash
git branch --unset-upstream feature-x
```

---

**Change tracking to a different remote branch:**

```bash
git branch -u origin/different-branch feature-x
```

Now your local `feature-x` tracks `origin/different-branch` instead.
