```bash
git show HEAD
```

This shows the commit message plus the full diff of all changes in the last commit.

## Variations:

```bash
# Just the diff, no commit metadata
git diff HEAD~1..HEAD

# Include file stats summary at the top
git show --stat HEAD

# Show last N commits
git show HEAD~2..HEAD

# Specific file only
git show HEAD -- path/to/file.py
```

## If you want to copy directly to clipboard:

```bash
# macOS
git show HEAD | pbcopy

# Linux (with xclip)
git show HEAD | xclip -selection clipboard
```
