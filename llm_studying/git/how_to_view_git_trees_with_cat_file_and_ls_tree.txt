You can inspect Git trees using a few built-in commands.

**View the tree of your current commit:**
```bash
git cat-file -p HEAD^{tree}
```

This outputs something like:
```
100644 blob 4c1a9e...    README.md
100644 blob 7d2f8c...    main.py
040000 tree 2b9e4a...    src
```

**View a specific tree by its hash:**
```bash
git cat-file -p 2b9e4a
```

**View the tree of a specific commit:**
```bash
git cat-file -p abc123^{tree}
```

**View the tree at a specific path:**
```bash
git ls-tree HEAD src/
```

**Recursively list all files in a tree:**
```bash
git ls-tree -r HEAD
```

**Show the tree hash for a commit:**
```bash
git rev-parse HEAD^{tree}
```

**What the output means:**
- `100644` — regular file
- `100755` — executable file
- `040000` — directory (subtree)
- `blob` or `tree` — object type
- The hash — pointer to the actual object
- The name — filename or directory name
