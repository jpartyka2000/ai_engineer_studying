Yes! `.gitignore` is **not created automatically** — you create it when you need it.

---

## Quick Setup

```bash
# Create it
touch .gitignore

# Or create and open in one step
code .gitignore      # VSCode
vim .gitignore       # vim
nano .gitignore      # nano
```

Then add your patterns:

```
# Example .gitignore
node_modules/
.env
*.log
__pycache__/
.DS_Store
```

---

## One Thing to Check

Files starting with `.` are hidden by default on Mac/Linux.

```bash
# This won't show it
ls

# This will
ls -la
```

In VSCode, you may need to enable "Show Hidden Files" in the explorer.
