**Overview**
The **OpenAI Agents SDK** (available as the `openai-agents` PyPI package) is a lightweight, Python-first framework for building multi-agent workflows with large language models. It’s provider-agnostic (works with OpenAI’s Chat/Response APIs and 100+ other LLMs) and designed to let you “orchestrate and chain agents” using familiar Python idioms rather than custom DSLs ([OpenAI GitHub][1], [GitHub][2]).

---

**Installation**

```bash
pip install openai-agents
```

(The package was published just recently.) ([PyPI][3])

---

**Core Concepts**

* **AgentLoop**
  A built-in loop that handles tool invocation, LLM calls, and iteration until your agent’s task is complete.
* **Python-first**
  You write standard Python functions and classes; the SDK auto-wraps them into “tools” or “agents.” No new abstractions to learn.
* **Handoffs**
  Coordinate between multiple agents via explicit “handoff” descriptions—one agent can pass control to another when its sub-task is done.
* **Guardrails**
  Define input validators or checks that run in parallel; if they fail, the agent stops early with a clear error.

---

**Tools**
Agents can invoke three classes of tools:

1. **Hosted Tools**

   * Built-in, server-hosted capabilities (web search, file search, code execution, image generation, etc.).
2. **Function Tools**

   * Any Python function can be exposed as a tool; arguments and docstrings are parsed automatically.
3. **Agent-as-Tool**

   * Treat one agent itself as a callable tool for another agent, enabling recursive or hierarchical workflows. ([OpenAI GitHub][4])

---

**Typical Usage**

```python
from agents import Agent, Tool, ModelSettings

# 1) Define your tools
def fetch_data(query: str) -> str:
    ...

# 2) Wrap them
tools = [Tool.from_function(fetch_data)]

# 3) Configure your agent
agent = Agent(
    name="DataFetcher",
    handoff_description="Fetch data from an API",
    tools=tools,
    model_settings=ModelSettings(temperature=0.2)
)

# 4) Run the agent loop
result = agent.run("Retrieve the latest sales figures for Q1.")
```
