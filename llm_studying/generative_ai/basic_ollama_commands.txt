`ollama list` is a command for **Ollama**, a tool for running large language models (LLMs) locally on your machine.

## What It Does

Lists all models you have downloaded/installed locally.

```bash
ollama list
```

---

## Other Common Ollama Commands

| Command | Description |
|---------|-------------|
| `ollama list` | List downloaded models |
| `ollama pull <model>` | Download a model |
| `ollama run <model>` | Run/chat with a model |
| `ollama rm <model>` | Remove a model |
| `ollama show <model>` | Show model details |
| `ollama ps` | Show currently running models |
| `ollama serve` | Start the Ollama server |

---

## Examples

```bash
# Download a model
ollama pull llama3.2

# Run a model interactively
ollama run llama3.2

# Run with a prompt directly
ollama run llama3.2 "Explain Python decorators"

# Remove a model
ollama rm mistral

# Show model info (parameters, template, etc.)
ollama show llama3.2

# See what's currently running
ollama ps
```

## Installation

If you don't have Ollama installed:

**macOS:**
```bash
brew install ollama
```

**Linux:**
```bash
curl -fsSL https://ollama.com/install.sh | sh
```

**Windows:**
Download from [ollama.com](https://ollama.com)
