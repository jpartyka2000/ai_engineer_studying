The screenshot shows `black-forest-labs/FLUX.2-max` — this is a **model identifier** (or **model ID/model string**).

The official engineering term for this format is a **namespaced model identifier**, following the `<organization>/<model-name>` convention. In API contexts it's typically called the **model endpoint** or just the **model ID** — it's the string you pass to the `model` parameter in your API call.

In this case: **Black Forest Labs** is the organization, and **FLUX.2-max** is the model name. Together.ai (and Hugging Face, which popularized this convention) uses this `org/model` namespacing pattern to avoid collisions between models from different providers.
