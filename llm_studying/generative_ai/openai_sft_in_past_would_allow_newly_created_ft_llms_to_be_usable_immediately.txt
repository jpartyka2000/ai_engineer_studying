With **OpenAI fine-tuning via SFT** (both legacy and current), once a fine-tuned model finishes training successfully:

1. **It automatically becomes available in the Playground** under the dropdown for your project (or under “personal” if created in your default personal project). You don’t have to “register” it manually.
2. **It is immediately reachable from the API** — you just call it like any other model using its full model name (e.g. `ft:gpt-4o-mini-2025-08-01:your-org::abcdef`).
3. **Visibility depends on project context**: if you fine-tuned it under a specific project, you’ll only see it in the Playground when that project is selected as the active one. If you accidentally created it under your “Default/personal” project, it will only appear there.

So yes — historically and now, a fine-tuned model should *always* appear automatically in both the Playground (scoped to the project you created it under) and the API.
