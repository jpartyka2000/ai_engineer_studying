It'll work, but tight. A few tips:

## Use a quantized version

```bash
ollama pull mistral:7b-instruct-q4_0
```

The `q4_0` quantization uses ~4GB instead of ~14GB, with minimal quality loss. This leaves headroom for your other apps.

## Close memory-hungry apps

When running inference, close Chrome, Slack, etc. — they compete for unified memory.

## Expect decent performance

~20-30 tokens/second is realistic on your setup.

## Getting started

```bash
brew install ollama
ollama pull mistral:7b-instruct-q4_0
ollama run mistral:7b-instruct-q4_0 "test prompt"
```

Once it's running, you can have Claude Code write the integration code to call it from your project.
