On a Mac, Modal credentials are stored in a TOML file in your home directory:

```
~/.modal.toml
```

The file looks like this:

```toml
[default]
token_id = "ak-12345..."
token_secret = "as-12345..."
```

If you have multiple workspaces/profiles, each gets its own section (e.g., `[my-company]`, `[personal]`).

You can view the file directly:
```bash
cat ~/.modal.toml
```

You can also customize the location by setting the `MODAL_CONFIG_PATH` environment variable if you need the config stored elsewhere.
