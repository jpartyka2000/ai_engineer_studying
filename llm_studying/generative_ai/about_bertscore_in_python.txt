## **✅ How to Compute BERTScore in Python**
Here's an example of how to compute **BERTScore** for a list of predicted and reference sentences:

```python
import bert_score

# Example predictions (hypotheses) and reference texts
predictions = ["The cat sat on the mat."]
references = ["A cat was sitting on the mat."]

# Compute BERTScore
P, R, F1 = bert_score.score(predictions, references, lang="en")

# Display results
print(f"Precision: {P.item():.4f}, Recall: {R.item():.4f}, F1 Score: {F1.item():.4f}")
```

### **📌 Explanation**
- **Predictions (`hypotheses`)**: The generated text or candidate text you want to evaluate.
- **References (`gold truth`)**: The expected or ground-truth text.
- **`lang="en"`**: Specifies the language (change this for different languages).
- **Returns `P, R, F1`**: Precision, Recall, and F1-score for BERT-based text similarity.

## **✅ Running BERTScore on Multiple Sentences**
```python
predictions = [
    "The quick brown fox jumps over the lazy dog.",
    "A journey of a thousand miles begins with a single step."
]
references = [
    "A fast brown fox leaps over a sleeping dog.",
    "Every long journey starts with a first step."
]

P, R, F1 = bert_score.score(predictions, references, lang="en")

for i in range(len(predictions)):
    print(f"Sentence {i+1}:")
    print(f"  Precision: {P[i].item():.4f}")
    print(f"  Recall: {R[i].item():.4f}")
    print(f"  F1 Score: {F1[i].item():.4f}\n")
```

---

## **✅ Use a Specific Transformer Model for BERTScore**
By default, `bertscore` selects a pre-trained **BERT-based model** for your language, but you can specify a different one:

```python
P, R, F1 = bert_score.score(predictions, references, model_type="roberta-large")
```

Popular model options:
- `"bert-base-uncased"` (default)
- `"roberta-large"`
- `"microsoft/deberta-xlarge-mnli"`
