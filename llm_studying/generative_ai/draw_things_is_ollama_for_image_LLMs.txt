**Draw Things** is a native Mac/iOS app that runs image generation models locally, similar to how Ollama runs text LLMs locally.

| | Ollama | Draw Things |
|--|--------|-------------|
| Type | Text LLMs | Image models |
| Interface | CLI + API | GUI |
| Models | Mistral, Llama, etc. | SD 1.5, SDXL, etc. |
| Model management | `ollama pull` | Built-in browser |
| Platform | Mac, Linux, Windows | Mac, iOS |
| Price | Free | Free |

## Key features

- Optimized for Apple Silicon (Metal acceleration)
- Built-in model downloader
- Supports ControlNet, LoRAs, upscaling
- No Python/dependencies to install — just download and run

It's the closest thing to "Ollama for images" on Mac.
