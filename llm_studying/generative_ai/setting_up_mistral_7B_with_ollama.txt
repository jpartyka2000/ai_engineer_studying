Yes, the M3 Pro has integrated GPU cores and unified memory — it's actually great for local LLM inference.

## Good news

- Apple Silicon shares RAM between CPU and GPU
- M3 Pro has 18 GPU cores
- Ollama and llama.cpp are optimized for Metal (Apple's GPU framework)

## How much RAM do you have?

| RAM | Can run |
|-----|---------|
| 18GB | Mistral-7B (tight, but works) |
| 36GB | Mistral-7B comfortably, some larger models |

## Quickest setup (Ollama)

```bash
brew install ollama
ollama serve  # Start the server (or it runs automatically)
ollama pull mistral
```

Then test it:
```bash
ollama run mistral "Hello, how are you?"
```

Performance should be decent — expect ~20-40 tokens/second on M3 Pro depending on your RAM and quantization.
