Yes, there are several good options for free or cheap open-source LLM API access with solid speed:

**Best free/cheap options:**

- **Groq** — Groq is a hardware company that runs LLMs on their own chips. They're known for extremely fast inference (often the fastest available) and offer a generous free tier for models like Llama 3 and Mixtral. Great for real-time applications.

- **OpenRouter** — With a Groq API key you can access a selection of supported models and switch between them by changing the model name. OpenRouter similarly aggregates many providers and offers some models for free (look for models with `:free` suffix). Useful as a unified gateway to try different models.

- **GitHub Models** — GitHub Models offers some of the most recent models with a free tier (you don't even need a Copilot subscription) that you can test already in the browser. You just need a GitHub Personal Access Token to authenticate.

- **HuggingFace Serverless Inference** — HuggingFace Serverless Inference limited to models smaller than 10GB. Some popular models are supported even if they exceed 10GB.

- **Together.ai** — Offers free credits and competitive pricing for open-source models like Llama and Mistral.

- **Google AI Studio** — Google's free tier for Gemini models is quite generous and fast.

**Notable open-source models to use with these services:**
- Llama 3.3 70B and Llama 4 Scout (Meta)
- DeepSeek V3 (strong reasoning, very cost-efficient)
- Mixtral and Mistral models
- Qwen models (good multilingual support)

**For local/self-hosted (totally free):**
- **Ollama** — They made it so easy it's like pulling docker images. There's literally a ollama pull command. You can run smaller models locally if you have decent hardware.

For most use cases, I'd start with **Groq** for speed or **OpenRouter** for flexibility. Both have OpenAI-compatible APIs, making integration straightforward.
