## `OpenAIResponsesModelSettings` Overview

`OpenAIResponsesModelSettings` is the **configuration class** that controls how OpenAI models behave in PydanticAI - essentially all the parameters you'd normally pass to OpenAI's API.

## Common Settings:

```python
from pydantic_ai.models.openai import OpenAIResponsesModelSettings

settings = OpenAIResponsesModelSettings(
    # Core parameters
    max_tokens=500,          # Max response length
    temperature=0.7,         # Creativity (0=deterministic, 2=very creative)
    top_p=0.9,              # Nucleus sampling threshold
    
    # Response control
    stream=False,           # Stream responses or get all at once
    stop=["END", "\n\n"],   # Stop sequences
    
    # Behavior
    frequency_penalty=0.5,   # Reduce repetition (-2 to 2)
    presence_penalty=0.0,    # Encourage new topics (-2 to 2)
    
    # Optional
    seed=42,                # For reproducible outputs
    response_format={"type": "json_object"}  # Force JSON mode
)
```

## Usage Pattern:

```python
# Apply settings when creating an agent
agent = Agent(
    'openai:gpt-4',
    model_settings=settings  # Your configured settings
)

# Or update settings later
agent.update_settings(
    temperature=0.2  # Make responses more focused
)
```

## Key Parameters:
- **`temperature`**: Controls randomness (0 = precise, higher = creative)
- **`max_tokens`**: Limits response length
- **`stream`**: Enable/disable streaming responses
- **`stop`**: Custom stop sequences
- **`response_format`**: Enforce structured output (like JSON)

**Bottom line**: It's your control panel for fine-tuning OpenAI's behavior - same parameters as OpenAI's API, but wrapped in a Pydantic-validated settings class.
