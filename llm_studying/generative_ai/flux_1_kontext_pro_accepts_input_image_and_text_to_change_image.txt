Yes — that's actually its core design. FLUX.1 Kontext goes beyond traditional text-to-image by enabling you to modify input images via simple text instructions. It combines text and image inputs for precise edits.

You pass in your album cover image as the input, then provide a text prompt describing what to change. The model analyzes the image, understands what's in it, and modifies only what you ask while preserving everything else.

Here's what a basic API call looks like on Together AI:

```python
from together import Together

client = Together()

response = client.images.create(
    model="black-forest-labs/FLUX.1-kontext-pro",
    prompt="Change the musician's outfit to a black leather jacket and place him on a rooftop at sunset, while preserving his exact facial features, body build, and hairstyle.",
    image_url="https://your-bucket.com/album-cover-1.png",
    width=1024,
    height=1024,
)

print(response.data[0].url)
```

You provide the image via `image_url` (a URL to your source image) and the text prompt describes what to change. That's it.
