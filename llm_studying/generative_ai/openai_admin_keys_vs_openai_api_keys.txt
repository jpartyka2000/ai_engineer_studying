## OpenAI Admin Key vs Regular API Key

**Regular API Key** (`sk-...`)
- Used to make API calls (chat completions, embeddings, etc.)
- Scoped to a specific project
- Can be created by project members

**Admin API Key** (`sk-admin-...`)
- Used for organization management tasks only
- Cannot be used to make model API calls
- Only Organization Owners can create them
- Used for: managing users, projects, API keys, and accessing usage/billing data

**Key point:** If you want to pull usage data from the Usage API, you need an Admin Key. Your regular API key won't work for that endpoint.

Create admin keys at: [platform.openai.com/settings/organization/admin-keys](https://platform.openai.com/settings/organization/admin-keys)
