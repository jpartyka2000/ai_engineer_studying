To log in to Modal using the Modal CLI, run:

```bash
modal setup
```

This command will open a browser window where you can authenticate with your Modal account. You can sign up or link an existing account through this process.

Alternatively, you can run it as a Python module:

```bash
python3 -m modal setup
```

This lets you create a Modal account or link your existing one directly from the command line.

If you need to set credentials manually (for example, in CI/CD environments), you can use the `modal token set` command with your `MODAL_TOKEN_ID` and `MODAL_TOKEN_SECRET`.
