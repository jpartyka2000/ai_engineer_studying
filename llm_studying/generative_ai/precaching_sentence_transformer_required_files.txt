When using the `SentenceTransformers` library to create a new embedding model, it downloads required files (like `.json` and `.bin`) during initialization to ensure the model is properly set up. To prevent this behavior, you can either pre-download and cache the necessary files or work entirely offline using a pre-specified directory where these files are already available. Here's how you can do it:

---

### 1. **Pre-download and Use a Local Directory**
   - Download the model files and cache them manually in a directory.
   - Initialize the model by pointing to the local directory where these files are stored.

```python
from sentence_transformers import SentenceTransformer

# Use a locally cached model directory
model = SentenceTransformer('/path/to/local/model')
```

Make sure all required files are in the specified directory. You can download the necessary files from [Hugging Face](https://huggingface.co/models).

---

### 2. **Disable Internet Access**
   If you’re working in an environment where downloads are not allowed or want to ensure no external requests are made, set the `SentenceTransformers` library to operate offline. You can:
   - Set the environment variable `TRANSFORMERS_OFFLINE` to `1`.

```bash
export TRANSFORMERS_OFFLINE=1
```

In Python, this can be achieved programmatically:

```python
import os
os.environ["TRANSFORMERS_OFFLINE"] = "1"
```

This will force the `transformers` library (used internally by `SentenceTransformers`) to avoid making external network requests.

---

### 3. **Use a Custom Cache Directory**
   Specify a custom cache directory where the model files are preloaded:

```python
from sentence_transformers import SentenceTransformer

# Set a custom cache directory
os.environ["TRANSFORMERS_CACHE"] = "/path/to/custom/cache"

model = SentenceTransformer('path/to/preloaded/model')
```

---

### 4. **Debugging Downloads**
   If you're unsure what files are being downloaded, run the program with debug logging enabled:

```bash
export TRANSFORMERS_VERBOSITY=debug
```

This will show what files the library is attempting to fetch.

---

### 5. **Alternative Method: Packaging Models**
If you are packaging your code and want to include the model files directly, you can ship the pre-downloaded model files with your project so no download is required during runtime.

---

By using one of these strategies, you can avoid unnecessary downloads during the initialization of `SentenceTransformers` models.
