## Weave Built-In Local Scorers

Weave provides these local scorers that run as small language models on your machine:

### 1. **WeaveBiasScorerV1**
- Assesses gender and race/origin bias along two dimensions: Race/Origin and Gender/Sexuality
- Uses fine-tuned deberta model
- Example: `WeaveBiasScorerV1()`

### 2. **WeaveToxicityScorerV1**
- Assesses toxicity across five dimensions: Race/Origin, Gender/Sexuality, Religious, Ability, and Violence/Abuse
- Uses the open-source Celadon model
- Example: `WeaveToxicityScorerV1()`

### 3. **WeaveHallucinationScorerV1**
- Checks if your AI system's output contains hallucinations based on the input data
- Uses the HHEM 2.1 model from Vectara
- Requires `query`, `context`, and `output` parameters
- Example: `WeaveHallucinationScorerV1()`

### 4. **WeaveContextRelevanceScorerV1**
- Designed for evaluating RAG systems, scores the relevance of context to the query
- Uses fine-tuned deberta model
- Example: `WeaveContextRelevanceScorerV1()`

### 5. **WeaveCoherenceScorerV1** ✅ (You have this!)
- Checks whether the input text is coherent
- Uses fine-tuned deberta model
- Example: `WeaveCoherenceScorerV1()`

### 6. **WeaveFluencyScorerV1** ✅ (You have this!)
- Checks whether text is fluent—easy to read and understand, similar to natural human language, evaluating grammar, syntax, and readability
- Uses fine-tuned ModernBERT model
- Example: `WeaveFluencyScorerV1()`

### 7. **WeaveTrustScorerV1**
- Composite scorer for RAG systems that evaluates trustworthiness by combining other scorers into Critical and Advisory categories, returning trust levels: high, medium, or low
- Includes: Toxicity, Hallucination, Context Relevance (Critical) + Fluency, Coherence (Advisory)
- Example: `WeaveTrustScorerV1()`

### 8. **PresidioScorer**
- Uses the Presidio library to detect Personally Identifiable Information (PII) in inputs and outputs
- Can detect emails, phone numbers, and other PII
- Example: `PresidioScorer()`

### 9. **MultiTaskBinaryClassificationF1**
- Built-in scorer for classification tasks
- Example: `MultiTaskBinaryClassificationF1(class_names=["fruit", "color", "flavor"])`

## Installation

```python
pip install weave[scorers]  # Required for local scorers
```

## Usage Example

```python
from weave.scorers import (
    WeaveBiasScorerV1,
    WeaveToxicityScorerV1,
    WeaveHallucinationScorerV1,
    WeaveContextRelevanceScorerV1,
    WeaveCoherenceScorerV1,
    WeaveFluencyScorerV1,
    WeaveTrustScorerV1,
    PresidioScorer,
    MultiTaskBinaryClassificationF1
)

# Add to your evaluation
evaluation = weave.Evaluation(
    name="my_eval",
    scorers=[
        WeaveCoherenceScorerV1(),
        WeaveFluencyScorerV1(),
        WeaveHallucinationScorerV1(),
        # ... others
    ],
    dataset=dataset
)
```

So you already have **coherence and fluency** - the other 7 built-in scorers cover bias, toxicity, hallucination, context relevance, trust (composite), PII detection, and classification!
