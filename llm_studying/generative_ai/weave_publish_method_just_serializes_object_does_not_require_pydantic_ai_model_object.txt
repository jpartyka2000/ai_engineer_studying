Great question! **`weave.publish()` does NOT require a pydantic-ai model** - it can publish any Python object. Weave is model-framework agnostic.

## What `weave.publish()` Actually Does

It serializes and versions **any Python object** - models, datasets, configs, custom classes, etc. It just needs to be able to serialize the object's attributes.

## Options for Legacy Agent

You have several choices:

### Option 1: Publish the Wrapper Instance (Recommended)
```python
if not use_legacy_agent:
    model_ref = weave.publish(model, name="CueEvaluationModel")
    model_uri = model_ref.uri()
else:
    # Publish the wrapper - Weave can track it
    wrapper_ref = weave.publish(
        cue_agents[list(cue_agents.keys())[0]],  # Publish one wrapper as representative
        name="LegacyCueAgentWrapper"
    )
    model_uri = wrapper_ref.uri()
```

### Option 2: Publish Model Config Dictionary
```python
if not use_legacy_agent:
    model_ref = weave.publish(model, name="CueEvaluationModel")
    model_uri = model_ref.uri()
else:
    # Publish a config dictionary describing the model
    legacy_model_config = {
        "model_type": "o4-mini",
        "framework": "openai_agents_sdk",
        "agent_class": "CueReactAgent",
        "wrapper_class": "LegacyCueAgentWrapper"
    }
    model_ref = weave.publish(legacy_model_config, name="LegacyModelConfig")
    model_uri = model_ref.uri()
```

### Option 3: Create a Simple Model Info Class
```python
from dataclasses import dataclass

@dataclass
class LegacyModelInfo:
    model_name: str = "o4-mini"
    framework: str = "openai_agents_sdk"
    agent_type: str = "CueReactAgent"
    
# Then in runner:
if not use_legacy_agent:
    model_ref = weave.publish(model, name="CueEvaluationModel")
    model_uri = model_ref.uri()
else:
    legacy_info = LegacyModelInfo()
    model_ref = weave.publish(legacy_info, name="LegacyModelInfo")
    model_uri = model_ref.uri()
```

## The Key Insight

Weave doesn't care about the model framework - it just serializes Python objects. OpenAI Agents SDK, LangChain, pydantic-ai, or raw OpenAI API calls all work fine with `weave.publish()`.
