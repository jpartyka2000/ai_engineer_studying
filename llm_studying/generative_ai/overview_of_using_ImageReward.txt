It's very straightforward. ImageReward is available as a Python package that you can install and use in just a few lines of code:

**Install:**
```bash
pip install image-reward
```

**Basic usage:**
```python
import ImageReward as RM

model = RM.load("ImageReward-v1.0")

# Score a single image against a prompt
score = model.score("a painting of an ocean with clouds and birds", ["path/to/image.png"])
```

You can also pass multiple images to compare them:
```python
scores = model.score("your prompt here", ["image1.png", "image2.png", "image3.png"])
```

A couple of things to note about the scores: ImageReward roughly follows a standard normal distribution, with a mean of 0 and a variance of 1. So scores aren't natively 0–1. Positive scores mean the image aligns well with the prompt, negative scores mean poor alignment. If you need a 0–1 range, you can apply a sigmoid transformation:

```python
import torch
normalized_score = torch.sigmoid(torch.tensor(score)).item()
```

The model weights are hosted on Hugging Face and download automatically the first time you call `RM.load()`. You'll need a GPU for reasonable performance, though it can run on CPU for small batches.

The GitHub repo is at [github.com/THUDM/ImageReward](https://github.com/THUDM/ImageReward) if you want more details.
