Yes, Draw Things has a local API server you can enable:

1. Open Draw Things
2. Load SDXL Lightning
3. Enable the API server in settings (it runs on `http://127.0.0.1:7860` by default)
4. Your Python script calls it via HTTP

## Basic Python usage

```python
import requests
import base64

response = requests.post('http://127.0.0.1:7860/sdapi/v1/txt2img', json={
    'prompt': 'album cover, synthwave, neon city skyline',
    'negative_prompt': 'blurry, low quality',
    'steps': 4,  # SDXL Lightning uses few steps
    'width': 1024,
    'height': 1024
})

# Decode and save image
image_data = base64.b64decode(response.json()['images'][0])
with open('album_cover.png', 'wb') as f:
    f.write(image_data)
```

## Workflow

1. Launch Draw Things manually (or script it to launch on startup)
2. Load your model
3. Enable API server
4. Run your Python app

## Alternative: ComfyUI

If you want more programmatic control without a GUI app running, **ComfyUI** might be better — it's designed for scripting/automation and has a proper API. But it requires more setup (Python environment, dependencies, etc.).

For simplicity, Draw Things + API is the easiest path.
