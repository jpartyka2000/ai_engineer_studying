Congratulations on successfully training the model! 🎉 Now you can test it with your fine-tuned weights. Here's the CLI command to test your newly trained model:

## Test Your Fine-Tuned Model

```bash
# Test with a custom prompt
modal run qwen_finetune.py::main --action test --test-prompt "Your question here"

# Or run the default test prompts
modal run qwen_finetune.py::main --action test
```

## Examples to Try

```bash
# Test with specific questions
modal run qwen_finetune.py::main --action test --test-prompt "What is deep learning?"
modal run qwen_finetune.py::main --action test --test-prompt "Explain neural networks"
modal run qwen_finetune.py::main --action test --test-prompt "How does machine learning work?"
```

## Direct Function Call

You can also call the test function directly:

```bash
modal run qwen_finetune.py::test_model --prompt "What is artificial intelligence?"
```

The test function will:
1. Load your base Qwen model
2. Apply your fine-tuned LoRA weights from the `/models/fine-tuned-qwen` directory
3. Generate a response to your prompt
4. Display the result

Try it out and see how your fine-tuning affected the model's responses! The model should now be more aligned with the training data you provided.
