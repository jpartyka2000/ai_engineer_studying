`modal app list` shows your Modal apps and their status. By default it only shows **running** apps, so to see your stopped ones you'd need:

```bash
modal app list --stop
```

or depending on your Modal CLI version:

```bash
modal app list --all
```

That said, I'm not 100% certain of the exact flag name since Modal's CLI evolves frequently. If neither of those works, run:

```bash
modal app list --help
```

That'll show you the available filters for your installed version.
