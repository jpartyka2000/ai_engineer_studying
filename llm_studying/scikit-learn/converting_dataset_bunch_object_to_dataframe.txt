When you call:

```python
diabetes = fetch_openml("diabetes", version=1, as_frame=True)
```

with `as_frame=True`, the returned object is a Bunch that contains a Pandas DataFrame under the attribute `.frame`. This DataFrame already combines both the feature data and the target. You can simply do:

```python
df_diabetes = diabetes.frame
```

If for some reason you want to build it manually (for example, if you want to customize the columns), you can combine the features and target like so:

```python
df_diabetes = pd.concat([diabetes.data, diabetes.target], axis=1)
```

Either approach gives you a Pandas DataFrame that you can then use for logging or further analysis.
