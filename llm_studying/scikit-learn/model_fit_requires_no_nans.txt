By default, scikit‑learn’s ExtraTreesRegressor does not support missing values in X. If your data has NaN values, you'll need to impute or remove them before calling model.fit(). For example, you could use scikit‑learn’s SimpleImputer:

```python
from sklearn.impute import SimpleImputer

imputer = SimpleImputer(strategy='mean')  # or another strategy
X_imputed = imputer.fit_transform(X)
model.fit(X_imputed, y)
```

Without handling missing values, you'll likely get an error during fitting.
