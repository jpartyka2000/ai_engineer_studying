You can just call the built-in `get_params()` method on any scikit-learn estimator to dump out exactly the hyper­parameter names and values it was instantiated with. For example:

```python
from sklearn.linear_model import LogisticRegression

# instantiate with whatever settings you like
clf = LogisticRegression(
    penalty='l2',
    solver='saga',
    C=0.1,
    max_iter=200,
    random_state=42
)

# this returns a dict of all hyperparameters and their values
hparams: dict = clf.get_params()
print(hparams)
```

That will print something like:

```
{
 'C': 0.1,
 'class_weight': None,
 'dual': False,
 'fit_intercept': True,
 'intercept_scaling': 1,
 'l1_ratio': None,
 'max_iter': 200,
 'multi_class': 'auto',
 'n_jobs': None,
 'penalty': 'l2',
 'random_state': 42,
 'solver': 'saga',
 'tol': 0.0001,
 'verbose': 0,
 'warm_start': False
}
```

If you ever need to see exactly which arguments are accepted (and their defaults), you can also use Python’s `inspect` module:

```python
import inspect
print(inspect.signature(LogisticRegression.__init__))
```

But for “here’s what I actually passed in,” `clf.get_params()` is what you want.
