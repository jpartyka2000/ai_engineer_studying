No, scikit-learn doesn't include a built-in function that produces a comprehensive regression report like the one produced by `classification_report` for classification tasks. Instead, you'll need to compute individual regression metrics—such as Mean Absolute Error (MAE), Mean Squared Error (MSE), Root Mean Squared Error (RMSE), and R² Score—using functions from `sklearn.metrics` and then format or combine them into a report if you want a single consolidated view.

For example, you could create a custom function like this:

```python
import numpy as np
from sklearn.metrics import mean_absolute_error, mean_squared_error, r2_score

def regression_report(y_true, y_pred):
    report = {
        "Mean Absolute Error": mean_absolute_error(y_true, y_pred),
        "Mean Squared Error": mean_squared_error(y_true, y_pred),
        "Root Mean Squared Error": np.sqrt(mean_squared_error(y_true, y_pred)),
        "R2 Score": r2_score(y_true, y_pred)
    }
    return report

# Example usage:
# y_true = [actual target values]
# y_pred = [predicted target values]
print(regression_report(y_true, y_pred))
```

This way, you can get a summary of multiple regression metrics in one place, similar to the idea behind `classification_report`.
