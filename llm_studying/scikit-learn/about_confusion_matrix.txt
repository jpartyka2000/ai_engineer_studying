The `sklearn.metrics.confusion_matrix(y_true, y_pred)` function returns a 2‑D NumPy array of shape `(n_classes, n_classes)`, where the element at `[i, j]` is the number of samples whose true label is class *i* but were predicted as class *j*. For a binary problem, this is a 2×2 array:

```
[[TN, FP],
 [FN, TP]]
```

----------------------------------------

You can wrap the NumPy array returned by `confusion_matrix` in a `pd.DataFrame`, supplying meaningful row and column names. For binary classification, for example:

```python
from sklearn.metrics import confusion_matrix
import pandas as pd

# 1) Compute the raw confusion matrix
cm = confusion_matrix(y_true, y_pred)

# 2) Convert to DataFrame with labels
cm_df = pd.DataFrame(
    cm,
    index=['Actual 0', 'Actual 1'],      # True labels
    columns=['Predicted 0', 'Predicted 1']  # Predicted labels
)

print(cm_df)
```

That will print something like:

|           | Predicted 0 | Predicted 1 |
|-----------|-------------|-------------|
| **Actual 0** | TN          | FP          |
| **Actual 1** | FN          | TP          |

If you have more than two classes, you can generalize by passing your list of class names to both `index=` and `columns=` (make sure to compute `cm` with the same `labels=` ordering if needed):

```python
labels = ['cat','dog','rabbit']
cm = confusion_matrix(y_true, y_pred, labels=labels)
cm_df = pd.DataFrame(cm, index=labels, columns=labels)
```

This gives you a nicely labeled DataFrame for further inspection or plotting.
