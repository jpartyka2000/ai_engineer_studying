model: BaseEstimator = get_model()

# Later, if you need to use a method specific to RandomForestRegressor:

**Key Points:**

- **Return Type:** Annotate your function as returning `BaseEstimator` since both potential models are subclasses of it.
- **Runtime Type Checking:** Use `isinstance` to check the type of the returned model at runtime.
- **Static Type Casting:** Use `cast` from the `typing` module if you want to inform static type checkers of the expected type, but remember that it does not enforce type safety at runtime.

This approach is standard in Python when a function can return multiple subclass types.
