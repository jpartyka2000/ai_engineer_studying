For visualizing GBDTs from LightGBM specifically, here are the best approaches:

## Built-in LightGBM Visualization Options

### 1. **Plot Individual Trees**
```python
import lightgbm as lgb
from lightgbm import plot_tree
import matplotlib.pyplot as plt

# After training your model
lgb.plot_tree(model, tree_index=0, figsize=(20, 8), show_info=['split_gain'])
plt.show()

# Or create a figure with the tree
ax = lgb.plot_tree(model, tree_index=0, figsize=(30, 10), 
                   show_info=['split_gain', 'leaf_count', 'internal_value'])
```

### 2. **Plot Feature Importance**
```python
lgb.plot_importance(model, max_num_features=20, figsize=(10, 8))
plt.show()

# Or with different importance types
lgb.plot_importance(model, importance_type='gain', max_num_features=20)
```

### 3. **Create Graphviz Visualizations**
```python
# Export to graphviz for better tree visualization
graph = lgb.create_tree_digraph(model, tree_index=0, 
                                show_info=['split_gain', 'leaf_count'],
                                name='Tree0')
graph.render(filename='tree', format='png', view=True)
```

### 2. **SHAP** - Best for Understanding Predictions ⭐
```python
import shap

# Create SHAP explainer
explainer = shap.TreeExplainer(model)
shap_values = explainer.shap_values(X)

# Summary plot - shows feature importance and effects
shap.summary_plot(shap_values, X)

# Force plot - individual prediction explanation
shap.force_plot(explainer.expected_value, shap_values[0], X.iloc[0])

# Waterfall plot - breakdown of single prediction
shap.waterfall_plot(shap.Explanation(values=shap_values[0], 
                                     base_values=explainer.expected_value,
                                     data=X.iloc[0],
                                     feature_names=X.columns.tolist()))

# Dependence plots - how features affect predictions
shap.dependence_plot("feature_name", shap_values, X)
```

## Complete Example: Comprehensive GBDT Visualization

```python
import lightgbm as lgb
import shap
import matplotlib.pyplot as plt
from sklearn.datasets import load_breast_cancer
from sklearn.model_selection import train_test_split

# Load data
data = load_breast_cancer()
X_train, X_test, y_train, y_test = train_test_split(
    pd.DataFrame(data.data, columns=data.feature_names),
    data.target, test_size=0.2, random_state=42
)

# Train model
model = lgb.LGBMClassifier(n_estimators=100, max_depth=5, random_state=42)
model.fit(X_train, y_train)

# 1. Plot first few trees
fig, axes = plt.subplots(nrows=1, ncols=3, figsize=(20, 6))
for idx, ax in enumerate(axes):
    lgb.plot_tree(model, tree_index=idx, ax=ax, show_info=['split_gain'])
    ax.set_title(f'Tree {idx}')
plt.tight_layout()
plt.show()

# 2. Feature importance
lgb.plot_importance(model, max_num_features=10, importance_type='gain')
plt.title('Feature Importance (Gain)')
plt.show()

# 3. SHAP analysis
explainer = shap.TreeExplainer(model)
shap_values = explainer.shap_values(X_test)

# For binary classification, shap_values might be a list
if isinstance(shap_values, list):
    shap_values = shap_values[1]  # Use positive class

# Global feature importance via SHAP
shap.summary_plot(shap_values, X_test)

# Individual prediction explanation
shap.force_plot(explainer.expected_value[1], shap_values[0], X_test.iloc[0])
```

## Online Interactive GBDT Visualizations

Unfortunately, there aren't many websites specifically for visualizing GBDTs interactively like TensorFlow Playground for neural networks. However:

1. **[XGBoost.js Playground](https://githubdragonfly.github.io/projects/xgboost/index.html)** - Limited but shows XGBoost in browser
2. **[SHAP Documentation](https://shap.readthedocs.io/en/latest/example_notebooks/overviews/An%20introduction%20to%20explainable%20AI%20with%20Shapley%20values.html)** - Interactive notebooks with GBDT explanations
3. **[Random Forests Demo](https://cs.stanford.edu/~karpathy/svmjs/demo/demoforest.html)** - Not GBDT but shows ensemble concepts

For LightGBM specifically, SHAP is your best bet for understanding how the model makes predictions, while `dtreeviz` creates the most visually appealing individual tree visualizations.
