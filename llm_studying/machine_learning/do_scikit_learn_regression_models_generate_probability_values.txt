No, a RandomForestRegressor in scikit‑learn outputs a continuous value (the predicted number) and does not provide an associated probability value like classifiers do. 

In classification, models can output probabilities (via methods like `predict_proba`) that indicate the likelihood of an instance belonging to each class. In contrast, regression models produce a single continuous prediction (for example, a predicted CLV) without an inherent probability distribution over possible outcomes. 

That said, some advanced techniques can estimate prediction uncertainty (e.g., using quantile regression forests), but by default, RandomForestRegressor does not generate probability estimates.
