One of the most direct ways to do exactly this is via **subgroup‑discovery** libraries, which are built to find “if‑then” style rules that identify subsets of your data with an unusually high (or low) rate of some target condition.  In your case you can treat your **prediction** (or a boolean flag like `pred == 1`) as the target, and then search for subgroups whose positive‐prediction rate exceeds 90%.

---

## 1. pysubgroup

A Python package for subgroup discovery on pandas DataFrames.

* **Install:**

  ```bash
  pip install pysubgroup
  ```
* **Basic workflow:**

  ```python
  import pysubgroup as ps

  # 1. wrap your df
  df = your_dataframe

  # 2. define your target: here “prediction == 1”
  target = ps.BinaryTarget('prediction', value=1)

  # 3. choose a quality function (e.g. weighted relative accuracy)
  qf = ps.WRAccQF()           

  # 4. run a beam search (or DFS) to find top‐k subgroups
  result = ps.BeamSearch(task=ps.SubgroupDiscoveryTask(
                           df, target, qf, depth=3),
                         beam_width=5, result_set_size=10).run()

  # 5. inspect the best rules
  for sg in result:
      print(sg.subgroup_description, sg.quality_, sg.cover / sg.subgroup_size)
  ```

  Each `sg.subgroup_description` is a conjunction of simple selectors (e.g. `age < 30 AND country == 'US'`), and `sg.cover/sg.subgroup_size` tells you the positive‐rate in that subgroup ([GitHub][1]).

---

## 2. pySubDisc

A Python wrapper around the Java‑based SubDisc subgroup‐discovery engine.  Very similar in spirit to pysubgroup, but leverages the original SubDisc algorithms.

* **Install:**

  ```bash
  pip install pysubdisc
  ```
* **Quick start:** see the `examples/` folder once installed ([GitHub][2]).

---

## 3. Rule‑learning as a surrogate

If you’re already comfortable with scikit‑learn, you can also **treat your predictions as labels** and train a rule‐based model:

* **wittgenstein** (implements RIPPER)
* **sklearn‑rulefit** (fit a RuleFit model to explain the predictions)

Then extract the learned “if‑then” rules that have high precision (≥ 0.9) on a held‐out set.

---

### Which to choose?

* **Subgroup discovery** (`pysubgroup`/`pySubDisc`) is tailor‑made for the task “find me all little sub‑populations where my target is unusually high (or low).”
* **RuleFit / RIPPER** is more of a general‐purpose rule‐based classifier: you get rules, but you first need to “fit” to replicate your existing predictions.

Either approach will give you human‑readable conditions describing exactly which slices of your DataFrame carry a ≥ 90% positive‐prediction rate.
