The simplest way to **map a value from \([-1, 1]\) to \([0, 1]\)** is to use the following transformation:  

\[
x_{\text{scaled}} = \frac{x + 1}{2}
\]

### **Explanation:**
- When **\( x = -1 \)** → \( x_{\text{scaled}} = \frac{-1 + 1}{2} = 0 \)  
- When **\( x = 1 \)** → \( x_{\text{scaled}} = \frac{1 + 1}{2} = 1 \)  
- When **\( x = 0 \)** → \( x_{\text{scaled}} = \frac{0 + 1}{2} = 0.5 \)  

### **Python Code:**
```python
def scale_neg1_to_1_to_0_to_1(x):
    return (x + 1) / 2
```

This transformation is **linear** and **preserves ordering**, making it an ideal choice for normalizing metrics in this range. 🚀
