Yes, the BG/NBD model is typically built using three summary inputs for each customer:

- **Frequency:** The number of repeat transactions (i.e., the number of purchases beyond the first one).
- **Recency:** The time between the customer's first and last transaction.
- **T (Age):** The total time period over which the customer has been observed (time since the first purchase).

These three metrics capture the essential purchasing behavior for modeling how often a customer is likely to transact in the future. If you're also interested in predicting the monetary value of these transactions, you might complement the BG/NBD model with a Gamma-Gamma model, which would require additional inputs related to transaction values.
