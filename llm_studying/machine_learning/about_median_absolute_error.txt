Yes, there is a measure called the **Median Absolute Error (MedAE)**. Unlike the Mean Absolute Error (MAE), which computes the average of the absolute errors, the MedAE calculates the median of all absolute differences between predictions and actual values. This makes it more robust to outliers, as the median is less affected by extreme values.

In scikit-learn, you can compute it using the function `median_absolute_error` from the `sklearn.metrics` module:

```python
from sklearn.metrics import median_absolute_error

medae = median_absolute_error(y_true, y_pred)
print(f"Median Absolute Error: {medae:.2f}")
```

This metric provides a clear indication of the typical error magnitude while reducing the influence of outliers on the evaluation.
