Yes — in binary classification, cross-entropy loss and log loss are the same thing.


---

✅ Definitions:

📉 Log Loss (used in scikit-learn):

For a single prediction:

\text{Log Loss} = -\left[ y \log(\hat{y}) + (1 - y) \log(1 - \hat{y}) \right]

Where:

is the true label

is the predicted probability for class 1



---

🤖 Binary Cross-Entropy:

It has the same formula:

\text{Binary Cross-Entropy} = -\left[ y \log(\hat{y}) + (1 - y) \log(1 - \hat{y}) \right]


---

📌 Summary:

✅ Same formula

✅ Same usage for binary classification

🧠 "Cross-entropy" is the information theory term

📊 "Log loss" is the common ML term (especially in sklearn)
