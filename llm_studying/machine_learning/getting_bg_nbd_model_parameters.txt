After fitting your BG/NBD model (using, for example, the `BetaGeoFitter` from the lifetimes library), you can retrieve the fitted parameters in a couple of ways:

1. **Using the `params_` attribute:**  
   This attribute is a Pandas Series that contains the estimated model parameters. For the BetaGeoFitter, these parameters typically include `r`, `alpha`, `a`, and `b`.

   ```python
   print(bgf.params_)
   ```

2. **Using the `summary` attribute:**  
   This provides a Pandas DataFrame with a detailed summary of the parameter estimates, including standard errors and other statistics.

   ```python
   print(bgf.summary)
   ```

Both of these methods let you inspect the model parameters after fitting the model with your data.
