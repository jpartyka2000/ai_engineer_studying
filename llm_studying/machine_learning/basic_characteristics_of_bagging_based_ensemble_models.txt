Bagging-based ensemble models (like Random Forests and Bagged Trees) are based on bootstrapping, but how they handle instances and features (columns) depends on the specific variant:

1. Bagging (Bootstrap Aggregating):

Bootstraps instances: YES — it samples with replacement from the training data to create each base learner’s dataset.

Subsets columns: NO — classic bagging uses all features when training each base model.


2. Random Forests (a special case of bagging):

Bootstraps instances: YES — same as standard bagging.

Subsets columns: YES — at each split in the decision tree, a random subset of features is considered, not all features.
