To label the bins instead of the lines in Matplotlib, you need to place the labels at the midpoint of each bin. This can be done by adjusting the `xticks` to position them between the bin edges rather than directly on them.

Here’s how you can adjust your code:

```python
import matplotlib.pyplot as plt
import numpy as np

# Example data
quartile_labels = ['Q1', 'Q2', 'Q3', 'Q4']
bin_edges = [0, 1, 2, 3, 4]  # Assuming these are your bin edges (e.g., for 4 quartiles)

# Calculate midpoints of each bin
bin_midpoints = [(bin_edges[i] + bin_edges[i+1]) / 2 for i in range(len(bin_edges) - 1)]

# Plot your data
plt.plot(...)  # Your plotting code here

# Set x-ticks at bin midpoints and use bin labels
plt.xticks(bin_midpoints, quartile_labels)

plt.xlabel('Quartiles')
plt.ylabel('Your Y-axis Label')
plt.title('Your Plot Title')
plt.legend()
plt.grid(True)
plt.show()
```

### Explanation
- `bin_edges` represents where each bin starts and ends (e.g., `[0, 1, 2, 3, 4]` for quartiles).
- `bin_midpoints` calculates the center of each bin using the average of two adjacent bin edges.
- `plt.xticks(bin_midpoints, quartile_labels)` sets the labels to be centered within each bin.

This approach will ensure that the labels appear in the center of each bin rather than directly at the dividing lines. Adjust `bin_edges` according to your actual data range and plotting needs.
