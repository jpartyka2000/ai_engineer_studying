Databricks doesn't have native Black formatting built-in, but there are a few options:

## Option 1: Built-in Formatter (Not Black)

Databricks has a basic built-in formatter you can trigger with:

**Mac:** `Cmd + Shift + F`
**Windows:** `Ctrl + Shift + F`

This uses a basic Python formatter, not Black specifically.

---

## Option 2: Install and Run Black Manually

```python
%pip install black

# Then in a cell:
import black
# Format code string
formatted_code = black.format_str(your_code_string, mode=black.Mode())
```

---

## Option 3: Use `%load_ext` Magic (if available)

```python
%pip install black
%pip install nb_black

# Load the extension
%load_ext nb_black
```

This auto-formats cells when you run them, though support varies by Databricks version.

---

## Option 4: Format Externally

1. Copy code to local IDE (VS Code, PyCharm)
2. Format with Black there (`Shift + Alt + F` in VS Code)
3. Paste back into Databricks

---

## Check Databricks Settings

In newer Databricks versions, check:
- **User Settings** → **Editor** → Look for formatting options
- Some workspaces have formatting preferences under **Admin Console** → **Workspace Settings**

Unfortunately, there's no single keyboard shortcut for Black specifically in Databricks as of now. The built-in formatter (`Cmd/Ctrl + Shift + F`) is the closest native option.
