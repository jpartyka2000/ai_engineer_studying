A **`.dbc`** file in Databricks is a workspace archive format used to bundle together one or more notebooks (and optional folders or libraries) into a single file for import/export. Under the hood, it’s a base64-encoded ZIP containing JSON representations of each notebook’s metadata and source. You can:

* **Export** parts of your workspace (notebooks, folders) as a `.dbc` for backup or sharing.
* **Import** a `.dbc` into another Databricks workspace to recreate the same structure and content.

This makes it easy to move collections of notebooks or project materials between environments without handling each file individually.
