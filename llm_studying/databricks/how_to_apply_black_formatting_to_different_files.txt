## **✅ Recap: Which Command Does What?**
| Command | What It Does |
|---------|-------------|
| `!black --check file.py` | Checks if formatting is correct, but does not change anything |
| `!black file.py` | Formats the file automatically (modifies it in place) |
| `!black --diff file.py` | Shows what changes would be made, but does not apply them |
| `!black --check --diff file.py` | Shows formatting issues and the expected changes |
