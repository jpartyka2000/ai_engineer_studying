You can list Databricks scopes and secret keys in a notebook using `dbutils.secrets`. Here's how to do it:

---

### 1. **List All Secret Scopes**
To see all the available secret scopes in your workspace:

```python
dbutils.secrets.listScopes()
```

This will return a list of secret scopes with their metadata.

---

### 2. **List All Secret Keys in a Specific Scope**
To list all the keys in a particular scope, use the following command (replace `your_scope_name` with the name of your scope):

```python
dbutils.secrets.list(scope="your_scope_name")
```

This will return a list of all secret keys available in that scope.

---

### Example Output
For the `listScopes()` command:
```python
[SecretScope(name='databricks_pat_scope', backend_type='DATABRICKS'),
 SecretScope(name='openai_scope', backend_type='DATABRICKS')]
```

For the `list(scope="openai_scope")` command:
```python
[SecretMetadata(key='api_key'),
 SecretMetadata(key='databricks_pat')]
```

---

### 3. **Retrieve a Specific Secret Value**
If you want to retrieve a specific secret value for testing, use:

```python
dbutils.secrets.get(scope="your_scope_name", key="your_key_name")
```

**Note:** Avoid printing secret values directly to maintain security.

---

### 4. **Delete a Secret or Scope (CLI Required)**
If you find old or unwanted keys, you can delete them using the Databricks CLI, as secret deletion is not supported directly in notebooks:
```bash
databricks secrets delete --scope your_scope_name --key your_key_name
```

To delete an entire scope:
```bash
databricks secrets delete-scope --scope your_scope_name
```
