Almost—your **host** is the full protocol + domain. In this case you’d use:

```
https://dbc-1ed7befc-b8c0.cloud.databricks.com
```

(not just the bare `dbc-1ed7befc-b8c0.cloud.databricks.com`). That complete URL is what you’d pass as `--host` (or put in `DATABRICKS_HOST`) when calling the Jobs or Model Serving REST API.
