The trick is to use the **"Group by"** setting in the chart editor. To assign different colors to each bar, set the "Group by" field to the same column as your X axis — in your case, set Group by to `lead_quality_bucket`. This makes Databricks treat each bucket as a separate series, giving each bar its own color.

Then if you want to customize the specific colors, click the "Colors" tab in the visualization editor to override the automatic colors and configure custom ones.
