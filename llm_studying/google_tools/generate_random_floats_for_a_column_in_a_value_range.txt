You can use Excel’s built‑in random functions to do this. Two easy options:

---

### 1) Using `RAND()`

Put this formula in your first cell (e.g. B2) and copy/fill down:

```excel
=RAND()*(25000-500) + 500
```

* `RAND()` returns a uniform random 0–1
* Multiply by `(25000–500)` to stretch it to 0–24500
* Add 500 to shift it to 500–25000
* You’ll get a new decimal value each time the sheet recalculates.

---

### 2) Using `RANDARRAY()` (Excel 365+)

If you have the newer dynamic‐array functions, you can generate an entire spill range in one go. In cell B2 for example:

```excel
=RANDARRAY(100,1, 500,25000, FALSE)
```

* `100,1` → 100 rows × 1 column
* `500,25000` → min = 500, max = 25000
* `FALSE` → generate decimals (set to `TRUE` for integers)

This will “spill” 100 random floats in B2\:B101. Change `100` to however many rows you need.
