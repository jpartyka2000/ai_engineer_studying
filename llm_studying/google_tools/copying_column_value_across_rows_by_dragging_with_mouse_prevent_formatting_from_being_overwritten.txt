When you drag a cell down in Google Sheets, it copies both the value and the formatting (including background color) by default. Here are a few ways to copy just the value without overwriting the background colors:

**Option 1: Paste Values Only (easiest)**
1. Copy the cell (Ctrl+C / Cmd+C)
2. Select the range of cells you want to fill
3. Right-click → **Paste special** → **Values only** (or use Ctrl+Shift+V / Cmd+Shift+V)

**Option 2: Use the fill handle + undo formatting**
1. Drag the cell down as usual
2. Immediately after, press Ctrl+Z / Cmd+Z to undo — but this undoes everything
3. This method doesn't work well, so Option 1 is better

**Option 3: Use a formula instead**
If the source value is in cell A1, in the cells below you can enter:
```
=A1
```
Or for a whole range, use:
```
=A$1
```
This references the value without affecting formatting.

**Option 4: Re-apply background colors after dragging**
1. Drag to fill the values
2. Select the affected cells
3. Re-apply your background colors

**Option 1 (Paste special → Values only)** is typically the quickest and cleanest solution for your situation.
