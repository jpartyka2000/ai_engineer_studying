In Google Sheets, you can adjust color brightness by using a custom color. Here's how:

1. **Select the row** you want to color
2. Click the **Fill color** icon (paint bucket) in the toolbar
3. At the bottom of the color palette, click **Custom**
4. In the custom color dialog, you have two options:

**Option A: Use the color picker**
- Click somewhere in the green area of the color gradient
- Then use the **vertical slider on the right** to adjust brightness — drag it **down** to make the color darker/less bright

**Option B: Use hex codes**
- Enter a darker green hex code directly, such as:
  - `#90EE90` - light green
  - `#228B22` - forest green (darker)
  - `#006400` - dark green
  - `#2E7D32` - muted/less bright green
  - `#A5D6A7` - soft pale green (less saturated)

**Option C: Adjust the HSL/RGB values**
- In the custom color picker, you can manually adjust values
- For "less brightness," you typically want to either:
  - **Lower the lightness** (makes it darker)
  - **Lower the saturation** (makes it more muted/grayish)

If you want a green that's easy on the eyes (not too bright), try something like `#C8E6C9` (very soft green) or `#81C784` (medium muted green).
