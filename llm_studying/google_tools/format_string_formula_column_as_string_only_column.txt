The quickest way is to **replace the formulas with their current values** by using “Paste values only.” Here’s how:

1. **Select** the entire column (e.g. click the column letter at the top).
2. **Copy** it (⌘+C on Mac or Ctrl+C on Windows).
3. Without changing your selection, right‑click anywhere in that column → **Paste special → Paste values only**.

   * Or use the keyboard shortcut **⌘+Shift+V** (Mac) or **Ctrl+Shift+V** (Windows).

That will overwrite every formula in the column with its evaluated text, leaving you with plain strings.

---

### Alternative: Apps Script

If you need to do this automatically (e.g. on a trigger), you can run a tiny Apps Script:

1. **Extensions → Apps Script**
2. Paste:

   ```javascript
   function pasteValuesOnly() {
     const sheet = SpreadsheetApp.getActiveSheet();
     const range = sheet.getRange("D2:D");   // adjust to your column
     range.copyTo(range, { contentsOnly: true });
   }
   ```
3. **Save** and run (or bind to a menu/trigger)—it will rip out formulas from D2\:D and leave only values.

But for a one‑off, the copy → “Paste values only” is by far the fastest.
