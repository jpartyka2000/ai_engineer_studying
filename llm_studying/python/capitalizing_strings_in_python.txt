You have a few built-in options on Python’s `str` to change capitalization:

1. **Capitalize just the first character** (and lowercase the rest)

   ```python
   s = "hello WORLD"
   s2 = s.capitalize()
   print(s2)   # "Hello world"
   ```

2. **Uppercase the entire string**

   ```python
   s = "hello"
   s2 = s.upper()
   print(s2)   # "HELLO"
   ```

3. **Title case every word** (capitalize the first letter of each word)

   ```python
   s = "hello world from python"
   s2 = s.title()
   print(s2)   # "Hello World From Python"
   ```

4. **Capitalize only the first letter of each sentence** isn’t built-in, but you can combine splitting and `.capitalize()`:

   ```python
   text = "hello world. this is python."
   sentences = [sent.strip().capitalize() for sent in text.split('.')]
   result = '. '.join(sentences).strip() + '.'
   print(result)  
   # "Hello world. This is python."
   ```

Choose the one that fits your need:

* Use **`.capitalize()`** to make a single word/sentence have an initial capital.
* Use **`.upper()`** or **`.title()`** for more aggressive capitalization.
