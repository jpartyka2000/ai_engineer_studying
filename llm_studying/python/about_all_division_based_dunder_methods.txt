## 📋 True Division vs Integer (Floor) Division

---

### The Two Division Operators

| Operator | Name | Example | Result |
|----------|------|---------|--------|
| `/` | True division | `7 / 2` | `3.5` |
| `//` | Floor/Integer division | `7 // 2` | `3` |

---

### The Corresponding Dunders

| Operator | Method |
|----------|--------|
| `/` | `__truediv__` |
| `//` | `__floordiv__` |

---

### Example

```python
class Number:
    def __init__(self, value):
        self.value = value
    
    def __truediv__(self, other):
        print("True division called")
        return Number(self.value / other.value)
    
    def __floordiv__(self, other):
        print("Floor division called")
        return Number(self.value // other.value)

a = Number(7)
b = Number(2)

result1 = a / b   # True division called → 3.5
result2 = a // b  # Floor division called → 3
```

---

### The Full Division Family

| Operator | Regular | Reflected | In-place |
|----------|---------|-----------|----------|
| `/` | `__truediv__` | `__rtruediv__` | `__itruediv__` |
| `//` | `__floordiv__` | `__rfloordiv__` | `__ifloordiv__` |
| `%` | `__mod__` | `__rmod__` | `__imod__` |

**Bonus:** There's also `__divmod__` for the `divmod()` function, which returns both quotient and remainder:

```python
print(divmod(7, 2))  # (3, 1) → (quotient, remainder)
```
