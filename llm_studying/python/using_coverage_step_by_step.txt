**Step-by-step:**

**1. Install coverage (if not done):**
```bash
pip install coverage
```

**2. Run your evaluation with coverage:**
```bash
coverage run -m cue_eval.runner "test_run" cue_eval/configs/dev.yaml true
```

**3. Generate the XML report (Coverage Gutters needs this):**
```bash
coverage xml
```

This creates `coverage.xml` in your project root.

**4. In VSCode, display the coverage:**
- `Cmd + Shift + P` → "Coverage Gutters: Display Coverage"

**5. Open any Python file** — you'll see:
- Green = executed lines
- Red = not executed

**6. To see a summary of all files:**
```bash
coverage report
```

Or for an interactive HTML report:
```bash
coverage html
open htmlcov/index.html
```

---

**To clear and re-run:**
```bash
coverage erase
coverage run -m cue_eval.runner ...
coverage xml
```

Then in VSCode: `Cmd + Shift + P` → "Coverage Gutters: Watch" to auto-refresh when the XML updates.
