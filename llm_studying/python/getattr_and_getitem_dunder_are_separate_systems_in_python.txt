Got it. In pure Python terms:

**`getattr(obj, 'foo')`** is for attribute access — it's equivalent to `obj.foo`

**`obj['foo']`** triggers `obj.__getitem__('foo')` — bracket/subscript access

These are two completely separate systems in Python. They don't call each other, they don't know about each other.

```python
obj.foo      # attribute access
obj['foo']   # subscript access — calls __getitem__
```

Dicts happen to use `__getitem__` for their key lookups. Objects happen to use attribute access for their properties. They're different mechanisms that exist in parallel.
