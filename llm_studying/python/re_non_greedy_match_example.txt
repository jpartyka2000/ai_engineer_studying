To make your regex pattern **non-greedy**, you can use the **`?`** qualifier after the `*` quantifier. In regular expressions, `*` is greedy by default, meaning it will match as much as possible. To make it non-greedy (i.e., match the smallest possible string), use `*?` instead.

Here’s how you can modify your pattern:

### Modified Pattern:
```regex
YEARWEEK\(\s*.*?\)
```

### Explanation:
- **`YEARWEEK\(`**: Matches the literal text "YEARWEEK(".
- **`\s*`**: Matches any whitespace characters (spaces, tabs, etc.), zero or more times.
- **`.*?`**: Matches any character (`.`), zero or more times (`*`), but in a non-greedy way (`?`), meaning it will match the smallest possible string.
- **`\)`**: Matches the literal closing parenthesis `)`.

### Example:
This pattern would now match something like `YEARWEEK( 2024, 12 )` but stop at the first closing parenthesis after the opening one, rather than consuming the entire string.

---

### Greedy vs. Non-Greedy:

- **Greedy (`*`)**: Matches as many characters as possible.
  - Example: `YEARWEEK( 2024, 12 )` would match everything from `YEARWEEK(` to the last closing parenthesis.
  
- **Non-Greedy (`*?`)**: Matches the smallest possible number of characters.
  - Example: `YEARWEEK( 2024, 12 )` would match from `YEARWEEK(` to the first closing parenthesis.
