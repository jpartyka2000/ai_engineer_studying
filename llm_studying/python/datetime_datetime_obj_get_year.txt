That won’t work, because the `.dt` accessor only exists on pandas Series/Index, not on a plain `datetime.datetime` object. If all you want is the current year as an integer, do:

```python
from datetime import datetime
last_year_int = datetime.now().year
```

If you actually had a pandas Series of datetimes (say `s = pd.Series([...])`), then you’d do:

```python
last_years = s.dt.year
```

but for a single `datetime`, just use `.year`.
