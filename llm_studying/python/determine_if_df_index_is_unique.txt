You can check this in a couple of ways:

1. **Boolean flag**

   ```python
   if df.index.is_unique:
       print("All index values are unique")
   else:
       print("There are duplicates in the index")
   ```

2. **Find exactly which labels are duplicated**

   ```python
   dup_idx = df.index[df.index.duplicated()]
   if len(dup_idx):
       print("Duplicated index values:", dup_idx.unique())
   else:
       print("Index is fully unique")
   ```

3. **Using `.duplicated()` directly**

   ```python
   has_dupes = df.index.duplicated().any()
   ```

– `df.index.is_unique` is the simplest: it returns `True` if and only if no two rows share the same index label.
