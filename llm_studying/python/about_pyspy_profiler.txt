`py-spy` (usually just called **pyspy**) is a **sampling profiler** for Python programs.

Here’s what that means:

---

## 1. What It Does

* It lets you **see where your Python program spends its time** (which functions, which lines) without modifying your code.
* It works by **sampling the call stack** of your running process many times per second and building a statistical profile.
* Output can be a **live view** in the terminal or a **flame graph** you can open in your browser.

---

## 2. Key Features

* **No code changes needed**: you don’t have to instrument your program; you can attach to any running Python process.
* **Low overhead**: written in Rust, so it samples efficiently and minimally slows down your program.
* **Safe to use in production**: it only reads process memory and doesn’t interfere with the interpreter’s execution.
* **Works across Python versions**: including CPython 2.7 and 3.3+ on Linux, macOS, and Windows.
* **Flamegraphs**: generates interactive HTML flamegraphs for visualizing hot paths.

---

## 3. Installation

```bash
pip install py-spy
```

On some systems, you might also install it via package managers or download binaries.

---

## 4. Common Usage

* **Record a flamegraph of a script:**

  ```bash
  py-spy record -o profile.svg -- python myscript.py
  ```

  → Produces `profile.svg`, which you can open in a browser.

* **Attach to a running process by PID:**

  ```bash
  py-spy top --pid 12345
  ```

  → Shows a live “top-like” view of which functions are consuming CPU.

* **Generate a flamegraph of a running process:**

  ```bash
  py-spy record -o profile.svg --pid 12345
  ```

---

## 5. When to Use

* Your Python program feels slow, and you want to know **which functions are bottlenecks**.
* You want performance insights **without modifying code** or restarting the program.
* You want a **visual breakdown** of CPU time across your code paths.

---

👉 In short: **py-spy is like `top` or `perf` for Python**—a safe, zero-instrumentation profiler that helps you find where your program spends time.
