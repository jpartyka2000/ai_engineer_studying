You can do this with the built‑in `datetime` and `timezone` modules (Python 3.6+), without any external libraries:

```python
from datetime import datetime, timezone

# 1. Get your current local time *with* timezone info
local_dt = datetime.now().astimezone()

# 2. Convert it to UTC
utc_dt = local_dt.astimezone(timezone.utc)

print("Local time:", local_dt)
print("UTC time:  ", utc_dt)
```

#### How it works

1. **`datetime.now().astimezone()`**

   * Calling `.astimezone()` with no arguments attaches your system’s local timezone to the naive `datetime.now()`.
2. **`.astimezone(timezone.utc)`**

   * Converts that timezone‐aware datetime into UTC, adjusting the hour appropriately.

---

### One‑liner

If you just need UTC right away, you can skip the local step:

```python
from datetime import datetime, timezone

utc_now = datetime.now(timezone.utc)
print("UTC now:", utc_now)
```

* **Note:** `datetime.utcnow()` returns a “naive” `datetime` (no timezone info), so it’s better to use `datetime.now(timezone.utc)` to get an aware UTC timestamp.
