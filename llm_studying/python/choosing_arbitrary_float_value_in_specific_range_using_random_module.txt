In Python you can do this with the `random.uniform(a, b)` function:

```python
import random

value = random.uniform(0.8, 1.2)
print(value)
```

---

### Notes:

* `random.uniform(0.8, 1.2)` returns a random float in the range $[0.8, 1.2]$.
* The range is **inclusive** at both ends in theory, though practically the exact upper bound (1.2) is almost never produced because floating-point draws are continuous.
* If you want many samples:

```python
samples = [random.uniform(0.8, 1.2) for _ in range(10)]
print(samples)
```

---

✅ That will give you floats uniformly spread between 0.8 and 1.2.
