In modern Python (3.10+), use `| None`:

```python
def analyze(name: str, timeout: int | None = None):
    ...
```

The legacy equivalent is `Optional` from typing:

```python
from typing import Optional

def analyze(name: str, timeout: Optional[int] = None):
    ...
```

A common misconception: `Optional[int]` doesn't mean the parameter is optional — it just means the type can be `int` or `None`. The parameter is optional because of the `= None` default value. So `Optional[int]` is exactly equivalent to `int | None` or `Union[int, None]`.

Again, if you want modern syntax on older Python:

```python
from __future__ import annotations

def analyze(name: str, timeout: int | None = None):
    ...
```
