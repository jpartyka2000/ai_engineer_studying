Python sets support several comparison operators that test subset and superset relationships between sets.

**Subset operators: `<=` and `<`**

```python
# <= tests "is subset of" (can be equal)
{1, 2} <= {1, 2, 3}      # True
{1, 2, 3} <= {1, 2, 3}   # True (equal sets)

# < tests "is proper subset of" (must be strictly smaller)
{1, 2} < {1, 2, 3}       # True
{1, 2, 3} < {1, 2, 3}    # False (equal sets don't count)
```

**Superset operators: `>=` and `>`**

```python
# >= tests "is superset of" (can be equal)
{1, 2, 3} >= {1, 2}      # True
{1, 2, 3} >= {1, 2, 3}   # True (equal sets)

# > tests "is proper superset of" (must be strictly larger)
{1, 2, 3} > {1, 2}       # True
{1, 2, 3} > {1, 2, 3}    # False (equal sets don't count)
```

**Equality: `==` and `!=`**

```python
{1, 2, 3} == {3, 2, 1}   # True (order doesn't matter)
{1, 2} != {1, 2, 3}      # True
```

**Method equivalents:**

Each operator has a corresponding method:

| Operator | Method | Meaning |
|----------|--------|---------|
| `<=` | `a.issubset(b)` | a is subset of b |
| `<` | No direct method | a is proper subset of b |
| `>=` | `a.issuperset(b)` | a is superset of b |
| `>` | No direct method | a is proper superset of b |

```python
{1, 2}.issubset({1, 2, 3})      # True, same as {1, 2} <= {1, 2, 3}
{1, 2, 3}.issuperset({1, 2})    # True, same as {1, 2, 3} >= {1, 2}
```

**Edge cases with empty sets:**

```python
set() <= {1, 2, 3}    # True - empty set is subset of everything
set() < {1, 2, 3}     # True - empty set is proper subset of any non-empty set
set() <= set()        # True - empty set is subset of itself
set() < set()         # False - empty set is not a PROPER subset of itself
```

**Disjoint sets (no relationship):**

When sets have no subset/superset relationship, comparisons return False in both directions:

```python
{1, 2} <= {3, 4}      # False
{1, 2} >= {3, 4}      # False
{3, 4} <= {1, 2}      # False
{3, 4} >= {1, 2}      # False
```
