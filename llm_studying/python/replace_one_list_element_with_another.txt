The easiest way is using index assignment:

```python
my_list = ['a', 'b', 'c', 'd']

# Replace by index
my_list[1] = 'new_value'
# Result: ['a', 'new_value', 'c', 'd']
```

If you need to find and replace by value (not index):

```python
my_list = ['a', 'b', 'c', 'd']

# Find index first, then replace
idx = my_list.index('b')
my_list[idx] = 'new_value'
# Result: ['a', 'new_value', 'c', 'd']
```

Or as a one-liner using list comprehension:

```python
my_list = ['a', 'b', 'c', 'd']

# Replace all occurrences of 'b' with 'new_value'
my_list = ['new_value' if x == 'b' else x for x in my_list]
# Result: ['a', 'new_value', 'c', 'd']
```
