Even if you don't know which member of an Enum you have, you can still access its value using the `.value` attribute. For example:

```python
from enum import Enum

class Color(Enum):
    RED = 1
    GREEN = 2
    BLUE = 3

# Suppose you have an unknown Color enum instance:
color_instance = Color.RED  # It might be RED, GREEN, or BLUE

# To print the underlying value:
print(color_instance.value)  # This will print 1 if it's RED, 2 if it's GREEN, etc.

# If you also want to print the member name:
print(color_instance.name)  # This prints "RED"
```

No matter which enum member you have, using `.value` (or `.name`) gives you the information associated with that instance.
