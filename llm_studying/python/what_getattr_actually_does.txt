`getattr()` is a built-in function that does the same thing as dot notation, but lets you specify the attribute name as a string:

```python
obj.foo
getattr(obj, 'foo')
```

These are equivalent. The reason `getattr()` exists is for situations where you don't know the attribute name ahead of time:

```python
field_name = 'foo'
getattr(obj, field_name)  # works
obj.field_name            # doesn't work — looks for literal attribute called "field_name"
```

It also lets you provide a default if the attribute doesn't exist:

```python
getattr(obj, 'foo', None)  # returns None instead of raising AttributeError
```
