To control your log level via the `.env` file, just add this line to your `.env`:

```
LOG_LEVEL=DEBUG
```

Then when you run your script with `uv run --env-file .env ...`, the environment variable will be loaded and your logger will pick it up via this line in the code:

```python
logging.basicConfig(level=os.getenv("LOG_LEVEL", "INFO"))
```

Common values you might use:

| Value | What you'll see |
|-------|-----------------|
| `DEBUG` | Everything (verbose) |
| `INFO` | Progress updates and summaries |
| `WARNING` | Only problems and failures |
| `ERROR` | Only actual errors |
