Great question — `sys.path` and `PYTHONPATH` are **closely related but not the same**. Here's the breakdown:

---

### ✅ `PYTHONPATH` (Environment Variable)

* **What it is**: An **environment variable** you set outside Python (in your shell, IDE, or notebook config).
* **What it does**: When Python starts, it **reads `PYTHONPATH`** and **prepends those paths to `sys.path`**.
* **Use case**: Add project directories globally (e.g., for scripts, notebooks, or packages not installed via `pip`).

🧠 Think of it as:

> "Tell Python where to look *before* anything else, every time it starts."

#### Example:

```bash
export PYTHONPATH=/home/user/my_project/src
python script.py
```

---

### ✅ `sys.path` (Python Variable)

* **What it is**: A **Python list** of directories Python will search when you use `import`.

* **What it contains**:

  1. The script’s directory (`""`)
  2. Standard library paths
  3. Site-packages (installed libs)
  4. Anything from `PYTHONPATH`
  5. Anything manually added via `sys.path.append()` or `sys.path.insert()`

* **Modifiable at runtime**: You can dynamically add paths in code:

  ```python
  import sys
  sys.path.insert(0, "/some/new/path")
  ```

🧠 Think of it as:

> "The actual list of places Python will check to resolve imports."

---

### ✅ Summary

| Feature     | `PYTHONPATH`                     | `sys.path`                               |
| ----------- | -------------------------------- | ---------------------------------------- |
| Type        | Environment variable             | Python list (`List[str]`)                |
| When set    | Before starting Python           | During runtime (or boot via PYTHONPATH)  |
| Purpose     | Inject search paths on startup   | Control import resolution order          |
| Modifiable? | No (only before Python runs)     | Yes (at runtime via code)                |
| Used by     | Python startup (bootstrap logic) | The import system (e.g., `import mylib`) |

---

### ✅ Example Together

```bash
export PYTHONPATH=/my/project/src
python script.py
```

Inside `script.py`:

```python
import sys
print(sys.path)  # You'll see /my/project/src at or near the top
```
