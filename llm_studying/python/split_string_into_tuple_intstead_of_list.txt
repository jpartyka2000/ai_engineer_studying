You can simply take the list that `str.split()` returns and wrap it in a `tuple()` call. For example:

```python
s = "apple,banana,cherry"
# this gives you a list:
lst = s.split(",")                # ['apple', 'banana', 'cherry']

# wrap it in tuple():
tup = tuple(lst)                  # ('apple', 'banana', 'cherry')
```

Or in one line:

```python
tup = tuple(s.split(","))         # ('apple', 'banana', 'cherry')
```

If you know exactly how many pieces there’ll be, you can even unpack directly into a tuple:

```python
a, b, c = s.split(",")
# now (a, b, c) is a tuple of three strings
```

Finally, if you really want to split into a tuple of single characters, you can do:

```python
chars = tuple(s)                  # ('a','p','p','l','e',',','b',…)
```

—but for delimited splitting, just wrap the `.split()` in `tuple()`.
