By default, when you use `append()` on a `deque`, it adds the element to the **right side** (the end) of the deque. 

So in your code:

```python
self.message_deque.append({"role": role, "content": content})
```

The new message will be added to the **right side** (end) of `self.message_deque`. If you want to add a message to the left side (front), you can use `appendleft()` instead:

```python
self.message_deque.appendleft({"role": role, "content": content})
```

But in your case, since you want the newest messages at the end of the deque, `append()` is the correct method to use.
