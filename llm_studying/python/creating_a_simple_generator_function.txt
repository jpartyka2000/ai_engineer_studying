QUESTION:
Complete the following code to create a generator function 
that yields the numbers 1, 2, and 3:

    ___ simple_generator():
        ___ 1
        ___ 2
        ___ 3

What two keywords should fill in the blanks?

------------------------------------------------------------
YOUR ANSWER:
def simple_generator():
    yield 1
    yield 2
    yield 3

CORRECT ANSWER:
def, yield

RESULT: ✓ Correct

EXPLANATION:
- `def` defines the function.
- `yield` makes it a generator function, pausing execution 
  and returning a value each time it's called.
