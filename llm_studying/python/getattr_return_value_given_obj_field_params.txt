`getattr(obj, field)` looks up the attribute whose name is the string held in `field` on the object `obj` and returns its value.

* **Equivalent to**:

  ```python
  obj.field_name   # if field == "field_name"
  ```
* **If the attribute doesn’t exist**, it raises `AttributeError`.
* **You can supply a default** as a third argument to avoid the exception:

  ```python
  getattr(obj, field, default_value)
  ```

  which returns `default_value` when `obj` has no such attribute.
