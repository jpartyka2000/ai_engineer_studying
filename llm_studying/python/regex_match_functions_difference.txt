If you want to **match** the string `"DELETE FROM"` in a case-insensitive manner without replacing anything, you can use Python's **`re.search()`** or **`re.findall()`** functions from the `re` module.

---

### Using `re.search()` to Match Once

If you just want to check if the pattern exists in the string (case-insensitively):

```python
import re

text = "Some text with DELETE FROM, delete from, and DeLeTe FrOm in it."

# Check for a case-insensitive match
match = re.search(r"DELETE FROM", text, flags=re.IGNORECASE)

if match:
    print("Match found!")
else:
    print("No match found.")
```

**Output:**
```
Match found!
```

---

### Using `re.findall()` to Find All Matches

If you want to find **all occurrences** of `"DELETE FROM"` in a case-insensitive manner:

```python
import re

text = "Some text with DELETE FROM, delete from, and DeLeTe FrOm in it."

# Find all matches
matches = re.findall(r"DELETE FROM", text, flags=re.IGNORECASE)

print(f"Matches found: {matches}")
print(f"Number of matches: {len(matches)}")
```

**Output:**
```
Matches found: ['DELETE FROM', 'delete from', 'DeLeTe FrOm']
Number of matches: 3
```

---

### Using `re.match()` (Matches Only the Start of a String)

If you want to check only the **beginning of the string** for the pattern:

```python
import re

text = "DELETE FROM table_name WHERE ..."

# Check only the start of the string
if re.match(r"DELETE FROM", text, flags=re.IGNORECASE):
    print("Match found at the start of the string!")
else:
    print("No match at the start.")
```

---

### Key Notes:
- **`re.search()`**: Checks for the first occurrence of the pattern anywhere in the string.
- **`re.findall()`**: Returns a list of all non-overlapping matches in the string.
- **`re.match()`**: Checks only the beginning of the string.
- **`re.IGNORECASE`**: Makes the pattern case-insensitive.

These functions allow you to match the string `"DELETE FROM"` in a case-insensitive way without replacing it.
