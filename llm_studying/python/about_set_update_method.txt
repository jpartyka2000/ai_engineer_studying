## `set.update()`

The `update()` method adds all elements from one or more iterables into an existing set, modifying it in place.

### Basic Syntax

```python
my_set.update(iterable)
```

### Accepted Iterables

You can pass virtually any iterable as input:

**Sets**
```python
a = {1, 2, 3}
a.update({4, 5})
# a is now {1, 2, 3, 4, 5}
```

**Lists**
```python
a = {1, 2, 3}
a.update([4, 5, 6])
# a is now {1, 2, 3, 4, 5, 6}
```

**Tuples**
```python
a = {1, 2, 3}
a.update((4, 5))
# a is now {1, 2, 3, 4, 5}
```

**Strings** (iterates over characters)
```python
a = {"a", "b"}
a.update("cd")
# a is now {"a", "b", "c", "d"}
```

**Dictionaries** (iterates over keys only)
```python
a = {1, 2}
a.update({"x": 10, "y": 20})
# a is now {1, 2, "x", "y"}
```

**Generators**
```python
a = {1, 2}
a.update(x * 2 for x in range(3))
# a is now {0, 1, 2, 4}
```

**Range objects**
```python
a = {1, 2}
a.update(range(5, 8))
# a is now {1, 2, 5, 6, 7}
```

### Multiple Iterables at Once

You can pass multiple iterables in a single call:

```python
a = {1}
a.update([2, 3], {4, 5}, (6, 7))
# a is now {1, 2, 3, 4, 5, 6, 7}
```

### Return Value

`update()` returns `None`—it modifies the set in place rather than returning a new set.

```python
a = {1, 2}
result = a.update({3, 4})
print(result)  # None
print(a)       # {1, 2, 3, 4}
```
