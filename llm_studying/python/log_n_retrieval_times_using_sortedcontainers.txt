Yes, there is a data structure in Python called `sortedcontainers.SortedList` that provides list-like functionality with logarithmic (`O(log n)`) time retrieval. Unlike a standard Python list, which has `O(n)` time complexity for lookups, `SortedList` from the `sortedcontainers` module maintains a sorted order and allows for efficient insertion, deletion, and retrieval.

Here's how `SortedList` can be used:

### Installation

You need to install `sortedcontainers` first, as it is not part of the standard library:
```bash
pip install sortedcontainers
```

### Example Usage:
```python
from sortedcontainers import SortedList

# Create a SortedList
slist = SortedList([5, 3, 8, 1, 4])

# Automatically sorts the elements
print(slist)  # Output: SortedList([1, 3, 4, 5, 8])

# O(log n) retrieval by index
print(slist[2])  # Output: 4

# O(log n) time for adding/removing elements while maintaining order
slist.add(7)
print(slist)  # Output: SortedList([1, 3, 4, 5, 7, 8])

# Check if an element exists (O(log n) time)
if 5 in slist:
    print("Found!")  # Output: Found!
```

### Explanation:

- **`SortedList`** maintains the order of elements automatically.
- **`O(log n)` operations:** Retrieval, addition, and removal are more efficient than a typical list for sorted collections.
- **Access by index:** This allows it to function similarly to a list but with faster lookups for sorted data.

`SortedList` is ideal when you need both the properties of a list (indexed access) and fast retrieval or insertion in a sorted collection.
