In Python, to check if a variable (especially a **primitive** like `float`) is `NaN`, use `math.isnan()` or `numpy.isnan()`:

---

### ✅ Recommended approaches

#### 1. **With `math.isnan()`** (for scalars):

```python
import math

x = float('nan')
print(math.isnan(x))  # True
```

> Use this if you're working with plain Python floats and not using NumPy.

---

#### 2. **With `numpy.isnan()`** (works for both scalars and arrays):

```python
import numpy as np

x = np.nan
print(np.isnan(x))  # True
```

> Use this if you're already using NumPy or want to handle arrays.

---

### ⚠️ Don't use `== np.nan`

```python
x = float('nan')
print(x == np.nan)  # False  ← doesn't work!
```

This fails because `NaN != NaN` by definition (IEEE 754 standard).
