You can use `collections.Counter` for this:

```python
from collections import Counter

values = ["apple", "banana", "apple", "cherry", "banana", "apple"]
counts = dict(Counter(values))
# {'apple': 3, 'banana': 2, 'cherry': 1}
```

`Counter` is already dict-like, so the `dict()` call is optional unless you specifically need a plain dict.

**Alternative without imports:**

```python
counts = {}
for value in values:
    counts[value] = counts.get(value, 0) + 1
```
