Yes—the exact Python `strftime`/`strptime` pattern is:

```python
"%m/%d/%Y %H:%M:%S"
```

Where:

* `%m` → month (01–12)
* `%d` → day (01–31)
* `%Y` → 4‑digit year (e.g. 2025)
* `%H` → hour in 24 h (00–23)
* `%M` → minute (00–59)
* `%S` → second (00–59)

### Examples

```python
from datetime import datetime

now = datetime.now()
# format to string:
s = now.strftime("%m/%d/%Y %H:%M:%S")
print(s)  # e.g. "05/14/2025 13:45:07"

# parse back from string:
dt = datetime.strptime("05/14/2025 13:45:07", "%m/%d/%Y %H:%M:%S")
print(dt)  # datetime.datetime(2025, 5, 14, 13, 45, 7)
```

That will always zero‑pad month, day, hour, minute, and second.
