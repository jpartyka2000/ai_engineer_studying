## `textwrap` Library Overview

`textwrap` is Python's built-in library for **formatting text** - wrapping long lines, adding indentation, and cleaning up multi-line strings.

## Main Functions:

### 1. **`wrap()` - Break Long Text into Lines**
```python
import textwrap

text = "This is a very long sentence that needs to be wrapped because it exceeds the maximum line width we want."

lines = textwrap.wrap(text, width=30)
for line in lines:
    print(line)
# Output:
# This is a very long sentence
# that needs to be wrapped
# because it exceeds the
# maximum line width we want.
```

### 2. **`fill()` - Wrap and Join into String**
```python
# Same as wrap() but returns a single string
wrapped = textwrap.fill(text, width=30)
print(wrapped)
# This is a very long sentence
# that needs to be wrapped
# because it exceeds the
# maximum line width we want.
```

### 3. **`dedent()` - Remove Common Leading Whitespace**
```python
# Perfect for multi-line strings in code
text = """
    Hello World!
    This is indented text.
        Extra indent here.
"""

print(textwrap.dedent(text))
# Output (indentation removed):
# Hello World!
# This is indented text.
#     Extra indent here.
```

### 4. **`indent()` - Add Prefix to Lines**
```python
text = "Line 1\nLine 2\nLine 3"
indented = textwrap.indent(text, prefix="    ")
print(indented)
# Output:
#     Line 1
#     Line 2
#     Line 3
```

## Common Use Cases:

### Terminal Output Formatting
```python
def print_help(description):
    # Wrap to terminal width
    wrapped = textwrap.fill(description, width=80)
    print(wrapped)
```

### Clean Multi-line Strings
```python
def get_sql_query():
    return textwrap.dedent("""
        SELECT user_id, name, email
        FROM users
        WHERE active = true
        ORDER BY created_at DESC
    """).strip()
```

### Creating Formatted Messages
```python
error_msg = textwrap.dedent(f"""
    Error: File not found
    
    The file '{filename}' could not be located.
    Please check the path and try again.
""").strip()
```

**Bottom line**: `textwrap` helps format text for display - especially useful for terminal output, docstrings, and cleaning up indented multi-line strings in your code.
