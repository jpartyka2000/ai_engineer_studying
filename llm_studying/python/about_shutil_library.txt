Great question — `shutil` is a **standard Python library** for **high-level file and directory operations** (its name comes from “shell utilities”).

It sits above the lower-level `os` and `pathlib` modules and provides convenient functions for **copying, moving, archiving, and deleting files and directories** — basically, things you’d often do from a command line (“shell”) script, but in Python.

---

## 🧠 Key Capabilities of `shutil`

Here’s what it can do:

### 🗂️ 1. Copy files and directories

```python
import shutil

shutil.copy("data.txt", "backup/data.txt")         # copy a file
shutil.copy2("data.txt", "backup/data.txt")        # copy + preserve metadata
shutil.copytree("src_folder", "dest_folder")       # recursively copy a directory
```

---

### 🚚 2. Move or rename

```python
shutil.move("old_name.txt", "new_folder/new_name.txt")
```

Equivalent to a Unix `mv` — moves or renames a file/directory.

---

### 🧹 3. Remove directories recursively

```python
shutil.rmtree("temp_folder")
```

Deletes a folder *and everything inside it* (⚠️ use carefully).

---

### 📦 4. Make or extract archives (zip, tar, etc.)

```python
# Create an archive
shutil.make_archive("backup", "zip", "my_folder")

# Extract an archive
shutil.unpack_archive("backup.zip", "extracted_folder")
```

You can specify formats like `"zip"`, `"gztar"`, `"bztar"`, etc.

---

### ⚙️ 5. Query or manage disk usage

```python
usage = shutil.disk_usage("/")
print(usage.total, usage.used, usage.free)
```

---

### 💻 6. Run shell commands with safety

```python
shutil.which("python")
```

Finds the path of a command, similar to `which` in Unix shells.

---

## ⚡ Typical Uses

* Backup scripts (copy/move/archive files)
* Deployment scripts (replace directories safely)
* Temporary workspace cleanup (`shutil.rmtree`)
* Packaging or data export automation
* Disk monitoring or management tasks

---

## 🧩 Example: Copy & Zip a Project Folder

```python
import shutil

# Copy project folder
shutil.copytree("project", "backup/project_copy")

# Zip the copy
shutil.make_archive("project_backup", "zip", "backup/project_copy")

# Clean up
shutil.rmtree("backup/project_copy")
```
