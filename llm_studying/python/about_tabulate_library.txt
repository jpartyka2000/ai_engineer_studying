The **tabulate** library is a small, lightweight Python package for **pretty-printing tabular data** in a variety of plain-text (and other) formats. Rather than manually building strings with fixed-width columns or worrying about alignment, you can just hand your data to `tabulate()` and get a nicely formatted table.

---

## Key Features

1. **Supports multiple input types**

   * Lists of lists/tuples
   * Lists of dicts
   * Dicts of iterables
   * Any iterable of iterables (e.g. generator of rows)
   * Pandas DataFrames (by converting `.values` plus `.columns`)

2. **Multiple output formats**

   * Plain text (with automatic column widths)
   * Grid-style tables (ASCII-art borders)
   * GitHub-flavored Markdown
   * reStructuredText
   * HTML
   * LaTeX
   * And more (you can even register your own table formats)

3. **Automatic alignment & padding**

   * Numbers right-aligned, text left-aligned by default (configurable)
   * Automatic column-width calculation
   * Optional number formatting (e.g. float precision)

4. **Lightweight & zero dependencies**

   * Pure Python (only requires the standard library)
   * Very small footprint

---

## Basic Usage

```python
from tabulate import tabulate

data = [
    ["Alice",   24, 3.5],
    ["Bob",     19, 3.7],
    ["Charlie", 22, 3.9],
]
headers = ["Name", "Age", "GPA"]

print(tabulate(data, headers=headers, tablefmt="github"))
```

**Output (GitHub-flavored Markdown):**

```markdown
| Name    |   Age |   GPA |
|---------|-------|-------|
| Alice   |    24 |   3.5 |
| Bob     |    19 |   3.7 |
| Charlie |    22 |   3.9 |
```

---

## Other Formats

```python
print(tabulate(data, headers, tablefmt="grid"))
```

```
+-----------+-------+-------+
| Name      |   Age |   GPA |
+===========+=======+=======+
| Alice     |    24 |   3.5 |
+-----------+-------+-------+
| Bob       |    19 |   3.7 |
+-----------+-------+-------+
| Charlie   |    22 |   3.9 |
+-----------+-------+-------+
```

```python
# LaTeX tabular
print(tabulate(data, headers, tablefmt="latex"))
```

```latex
\begin{tabular}{lrr}
\hline
 Name    &   Age &   GPA \\
\hline
 Alice   &    24 &   3.5 \\
 Bob     &    19 &   3.7 \\
 Charlie &    22 &   3.9 \\
\hline
\end{tabular}
```

## Handy Tips

* **Number formatting**:

  ```python
  print(tabulate(data, headers, floatfmt=".2f"))
  ```
* **Custom alignment**:

  ```python
  # left-align all columns
  print(tabulate(data, headers, stralign="left", numalign="left"))
  ```
* **Dict input**:

  ```python
  dicts = [{"Name":"Alice","Age":24},
           {"Name":"Bob","Age":19}]
  print(tabulate(dicts, headers="keys", tablefmt="plain"))
  ```
* **Pandas integration**:

  ```python
  import pandas as pd
  df = pd.DataFrame(data, columns=headers)
  print(tabulate(df, headers="keys", tablefmt="psql"))
  ```

---

### ✅ Why Use It?

* **Readable CLI output**: great for command-line tools and REPL explorations.
* **Documentation & reports**: easily embed tables in Markdown, HTML, or LaTeX docs.
* **Zero boilerplate**: no manual string formatting or column-width calculations.

In short, **tabulate** makes it effortless to turn raw rows & columns into elegant tables.
