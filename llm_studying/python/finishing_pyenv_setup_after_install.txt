Since you've installed `pyenv` using Homebrew, you now need to **configure your shell** so that `pyenv` is recognized properly. Follow these steps:

---

### **Step 1: Add pyenv to Your Shell**
Run the following commands to add `pyenv` to your shell configuration file (`~/.zshrc` if using zsh or `~/.bashrc` if using bash):

#### **For zsh (default shell in macOS Catalina and later)**
```sh
echo 'export PYENV_ROOT="$HOME/.pyenv"' >> ~/.zshrc
echo 'export PATH="$PYENV_ROOT/bin:$PATH"' >> ~/.zshrc
echo 'eval "$(pyenv init --path)"' >> ~/.zshrc
echo 'eval "$(pyenv init -)"' >> ~/.zshrc
```

#### **For bash (if using an older macOS version or explicitly using bash)**
```sh
echo 'export PYENV_ROOT="$HOME/.pyenv"' >> ~/.bashrc
echo 'export PATH="$PYENV_ROOT/bin:$PATH"' >> ~/.bashrc
echo 'eval "$(pyenv init --path)"' >> ~/.bashrc
echo 'eval "$(pyenv init -)"' >> ~/.bashrc
```

---

### **Step 2: Apply Changes**
After updating your shell configuration, reload it:
```sh
source ~/.zshrc  # For zsh users
```
or
```sh
source ~/.bashrc  # For bash users
```

---

### **Step 3: Verify Installation**
Check if `pyenv` is working correctly by running:
```sh
pyenv --version
```
If it outputs the installed version (e.g., `pyenv 2.3.22`), then `pyenv` is properly set up.

---

### **Step 4: Install a Python Version**
Now you can install a Python version using:
```sh
pyenv install 3.11.6  # Replace with your desired Python version
```

List installed versions:
```sh
pyenv versions
```

Set a global Python version:
```sh
pyenv global 3.11.6
```

Confirm that `pyenv` is managing Python:
```sh
python --version
```
It should return `Python 3.11.6` (or the version you installed).
