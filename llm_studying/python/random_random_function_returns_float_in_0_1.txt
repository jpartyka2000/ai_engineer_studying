The function call

```python
import random
random.random()
```

returns a pseudorandom floating-point number **in the half-open interval** \[0.0, 1.0) — that is, it can be 0.0 but will always be less than 1.0.

It’s drawn from a uniform distribution, so every value in that range is (theoretically) equally likely.

**Example usage**:

```python
>>> import random
>>> random.random()
0.3745401188473625
>>> random.random()
0.9507143064099162
```

Each call will produce a different float between 0.0 (inclusive) and 1.0 (exclusive).
