Yes, normal Python classes are dynamic by default. You can add new attributes (fields) to an instance at runtime simply by assigning to them. For example:

```python
class MyClass:
    def __init__(self, value):
        self.value = value

instance = MyClass(10)
instance.new_field = "I was added dynamically"
print(instance.new_field)  # Outputs: I was added dynamically
```

This works because Python uses a dynamic attribute lookup mechanism, and every instance has a `__dict__` that holds its attributes. However, if you want to restrict this behavior, you can define `__slots__` in your class, which limits the attributes that instances can have.
