### **Getting the Value from an Enum in Python**
In Python, you can retrieve the value associated with an `Enum` member using the `.value` attribute.

---

### **1️⃣ Example: Basic Enum Usage**
```python
from enum import Enum

class Status(Enum):
    SUCCESS = 200
    NOT_FOUND = 404
    SERVER_ERROR = 500

# Get value from an enum member
print(Status.SUCCESS.value)  # Output: 200
print(Status.NOT_FOUND.value)  # Output: 404
```

- `Status.SUCCESS.value` → **200**
- `Status.NOT_FOUND.value` → **404**

---

### **2️⃣ Example: Enum with String Values**
```python
class UserRole(Enum):
    ADMIN = "Administrator"
    USER = "Regular User"
    GUEST = "Guest User"

print(UserRole.ADMIN.value)  # Output: "Administrator"
```
- Works the same way for **string values**.

---

### **3️⃣ Getting Enum from a Value**
If you **only have the value** and want to retrieve the corresponding `Enum` member, use `Enum(value)`:
```python
print(Status(200))  # Output: Status.SUCCESS
print(Status(404))  # Output: Status.NOT_FOUND
```
⚠ **Raises `ValueError` if the value doesn’t exist**:
```python
print(Status(999))  # ❌ Raises ValueError: 999 is not a valid Status
```

---

### **4️⃣ Checking if a Value Exists in an Enum**
If you're unsure whether a value exists in an Enum:
```python
def is_valid_status(code):
    return code in {s.value for s in Status}

print(is_valid_status(200))  # ✅ True
print(is_valid_status(999))  # ❌ False
```

---

### **Key Takeaways**
✅ Use `.value` to get the value from an `Enum`.  
✅ Use `Enum(value)` to get the Enum member from a value.  
✅ Use a set comprehension `{e.value for e in EnumClass}` to check if a value is valid.
