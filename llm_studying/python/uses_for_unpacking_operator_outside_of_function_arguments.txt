Yes, you can. Python's unpacking operators (both * and **) are not limited to function arguments. They’re also used in several other contexts:

- **Assignment Unpacking:**  
  You can use the * operator for extended iterable unpacking. For example:  
  ```python
  first, *middle, last = [1, 2, 3, 4, 5]
  # first = 1, middle = [2, 3, 4], last = 5
  ```

- **List and Tuple Literals:**  
  Since Python 3.5 (thanks to PEP 448), you can use * to unpack iterables into lists or tuples:  
  ```python
  a = [1, 2, 3]
  b = [*a, 4, 5]  # b becomes [1, 2, 3, 4, 5]
  ```

- **Dictionary Literals:**  
  Similarly, you can use ** to merge dictionaries:  
  ```python
  dict1 = {'a': 1, 'b': 2}
  dict2 = {'c': 3}
  merged = {**dict1, **dict2}  # merged becomes {'a': 1, 'b': 2, 'c': 3}
  ```

These features make your code more concise and expressive beyond just function argument passing.
