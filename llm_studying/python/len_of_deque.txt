To get the number of items in a `deque`, you can simply use the `len()` function, just like you would with a regular Python list. The `len()` function will return the current size of the deque.

### Example:
```python
from collections import deque

# Create a deque and add some items
messages = deque(["Message 1", "Message 2", "Message 3"])

# Get the number of items in the deque
num_items = len(messages)

print(num_items)  # Output: 3
```

This gives you the current number of elements in the `deque`.
