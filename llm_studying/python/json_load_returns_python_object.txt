## What `json.load()` Returns:

`json.load()` returns **native Python objects** - the JSON is automatically converted to Python types:

### **Type Conversion:**
```python
import json

with open('data.json', 'r') as f:
    result = json.load(f)
    # Returns Python objects based on JSON content:
```

| JSON Type | → | Python Type |
|-----------|---|-------------|
| `{...}` object | → | `dict` |
| `[...]` array | → | `list` |
| `"string"` | → | `str` |
| `123` number | → | `int` or `float` |
| `true`/`false` | → | `bool` |
| `null` | → | `None` |

### **Examples:**
```python
# If data.json contains: {"name": "Alice", "age": 30}
data = json.load(f)
print(type(data))  # <class 'dict'>
print(data)        # {'name': 'Alice', 'age': 30}

# If data.json contains: [1, 2, 3]
data = json.load(f)
print(type(data))  # <class 'list'>
print(data)        # [1, 2, 3]

# If data.json contains: "just a string"
data = json.load(f)
print(type(data))  # <class 'str'>
print(data)        # 'just a string'
```

### **Complex Example:**
```python
# data.json:
{
    "users": [
        {"id": 1, "active": true},
        {"id": 2, "active": false}
    ],
    "count": 2,
    "next": null
}

# Loading it:
data = json.load(f)
# Returns a dict with:
# - data["users"] is a list
# - data["users"][0] is a dict
# - data["users"][0]["active"] is bool (True)
# - data["next"] is None
```

**Bottom line**: `json.load()` returns regular Python objects (usually a `dict` or `list`) that you can immediately use in your code - no further conversion needed!
