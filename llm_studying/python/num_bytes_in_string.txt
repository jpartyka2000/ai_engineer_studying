To get the total number of bytes in a string in Python, you can encode the string to bytes and then get the length of the byte representation. Here's an example:

```python
# Your string
my_string = "Hello, World!"

# Encode the string to bytes (using UTF-8 by default) and get the length
num_bytes = len(my_string.encode('utf-8'))

print(f"Number of bytes: {num_bytes}")
```

### Explanation:
- `my_string.encode('utf-8')` converts the string into its byte representation using UTF-8 encoding.
- `len()` then returns the number of bytes in that byte string.

This will give you the number of bytes that the string occupies when encoded in UTF-8.
