============================================================
GENERATOR EXPRESSIONS — A BRIEF OVERVIEW
============================================================

WHAT ARE THEY?
Generator expressions are a concise, memory-efficient way to 
create generators. They look just like list comprehensions, 
but use parentheses () instead of brackets [].

    # List comprehension — creates entire list in memory
    squares_list = [x ** 2 for x in range(1000000)]

    # Generator expression — creates values one at a time
    squares_gen = (x ** 2 for x in range(1000000))

------------------------------------------------------------
SYNTAX
------------------------------------------------------------

    (expression for item in iterable if condition)

    - expression        What to yield (e.g., x ** 2)
    - for item in iter  Loop over a sequence
    - if condition      Optional filter

------------------------------------------------------------
KEY CHARACTERISTICS
------------------------------------------------------------

Feature          Generator Expression    List Comprehension
-----------      --------------------    ------------------
Syntax           ( )                     [ ]
Memory           Lazy (one at a time)    Eager (all at once)
Reusable?        No (exhausted once)     Yes
Speed            Slightly slower         Slightly faster
Best for         Large/infinite data     Small datasets

------------------------------------------------------------
EXAMPLES
------------------------------------------------------------

# Basic: squares of 0-9
squares = (x ** 2 for x in range(10))

# With filter: even squares only
even_sq = (x ** 2 for x in range(10) if x % 2 == 0)

# Nested: flatten a 2D list
matrix = [[1,2], [3,4], [5,6]]
flat = (num for row in matrix for num in row)

# Using with functions (no extra parentheses needed)
total = sum(x ** 2 for x in range(10))

------------------------------------------------------------
HOW TO CONSUME GENERATOR EXPRESSIONS
------------------------------------------------------------

gen = (x ** 2 for x in range(5))

# Option 1: Loop
for value in gen:
    print(value)

# Option 2: next()
print(next(gen))  # 0
print(next(gen))  # 1

# Option 3: Convert to list (defeats memory benefit)
list(gen)  # [0, 1, 4, 9, 16]

# Option 4: Pass directly to functions
sum(x ** 2 for x in range(5))   # 30
max(x ** 2 for x in range(5))   # 16
''.join(str(x) for x in range(5))  # '01234'

------------------------------------------------------------
WHEN TO USE GENERATOR EXPRESSIONS
------------------------------------------------------------

✓ Processing large files line by line
✓ Working with large datasets that won't fit in memory
✓ Chaining operations in a pipeline
✓ When you only need to iterate once
✓ Passing to functions like sum(), max(), min(), join()

✗ When you need to access items multiple times
✗ When you need indexing or slicing
✗ When the dataset is small (list is fine)
