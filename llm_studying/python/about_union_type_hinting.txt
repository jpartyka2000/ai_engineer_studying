### **Overview of Union Type Hinting in Python**  

#### **What is Union Type Hinting?**  
Union type hinting in Python allows you to specify that a variable or function parameter **can accept multiple types**. This is useful when a value can have different types depending on context.  

---

### **1️⃣ Using `Union` from `typing` (Before Python 3.10)**  
Before Python 3.10, you had to use `Union` from the `typing` module:

```python
from typing import Union

def process_value(value: Union[int, float, str]) -> str:
    return f"Processed value: {value}"

print(process_value(42))        # int
print(process_value(3.14))      # float
print(process_value("hello"))   # str
```
Here, `value` can be an **int, float, or str**.

---

### **2️⃣ Using the `|` Operator (Python 3.10+)**
Starting in **Python 3.10**, you can use the `|` (pipe) operator instead of `Union`:

```python
def process_value(value: int | float | str) -> str:
    return f"Processed value: {value}"
```
This is more **concise** and **readable**.

---

### **3️⃣ Using `Optional[T]` (Shorthand for `T | None`)**
If a variable can be **`None` or another type**, you can use `Optional[T]`, which is equivalent to `Union[T, None]`:

#### ✅ **Before Python 3.10:**
```python
from typing import Optional

def get_user_name(user_id: int) -> Optional[str]:  # Same as Union[str, None]
    return "Alice" if user_id == 1 else None
```

#### ✅ **Python 3.10+:**
```python
def get_user_name(user_id: int) -> str | None:
    return "Alice" if user_id == 1 else None
```

---

### **4️⃣ Example Use Cases for Union Types**
✅ **Handling Multiple Input Types:**
```python
def add(x: int | float, y: int | float) -> float:
    return x + y

print(add(5, 3.2))  # 8.2
```

✅ **Defining Class Attributes in Dataclasses:**
```python
from dataclasses import dataclass

@dataclass
class Task:
    name: str
    response: str | None = None  # Optional response

task1 = Task(name="Example")
print(task1.response)  # Output: None
```

✅ **Using `Union` in Type Annotations for Lists:**
```python
from typing import List

data: List[int | str] = [1, "hello", 3, "world"]
```

---

### **Key Takeaways**
🔹 **Before Python 3.10** → Use `Union[T1, T2]`  
🔹 **Python 3.10+** → Use `T1 | T2` for cleaner syntax  
🔹 **Use `Optional[T]` or `T | None`** for values that may be `None`  
🔹 **Helps improve code readability and type safety**
