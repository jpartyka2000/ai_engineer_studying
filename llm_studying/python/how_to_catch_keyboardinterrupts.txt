You can catch a `KeyboardInterrupt` in Python with a `try`/`except` block. This is the error that’s raised when you press **Ctrl+C** (or **Cmd+.** on some Macs) to stop a running script.

Here’s a simple example:

```python
try:
    print("Press Ctrl+C to stop...")
    while True:
        pass  # your long-running code here
except KeyboardInterrupt:
    print("\nKeyboardInterrupt caught! Cleaning up and exiting...")
```

---

### ✅ Key details:

* `KeyboardInterrupt` is a built-in exception, so you don’t need to import anything.
* The `try` block is where you put the code that might be interrupted.
* The `except KeyboardInterrupt` block handles it gracefully instead of just throwing a traceback.
* You can also combine it with `finally` to ensure cleanup:

```python
try:
    while True:
        pass
except KeyboardInterrupt:
    print("Stopped by user")
finally:
    print("Goodbye!")
```
