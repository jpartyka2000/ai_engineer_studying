You can reference a function's **docstring** programmatically using the `.__doc__` attribute or the built-in `help()` function.

---

### **1️⃣ Using `.__doc__`**
Every function in Python has a special attribute `.__doc__` that stores its **docstring**.

```python
def greet(name):
    """This function greets the person whose name is passed as an argument."""
    return f"Hello, {name}!"

print(greet.__doc__)  
```
**Output:**
```
This function greets the person whose name is passed as an argument.
```

---

### **2️⃣ Using `help()`**
You can also use the built-in `help()` function to display a function's docstring.

```python
help(greet)
```
This prints a nicely formatted version of the docstring.

---

### **3️⃣ Accessing Docstring for a Method in a Class**
```python
class Math:
    """A simple math utility class."""

    def add(a, b):
        """Returns the sum of a and b."""
        return a + b

print(Math.add.__doc__)
```
**Output:**
```
Returns the sum of a and b.
```

---

### **4️⃣ Accessing Module-Level Docstrings**
If you want to get the **docstring of an entire module**, use:
```python
import math
print(math.__doc__)
```

---

### **5️⃣ Using `inspect` for Multi-line Docstrings**
For better formatting, use `inspect.getdoc()`, which removes excess indentation.

```python
import inspect
print(inspect.getdoc(greet))
```

---

### **Key Takeaways**
✅ Use `.__doc__` to directly access a function’s docstring.  
✅ Use `help(function)` for a formatted display.  
✅ Use `inspect.getdoc()` to clean up formatting.
