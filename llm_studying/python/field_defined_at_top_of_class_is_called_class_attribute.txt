It's a **class attribute** (Python's term for what other languages call a static field).

```python
class CueReactAgent:
    is_legacy: bool = True  # Class attribute - shared across all instances
```

**Difference:**

```python
# Class attribute (shared)
class Foo:
    is_legacy: bool = True

# Instance attribute (per-instance)
class Bar:
    def __init__(self):
        self.is_legacy: bool = True
```

**For your use case, either works** because:

1. You're only reading it, not modifying per-instance
2. `getattr(agent, 'is_legacy', False)` finds both class and instance attributes

But class attribute is the right choice here — it's a property of the class itself, not something that varies per instance.
