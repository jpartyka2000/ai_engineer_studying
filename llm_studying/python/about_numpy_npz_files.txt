An `.npz` file in NumPy is a **compressed archive file** that stores multiple NumPy arrays. It is part of the NumPy file format designed to efficiently save and load array data to and from disk.

### Key Features
1. **Compressed Archive**:
   - The `.npz` file uses **ZIP compression** to store data, making it efficient in terms of disk space.
   
2. **Multiple Arrays**:
   - It can contain multiple arrays, each saved as a separate `.npy` file within the archive. Each array is accessible by a key (a filename inside the archive).

3. **Dictionary-Like Structure**:
   - When loaded, the `.npz` file behaves like a dictionary, allowing you to access individual arrays by their names.

---

### How to Create an `.npz` File
You can save multiple arrays into a single `.npz` file using the `numpy.savez()` or `numpy.savez_compressed()` function:

```python
import numpy as np

# Example arrays
arr1 = np.array([1, 2, 3])
arr2 = np.array([[4, 5, 6], [7, 8, 9]])

# Save to an .npz file
np.savez("example.npz", array1=arr1, array2=arr2)

# Save with compression
np.savez_compressed("example_compressed.npz", array1=arr1, array2=arr2)
```

---

### How to Load an `.npz` File
You can load an `.npz` file using `numpy.load()`. This returns a dictionary-like object where arrays can be accessed by their names:

```python
# Load the .npz file
data = np.load("example.npz")

# Access arrays
arr1 = data["array1"]
arr2 = data["array2"]

print(arr1)  # Output: [1 2 3]
print(arr2)  # Output: [[4 5 6] [7 8 9]]
```

---

### Common Use Cases
- Storing results of experiments or models (e.g., training data and weights).
- Archiving data with multiple arrays.
- Sharing datasets in a compact format.
