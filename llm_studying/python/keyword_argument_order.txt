Yes, when calling a function, you can specify keyword arguments in any order because they are matched by name rather than position. For example, if you have:

```python
def my_function(a, b, c):
    print(a, b, c)

# You can call it like this:
my_function(c=3, a=1, b=2)  # Prints: 1 2 3
```

However, if you're mixing positional and keyword arguments, all positional arguments must come before any keyword arguments.
