Yes—PEP 8 is quite clear about that:

* **Block comments** (i.e. whole‐line comments) should look like this:

  ```python
  # This is a block comment, indented to the same level as the code
  ```

  Notice there’s exactly **one** space between the `#` and the start of the text.

* **Inline comments** (i.e. comments on the same line as code) should be separated from the code by **at least two spaces**, and then start with `#` plus a single space:

  ```python
  x = x + 1    # Increment x by one
  ```

So in all cases you should use **one space after the `#`**, and if it’s inline, leave two spaces between the end of your code and the `#`.
