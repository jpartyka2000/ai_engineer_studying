You can access a function’s docstring at runtime in a few different ways:

1. **Via the `__doc__` attribute**
   Every Python object gets a `__doc__` attribute containing its docstring (or `None` if it has none):

   ```python
   def foo(x, y):
       """Add x and y and return the result."""
       return x + y

   print(foo.__doc__)
   # → Add x and y and return the result.
   ```

2. **With the built-in `help()`**
   This prints the signature *and* the docstring in a nicely formatted pager:

   ```python
   help(foo)
   ```

3. **Using the `inspect` module**

   ```python
   import inspect

   print(inspect.getdoc(foo))
   # → Add x and y and return the result.
   ```

   * `inspect.getdoc()` also cleans up indentation for you.

4. **Via `pydoc`**

   ```bash
   pydoc foo
   ```

   Or from within Python:

   ```python
   import pydoc
   print(pydoc.render_doc(foo, "Help on %s"))
   ```

Choose whichever feels most convenient for your use case—`__doc__` and `inspect.getdoc()` if you need the string in code, or `help()` / `pydoc` if you just want to read it interactively.
