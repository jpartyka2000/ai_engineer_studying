You can randomly shuffle the elements of a list in Python using the **`random.shuffle()`** function from the `random` module. This function shuffles the list **in place**, meaning the original list is modified.

### Example:

```python
import random

# Example list
my_list = [1, 2, 3, 4, 5]

# Shuffle the list in place
random.shuffle(my_list)

# Print the shuffled list
print(my_list)
```

### Output (varies due to randomness):
```
[4, 2, 5, 1, 3]
```

### Notes:
- **`random.shuffle()`** shuffles the list in place and returns `None`. If you want to keep the original list unchanged and create a shuffled copy, you can first make a copy of the list using the `copy()` method.

### Shuffling without Modifying the Original List:
If you want to keep the original list intact and get a shuffled version of it, you can use `random.sample()` to return a shuffled copy of the list:

```python
shuffled_list = random.sample(my_list, len(my_list))
print(shuffled_list)
```

This way, **`shuffled_list`** will be a new list with the elements shuffled, and the original list will remain unchanged.
