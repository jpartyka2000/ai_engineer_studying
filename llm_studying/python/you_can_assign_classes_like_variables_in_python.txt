You can just put them in a list directly:

```python
classes = [
    PandasBasicTransformUtils,
    PandasAdvancedAnalytics,
    PandasColumnTypeUtils,
    PandasSummaryAnalytics,
    PandasFeatureCreationUtils,
    PandasOutcomeVariableUtils,
    PandasProbabilityStatsModel,
    SparkBasicTransformUtils,
    SparkColumnTypeUtils,
    SparkSummaryAnalytics,
    SparkFeatureCreationUtils,
    SparkOutcomeVariableUtils,
    SparkProbabilityStatsModel,
]
```

Classes are first-class objects in Python, so you can pass them around like any other variable. Then you can iterate over them:

```python
for cls in classes:
    list_methods(cls)
```

If you want to do it more dynamically from the module itself:

```python
import inspect
import src.ecue_ingest.eda.eda_pipeline_exp as eda_module

classes = [
    obj for name, obj in inspect.getmembers(eda_module, inspect.isclass)
    if not name.startswith('_')
]
```

This auto-discovers all public classes in the module without having to list them manually.
