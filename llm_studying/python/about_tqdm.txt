`tqdm` is a Python library used to display **progress bars** for loops, making it easy to monitor the progress of long-running operations.  

### **Basic Usage**
```python
from tqdm import tqdm
import time

for i in tqdm(range(10)):
    time.sleep(0.5)  # Simulating work
```
🔹 This will display a dynamic progress bar in the console as the loop runs.  

---

### **Where is `tqdm` Useful?**
- **Processing large datasets** (e.g., iterating over a Pandas DataFrame)
- **Downloading files**
- **Training machine learning models**
- **Executing long-running loops in scripts**

---

### **Using `tqdm` with Pandas**
```python
from tqdm import tqdm
import pandas as pd

tqdm.pandas()  # Enable tqdm for Pandas operations

df["new_column"] = df["existing_column"].progress_apply(lambda x: x * 2)
```
🔹 This shows a progress bar while applying a function to a DataFrame column.

---

### **Adding `tqdm` to a `map()` Function**
```python
from tqdm import tqdm

results = list(tqdm(map(lambda x: x**2, range(1000)), total=1000))
```
🔹 Useful when using `map()` instead of a loop.

---

### **Want to Use `tqdm` in Jupyter Notebook?**
Use **`tqdm.notebook`** for better visuals in Jupyter:
```python
from tqdm.notebook import tqdm

for i in tqdm(range(10)):
    time.sleep(0.5)
```
