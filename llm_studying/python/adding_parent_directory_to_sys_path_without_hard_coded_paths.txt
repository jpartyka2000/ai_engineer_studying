### 📌 How to Add `'..'` (Parent Directory) to `sys.path` Without Hardcoding

In Python, you can dynamically add the parent directory of your current script or notebook to the import path using `os.path` and `sys`:

```python
import os
import sys

# Get the current working directory
cwd = os.getcwd()

# Resolve the parent directory (i.e., '..')
parent_dir = os.path.abspath(os.path.join(cwd, ".."))

# Add to sys.path only if it's not already there
if parent_dir not in sys.path:
    sys.path.insert(0, parent_dir)
```

### ✅ Why this is useful:

* Avoids hardcoding full paths like `/Workspace/Users/...`
* Keeps your code portable across environments (e.g., Databricks, local dev, CI/CD)
* Enables you to import from sibling or higher-level modules

You can also use `__file__` instead of `os.getcwd()` in regular `.py` scripts to get the path relative to the script file:

```python
current_file = os.path.abspath(__file__)
parent_dir = os.path.abspath(os.path.join(os.path.dirname(current_file), ".."))
```
