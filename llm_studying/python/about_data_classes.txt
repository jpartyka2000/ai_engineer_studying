Here’s one more Python gem: **Use `dataclasses` for clean and efficient data handling.**

---

### Why it’s useful:
`dataclasses` simplify the creation of classes that primarily store data. They automatically generate boilerplate code like `__init__`, `__repr__`, `__eq__`, and more.

---

### Examples:

#### 1. **Basic Usage**
```python
from dataclasses import dataclass

@dataclass
class Point:
    x: int
    y: int

p1 = Point(3, 4)
p2 = Point(3, 4)

print(p1)  # Point(x=3, y=4)
print(p1 == p2)  # True (comparison based on field values)
```

---

#### 2. **Add Default Values**
```python
from dataclasses import dataclass

@dataclass
class Rectangle:
    width: int = 10
    height: int = 20

r = Rectangle()
print(r)  # Rectangle(width=10, height=20)
```

---

#### 3. **Immutable Data with `frozen=True`**
Make your data immutable (like tuples) using `frozen=True`.
```python
from dataclasses import dataclass

@dataclass(frozen=True)
class User:
    name: str
    age: int

u = User("Alice", 30)
# u.age = 31  # Raises: FrozenInstanceError
```

---

#### 4. **Post-Initialization**
Sometimes, you need to calculate additional fields after initialization:
```python
from dataclasses import dataclass, field

@dataclass
class Circle:
    radius: float
    area: float = field(init=False)

    def __post_init__(self):
        self.area = 3.14159 * self.radius ** 2

c = Circle(5)
print(c.area)  # 78.53975
```

---

#### 5. **Integrate with Type Hints and Validation**
Combine with type hints and `dataclasses.field` for even more control:
```python
from dataclasses import dataclass, field
from typing import List

@dataclass
class Student:
    name: str
    grades: List[int] = field(default_factory=list)

s = Student("John")
s.grades.append(90)
print(s)  # Student(name='John', grades=[90])
```

---

### Why it’s "weird":
Before `dataclasses` (introduced in Python 3.7), creating classes for simple data storage involved lots of boilerplate code. Now, it feels almost magical how Python can infer and generate so much for you.

---

**Pro Tip:** You can use `dataclasses.asdict()` to convert instances to dictionaries and `dataclasses.astuple()` to convert them to tuples:
```python
from dataclasses import asdict, astuple

p = Point(3, 4)
print(asdict(p))  # {'x': 3, 'y': 4}
print(astuple(p))  # (3, 4)
```

---

This makes `dataclasses` a must-have for clean and Pythonic code!
