To list all available Python versions that `pyenv` can install, use the following command:

```bash
pyenv install --list
```

### **Filtering the List**
The output can be long, so you can filter it using `grep`:

- **List only stable Python 3 versions:**
  ```bash
  pyenv install --list | grep " 3\.[0-9]\.[0-9]$"
  ```
- **List only Python 3.11 versions:**
  ```bash
  pyenv install --list | grep " 3.11"
  ```
- **List only PyPy versions:**
  ```bash
  pyenv install --list | grep "pypy"
  ```

### **Installing a Specific Version**
Once you've found the version you need, install it using:

```bash
pyenv install 3.10.6
```
