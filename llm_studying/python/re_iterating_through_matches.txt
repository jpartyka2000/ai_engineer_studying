Here’s an example of how you can iterate through all matches in a given test string using the **`re.finditer()`** function:

---

### Example Code:
```python
import re

# Define the pattern and the test string
pattern = r"\b\w{4}\b"  # Matches words with exactly 4 letters
test_string = "This is a test string with many words like tree, book, and play."

# Use re.finditer() to get an iterator of Match objects
matches = re.finditer(pattern, test_string)

# Iterate through the matches
for match in matches:
    print(f"Match: '{match.group()}' at index {match.start()}-{match.end()}")
```

---

### Explanation:
1. **Regex Pattern:**
   - `\b`: Matches a word boundary (ensures we capture whole words).
   - `\w{4}`: Matches exactly 4 word characters (letters, digits, or underscores).
   - `\b`: Ensures the match ends at a word boundary.

2. **`re.finditer()`**:
   - Returns an iterator of `re.Match` objects for all non-overlapping matches in the string.

3. **Accessing Match Data:**
   - `match.group()`: The matched text.
   - `match.start()`: The start index of the match.
   - `match.end()`: The end index of the match.
