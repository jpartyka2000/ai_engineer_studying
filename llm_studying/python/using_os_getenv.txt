Yes—`os.getenv(key, default=None)` looks up the environment variable named `key` and returns its value (a string) if it exists, otherwise it returns `default` (which is `None` if you don’t supply one). Under the hood it’s equivalent to:

```python
value = os.environ.get(key, default)
```

**Example:**

```python
import os

# Suppose you did in your shell:
# export MY_VAR="hello"

print(os.getenv("MY_VAR"))         # prints "hello"
print(os.getenv("NOT_SET"))        # prints None
print(os.getenv("NOT_SET", "N/A")) # prints "N/A"
```

Use `os.getenv` anytime you want to read configuration from environment variables.
