The @ symbol in Python is the matrix multiplication operator (introduced in Python 3.5 via PEP 465).

🔢 What it does:

z = X @ w

This is equivalent to:

z = np.dot(X, w)

But it's more readable when dealing with linear algebra.


---

✅ Example:

If:

X = np.array([[1, 2],
              [3, 4]])   # shape (2, 2)
w = np.array([0.5, 1.0]) # shape (2,)

Then:

X @ w

gives:

[1*0.5 + 2*1.0, 3*0.5 + 4*1.0] = [2.5, 5.5]


---

So in your logistic regression example, X @ w computes the linear combination of the input features and weights for every row of data — a core part of logistic regression and all linear models.
