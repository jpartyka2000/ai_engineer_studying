You don’t need to hard-code any absolute path at all – you can either change into the parent directory directly:

```python
import os

# go “up” one directory
os.chdir('..')
print(os.getcwd())
```

Or, if you just want to compute the parent directory’s path (without actually cd’ing into it), use `os.path.dirname` on your current working directory:

```python
import os

cwd        = os.getcwd()
parent_dir = os.path.dirname(cwd)
print(parent_dir)
```

Under the covers you can also use `os.pardir` and `os.path.join`:

```python
import os

parent_dir = os.path.normpath(os.path.join(os.getcwd(), os.pardir))
print(parent_dir)
```

All three approaches will give you the directory one level up from whatever `getcwd()` returned, without ever typing out an absolute path.
