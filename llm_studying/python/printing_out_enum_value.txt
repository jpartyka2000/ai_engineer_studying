When you define an enum using Python’s built-in `enum` module, each enum member has a `.value` attribute that stores its associated value. For example:

```python
from enum import Enum

class Color(Enum):
    RED = 1
    GREEN = 2
    BLUE = 3

# Print the enum member's value:
print(Color.RED.value)  # Outputs: 1
```

If you print the enum member directly (e.g., `print(Color.RED)`), you’ll see something like `Color.RED`. To print just the underlying value, access the `.value` attribute.
