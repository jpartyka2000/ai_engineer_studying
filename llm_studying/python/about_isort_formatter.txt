You can install **isort** easily using **pip**:

```sh
pip install isort
```

### **Basic Usage**
To sort imports in a single file:
```sh
isort my_script.py
```
To sort imports in an entire project:
```sh
isort .
```

### **Making isort Compatible with Black**
Since **Black and isort can conflict**, use the **Black profile** for isort:
```sh
isort . --profile black
```
Or, set it permanently in `pyproject.toml`:
```toml
[tool.isort]
profile = "black"
```

### **Automating isort + Black**
You can format your code with both tools in one command:
```sh
isort . && black .
```
