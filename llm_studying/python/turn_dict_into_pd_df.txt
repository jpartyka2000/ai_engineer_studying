Sure—just wrap that single‐record dict in a list and hand it to `pd.DataFrame`:

```python
import pandas as pd

input_example = {
    "overall_budget": 250000.0,
    "n_iterations":   10000,
    "pct_min_max":    0.5,
    "ctr":            0.02,
    "input_table":    "pathahead.tealhq_ecue.campaign_mc_data_april_2025"
}

# Create a one-row DataFrame
input_df = pd.DataFrame([input_example])

print(input_df)
```

That will give you:

|    | overall\_budget | n\_iterations | pct\_min\_max |  ctr | input\_table                                           |
| -: | --------------: | ------------: | ------------: | ---: | :----------------------------------------------------- |
|  0 |          250000 |         10000 |           0.5 | 0.02 | pathahead.tealhq\_ecue.campaign\_mc\_data\_april\_2025 |

You can then pass `input_df` as the `input_example` to MLflow or directly into your `predict()` method.
