**QUESTION:**
Fill in the blank to correctly type hint a list of integers:

```python
def sum_numbers(numbers: ________) -> int:
    return sum(numbers)
```

**YOUR ANSWER:** List[int]

**CORRECT ANSWER:** List[int] (or list[int] in Python 3.9+)

**EXPLANATION:**
`List[int]` is correct! This indicates a list containing integer elements.

**Two valid syntaxes:**
- `List[int]` - Traditional syntax, requires `from typing import List`
- `list[int]` - Modern syntax (Python 3.9+), uses the built-in `list` type directly

The lowercase `list[int]` is now preferred in modern Python code as it's simpler and doesn't require an import.
