────────────────────────────────────────────────────────────────
QUESTION:
────────────────────────────────────────────────────────────────
from collections import Counter

words = ['apple', 'banana', 'apple', 'cherry', 'banana', 'apple']
c = Counter(words)

result = c.most_common(2)
print(result)

What is the output?

A) {'apple': 3, 'banana': 2}
B) [('apple', 3), ('banana', 2)]
C) ['apple', 'banana']
D) Counter({'apple': 3, 'banana': 2})

────────────────────────────────────────────────────────────────
YOUR ANSWER:      D
CORRECT ANSWER:   B
RESULT:           ❌ Incorrect
────────────────────────────────────────────────────────────────
EXPLANATION:
────────────────────────────────────────────────────────────────
The most_common(n) method returns a LIST OF TUPLES, not a Counter
object. Each tuple contains (element, count), ordered from most
frequent to least frequent.

- c.most_common()    → all elements as list of tuples
- c.most_common(2)   → top 2 most common as list of tuples

This is a common interview gotcha — remember the return type!
