Yes, **`re.sub`** in Python by default replaces **all instances** of the matched pattern in a string. However, you can configure it to only replace the **first instance** of the match by passing an additional argument `count=1`.

### Example:

```python
import re

# Test string
test_string = "The quick brown fox jumps over the lazy dog. The quick brown fox again."

# Pattern to match
pattern = r"quick brown fox"

# Replace only the first match
result = re.sub(pattern, "fast brown fox", test_string, count=1)

print(result)
```

### Output:
```
The fast brown fox jumps over the lazy dog. The quick brown fox again.
```

### Explanation:
- **`re.sub(pattern, replacement, string, count=1)`**:
  - **`pattern`**: The regular expression pattern to search for.
  - **`replacement`**: The string that will replace the matched pattern.
  - **`string`**: The input string to search within.
  - **`count=1`**: Limits the replacement to only the **first** occurrence of the match. If `count` is not provided or set to `0`, all occurrences will be replaced.

This allows you to replace just the first match of the pattern in the string.
