Use the `inspect` module:

```python
import inspect

sig = inspect.signature(my_function)
print(sig)
```

This prints the full signature with parameters and type hints, e.g.:

```
(df: SparkDataFrame, return_as_df: bool = True) -> SparkDataFrame | dict
```

A few other useful things you can do with it:

```python
# Get individual parameters
for name, param in sig.parameters.items():
    print(f"{name}: default={param.default}, annotation={param.annotation}")

# Just get the source code (includes signature + body)
print(inspect.getsource(my_function))

# Quick one-liner using help()
help(my_function)
```

For a classmethod, you'd reference it through the class:

```python
inspect.signature(MyClass.get_row_and_column_info)
```
