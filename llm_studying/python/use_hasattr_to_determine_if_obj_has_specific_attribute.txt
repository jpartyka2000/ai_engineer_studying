You can use Python’s built-in `hasattr()` function, which returns `True` if the object defines that attribute (even if it’s set to `None`). For example:

```python
if hasattr(event, "name"):
    # event.name exists
    print("This event has a name:", event.name)
else:
    print("No name attribute on this event")
```

If you just want a boolean flag in one line:

```python
has_name = hasattr(event, "name")
```

Alternatively, if you need to both test and fetch (with a default), you can use `getattr()`:

```python
name = getattr(event, "name", None)
if name is not None:
    print("Name is:", name)
else:
    print("No name or it's explicitly None")
```
