def my_gen():
    yield 1
    yield 2

result = my_gen()       # No code inside my_gen runs yet!
print(type(result))     # <class 'generator'>

# Now we can iterate:
print(next(result))     # 1 — NOW it runs until the first yield
print(next(result))     # 2 — resumes and runs until the second yield
```

This is a key distinction:
- **Regular function call** → Executes immediately, returns a value
- **Generator function call** → Returns a generator object, executes lazily
```
