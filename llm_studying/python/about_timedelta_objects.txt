A **`timedelta`** in Python‚Äôs standard library (the `datetime` module) represents a **duration**‚Äîthe difference between two dates or times. It‚Äôs the go-to object whenever you need to add or subtract spans of time.

---

## Creation

You construct a `timedelta` by specifying any combination of:

```python
from datetime import timedelta

td = timedelta(
    days=2,
    hours=5,
    minutes=30,
    seconds=10,
    microseconds=500,
    weeks=1
)
```

All of those arguments are optional (they default to zero), and you can mix & match. Internally they get **normalized** so that, e.g., 90 seconds becomes 1 minute + 30 seconds, etc.

---

## Core Attributes

Once you have a `timedelta`, you can inspect its components:

```python
td.days          # total whole days (including weeks)
td.seconds       # leftover seconds (0‚Äì86399)
td.microseconds  # leftover microseconds (0‚Äì999999)
```

> **Note:** There are no direct `.hours` or `.minutes` attributes; you convert from `.seconds` if you need them.

---

## Arithmetic

You most commonly use `timedelta` to shift a `date` or `datetime`:

```python
from datetime import date, datetime, timedelta

d0 = date(2025, 3, 1)
d1 = d0 + timedelta(weeks=1)       # ‚Üí date(2025, 3, 8)

dt0 = datetime(2025,3,1,9,30)
dt1 = dt0 - timedelta(hours=2)     # ‚Üí datetime(2025,3,1,7,30)
```

You can also subtract two datetimes (or dates) to get a `timedelta`:

```python
delta = datetime.now() - datetime(2025,1,1)
print(delta.days, delta.seconds)   # e.g. (87 days, 45230 seconds)
```

---

## Comparisons & Operations

- You can compare them (`td1 < td2`, `td == timedelta(hours=0)`).
- You can add/subtract them together.
- You can multiply/divide by numbers (`td * 2`, `td / 3`).

---

## Converting to Other Units

To get a single measure in seconds (including fractional):

```python
total_seconds = td.total_seconds()
```

If you want hours or minutes, you can divide:

```python
hours = td.total_seconds() / 3600
```

---

## Use Cases

- **Scheduling**: compute ‚Äú30 days from now,‚Äù or ‚Äúnext business day.‚Äù  
- **Elapsed time**: measure durations (`end_time - start_time`).  
- **Windowing**: slide time windows in time-series data.  
- **Timeouts/Delays**: specify wait intervals.  

---

## When You Need More

For calendar-aware shifts (e.g. ‚Äúadd 1 month‚Äù or ‚Äúnext quarter‚Äù) where month lengths vary, consider `dateutil.relativedelta.relativedelta`, which handles month/year arithmetic more intuitively than a fixed-day `timedelta`.

---

In short, `timedelta` is your lightweight, built-in way to work with spans of time in Python‚Äîadding, subtracting, comparing, and converting durations with ease.
