Yes, it will raise a `TypeError`. Generators don't have a length because they produce values lazily and don't know how many items they'll yield until they're exhausted.

```python
def my_generator():
    yield 1
    yield 2
    yield 3

gen = my_generator()
len(gen)  # TypeError: object of type 'generator' has no len()
```

If you need the length, you have a few options:

**1. Convert to a list first (consumes the generator):**
```python
gen_list = list(gen)
length = len(gen_list)
```

**2. Count by iterating (also consumes the generator):**
```python
length = sum(1 for _ in gen)
```

**3. If you need both length and values, convert to list once:**
```python
gen_list = list(gen)
length = len(gen_list)
for item in gen_list:
    # process item
```

Keep in mind that once a generator is consumed, it's exhausted—you can't iterate over it again.
