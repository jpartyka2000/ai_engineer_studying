**TeXShop** is a **TeX/LaTeX editor and viewer** specifically designed for macOS. It's an integrated application that makes it easy to write, compile, and view LaTeX documents.

## What TeXShop Does:

**1. Text Editor**
- Provides a dedicated editor for writing LaTeX code
- Syntax highlighting for LaTeX commands
- Auto-completion for common commands
- Bracket matching

**2. Compiles LaTeX Documents**
- Converts your `.tex` source files into PDFs
- Supports different compilation engines (pdfLaTeX, XeLaTeX, LuaLaTeX, etc.)
- Shows compilation errors and warnings

**3. Built-in PDF Viewer**
- Displays the compiled PDF right next to your code
- Automatically refreshes when you recompile
- Supports **SyncTeX** - click on the PDF to jump to that location in your code, or vice versa

**4. Integrated Workflow**
- Single "Typeset" button compiles and displays your document
- Split-screen view: code on one side, PDF preview on the other
- Makes the LaTeX workflow smooth and seamless

## Why People Use It:

- **macOS native** - looks and feels like a Mac app
- **Free and open-source**
- **Simple and straightforward** - less overwhelming than some other LaTeX editors
- **Comes with MacTeX** - the standard TeX distribution for Mac

## Typical Workflow:
1. Write LaTeX code in TeXShop's editor
2. Click "Typeset" button (or press Cmd+T)
3. View the generated PDF in the built-in viewer
4. Make edits and repeat

It's essentially an all-in-one LaTeX IDE for Mac users, handling everything from editing to compilation to viewing.
