## The `\usepackage` Command

`\usepackage` loads **packages** (extensions) that add extra functionality to LaTeX beyond what the basic document classes provide.

**Basic syntax:**
```latex
\usepackage{packagename}
```

or with options:

```latex
\usepackage[options]{packagename}
```

**Location:** `\usepackage` commands go in the **preamble** — after `\documentclass` but before `\begin{document}`.

---

### **Common Packages and What They Do:**

**Graphics and Images:**
- `graphicx` — include images (PNG, JPG, PDF)
- `tikz` — create diagrams and drawings

**Math:**
- `amsmath` — enhanced math environments and commands
- `amssymb` — additional math symbols
- `mathtools` — extends amsmath with more tools

**Formatting:**
- `geometry` — customize page margins and layout
- `setspace` — control line spacing
- `fancyhdr` — customize headers and footers

**Typography:**
- `fontenc` — font encoding (often `\usepackage[T1]{fontenc}`)
- `inputenc` — input encoding (e.g., `\usepackage[utf8]{inputenc}`)
- `babel` — language support and hyphenation

**Tables and Lists:**
- `booktabs` — professional-quality tables
- `tabularx` — tables with adjustable column widths
- `enumitem` — customize lists

**References:**
- `hyperref` — creates clickable links (citations, URLs, table of contents)
- `cite` — improved citation handling
- `natbib` or `biblatex` — bibliography management

**Colors:**
- `xcolor` — add colors to text, backgrounds, etc.

---

### **Examples:**

```latex
\documentclass{article}

\usepackage{graphicx}           % for images
\usepackage{amsmath}            % for better math
\usepackage[margin=1in]{geometry}  % 1-inch margins
\usepackage{hyperref}           % clickable links

\begin{document}
...
\end{document}
```

**Package with options:**
```latex
\usepackage[utf8]{inputenc}     % specify UTF-8 encoding
\usepackage[english]{babel}     % English language support
\usepackage[dvipsnames]{xcolor} % load extra color names
```

**Loading multiple packages at once:**
```latex
\usepackage{amsmath,amssymb,graphicx}
```

Packages extend LaTeX's capabilities, letting you do things like include images, create complex mathematical expressions, adjust page layout, add colors, and much more.
