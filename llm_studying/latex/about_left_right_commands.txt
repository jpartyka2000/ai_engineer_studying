`\left` and `\right` are LaTeX commands that create **automatically-sized delimiters**—brackets, parentheses, braces, etc. that scale to match the height of whatever they enclose.

## Basic usage

They always come in pairs:

```latex
\left( ... \right)
\left[ ... \right]
\left\{ ... \right\}
```

## What they do

Compare these two:

```latex
( \frac{a}{b} )        % fixed-size parentheses (too small)
\left( \frac{a}{b} \right)   % parentheses scale to fit the fraction
```

The first gives you normal-sized parentheses that look awkward next to a tall fraction. The second automatically stretches them.

## The delimiter can be any paired symbol

```latex
\left| x \right|           % absolute value
\left\| x \right\|         % norm (double bars)
\left\langle x \right\rangle   % angle brackets
\left\lfloor x \right\rfloor   % floor
\left\lceil x \right\rceil     % ceiling
```

## You can mix or omit delimiters

If you only want one side, use `\left.` or `\right.` (a dot means "invisible delimiter"):

```latex
\left. \frac{x^2}{2} \right|_0^1    % evaluation bar on right only
```

You can also mix different delimiters:

```latex
\left( 0, 1 \right]    % half-open interval
```

## The rule

Every `\left` must have a matching `\right` in the same math environment. They don't have to be the same symbol, but they must be paired.
