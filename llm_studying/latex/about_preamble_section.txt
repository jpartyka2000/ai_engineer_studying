## The Preamble in a LaTeX Document

The **preamble** is the section of a LaTeX document that comes **between** `\documentclass` and `\begin{document}`.

**Structure of a LaTeX document:**
```latex
\documentclass{article}        % Required first line

% ========== PREAMBLE STARTS HERE ==========
\usepackage{graphicx}
\usepackage{amsmath}
\title{My Document}
\author{Your Name}
\date{\today}
% ========== PREAMBLE ENDS HERE ==========

\begin{document}               % Document content starts

\maketitle

Hello world!

\end{document}                 % Document content ends
```

---

### **What Goes in the Preamble?**

The preamble is where you **configure** your document before the actual content begins. Common things include:

**1. Load packages:**
```latex
\usepackage{graphicx}
\usepackage[margin=1in]{geometry}
```

**2. Document metadata:**
```latex
\title{My Research Paper}
\author{Jane Smith}
\date{December 2025}
```

**3. Custom commands:**
```latex
\newcommand{\R}{\mathbb{R}}    % shortcut for real numbers
```

**4. Settings and configurations:**
```latex
\setlength{\parindent}{0pt}    % no paragraph indentation
\linespread{1.5}               % line spacing
```

**5. Bibliography setup:**
```latex
\bibliographystyle{plain}
```

**6. Define colors, counters, lengths, etc.:**
```latex
\definecolor{myblue}{RGB}{0,100,200}
```

---

### **Key Point:**

- **Preamble** = configuration and setup
- **Document body** (between `\begin{document}` and `\end{document}`) = actual content

Nothing in the preamble appears in your final PDF output — it only affects *how* the document is formatted and what features are available.
