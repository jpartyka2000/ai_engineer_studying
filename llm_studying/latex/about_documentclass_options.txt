## Options for `\documentclass`

Options are specified in square brackets before the document class name:

```latex
\documentclass[option1,option2,...]{class}
```

### **Font Size Options**
- `10pt` — 10 point font (default)
- `11pt` — 11 point font
- `12pt` — 12 point font

These are the standard sizes. Some classes support additional sizes like `8pt`, `9pt`, `14pt`, etc.

### **Paper Size Options**
- `letterpaper` — US Letter size: 8.5" × 11" (default)
- `a4paper` — A4 size: 210mm × 297mm
- `a5paper` — A5 size: 148mm × 210mm
- `b5paper` — B5 size: 176mm × 250mm
- `legalpaper` — US Legal: 8.5" × 14"
- `executivepaper` — Executive: 7.25" × 10.5"

### **Layout Options**
- `oneside` — formatted for single-sided printing (default for article, report)
- `twoside` — formatted for double-sided printing (default for book)
- `onecolumn` — single column text (default)
- `twocolumn` — two-column layout
- `landscape` — landscape orientation
- `portrait` — portrait orientation (default)

### **Title Page Options**
- `titlepage` — puts title on separate page (default for report, book)
- `notitlepage` — title on same page as content (default for article)

### **Equation Numbering**
- `leqno` — equation numbers on left side
- `reqno` — equation numbers on right side (default)
- `fleqn` — left-aligned equations instead of centered

### **Draft/Final Options**
- `draft` — marks overfull lines, doesn't include images (faster compilation)
- `final` — normal output (default)

### **Opening Page Options**
- `openright` — chapters start on right-hand pages only (book default)
- `openany` — chapters can start on any page (report default)

### **Example with multiple options:**
```latex
\documentclass[12pt,a4paper,twoside,openright,draft]{book}
```

This creates a book with 12pt font on A4 paper, formatted for double-sided printing, chapters starting on right pages, in draft mode.
