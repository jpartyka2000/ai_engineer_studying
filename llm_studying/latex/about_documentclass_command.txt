`\documentclass` is a **required command** that must appear at the very beginning of every LaTeX document. It specifies the **type and overall layout** of the document you're creating.

**Basic syntax:**
```latex
\documentclass{class}
```

or with options:

```latex
\documentclass[options]{class}
```

**Common document classes:**

- `article` — for short documents, papers, journal articles (no chapters)
- `report` — for longer documents with chapters, theses
- `book` — for books (two-sided printing by default, chapters, parts)
- `letter` — for letters
- `beamer` — for presentations/slides

**Common options:**
- `10pt`, `11pt`, `12pt` — font size (default is 10pt)
- `a4paper`, `letterpaper` — paper size
- `twocolumn` — two-column layout
- `twoside`, `oneside` — two-sided or one-sided printing

**Example:**
```latex
\documentclass[12pt,a4paper]{article}

\begin{document}
Hello world!
\end{document}
```

This creates an article-style document with 12pt font on A4 paper. The `\documentclass` line defines the overall structure and formatting conventions that LaTeX will use throughout your document.
