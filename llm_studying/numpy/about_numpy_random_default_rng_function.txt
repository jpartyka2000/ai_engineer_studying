Yes! NumPy has several ways to randomly sample data, all within the `numpy.random` module.

**Modern approach (recommended):** Use the `Generator` object via `numpy.random.default_rng()`:

```python
import numpy as np

rng = np.random.default_rng()

# Random sample from an array
arr = np.array([10, 20, 30, 40, 50])
rng.choice(arr, size=3)              # 3 random elements (with replacement)
rng.choice(arr, size=3, replace=False)  # Without replacement

# Shuffle an array
rng.shuffle(arr)          # In-place shuffle
rng.permutation(arr)      # Returns a shuffled copy
```

**Sampling from distributions:**

```python
rng.random(5)                  # 5 uniform floats in [0, 1)
rng.integers(1, 10, size=5)    # 5 random ints from 1 to 9
rng.normal(0, 1, size=5)       # 5 samples from standard normal
rng.uniform(0, 100, size=5)    # 5 uniform floats in [0, 100)
```

**Legacy approach** (still works but less preferred):

```python
np.random.choice(arr, size=3)
np.random.rand(5)       # 5 uniform floats
np.random.randint(1, 10, 5)
```

The newer `default_rng()` approach is recommended because it offers better statistical properties and reproducibility. You can seed it for reproducible results:

```python
rng = np.random.default_rng(seed=42)
```
