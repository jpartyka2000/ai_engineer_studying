Here’s a concise overview of the two forms of `np.where`:

---

## 1. `np.where(condition)`

* **Purpose:** Find the indices where a boolean condition holds.
* **Signature:**

  ```python
  np.where(condition)
  ```
* **Return:**
  A `tuple` of integer arrays—one per dimension—containing the coordinates of `True` entries.
* **When to use:**
  Whenever you need the positions of elements meeting some test (e.g. for masking, fancy indexing, or counting).
* **Example:**

  ```python
  >>> import numpy as np
  >>> a = np.array([[1, 4, 2],
                    [0, 5, 3]])
  >>> np.where(a > 2)
  (array([0, 1, 1]), array([1, 1, 2]))
  # → positions (0,1), (1,1), (1,2)
  ```

---

## 2. `np.where(condition, x, y)`

* **Purpose:** Element‑wise “choose from two” array selection.
* **Signature:**

  ```python
  np.where(condition, x, y)
  ```
* **Return:**
  An `ndarray` shaped like `condition` (after broadcasting), where each element is

  * `x[i]` if `condition[i]` is `True`
  * `y[i]` if `condition[i]` is `False`
* **When to use:**
  As a vectorized “if‑else” to build a new array from two possibilities.
* **Example:**

  ```python
  >>> a = np.array([10, 15, 20, 25])
  >>> np.where(a % 2 == 0, 'even', 'odd')
  array(['even', 'odd', 'even', 'odd'], dtype='<U4')
  ```

---

### Quick Comparison

| Form                   | Returns                 | Use Case                              |
| ---------------------- | ----------------------- | ------------------------------------- |
| `np.where(cond)`       | `tuple` of index arrays | Locate positions of `True` elements   |
| `np.where(cond, x, y)` | `ndarray`               | Select between two arrays elementwise |

Both are fundamental building blocks for conditional logic and indexing in NumPy.
