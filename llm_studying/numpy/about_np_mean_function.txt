Yes! NumPy provides `numpy.mean()` (or equivalently, the `.mean()` method on arrays) to compute the arithmetic mean.

**Basic usage:**

```python
import numpy as np

arr = np.array([1, 2, 3, 4, 5])
np.mean(arr)  # Returns 3.0

# Or as a method
arr.mean()    # Also returns 3.0
```

**With multi-dimensional arrays**, you can compute the mean along specific axes:

```python
arr_2d = np.array([[1, 2, 3],
                   [4, 5, 6]])

np.mean(arr_2d)           # Mean of all elements: 3.5
np.mean(arr_2d, axis=0)   # Mean along columns: [2.5, 3.5, 4.5]
np.mean(arr_2d, axis=1)   # Mean along rows: [2., 5.]
```

**Other useful parameters:**
- `dtype` — specify the data type for the computation (useful for precision)
- `keepdims=True` — retain the reduced axis as a dimension of size 1

There's also `numpy.nanmean()` if your data contains NaN values and you want to ignore them.
