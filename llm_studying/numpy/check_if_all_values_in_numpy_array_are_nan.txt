You can use `numpy.isnan` combined with `all`:

```python
import numpy as np

arr = np.array([np.nan, np.nan, np.nan])

# returns True if every element is NaN
all_nan = np.isnan(arr).all()
print(all_nan)  # True
```

**Explanation:**

1. `np.isnan(arr)` returns a boolean array of the same shape, `True` where the element is `NaN`, else `False`.
2. Calling `.all()` on that boolean array checks that *every* entry is `True`.

---

### A couple of variants

* **2D (or higher-dimensional) arrays:**

  ```python
  arr2 = np.array([[np.nan, np.nan], [np.nan, np.nan]])
  np.isnan(arr2).all()              # True
  np.isnan(arr2).all(axis=0)        # [True, True] per column
  np.isnan(arr2).all(axis=1)        # [True, True] per row
  ```

* **Using `np.allclose` with `nan` tolerance:**

  ```python
  all_nan = np.allclose(arr, np.nan, equal_nan=True)
  ```

  but this is less direct than `np.isnan(arr).all()`.

---

That simple pattern—`np.isnan(your_array).all()`—is the canonical way to test “is every element NaN?” in NumPy.
