np.triu() is a NumPy function that returns the upper triangular part of an array (usually a 2D matrix).

Definition:

numpy.triu(m, k=0)

m: Input array (at least 2-D).

k (default = 0): Controls which diagonal to consider as the "cutoff":

k = 0: Keep the main diagonal and everything above it.

k > 0: Excludes diagonals above the main one (shifts upward).

k < 0: Includes more below the diagonal.



Behavior:
It sets all elements below the k-th diagonal to zero, leaving only the upper triangle.



---

Example

import numpy as np

a = np.arange(1, 10).reshape(3, 3)
print(a)
# [[1 2 3]
#  [4 5 6]
#  [7 8 9]]

print(np.triu(a))
# [[1 2 3]
#  [0 5 6]
#  [0 0 9]]

print(np.triu(a, k=1))   # Only above the main diagonal
# [[0 2 3]
#  [0 0 6]
#  [0 0 0]]


---

👉 In short: np.triu() is useful when you only want the upper triangular portion of a matrix, often for linear algebra tasks, adjacency matrices, or eliminating redundant values in symmetric matrices.
