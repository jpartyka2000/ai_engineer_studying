Yes—by default

```python
import numpy as np
a = np.array([10, 15, 23, 32])
np.diff(a)
```

will give you:

```
array([ 5,  8,  9])
```

which are

- 15 − 10 = 5  
- 23 − 15 = 8  
- 32 − 23 = 9  

---

### Key points

- **Order (`n`)**: `np.diff(a, n=2)` will take differences twice, i.e. the second differences.  
- **Axis**: works along any axis of an N-D array via the `axis` argument.  
- **Padding**: you can supply `prepend` or `append` values if you want the output to remain the same length as the input.  

But in its simplest form, `np.diff(x)` computes `x[i+1] - x[i]` for each adjacent pair in `x`.
