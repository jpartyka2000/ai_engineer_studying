To classify each probability value as being in either the bottom third or top two-thirds of all values in Google Sheets, you can:

1. **Calculate the Bottom Third Threshold**: Use the `PERCENTILE` function to find the value that represents the 33rd percentile. Any probability below this threshold will be in the "bottom third," and any probability equal to or above it will be in the "top two-thirds."

2. **Apply the Classification**: Use an `IF` statement to label each value accordingly.

Assuming your probabilities are in column `A` (starting from `A2`), here’s how to do it:

### Step 1: Calculate the 33rd Percentile Threshold
In a separate cell, say `C2`, calculate the 33rd percentile as follows:

```excel
=PERCENTILE(A2:A, 0.33)
```

This gives you the value that separates the bottom third from the top two-thirds.

### Step 2: Apply the Classification with `ARRAYFORMULA`
In cell `B2`, use `ARRAYFORMULA` with an `IF` statement to classify each probability value based on this threshold:

```excel
=ARRAYFORMULA(IF(A2:A350 < C2, "Bottom third", "Top two-thirds"))
```

### Explanation:
- `A2:A350` is the range for your probability values, limited to row 350.
- `C2` contains the threshold for the bottom third.
- The formula assigns "Bottom third" to values below the threshold and "Top two-thirds" to values at or above it.

This will automatically fill the new column with classifications based on each probability value’s position relative to the bottom third threshold.
