In Google Sheets, you can apply a formula to an entire column up to the last row by using **ArrayFormula**. This way, you don’t need to drag the formula down manually.

For your example, assuming your probability values start in `A2`, you can place the following formula in `B2`:

```excel
=ARRAYFORMULA(IF(A2:A <= 0.05, "hello!", "bye!"))
```

### Explanation:
- `ARRAYFORMULA` allows the formula to operate on a range (here, `A2:A` refers to all rows in column A starting from A2).
- `IF(A2:A <= 0.05, "hello!", "bye!")` applies the condition to each cell in column `A` and outputs "hello!" or "bye!" in each corresponding cell in column `B`.

Now, this formula will automatically apply to all rows in column `A`, and it will dynamically expand if you add or remove data.
