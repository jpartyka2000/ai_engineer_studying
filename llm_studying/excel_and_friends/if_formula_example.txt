In Google Sheets, you can use an `IF` statement to print "hello!" if the probability value is less than or equal to 0.05, and "bye!" otherwise. Assuming your probability values are in column `A` starting from `A2`, you can use this formula in a new column, say in cell `B2`:

```excel
=IF(A2 <= 0.05, "hello!", "bye!")
```

### Steps:
1. Enter the formula in cell `B2`.
2. Copy the formula down the column to apply it to each row.

This formula will check the value in each cell of column `A`:
- If the probability is `<= 0.05`, it will print "hello!" in the new column.
- If the probability is greater than 0.05, it will print "bye!".
