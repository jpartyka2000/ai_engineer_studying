**Check current formatter setting:**

1. `Cmd + Shift + P` (Mac) or `Ctrl + Shift + P` (Windows)
2. Type "Open Settings (JSON)" and select it
3. Look for these settings:

```json
{
    "[python]": {
        "editor.defaultFormatter": "ms-python.black-formatter"
    }
}
```

Or the older style:
```json
{
    "python.formatting.provider": "black"
}
```

**Or check via UI:**

1. `Cmd + ,` (Mac) or `Ctrl + ,` (Windows) to open Settings
2. Search "default formatter"
3. Look for the Python-specific setting — should show "Black Formatter"

**Quick test:**

1. Open a Python file
2. Intentionally mess up formatting (e.g., add extra spaces)
3. Run `Shift + Option + F` (Mac) or `Shift + Alt + F` (Windows)
4. If it reformats with Black's style (e.g., double quotes, specific line lengths), it's working

**If nothing happens or wrong formatter:**

1. Install the Black Formatter extension from marketplace
2. `Cmd + Shift + P` → "Python: Select Formatter" → choose Black
