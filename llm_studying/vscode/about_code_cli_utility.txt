# The `code` CLI for VS Code

The `code` command is a command-line utility that lets you open VS Code from your terminal.

## Common Uses

```bash
# Open current directory in VS Code
code .

# Open a specific file
code myfile.py

# Open a specific folder
code /path/to/project

# Open multiple files
code file1.js file2.js file3.js

# Open a file at a specific line number
code -g myfile.py:42
```

## Useful Flags

| Flag | Purpose |
|------|---------|
| `-n` | Open a new window |
| `-r` | Reuse the current window |
| `-g file:line` | Go to specific line |
| `-d file1 file2` | Diff two files |
| `--list-extensions` | List installed extensions |
| `--install-extension` | Install an extension |

## Installation

On **Mac/Linux**, you may need to install it manually:
1. Open VS Code
2. Press `Cmd+Shift+P` (Mac) or `Ctrl+Shift+P` (Linux)
3. Type "Shell Command: Install 'code' command in PATH"
4. Press Enter

On **Windows**, it's typically added to PATH during VS Code installation.

## Examples

```bash
# Diff two files
code -d old.json new.json

# Install an extension
code --install-extension ms-python.python

# Open project in new window
code -n ~/projects/my-app
```
