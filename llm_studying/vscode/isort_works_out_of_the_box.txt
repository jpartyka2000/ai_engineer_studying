## isort in VSCode (out of the box)

With the Python extension installed, just press:

**`Shift+Option+O`** (Mac) or **`Shift+Alt+O`** (Windows/Linux)

This runs "Organize Imports" using isort — no configuration needed.

---

## Additional isort configuration (optional)

**Run on save automatically:**
```json
{
    "[python]": {
        "editor.codeActionsOnSave": {
            "source.organizeImports": "always"
        }
    }
}
```

**Custom isort args (e.g., Black compatibility):**
```json
{
    "isort.args": ["--profile", "black"]
}
```

**Dedicated isort extension:**

For more control, install the [isort extension](https://marketplace.visualstudio.com/items?itemName=ms-python.isort) by Microsoft, then run via Command Palette (`Cmd+Shift+P`):
```
isort: Sort Imports
```
