A few options:

**Option 1: GitLens extension (best)**

1. Install "GitLens" from the extensions marketplace
2. Open the file
3. Click the GitLens icon in the sidebar, or
4. Right-click in the file → "Open File History"

**Option 2: Built-in Git**

1. Open the file
2. `Cmd + Shift + P` → "Git: View File History"

Or:
1. Click the Source Control icon in sidebar (branch icon)
2. Right-click the file → "Open Timeline"

**Option 3: Timeline view (built-in)**

1. Open the file
2. In the Explorer sidebar, scroll down to "Timeline" panel
3. It shows commits that touched this file

**Option 4: Command line**

```bash
git log --oneline cue_eval/runner.py
```

Or with diffs:
```bash
git log -p cue_eval/runner.py
```

---

**My recommendation:** Install GitLens — it's the most powerful option and shows blame annotations inline.
