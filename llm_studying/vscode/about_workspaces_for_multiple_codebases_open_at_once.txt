Yes! VSCode supports having multiple folders open simultaneously in what's called a **multi-root workspace**.

**How to add a second folder:**

1. **File menu approach:**
   - Go to `File` → `Add Folder to Workspace...`
   - Select the folder you want to add
   - Both folders will now appear in your Explorer sidebar

2. **Right-click approach:**
   - Right-click in the Explorer sidebar
   - Select `Add Folder to Workspace...`

**What you'll see:**

Once you have multiple folders open, the Explorer sidebar will show both folder structures separately. Each folder becomes a "root" in your workspace, and you can expand/collapse them independently.

**Saving the workspace:**

After adding folders, VSCode will prompt you to save the workspace configuration:
- Go to `File` → `Save Workspace As...`
- This creates a `.code-workspace` file
- Opening this file in the future will load all your folders at once

**Benefits:**

- Work on related projects simultaneously (like frontend/backend)
- Search across multiple folders at once
- Keep separate Git repositories visible
- Each folder maintains its own settings if needed

**Note:** This is different from just having multiple VSCode windows open - multi-root workspaces let you see and work with multiple folders in a single window, with unified search, find-and-replace, and other features working across all folders.

It's a really handy feature for managing related projects!
