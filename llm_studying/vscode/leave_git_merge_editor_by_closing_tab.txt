Just close the file/tab in VSCode.

When you close it, VSCode should prompt you to save. Save the file, and Git will detect that the editor has closed and continue the rebase.

Alternatively, you can:

1. Save the file (`Cmd+S` on Mac, `Ctrl+S` on Windows/Linux)
2. Close the tab (`Cmd+W` or `Ctrl+W`)

The rebase will then proceed to the next commit automatically.
