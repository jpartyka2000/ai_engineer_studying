## Disable Preview Mode in VS Code

If you're tired of files opening in preview mode (where each click replaces the previous tab), you can turn it off:

1. Open Settings: `Ctrl+,` (Windows/Linux) or `Cmd+,` (Mac)
2. Search for "enable preview"
3. Uncheck **"Workbench › Editor: Enable Preview"**

Now single clicks will always open files as permanent tabs instead of replacing each other.
