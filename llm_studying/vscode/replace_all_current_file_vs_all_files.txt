**Replace All only applies to the current file by default.**

However, VSCode's Find and Replace has two modes:

**1. Current File (default)**
- Press `Ctrl+H` / `Cmd+H`
- "Replace All" only affects the currently open file

**2. All Files in Project**
- Press `Ctrl+Shift+H` / `Cmd+Shift+H` (note the addition of Shift)
- This opens "Find in Files" mode where "Replace All" affects all files in your workspace

You can also toggle between these modes in the Find widget by clicking the arrow icon (▶) on the left side of the search box to expand it to search across files.

So you're safe - when you use the regular `Ctrl+H` Find and Replace, "Replace All" will only modify the file you're currently viewing.
