`Cmd+,` opens the **Settings UI** (a graphical interface), not a file directly.

Behind it is your **global user settings file** located at:

```
~/Library/Application Support/Code/User/settings.json
```

## VSCode settings hierarchy

| Level | Location | Scope |
|-------|----------|-------|
| User | `~/Library/Application Support/Code/User/settings.json` | All projects |
| Workspace | `.vscode/settings.json` in your project | That project only |

Workspace settings override user settings.

## Quick access

| Action | What it opens |
|--------|---------------|
| `Cmd+,` | Settings UI |
| `Cmd+Shift+P` → "Open User Settings (JSON)" | User `settings.json` file |
| `Cmd+Shift+P` → "Open Workspace Settings (JSON)" | Project `.vscode/settings.json` |
