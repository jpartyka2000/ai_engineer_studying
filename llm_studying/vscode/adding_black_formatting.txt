A few options:

**Option 1: Command Palette (single file)**
1. Open the file
2. `Cmd + Shift + P` (Mac) or `Ctrl + Shift + P` (Windows)
3. Type "Format Document" and select it

**Option 2: Keyboard shortcut (single file)**
- Mac: `Shift + Option + F`
- Windows: `Shift + Alt + F`

**Option 3: Format on save (automatic)**

Add to your VSCode settings (`Cmd + ,` → search "format on save"):
```json
{
    "editor.formatOnSave": true,
    "python.formatting.provider": "black"
}
```

Or for newer VSCode with the Black extension:
```json
{
    "[python]": {
        "editor.defaultFormatter": "ms-python.black-formatter",
        "editor.formatOnSave": true
    }
}
```

**Option 4: Command line (multiple files)**
```bash
black cue_eval/runner.py cue_eval/evaluation_agent.py cue_eval/legacy/cue_react/cue_react_agent.py
```

Or format a whole directory:
```bash
black cue_eval/
```

---

**Make sure Black is installed:**
```bash
pip install black
```

And if using the VSCode extension, install "Black Formatter" from the extensions marketplace.
