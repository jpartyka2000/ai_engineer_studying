## Finding All References to a Class in VSCode:

### **Primary Methods:**

### 1. **Find All References** (Most Complete)
- **Right-click** on the class name → **"Find All References"**
- Or place cursor on class name and press:
  - **`Shift+F12`** (Windows/Linux)
  - **`Shift+F12`** (Mac)
- Opens a panel showing every place the class is used

### 2. **Go to References** (Inline View)
- **`Alt+Shift+F12`** (Windows/Linux)
- **`Option+Shift+F12`** (Mac)
- Shows references inline in a peek window

### 3. **Call Hierarchy** (For Methods)
- Right-click on a method → **"Show Call Hierarchy"**
- Or **`Shift+Alt+H`**
- Shows incoming/outgoing calls in a tree view

### **Alternative Methods:**

### 4. **Search Across Files**
- **`Ctrl+Shift+F`** (or `Cmd+Shift+F` on Mac)
- Search for the class name across all files
- Less precise but works without language support

### 5. **Symbol Search**
- **`Ctrl+T`** (or `Cmd+T` on Mac)
- Type `#` followed by class name
- Shows where the symbol is defined and referenced

### **Visual Indicators:**
```python
class MyClass:  # ← Right-click here
    pass

# VSCode will find:
from module import MyClass  # Import
x = MyClass()              # Instantiation  
isinstance(obj, MyClass)   # Type check
class Child(MyClass):      # Inheritance
```

### **Pro Tips:**
- **CodeLens** (if enabled) shows reference count above class: `3 references`
- Results panel lets you filter by file type or path
- Works best with Python language server (Pylance) installed

**Fastest**: Put cursor on class name → **`Shift+F12`** → See all references!
