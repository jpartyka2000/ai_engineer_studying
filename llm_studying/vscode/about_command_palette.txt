## Command Palette - Brief Overview

The **Command Palette** is VSCode's universal search for all commands and actions.

### Opening It
- **Windows/Linux**: `Ctrl+Shift+P` or `F1`
- **Mac**: `Cmd+Shift+P` or `F1`

### What It Does
- Access **any VSCode command** without memorizing shortcuts
- Search settings, extensions, git operations, file operations, etc.
- Just start typing what you want to do

### Examples
- Type "format" → Format Document
- Type "git commit" → Git: Commit
- Type "settings" → Preferences: Open Settings
- Type "theme" → Change Color Theme
- Type "install" → Extensions: Install Extensions

### Pro Tips
- **`>`** prefix (auto-added) = commands
- **No `>`** = file search (like `Ctrl+P`)
- **`@`** = symbol search in current file
- **`#`** = symbol search across workspace
- **`:`** = go to line number

Think of it as **"Google for VSCode"** - when you don't know where something is, just open the Command Palette and search for it!
