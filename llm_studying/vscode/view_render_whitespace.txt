## VSCode Render Whitespace

**View → Render Whitespace** makes invisible characters visible in your editor:

### **What It Shows:**
- **Spaces** → Small dots (·)
- **Tabs** → Arrows (→)
- **Newlines** → ¬ symbol (optional)

### **Example View:**
```python
def·hello():
····name·=·"Alice"··  # Shows trailing spaces!
→   return·name        # Tab vs spaces visible
```

### **How to Toggle:**
1. **Menu**: View → Render Whitespace
2. **Command Palette**: `Ctrl+Shift+P` → "Toggle Render Whitespace"
3. **Settings**: Can set to `"selection"`, `"all"`, `"boundary"`, or `"none"`

### **Why It's Useful:**
- Spot **mixed tabs/spaces** (Python errors!)
- Find **trailing whitespace**
- Debug **YAML indentation** issues
- See **accidental double spaces**

### **Settings Control:**
```json
// settings.json
"editor.renderWhitespace": "all",      // Always show
"editor.renderWhitespace": "selection", // Only in selected text
"editor.renderWhitespace": "boundary",  // Smart visibility
"editor.renderWhitespace": "none"       // Hidden (default)
```

**Pro tip**: Use `"boundary"` for the best balance - shows whitespace where it matters without cluttering your view!
