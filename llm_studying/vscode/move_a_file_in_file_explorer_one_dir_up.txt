Yes, a few options:

## Drag and drop
Just drag the file and drop it onto the parent folder in the file explorer sidebar.

## Cut and paste (keyboard)
1. Select the file
2. `Ctrl+X` (or `Cmd+X` on Mac) to cut
3. Click the parent directory
4. `Ctrl+V` (or `Cmd+V`) to paste

## Quick method with `..`
1. Right-click the file → **Rename** (or press `F2`)
2. Change the name to `../filename.ext`
3. Press Enter

This last trick works because VSCode interprets the path — typing `../` moves it up one level.
