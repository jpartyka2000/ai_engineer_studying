In VS Code, you can see your installed extensions in a few ways:

## Using the Sidebar

1. Click the **Extensions icon** in the left sidebar (looks like four squares, one detached)
2. Or press `Ctrl+Shift+X` (Windows/Linux) or `Cmd+Shift+X` (Mac)
3. Click on **"Installed"** in the dropdown or scroll down to the "Installed" section

## Using the Command Palette

1. Press `Ctrl+Shift+P` (Windows/Linux) or `Cmd+Shift+P` (Mac)
2. Type `Extensions: Show Installed Extensions`
3. Press Enter

## Using the Terminal

If you prefer the command line:

```bash
code --list-extensions
```

This prints a plain list of extension IDs, useful for scripting or sharing your setup.
