In VSCode, you have several ways to open a file outside your current project:

## Quick Methods

### 1. **File → Open File** (Recommended)
- `Ctrl+O` (Windows/Linux) or `Cmd+O` (Mac)
- Browse to any file on your system
- Opens in current window alongside your project

### 2. **Drag and Drop**
- Simply drag the file from your file explorer into VSCode
- It will open in the editor

### 3. **Quick Open**
- `Ctrl+P` (Windows/Linux) or `Cmd+P` (Mac)
- Type the full path to the file (if you know it)
- Press Enter

### 4. **Command Palette**
- `Ctrl+Shift+P` (Windows/Linux) or `Cmd+Shift+P` (Mac)
- Type "File: Open File"
- Browse and select

## If You Want a Completely Separate Window

### **File → New Window**
- `Ctrl+Shift+N` (Windows/Linux) or `Cmd+Shift+N` (Mac)
- Opens a new VSCode window
- Then use `Ctrl+O` / `Cmd+O` to open your unrelated file

## Notes

- Opening a file with `Ctrl+O` / `Cmd+O` **doesn't change your workspace** - your project folder stays the same in the Explorer
- The file just appears as an editor tab
- The file won't show in your Explorer sidebar (since it's not part of the project)
- You can have files from multiple locations open simultaneously

The quickest approach is usually just **`Ctrl+O` / `Cmd+O`** and navigate to your file!
