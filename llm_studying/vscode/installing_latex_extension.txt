Here's how to install LaTeX Workshop in VS Code:

**Method 1: Extensions Sidebar (Easiest)**
1. Open VS Code
2. Click the **Extensions icon** in the left sidebar (or press **Cmd+Shift+X**)
3. Search for **"LaTeX Workshop"**
4. Click **Install** on the one by James Yu (it's the most popular one)

**Method 2: Command Palette**
1. Press **Cmd+Shift+P** to open Command Palette
2. Type **"Extensions: Install Extensions"**
3. Search for **"LaTeX Workshop"**
4. Click Install

**After Installation:**

You'll still need a LaTeX distribution installed on your Mac for it to work:

**Install MacTeX or BasicTeX:**
- **MacTeX** (full, ~4GB): https://www.tug.org/mactex/
- **BasicTeX** (minimal, ~100MB): https://www.tug.org/mactex/morepackages.html

**To verify it's working:**
1. Create a new file: `test.tex`
2. Add some basic LaTeX:
```latex
\documentclass{article}
\begin{document}
Hello World!
\end{document}
```
3. Save the file
4. LaTeX Workshop should auto-compile and show a PDF preview

**Useful VS Code shortcuts with LaTeX Workshop:**
- **Cmd+Alt+B** - Build LaTeX
- **Cmd+Alt+V** - View PDF
