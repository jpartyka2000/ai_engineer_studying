To start typing LaTeX in VS Code with the LaTeX Workshop extension, you just need to:

1. **Create a new file** with a `.tex` extension (e.g., `document.tex`)
2. **Start writing** your LaTeX code

That's basically it! The extension activates automatically when it detects a `.tex` file.

Here's a minimal document to get you started:

```latex
\documentclass{article}

\begin{document}

Hello, world!

\end{document}
```

**To compile and preview:**
- Save the file, and LaTeX Workshop will typically auto-build it (you'll see a PDF appear)
- Or press `Cmd + Option + B` to build manually
- Press `Cmd + Option + V` to open the PDF preview panel

**One thing to check:** LaTeX Workshop needs to know where your TeX distribution is. Since you have TeXShop installed, you likely have MacTeX or a similar distribution. If building doesn't work out of the box, open VS Code settings and search for `latex-workshop.latex.tools` to verify the paths are correct—but usually it just works if MacTeX is installed normally.
