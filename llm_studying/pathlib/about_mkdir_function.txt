QUESTION:
---------
Level: Hard | Area: Directory Operations | Type: Coding

Write code to create a nested directory structure /data/processed/2024/reports, 
creating all parent directories if they don't exist, without raising an error 
if the directory already exists:

    from pathlib import Path
    p = Path("/data/processed/2024/reports")

    ______

YOUR ANSWER:
------------
p.mkdir("/data/processed/2024/reports", create_directories=True)

CORRECT ANSWER:
---------------
p.mkdir(parents=True, exist_ok=True)

EXPLANATION:
------------
Two issues with your answer:

1. mkdir() doesn't take a path argument — the path is already stored in `p`
2. The parameter is `parents=True`, not `create_directories=True`

THE MKDIR() PARAMETERS:
-----------------------
    p.mkdir()                              # Basic — fails if parents missing
    p.mkdir(parents=True)                  # Creates parent directories
    p.mkdir(exist_ok=True)                 # No error if directory exists
    p.mkdir(parents=True, exist_ok=True)   # Both — the safest option

WHAT EACH PARAMETER DOES:
-------------------------
    parents=True   →  Creates /data, /data/processed, /data/processed/2024
                      if they don't exist (like `mkdir -p` in shell)
    
    exist_ok=True  →  Silently succeeds if directory already exists
                      (without this, FileExistsError is raised)

COMPARISON TO OS MODULE:
------------------------
    # os module
    os.makedirs("/data/processed/2024/reports", exist_ok=True)
    
    # pathlib
    Path("/data/processed/2024/reports").mkdir(parents=True, exist_ok=True)
