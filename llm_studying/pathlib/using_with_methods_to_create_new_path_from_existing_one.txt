QUESTION:
---------
Level: Medium | Area: Navigation Methods | Type: Coding

Given:

    from pathlib import Path
    p = Path("/data/reports/2024/sales.csv")

Write an expression that changes the file extension from .csv to .json, 
resulting in a new Path representing /data/reports/2024/sales.json.

(Don't modify the original path — just create a new one.)

YOUR ANSWER:
------------
p.with_suffix(".json")

CORRECT ANSWER:
---------------
p.with_suffix(".json")

EXPLANATION:
------------
The `with_suffix()` method returns a new Path with the suffix (extension) changed.
Important: Include the dot in the new suffix!

    p = Path("/data/reports/2024/sales.csv")
    
    p.with_suffix(".json")  →  Path("/data/reports/2024/sales.json")
    p.with_suffix(".txt")   →  Path("/data/reports/2024/sales.txt")
    p.with_suffix("")       →  Path("/data/reports/2024/sales")  # removes extension

Related methods:
    p.with_name("report.csv")   →  replaces entire filename
    p.with_stem("report")       →  replaces just the stem (Python 3.9+)

Note: These methods don't modify the original Path (paths are immutable) — 
they return a new Path object.
