QUESTION:
---------
Level: Medium | Area: Reading/Writing Files | Type: Coding

Write the simplest pathlib way to read the entire contents of a text file 
as a string:

    from pathlib import Path
    p = Path("notes.txt")

    content = ______

YOUR ANSWER:
------------
p.contents()

CORRECT ANSWER:
---------------
p.read_text()

EXPLANATION:
------------
Pathlib provides convenient methods for reading and writing files without 
needing to use open() or context managers:

READING:
    p.read_text()       →  returns entire file as a string
    p.read_text(encoding="utf-8")  →  specify encoding
    p.read_bytes()      →  returns file contents as bytes

WRITING:
    p.write_text("hello world")    →  writes string to file
    p.write_bytes(b"binary data")  →  writes bytes to file

COMPARISON — Old way vs Pathlib way:
    
    # Old way (os/open)
    with open("notes.txt", "r") as f:
        content = f.read()
    
    # Pathlib way — one line!
    content = Path("notes.txt").read_text()

Note: read_text() and write_text() handle opening and closing the file 
automatically. write_text() overwrites existing content (no append mode).
