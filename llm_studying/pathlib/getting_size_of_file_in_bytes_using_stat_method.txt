QUESTION:
---------
Level: Hard | Area: File Operations | Type: Coding

Write code to get the size in bytes of a file using pathlib:

    from pathlib import Path
    p = Path("data.csv")

    size_in_bytes = ______

YOUR ANSWER:
------------
p.size()

CORRECT ANSWER:
---------------
p.stat().st_size

EXPLANATION:
------------
Pathlib doesn't have a direct .size() method. Instead, you use .stat() 
which returns an os.stat_result object containing file metadata, then 
access the .st_size attribute.

    p = Path("data.csv")
    
    p.stat()           →  os.stat_result object with all file metadata
    p.stat().st_size   →  file size in bytes (integer)

THE STAT() OBJECT — USEFUL ATTRIBUTES:
--------------------------------------
    stat_info = p.stat()
    
    stat_info.st_size   →  size in bytes
    stat_info.st_mtime  →  modification time (timestamp)
    stat_info.st_ctime  →  creation time (timestamp)
    stat_info.st_atime  →  last access time (timestamp)
    stat_info.st_mode   →  file permissions

PRACTICAL EXAMPLE:
------------------
    from pathlib import Path
    
    p = Path("data.csv")
    size = p.stat().st_size
    
    print(f"File size: {size} bytes")
    print(f"File size: {size / 1024:.2f} KB")
    print(f"File size: {size / 1024 / 1024:.2f} MB")

Note: .stat() raises FileNotFoundError if the path doesn't exist.
