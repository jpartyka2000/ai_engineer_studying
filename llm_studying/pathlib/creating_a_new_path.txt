QUESTION:
---------
Level: Easy | Area: Path Creation & Basics | Type: Fill in the Blank

Complete the code to create a Path object representing the file /home/user/data.csv:

    from pathlib import ______
    file_path = ______("/home/user/data.csv")

YOUR ANSWER:
------------
Path (for both blanks)

CORRECT ANSWER:
---------------
Path (for both blanks)

EXPLANATION:
------------
The `Path` class is the main entry point for pathlib. You import it from the 
pathlib module and instantiate it with a string representing your path.

    from pathlib import Path
    file_path = Path("/home/user/data.csv")

Note: pathlib also offers `PurePath`, `PosixPath`, and `WindowsPath`, but `Path` 
is the most commonly used as it automatically selects the right type for your OS.
