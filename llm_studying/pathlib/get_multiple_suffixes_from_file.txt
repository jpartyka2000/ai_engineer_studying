QUESTION:
---------
Level: Medium | Area: Path Properties | Type: Coding

Given a file with multiple extensions like data.tar.gz, write an expression 
to get all the suffixes (not just the last one):

    from pathlib import Path
    p = Path("/backups/data.tar.gz")

    all_extensions = ______
    # Expected result: [".tar", ".gz"]

YOUR ANSWER:
------------
p.suffix(get_all=True)

CORRECT ANSWER:
---------------
p.suffixes

EXPLANATION:
------------
Pathlib has TWO separate properties for extensions — suffix (singular) 
and suffixes (plural). Note: these are PROPERTIES, not methods!

    p = Path("/backups/data.tar.gz")
    
    p.suffix    →  ".gz"              # Only the last extension
    p.suffixes  →  [".tar", ".gz"]    # All extensions as a list

RELATED PROPERTIES:
-------------------
    p = Path("/backups/data.tar.gz")
    
    p.name      →  "data.tar.gz"      # Full filename
    p.stem      →  "data.tar"         # Name without last suffix
    p.suffix    →  ".gz"              # Last suffix only
    p.suffixes  →  [".tar", ".gz"]    # All suffixes

PRACTICAL USE CASES:
--------------------
    # Check for compound extensions
    if p.suffixes == [".tar", ".gz"]:
        print("This is a gzipped tarball")
    
    # Get the "real" stem (without any extensions)
    p = Path("data.tar.gz")
    real_stem = p.name.replace("".join(p.suffixes), "")
    # real_stem → "data"
    
    # Rebuild filename with different compression
    new_name = p.stem + ".bz2"   # "data.tar.bz2"
