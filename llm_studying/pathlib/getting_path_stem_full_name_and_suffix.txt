QUESTION:
---------
Level: Medium | Area: Path Properties | Type: Coding

Given the following path:

    from pathlib import Path
    p = Path("/projects/webapp/src/utils/helpers.py")

Write a single expression that extracts just the filename without the extension 
(i.e., you want "helpers").

YOUR ANSWER:
------------
p.stem

CORRECT ANSWER:
---------------
p.stem

EXPLANATION:
------------
The `stem` property returns the final component of the path, without its suffix 
(extension).

    p = Path("/projects/webapp/src/utils/helpers.py")
    
    p.name    →  "helpers.py"   # full filename
    p.suffix  →  ".py"          # extension (with dot)
    p.stem    →  "helpers"      # filename without extension

This is especially useful when you need to rename files or create related files 
with different extensions (e.g., helpers.py → helpers.txt).
