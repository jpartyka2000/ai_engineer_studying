PATH.PARENT & PATH.PARENTS — NAVIGATING UP THE DIRECTORY TREE
==============================================================

IMPORTANT: These are PROPERTIES, not methods — no parentheses!
--------------------------------------------------------------
    p.parent    ✓  (correct)
    p.parent()  ✗  (wrong — TypeError!)

PATH.PARENT:
------------
Returns the immediate parent directory as a Path object.

    from pathlib import Path
    p = Path("/home/user/docs/file.txt")
    
    p.parent              →  Path("/home/user/docs")
    p.parent.parent       →  Path("/home/user")
    p.parent.parent.parent →  Path("/home")

PATH.PARENTS:
-------------
Returns a sequence of ALL ancestor directories (immutable sequence, not a list).

    p = Path("/home/user/docs/file.txt")
    
    p.parents[0]  →  Path("/home/user/docs")    # same as p.parent
    p.parents[1]  →  Path("/home/user")
    p.parents[2]  →  Path("/home")
    p.parents[3]  →  Path("/")
    
    list(p.parents)  →  [Path("/home/user/docs"), 
                         Path("/home/user"), 
                         Path("/home"), 
                         Path("/")]

COMMON USE CASES:
-----------------
    # Get the grandparent directory
    grandparent = p.parent.parent
    
    # Check if path is under a certain directory
    if Path("/home/user") in p.parents:
        print("File is in user's home directory tree")
    
    # Find project root by looking for a marker file
    for parent in p.parents:
        if (parent / "pyproject.toml").exists():
            project_root = parent
            break

EDGE CASES:
-----------
    # Root's parent is itself
    Path("/").parent  →  Path("/")
    
    # Relative path parents
    Path("a/b/c").parents  →  (Path("a/b"), Path("a"), Path("."))
