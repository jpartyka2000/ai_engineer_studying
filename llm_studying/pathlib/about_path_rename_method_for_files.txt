QUESTION:
---------
Level: Hard | Area: File Operations | Type: Coding

Write code to rename/move a file from old_name.txt to new_name.txt using pathlib:

    from pathlib import Path
    p = Path("old_name.txt")

    ______
    # File is now named "new_name.txt"

YOUR ANSWER:
------------
p.mv("new_name.txt")

CORRECT ANSWER:
---------------
p.rename("new_name.txt")

EXPLANATION:
------------
Pathlib uses rename() for both renaming and moving files — there's no mv().

    p = Path("old_name.txt")
    p.rename("new_name.txt")      # Rename in same directory
    p.rename("subdir/file.txt")   # Move to different directory

IMPORTANT — RENAME() RETURNS THE NEW PATH:
------------------------------------------
    p = Path("old_name.txt")
    new_p = p.rename("new_name.txt")
    
    # p still points to "old_name.txt" (which no longer exists!)
    # new_p points to "new_name.txt"

RENAME() VS REPLACE():
----------------------
    p.rename("target.txt")    # Fails if target exists (on some systems)
    p.replace("target.txt")   # Overwrites target if it exists (atomic)

CROSS-DEVICE MOVES:
-------------------
rename() may fail when moving across different filesystems/drives.
For cross-device moves, use shutil:

    import shutil
