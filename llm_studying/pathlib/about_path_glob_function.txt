PATH.GLOB() — PATTERN MATCHING FOR FILES AND DIRECTORIES
=========================================================

BASIC SYNTAX:
-------------
    from pathlib import Path
    p = Path("/some/directory")
    
    results = p.glob("pattern")   # returns a generator

COMMON PATTERNS:
----------------
    p.glob("*")           →  all files and folders (immediate directory)
    p.glob("*.txt")       →  all .txt files
    p.glob("data*")       →  all items starting with "data"
    p.glob("*.??")        →  files with 2-character extensions (.py, .js, etc.)
    p.glob("[abc]*")      →  items starting with a, b, or c
    
RECURSIVE PATTERNS (using **):
------------------------------
    p.glob("**/*")        →  all files/folders recursively
    p.glob("**/*.py")     →  all .py files recursively
    p.rglob("*.py")       →  shorthand for glob("**/*.py")

RETURNS:
--------
    - A generator (not a list!) — iterate or wrap in list()
    - Each item is a Path object

EXAMPLE USAGE:
--------------
    # List all CSV files
    for csv_file in Path("data").glob("*.csv"):
        print(csv_file.name)
    
    # Get a list of all Python files recursively
    py_files = list(Path("src").rglob("*.py"))
