---------
Level: Medium | Area: Path Resolution | Type: Multiple Choice

What does Path.resolve() do?

A) Checks if the path exists and raises an error if not
B) Returns an absolute path with all symlinks and `..` components resolved
C) Splits the path into its individual components
D) Converts the Path object to a string

YOUR ANSWER:
------------
B) Returns an absolute path with all symlinks and `..` components resolved

CORRECT ANSWER:
---------------
B) Returns an absolute path with all symlinks and `..` components resolved

EXPLANATION:
------------
The resolve() method returns the absolute path, resolving any symlinks 
and eliminating any `.` or `..` components.
