QUESTION:
---------
Level: Easy | Area: File Operations | Type: Multiple Choice

Which method creates an empty file (like the Unix `touch` command)?

    from pathlib import Path
    p = Path("newfile.txt")

A) p.create()
B) p.touch()
C) p.new()
D) p.make_file()

YOUR ANSWER:
------------
B) p.touch()

CORRECT ANSWER:
---------------
B) p.touch()

EXPLANATION:
------------
The touch() method creates an empty file if it doesn't exist, or updates 
the modification timestamp if it does — exactly like the Unix `touch` command.

    from pathlib import Path
    
    p = Path("newfile.txt")
    p.touch()   # File now exists (empty)

TOUCH() PARAMETERS:
-------------------
    p.touch()               # Create file, or update timestamp if exists
    p.touch(exist_ok=True)  # Default behavior (no error if exists)
    p.touch(exist_ok=False) # Raises FileExistsError if file exists

COMMON USE CASES:
-----------------
    # Create an empty config file
    Path("config.ini").touch()
    
    # Create a .gitkeep to track empty directories
    Path("logs/.gitkeep").touch()
    
    # Create file then write to it
    p = Path("data.txt")
    p.touch()
    p.write_text("Hello!")

Note: Parent directories must exist — touch() won't create them.
Use p.parent.mkdir(parents=True, exist_ok=True) first if needed.
