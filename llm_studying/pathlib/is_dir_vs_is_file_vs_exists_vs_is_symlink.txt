QUESTION:
---------
Level: Easy | Area: File System Checks | Type: Multiple Choice

Which method would you use to check if a Path points to an existing directory?

    from pathlib import Path
    p = Path("/home/user/documents")

A) p.is_folder()
B) p.is_dir()
C) p.isdir()
D) p.check_directory()

YOUR ANSWER:
------------
B) p.is_dir()

CORRECT ANSWER:
---------------
B) p.is_dir()

EXPLANATION:
------------
The `is_dir()` method returns True if the path points to an existing directory, 
False otherwise.

Common file system check methods in pathlib:
    p.exists()    →  True if path exists (file or directory)
    p.is_file()   →  True if path is an existing regular file
    p.is_dir()    →  True if path is an existing directory
    p.is_symlink() → True if path is a symbolic link

Note: Option C (`isdir()`) is the os.path style — pathlib uses underscores 
(is_dir). This is a common gotcha when transitioning from os.path to pathlib!
