QUESTION:
---------
Level: Hard | Area: Directory Operations | Type: Coding

Write code that finds all Python files (.py) in a directory and all its 
subdirectories recursively.

    from pathlib import Path
    project_dir = Path("/myproject")

    # Write an expression that returns all .py files recursively
    python_files = ______

YOUR ANSWER:
------------
project_dir.rglob("*.py")

CORRECT ANSWER:
---------------
project_dir.rglob("*.py")
    — OR —
project_dir.glob("**/*.py")

EXPLANATION:
------------
The `rglob()` method performs a recursive glob search, finding all matching 
files in the directory and all subdirectories.

    project_dir.rglob("*.py")      →  all .py files, recursively
    project_dir.glob("**/*.py")    →  equivalent using glob with ** pattern

Comparison of directory iteration methods:
    p.iterdir()         →  all items in immediate directory (no pattern)
    p.glob("*.py")      →  matching files in immediate directory only
    p.rglob("*.py")     →  matching files recursively (all subdirectories)

Important notes:
- Both return generators, so wrap in list() if you need a list
- The ** pattern means "this directory and all subdirectories"
- rglob("*.py") is shorthand for glob("**/*.py")
