QUESTION:
---------
Level: Easy | Area: File System Checks | Type: Coding

Write an expression to check if a path is an absolute path (not relative):

    from pathlib import Path
    p = Path("/home/user/file.txt")

    is_abs = ______
    # Expected result: True

YOUR ANSWER:
------------
p.is_absolute

CORRECT ANSWER:
---------------
p.is_absolute()

EXPLANATION:
------------
is_absolute() is a METHOD that returns True/False, so parentheses are required.

    p = Path("/home/user/file.txt")
    p.is_absolute()   →  True
    
    p = Path("docs/file.txt")
    p.is_absolute()   →  False

WITHOUT PARENTHESES:
--------------------
    p.is_absolute     →  <bound method PosixPath.is_absolute of ...>
    
    # This is the method object itself, not the result!

METHODS VS PROPERTIES IN PATHLIB:
---------------------------------
PROPERTIES (no parentheses):
    p.name, p.stem, p.suffix, p.suffixes
    p.parent, p.parents, p.parts
    p.anchor, p.root, p.drive

METHODS (need parentheses):
    p.is_absolute(), p.is_relative_to()
    p.exists(), p.is_file(), p.is_dir()
    p.resolve(), p.absolute()
    p.stat(), p.glob(), p.rglob()
