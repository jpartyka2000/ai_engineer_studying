If you want to close every open window shown in your Dock (bottom bar), you essentially need to close all windows in each running app. There isn’t a single “Close All” menu item for every app at once, but you can do it per application in two main ways:

---

## 1. Option-Click the Red “Close” Button (Per App)

1. **Select any window** of the app you want to clear (for example, a Finder window, a Safari window, etc.).
2. **Hold down ⌥ (Option)**. You’ll see the red “×” button in the title bar turn into a slightly different icon.
3. **Click that red “×” once while still holding ⌥**. macOS will close *every* open window for that app.

Repeat for each app whose windows you wish to close.

* If an app has only one window open, ⌥-click on its “×” simply closes that one.
* If it has multiple windows, ⌥-click closes them all in one go.

---

## 2. Quit the App Entirely (Per App)

In many cases, simply quitting the app will close all its windows:

1. **Make sure the app is the frontmost (active) application** (click any of its windows).
2. Press **⌘ + Q** (or choose **App Name → Quit App Name** from the menu bar).
3. That app’s windows will all disappear, and the app itself will exit.

If you don’t want to quit but only want to close its windows, use option-click on the “×” (see method 1).

---

## 3. Close Every Window in All Apps at Once (via AppleScript)

If you really want to slam-shut every open window across every running app with a single command, you can use a tiny AppleScript. That will ask each GUI-enabled application to close all its windows. Here’s how:

1. Open **Script Editor** (in `/Applications/Utilities/Script Editor.app`).

2. Paste in the following script:

   ```applescript
   tell application "System Events"
       set frontmostProcessList to name of every process whose background only is false
   end tell

   repeat with procName in frontmostProcessList
       try
           tell application procName
               close every window
           end tell
       end try
   end repeat
   ```

3. Click ▶︎ (Run). It will send a “close every window” message to each front-end app.

   * Some apps will prompt “Do you want to save?” if there are unsaved documents.
   * If an app doesn’t support `close every window`, the `try`/`end try` block simply skips it without error.

4. If you like, save this script as an application bundle and drop it in your Dock for a one-click “Close All Windows”—just open the saved script app whenever you want to shut everything.

---

### Notes

* **Minimized windows** (those you’ve clicked to the Dock) still count as “open,” so AppleScript’s `close every window` will also clear them.
* The **Option-Click** method (section 1) is the fastest way to close all windows *of a single app* without quitting.
* The **Quit (⌘ + Q)** method (section 2) both closes windows *and* terminates the app.
* If an app has unsaved changes, any automatic “close all” will prompt you to save or discard before proceeding.

Choose whichever approach fits your workflow—⌥-click is ideal when you want to keep the app running but clear its windows; AppleScript is the only way to do everything in one pass.
