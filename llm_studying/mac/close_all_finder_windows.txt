To close all Finder windows on a Mac, follow these steps:

### Using a Keyboard Shortcut
1. Focus on Finder by clicking the desktop or selecting Finder from the Dock.
2. Press **Option + Command + W**.  
   - This will close all Finder windows simultaneously.

### Using the Menu Bar
1. Focus on Finder by clicking the desktop or selecting Finder from the Dock.
2. From the menu bar at the top, click **File** > **Close All** (if available).

These methods ensure all open Finder windows are closed at once.
