The `caffeinate` command is a built-in macOS utility that prevents your Mac from going to sleep or activating the screensaver. It's particularly useful for long-running processes.

## Basic Syntax
```bash
caffeinate [options] [command]
```

## Common Options

**`-i` (prevent idle sleep)**
```bash
caffeinate -i python script.py
# Prevents system sleep while script runs
```

**`-d` (prevent display sleep)**
```bash
caffeinate -d
# Keeps display awake (useful for presentations)
```

**`-m` (prevent disk sleep)**
```bash
caffeinate -m python backup_script.py
# Prevents disk from spinning down
```

**`-s` (prevent system sleep on AC power)**
```bash
caffeinate -s python script.py
# Only works when plugged into power
```

**`-u` (prevent user idle sleep)**
```bash
caffeinate -u -t 3600
# Simulates user activity for 1 hour
```

**`-t` (timeout in seconds)**
```bash
caffeinate -i -t 7200 python script.py
# Prevents sleep for 2 hours, then allows normal sleep
```

## Practical Examples

**Run a script without sleep:**
```bash
caffeinate -i python long_running_script.py
```

**Keep awake for specific duration:**
```bash
caffeinate -i -t 1800  # 30 minutes
```

**Prevent display sleep during presentation:**
```bash
caffeinate -d -t 3600  # 1 hour
```

**Multiple options combined:**
```bash
caffeinate -ims python data_processing.py
# Prevents idle, disk, and system sleep
```

## How to Use

1. **With a command:** Automatically ends when the command finishes
2. **Without a command:** Runs until you press Ctrl+C
3. **Background process:** Add `&` to run in background

## Check What's Preventing Sleep
```bash
pmset -g assertions
```
This shows active caffeinate processes and other sleep preventers.

## Tips
- `caffeinate` only prevents automatic sleep, not manual sleep (closing laptop lid)
- It's safer than changing system sleep settings permanently
- Use `-t` for safety - prevents accidentally leaving your Mac awake indefinitely
- Combine options for comprehensive prevention: `caffeinate -idms`

The command is perfect for your Python scripts since it automatically terminates when the script finishes, returning your Mac to normal sleep behavior.
