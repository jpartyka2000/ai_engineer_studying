```python
import pandas as pd
import matplotlib.pyplot as plt
import seaborn as sns
import missingno as msno
from pandas_profiling import ProfileReport

# 1️⃣ Data Overview & Basic Info
print("Data Shape:", df.shape)  # Data shape
print("Info:")
print(df.info())  # Column types & memory usage
print("Head of the DataFrame:")
print(df.head())  # First 5 rows
print("Tail of the DataFrame:")
print(df.tail())  # Last 5 rows
print("Random Sample of DataFrame:")
print(df.sample(5))  # Random 5 rows

# 2️⃣ Null Counts & Missing Values
print("Null Counts for All Columns:")
print(df.isnull().sum())  # Null counts
msno.matrix(df)  # Visual representation of missing values
plt.show()

# 3️⃣ Number of Unique Values per Column
print("Number of Unique Values for All Columns:")
print(df.nunique())  # Number of unique values per column

# 4️⃣ Summary Statistics for Numerical Columns
print("Descriptive Summary for Numerical Columns:")
print(df.describe())  # Summary statistics for numerical columns

# 5️⃣ Detect Duplicates
print(f"Duplicate Rows: {df.duplicated().sum()}")

# 6️⃣ High Cardinality Columns
high_cardinality_cols = [col for col in df.columns if df[col].nunique() > 0.8 * len(df)]
print("High Cardinality Columns:", high_cardinality_cols)

# 7️⃣ Outlier Detection (Basic - Boxplot)
sns.boxplot(data=df.select_dtypes(include=['number']))
plt.xticks(rotation=90)
plt.title("Boxplot of Numerical Features")
plt.show()

# 8️⃣ Correlation Matrix
plt.figure(figsize=(10,6))
sns.heatmap(df.corr(), annot=True, cmap="coolwarm", fmt=".2f")
plt.title("Correlation Matrix")
plt.show()

# 9️⃣ Categorical Feature Distribution
for col in df.select_dtypes(include=['object', 'category']):
    print(f"Value counts for {col}:\n", df[col].value_counts(), "\n")

# 🔟 Skewness & Kurtosis (Numerical Data)
print("Skewness of Numerical Columns:")
print(df.skew())  # Skewness (asymmetry)
print("Kurtosis of Numerical Columns:")
print(df.kurt())  # Kurtosis (tail heaviness)

# 1️⃣1️⃣ Pandas Profiling (Full Report - One-Liner)
profile = ProfileReport(df, explorative=True)
profile.to_notebook_iframe()
```

### Breakdown of What Each Section Does:
- **Data Overview & Basic Info**: Provides an overall look at the shape, types, and first/last rows of the dataset.
- **Null Counts & Missing Values**: Shows missing data counts and visualizes it.
- **Unique Values**: Displays how many unique values are present in each column.
- **Summary Statistics**: Provides statistics like mean, standard deviation, min, max, and percentiles for numerical columns.
- **Duplicate Detection**: Checks if there are duplicate rows in the dataset.
- **High Cardinality Columns**: Identifies columns with many unique values (potential categorical features that need attention).
- **Outlier Detection (Boxplot)**: Visualizes outliers in numerical data via boxplots.
- **Correlation Matrix**: Shows the correlation between numerical columns.
- **Categorical Feature Distribution**: Displays value counts for each categorical column.
- **Skewness & Kurtosis**: Measures the asymmetry (skewness) and the "tailedness" (kurtosis) of numerical columns.
- **Pandas Profiling**: Generates a detailed, interactive report summarizing the dataset.
