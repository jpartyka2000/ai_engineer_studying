The call

```python
returns = torch.stack(returns)
```

takes your **Python list** of tensors (each here is a 0-dim tensor, e.g. `tensor(3.5)`) and “stacks” them into a **single** tensor by adding a new first dimension.

* **Before**:

  ```python
  returns  # a list like [tensor(G0), tensor(G1), tensor(G2), …]
  ```
* **After**:

  ```python
  returns  # a tensor of shape (T,), e.g. tensor([G0, G1, G2, …])
  ```

Under the hood, `torch.stack` checks that every element has the same shape and dtype, then creates a new tensor whose 0ᵗʰ dimension indexes which element came from the list.

> **Why not use `torch.cat`?**
> • `torch.cat` concatenates along an **existing** dimension (so you’d need each tensor to have shape `(1,)` already).
> • `torch.stack` is the simpler way to turn a list of scalars into a vector.
