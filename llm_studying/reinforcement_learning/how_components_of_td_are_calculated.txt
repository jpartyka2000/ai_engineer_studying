In Advantage Actor-Critic (A2C) (and most actor-critic methods), the value network (critic) is trained to approximate the state value function:

V^\pi(s) = \mathbb{E}_\pi \Big[ \sum_{k=0}^\infty \gamma^k r_{t+k} \,\Big|\, s_t = s \Big]

This means that the value network learns a function  that takes only the state  as input and predicts the expected return starting from that state under the current policy.

Here's how it works step by step:


---

1️⃣ Data Collection

During an episode or batch of experience, the agent interacts with the environment and collects transitions:

(s_t, a_t, r_t, s_{t+1})

These transitions are stored for training.


---

2️⃣ Temporal Difference (TD) Target

We compute a bootstrapped target for the value function:

y_t = r_t + \gamma V_\theta(s_{t+1})

3️⃣ Value Network Update

We train the value network by minimizing the mean squared error (MSE) between its prediction for the current state  and the TD target:
