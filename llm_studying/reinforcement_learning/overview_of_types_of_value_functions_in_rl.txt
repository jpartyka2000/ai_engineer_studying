Here’s a quick rundown of the four core value‑function concepts in RL:

---

## 1. On‑Policy State‑Value Function $V^\pi(s)$

* **Definition**: The expected return when starting in state $s$ and thereafter following policy $\pi$:

  $$
    V^\pi(s) = \mathbb{E}_\pi\Bigl[\sum_{t=0}^\infty \gamma^t\,r_{t+1}\;\Big|\;s_0=s\Bigr].
  $$
* **Interpretation**: “How good is it to be in state $s$ if I act according to $\pi$?”
* **Bellman equation**:

  $$
    V^\pi(s) = \sum_{a}\pi(a\mid s)\sum_{s',r}P(s',r\mid s,a)\,\bigl[r + \gamma\,V^\pi(s')\bigr].
  $$

---

## 2. On‑Policy Action‑Value Function $Q^\pi(s,a)$

* **Definition**: The expected return when starting in state $s$, taking action $a$, and thereafter following $\pi$:

  $$
    Q^\pi(s,a) = \mathbb{E}_\pi\Bigl[\sum_{t=0}^\infty \gamma^t\,r_{t+1}\;\Big|\;s_0=s,a_0=a\Bigr].
  $$
* **Interpretation**: “How good is it to take action $a$ in state $s$ under $\pi$?”
* **Bellman equation**:

  $$
    Q^\pi(s,a) = \sum_{s',r}P(s',r\mid s,a)\,\bigl[r + \gamma\,V^\pi(s')\bigr].
  $$

---

## 3. Optimal State‑Value Function $V^*(s)$

* **Definition**: The maximum achievable expected return starting from $s$, over **all** policies:

  $$
    V^*(s) = \max_\pi V^\pi(s).
  $$
* **Interpretation**: “What’s the best possible value I can get from state $s$ if I act optimally?”
* **Bellman optimality equation**:

  $$
    V^*(s) = \max_a \sum_{s',r}P(s',r\mid s,a)\,\bigl[r + \gamma\,V^*(s')\bigr].
  $$

---

## 4. Optimal Action‑Value Function $Q^*(s,a)$

* **Definition**: The maximum expected return achievable by taking action $a$ in $s$ and thereafter acting optimally:

  $$
    Q^*(s,a) = \max_\pi Q^\pi(s,a).
  $$
* **Interpretation**: “If I take $a$ in $s$ and then behave best‑possible, how much can I expect?”
* **Bellman optimality equation**:

  $$
    Q^*(s,a)
    = \sum_{s',r}P(s',r\mid s,a)\,\Bigl[r + \gamma\,\max_{a'}Q^*(s',a')\Bigr].
  $$

---

### Relationships & Usage

* **Policy evaluation**: compute $V^\pi$ or $Q^\pi$ for a fixed $\pi$.
* **Policy improvement**: use $Q^\pi$ to derive a better policy (e.g. greedy w\.r.t. $Q^\pi$).
* **Optimal control**: seek $V^*$ or $Q^*$ directly (e.g. via value iteration or Q‑learning), then extract $\pi^*(s) = \arg\max_a Q^*(s,a)$.

These four functions form the backbone of virtually all RL algorithms, balancing “what happens if I keep doing $\pi$” versus “what’s the best I could do.”
