**Question:** What is the default logging level when you use `logging.basicConfig()` without specifying a level?

**Answer:** C) `WARNING`

By default, Python's logging module uses `WARNING` as its threshold level. This means only messages with severity `WARNING`, `ERROR`, and `CRITICAL` will be output—`DEBUG` and `INFO` messages are suppressed.

Running the code:

```python
import logging
logging.basicConfig()
logging.debug("Debug message")      # NOT displayed
logging.info("Info message")        # NOT displayed
logging.warning("Warning message")  # Displayed
```

Output:
```
WARNING:root:Warning message
```

Log Levels (lowest to highest):
- DEBUG = 10
- INFO = 20
- WARNING = 30 (default)
- ERROR = 40
- CRITICAL = 50

To display INFO and above, explicitly set it:
```python
logging.basicConfig(level=logging.INFO)
```
