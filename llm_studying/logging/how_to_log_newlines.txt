**No, that won't work!** You'll get a `TypeError` because the logger methods require at least one argument (the message).

```python
import logging

logger = logging.getLogger("test")
logger.setLevel(logging.INFO)
handler = logging.StreamHandler()
handler.setFormatter(logging.Formatter('%(message)s'))
logger.addHandler(handler)

# This will raise an error:
logger.info()  # ❌ TypeError: info() missing 1 required positional argument: 'msg'
```

## How to Log a Blank Line

You need to pass a message argument. Here are your options:

### Option 1: Empty String
```python
logger.info("")
```

**Output:**
```
(blank line)
```

### Option 2: Newline Character
```python
logger.info("\n")
```

**Output:**
```

(blank line)
```

### Option 3: Multiple Newlines
```python
logger.info("\n\n")
```

**Output:**
```


(two blank lines)
```

## Practical Example

```python
import logging
import sys

logger = logging.getLogger("app")
logger.setLevel(logging.INFO)
handler = logging.StreamHandler(sys.stdout)
handler.setFormatter(logging.Formatter('%(message)s'))
logger.addHandler(handler)

# Creating visual separation
logger.info("=" * 60)
logger.info("SUMMARY")
logger.info("=" * 60)
logger.info("")  # Blank line for spacing
logger.info("Total records: 1000")
logger.info("Errors: 5")
logger.info("")  # Another blank line
logger.info("=" * 60)
```

**Output:**
```
============================================================
SUMMARY
============================================================

Total records: 1000
Errors: 5

============================================================
```

## Why You Must Provide an Argument

The signature of logger methods is:
```python
def info(self, msg, *args, **kwargs):
    # msg is required
```

The `msg` parameter is **required** - it's not optional and has no default value.

## Tip: Helper Function

If you log blank lines frequently, you could create a helper:

```python
def log_blank(logger, count=1):
    """Log one or more blank lines"""
    for _ in range(count):
        logger.info("")

# Usage
log_blank(logger)        # One blank line
log_blank(logger, 3)     # Three blank lines
```

**Bottom line:** Always pass at least an empty string `""` to logger methods!
