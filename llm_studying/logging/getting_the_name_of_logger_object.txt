To get the name of a logger object in Python, simply access its `name` attribute:

```python
import logging

# Create a logger
logger = logging.getLogger("my_app.module")

# Get the name
logger_name = logger.name
print(logger_name)  # Output: my_app.module
```

## Complete Examples

### Example 1: Basic Usage

```python
import logging

logger = logging.getLogger("database")
print(f"Logger name: {logger.name}")
# Output: Logger name: database
```

### Example 2: With __name__

```python
import logging

# Create logger using module name
logger = logging.getLogger(__name__)

# Get the name back
print(f"This logger's name is: {logger.name}")
# If in module "my_module.py", output: This logger's name is: my_module
```

### Example 3: Root Logger

```python
import logging

# Root logger has an empty string as its name
root_logger = logging.getLogger()
print(f"Root logger name: '{root_logger.name}'")
# Output: Root logger name: 'root'

# Check if it's the root logger
if root_logger.name == 'root':
    print("This is the root logger")
```

### Example 4: Hierarchical Logger Names

```python
import logging

# Create hierarchical loggers
parent = logging.getLogger("app")
child = logging.getLogger("app.database")
grandchild = logging.getLogger("app.database.postgres")

print(f"Parent: {parent.name}")        # Output: Parent: app
print(f"Child: {child.name}")          # Output: Child: app.database
print(f"Grandchild: {grandchild.name}") # Output: Grandchild: app.database.postgres
```

### Example 5: Dynamic Logger Name Checking

```python
import logging

def log_with_context(logger, message):
    """Log a message with information about which logger is being used"""
    print(f"[{logger.name}] {message}")
    logger.info(message)

# Create different loggers
auth_logger = logging.getLogger("auth")
db_logger = logging.getLogger("database")

log_with_context(auth_logger, "User logged in")
log_with_context(db_logger, "Query executed")

# Output:
# [auth] User logged in
# [database] Query executed
```

## Practical Use Case: Debugging

```python
import logging

def configure_logger(name=None):
    logger = logging.getLogger(name)
    logger.setLevel(logging.INFO)
    
    handler = logging.StreamHandler()
    formatter = logging.Formatter(f'[{logger.name}] %(message)s')
    handler.setFormatter(formatter)
    logger.addHandler(handler)
    
    print(f"Configured logger: {logger.name}")
    return logger

# Usage
my_logger = configure_logger("app.service")
# Output: Configured logger: app.service

my_logger.info("Service started")
# Output: [app.service] Service started
```

## Summary

**The `name` attribute:**
- Type: `str`
- Always available on any logger object
- Read-only (you shouldn't modify it)
- For the root logger, the name is `'root'`
- For named loggers, it's the string you passed to `getLogger()`

```python
logger = logging.getLogger("my_logger")
logger_name = logger.name  # Simple as that!
```
