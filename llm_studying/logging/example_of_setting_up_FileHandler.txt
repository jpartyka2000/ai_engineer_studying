**Question:** Complete the code to add a handler that writes log messages to a file called `app.log`:

```python
import logging

logger = logging.getLogger("myapp")
logger.setLevel(logging.DEBUG)

handler = logging._____________("app.log")
logger.addHandler(handler)

logger.info("Application started")
```

**Your Answer:** `FileHandler`

**Correct Answer:** `FileHandler`

**Explanation:**

`logging.FileHandler` is the built-in handler class for writing log messages to a file. The complete code looks like:

```python
import logging

logger = logging.getLogger("myapp")
logger.setLevel(logging.DEBUG)

handler = logging.FileHandler("app.log")
logger.addHandler(handler)

logger.info("Application started")
```

This writes to `app.log` in the current directory. 

**Bonus tip:** `FileHandler` also accepts optional parameters:
- `mode` – defaults to `'a'` (append); use `'w'` to overwrite
- `encoding` – e.g., `encoding='utf-8'`

```python
handler = logging.FileHandler("app.log", mode='w', encoding='utf-8')
```
