LightGBM is a machine learning library, specifically designed for gradient boosting.

But more precisely:

LightGBM the library provides fast, scalable, and accurate implementations of gradient boosting algorithms.

Inside the library, the models it trains are gradient boosting decision tree (GBDT) models.


So, when people say “LightGBM model,” they usually mean a model trained using the LightGBM library, particularly a gradient boosted decision tree model.
