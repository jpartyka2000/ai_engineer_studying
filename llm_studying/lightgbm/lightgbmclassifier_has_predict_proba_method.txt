Yes—the **`lightgbm.LGBMClassifier`** class (the scikit‑learn API wrapper) implements `predict_proba()` just like any other scikit‑learn classifier. It returns a NumPy array of shape `(n_samples, n_classes)` giving the class probabilities.

### Quick example

```python
import lightgbm as lgb
from sklearn.datasets import load_iris
from sklearn.model_selection import train_test_split

# load data
X, y = load_iris(return_X_y=True)
X_train, X_test, y_train, _ = train_test_split(X, y, random_state=42)

# train
clf = lgb.LGBMClassifier(n_estimators=100, random_state=42)
clf.fit(X_train, y_train)

# predict probabilities
probs = clf.predict_proba(X_test)
print(probs.shape)     # (n_samples, 3) for the 3 Iris classes
print(probs[:5])
# e.g. [[0.01, 0.97, 0.02], [0.00, 0.99, 0.01], …]
```

If you’re loading a LightGBM model from MLflow with the LightGBM flavor:

```python
import mlflow.lightgbm

model_uri = "models:/MyLightGBMModel/Production"
lgb_model = mlflow.lightgbm.load_model(model_uri)

probs = lgb_model.predict_proba(X_new)  # same interface
```

Both the standalone `LGBMClassifier` and the MLflow‑loaded LightGBM model expose `predict_proba()` out of the box.
