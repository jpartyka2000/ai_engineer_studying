To get the smallest nonnegative number in an integer column in pandas, you can filter the column to only include nonnegative values and then use the `min()` function to find the smallest value. Here's how you can do it:

```python
smallest_nonnegative_value = df_all_opps[df_all_opps['your_column'] >= 0]['your_column'].min()
```

### Explanation:
- `df_all_opps['your_column'] >= 0` filters the rows where the column values are greater than or equal to 0 (nonnegative values).
- `['your_column']` selects the column you're interested in.
- `.min()` returns the smallest value from the filtered column.

This will give you the smallest nonnegative value in that column. If no nonnegative values exist, `min()` will return `NaN`.
