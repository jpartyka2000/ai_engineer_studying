Yes. When you do:

```python
df_closed.groupby('name').apply(_win_rate)
```

pandas will take your original **DataFrame** (`df_closed`), split it into chunks **per unique `name`**, and then call your helper function `_win_rate(g)` with each chunk `g` being a **DataFrame** containing just the rows for that one rep.  

- **`g`** has all the same columns as `df_closed`, but only the subset of rows where `name == that_rep`.  
- Whatever scalar value you return from `_win_rate(g)` becomes the element in the resulting Series at index `that_rep`.  

So yes—**each `g` passed into `_win_rate` is a DataFrame** of that rep’s closed‐opportunity records.
