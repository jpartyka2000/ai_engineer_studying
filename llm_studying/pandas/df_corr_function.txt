Yes, by default, `df.corr()` in pandas calculates the **Pearson correlation coefficient** for the columns in the DataFrame. The Pearson correlation measures the linear relationship between two continuous variables and ranges from -1 to +1.

### Explanation of the Pearson Correlation
- **+1**: Perfect positive linear relationship.
- **0**: No linear relationship.
- **-1**: Perfect negative linear relationship.

### Other Correlation Methods Supported by `df.corr()`
You can specify a different method if you want to use another type of correlation:
- `'pearson'`: Default method, calculates the Pearson correlation coefficient.
- `'kendall'`: Calculates the Kendall Tau correlation coefficient, a measure of ordinal association.
- `'spearman'`: Calculates the Spearman rank correlation, which assesses the monotonic relationship between variables.

### Example of Using Different Correlation Methods
```python
# Pearson correlation (default)
pearson_corr = df.corr(method='pearson')

# Kendall correlation
kendall_corr = df.corr(method='kendall')

# Spearman correlation
spearman_corr = df.corr(method='spearman')
```

### When to Use Each Method
- **Pearson**: Use when your data is continuous and you want to measure linear relationships.
- **Spearman/Kendall**: Use when your data is ordinal, not normally distributed, or when you want to capture monotonic relationships.

So, if you don’t specify a method, `df.corr()` uses Pearson correlation by default, which is suitable for most continuous datasets and linear relationships.
