`df.info()` provides a concise summary of a DataFrame's structure and content.

## What It Shows

**Key information displayed:**
- **Number of rows and columns**
- **Column names**
- **Data type of each column** (int64, float64, object, datetime64, etc.)
- **Non-null count** (how many non-missing values per column)
- **Memory usage** of the DataFrame

## Example Output

```python
import pandas as pd

df = pd.DataFrame({
    'name': ['Alice', 'Bob', None, 'David'],
    'age': [25, 30, 35, 40],
    'salary': [50000.0, 60000.0, 75000.0, None]
})

df.info()
```

**Output:**
```
<class 'pandas.core.frame.DataFrame'>
RangeIndex: 4 entries, 0 to 3
Data columns (total 3 columns):
 #   Column   Non-Null Count  Dtype  
---  ------   --------------  -----  
 0   name     3 non-null      object 
 1   age      4 non-null      int64  
 2   salary   3 non-null      float64
dtypes: float64(1), int64(1), object(1)
memory usage: 224.0+ bytes
```

## Why It's Useful

**1. Quick data inspection:**
- See shape without counting manually
- Spot missing values immediately

**2. Data type checking:**
- Verify columns are correct type (numbers vs. strings)
- Identify type conversion needs

**3. Memory diagnostics:**
- Check DataFrame size
- Optimize memory if needed

**4. Missing data detection:**
- Non-null count shows where you have gaps
- Plan data cleaning strategy

## Common Use Cases

```python
# First thing to do when loading data
df = pd.read_csv('data.csv')
df.info()  # Get overview before analysis

# Check after data cleaning
df.info()  # Verify no missing values remain

# Before/after type conversion
df.info()  # See current types
df['date'] = pd.to_datetime(df['date'])
df.info()  # Verify conversion worked
```

## Related Methods

- `df.shape` - Just dimensions (rows, cols)
- `df.dtypes` - Just data types
- `df.describe()` - Statistical summary (mean, std, etc.)
- `df.head()` - First few rows of actual data

**TL;DR:** `df.info()` is your "quick health check" for a DataFrame - shows structure, types, missing values, and memory usage all at once.
