The difference is that:

- **`sum()`** returns a single scalar that is the total of all values in the column.
- **`cumsum()`** returns a Series where each element is the cumulative sum of the values up to that row.

For example:

```python
import pandas as pd

df = pd.DataFrame({'values': [1, 2, 3, 4]})
total = df['values'].sum()      # 10
cumulative = df['values'].cumsum()  # [1, 3, 6, 10]
```

So, `sum()` gives you one overall sum, while `cumsum()` gives you the running total at each step.
