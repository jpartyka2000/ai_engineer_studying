You can convert the Series into a DataFrame using the `to_frame()` method and then reset the index so that the original index becomes a column. For example:

```python
import pandas as pd

# Assuming your series is named 's'
df = s.to_frame(name='correlation')
df = df.reset_index()
df.columns = ['feature', 'correlation']
```

This produces a DataFrame with two columns: one for the feature names and one for the correlation values.
