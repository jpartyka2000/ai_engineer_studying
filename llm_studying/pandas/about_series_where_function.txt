Yes, Pandas DataFrames have a `.where()` method. It returns a DataFrame of the same shape, where each element is replaced by a specified value (default is NaN) if a given condition is False. For example:

```python
import pandas as pd

df = pd.DataFrame({
    "A": [1, 2, 3, 4],
    "B": [10, 20, 30, 40]
})

# Keep only values greater than 2, replace others with NaN
df_where = df.where(df > 2)
print(df_where)
```

This will output:

```
      A     B
0   NaN  10.0
1   NaN  20.0
2   3.0  30.0
3   4.0  40.0
```

You can also specify a different fill value using the `other` parameter:

```python
df_where = df.where(df > 2, other=-1)
print(df_where)
```

Which outputs:

```
    A   B
0  -1  -1
1  -1  -1
2   3  30
3   4  40
```

So yes, the `.where()` method is available and works similarly to NumPy’s `where` function but returns a DataFrame with the same shape.
