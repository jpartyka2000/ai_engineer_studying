You can slice the `.index` attribute just like any other sequence. For example, to grab and print the first 50 values:

```python
# For a DataFrame
print(df.index[:50])

# If you want them as a plain Python list:
print(df.index[:50].tolist())
```

And exactly the same for a Series:

```python
# For a Series
print(ser.index[:50])
# or
print(ser.index[:50].tolist())
```

If you’d rather loop through them one by one:

```python
for i, idx in enumerate(df.index):
    if i == 50:
        break
    print(idx)
```

All of these will show you the first fifty index labels.
