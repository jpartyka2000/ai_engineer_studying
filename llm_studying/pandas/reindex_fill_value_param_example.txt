Here’s a concrete example using `fill_value=0` to add new index labels and fill them with zeros:

```python
import pandas as pd

# Original DataFrame with sales for Jan–Mar
df = pd.DataFrame(
    {'Sales': [100, 200, 150]},
    index=['Jan', 'Feb', 'Mar']
)

# Suppose we want to extend this to Jan–May, 
# filling any missing months with 0
new_index = ['Jan', 'Feb', 'Mar', 'Apr', 'May']

df_reindexed = df.reindex(new_index, fill_value=0)

print(df_reindexed)
```

**Output:**

```
     Sales
Jan    100
Feb    200
Mar    150
Apr      0
May      0
```

* **What happened?**

  * Rows for **Apr** and **May** didn’t exist in the original, so they’ve been inserted.
  * Because we passed `fill_value=0`, those new rows are filled with `0` instead of the default `NaN`.
  * The original months keep their original values.
