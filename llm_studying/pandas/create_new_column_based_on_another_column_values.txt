You can create a new column based on the condition using the `numpy.where()` function or by using simple conditional logic in Pandas. Here’s how you can do it:

### Using `numpy.where()`:
```python
import pandas as pd
import numpy as np

# Example DataFrame
df = pd.DataFrame({
    'anomaly_score': [0.5, -1.2, 0.3, -0.8, 1.0, -0.5]
})

# Create the 'outlier' column based on the condition
df['outlier'] = np.where(df['anomaly_score'] < 0, True, False)

print(df)
```

### Using Conditional Logic Directly:
```python
# Create the 'outlier' column using conditional logic
df['outlier'] = df['anomaly_score'] < 0

print(df)
```

Both methods will result in a new column named `outlier` with `True` if the `anomaly_score` is less than 0 and `False` otherwise. The resulting DataFrame will look like this:

```
   anomaly_score  outlier
0            0.5    False
1           -1.2     True
2            0.3    False
3           -0.8     True
4            1.0    False
5           -0.5     True
```
