You can drop all columns with a suffix of `'_y'` in a Pandas DataFrame using the `df.filter` method combined with `df.drop`. Here’s how you can do it:

```python
# Assuming 'df' is your DataFrame
df = df.drop(df.filter(regex='_y$').columns, axis=1)
```

### Explanation
- `df.filter(regex='_y$')`: Selects all columns that end with `'_y'` using a regular expression (`$` denotes the end of the column name).
- `.columns`: Extracts the column names that match the filter.
- `df.drop(..., axis=1)`: Drops these columns from the DataFrame.

This will remove all columns ending with `'_y'` from your DataFrame.
