To **drop duplicate columns** in a **Pandas DataFrame**, you can use the following approach, which involves transposing the DataFrame, dropping duplicate rows (since columns become rows when transposed), and then transposing it back to its original form.

### Example:

```python
import pandas as pd

# Sample DataFrame with duplicate columns
data = {'col1': [1, 2, 3],
        'col2': [4, 5, 6],
        'col1': [1, 2, 3],  # Duplicate column
        'col3': [7, 8, 9]}

df = pd.DataFrame(data)

# Drop duplicate columns
df = df.T.drop_duplicates().T

# Show the result
print(df)
```

### Explanation:
1. **`df.T`**: Transposes the DataFrame, so columns become rows.
2. **`.drop_duplicates()`**: Drops duplicate rows (originally columns before transposing).
3. **`.T`**: Transposes the DataFrame back to its original orientation.

### Output:
```
   col1  col2  col3
0     1     4     7
1     2     5     8
2     3     6     9
```

This will remove any duplicate columns based on column values, keeping only the first occurrence.
