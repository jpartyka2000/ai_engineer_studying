The `key` parameter specifies a function to apply to the values *before* sorting, without actually changing the data itself.

In this case, `key=abs` means:

- Sort by the **absolute value** of the correlation
- But keep the original (positive or negative) values in the output

So correlations of `-0.9` and `0.8` would be sorted as `0.9` and `0.8` for comparison purposes, putting `-0.9` first (since `ascending=False` means largest first). The actual `-0.9` value remains in your dataframe.

This is useful for correlations because you often care about the **strength** of the relationship regardless of direction — a correlation of `-0.85` is just as strong as `0.85`, just in the opposite direction.
