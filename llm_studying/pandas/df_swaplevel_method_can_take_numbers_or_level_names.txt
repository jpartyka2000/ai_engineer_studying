Yes—you can pass the **level names** directly to `swaplevel()`, not just integers. Its signature is:

```python
DataFrame.swaplevel(i: Union[int, str], j: Union[int, str], axis: int = 0)
```

So if your MultiIndex levels are named, you can do:

```python
# Suppose df.index.names == ['Location','Day']
df_swapped = df.swaplevel('Location', 'Day')
```

This is equivalent to using their positions:

```python
df_swapped = df.swaplevel(0, 1)
```

Both calls will swap the two levels, but using names can make your code more self‑documenting—especially if you ever reorder or insert new levels.
