Yes—you can. When you do:

```python
import pandas as pd

d = {'rule1': 0.10, 'rule2': 0.35, 'rule3': 0.75}
s = pd.Series(d)
print(s)
```

you’ll get:

```
rule1    0.10
rule2    0.35
rule3    0.75
dtype: float64
```

Here the **dict keys** become the **Series index** and the dict values become the Series values. You can even control the order (and fill missing keys) with the `index=` parameter:

```python
s = pd.Series(d, index=['rule3','rule1','rule4'], dtype=float).fillna(0)
print(s)
```

```
rule3    0.75
rule1    0.10
rule4    0.00
dtype: float64
```

In this example:

* `'rule3'` and `'rule1'` pull from your dict.
* `'rule4'` wasn’t in the dict, so you can `.fillna(...)` or leave it as `NaN`.
