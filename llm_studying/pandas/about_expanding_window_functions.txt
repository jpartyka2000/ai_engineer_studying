Here’s a simple example showing how to use expanding windows in Pandas.

Expanding windows are like rolling windows, except instead of a fixed size, the window starts small and grows with each row, recalculating the statistic as more data is included.


---

Example: Expanding sum and mean

import pandas as pd

# Sample data
df = pd.DataFrame({
    'values': [10, 20, 15, 25, 30]
})

# Expanding sum
df['exp_sum'] = df['values'].expanding().sum()

# Expanding mean
df['exp_mean'] = df['values'].expanding().mean()

print(df)

Output:

values  exp_sum  exp_mean
0      10     10.0      10.0
1      20     30.0      15.0
2      15     45.0      15.0
3      25     70.0      17.5
4      30    100.0      20.0


---

Key points

expanding() starts with the first row and grows the window to include all previous rows up to the current one.

You can pass parameters like min_periods:

df['exp_mean_min2'] = df['values'].expanding(min_periods=2).mean()

This will return NaN until at least 2 observations are present in the window.

Works with many aggregation functions: .sum(), .mean(), .std(), .max(), etc.
