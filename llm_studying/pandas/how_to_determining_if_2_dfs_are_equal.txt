Here are a few ways to check DataFrame equality in pandas:

---

## 1. Use the built‑in `DataFrame.equals()`

```python
df1.equals(df2)
```

* Returns `True` iff **shape**, **index**, **columns**, and **all element values** (including `NaN` in the same positions) match exactly.
* Easy one‑liner, but it won’t tell you *where* they differ.

---

## 2. Manual checks for more control

```python
# 1) Same shape?
if df1.shape != df2.shape:
    print("Different shape:", df1.shape, df2.shape)

# 2) Same index?
if not df1.index.equals(df2.index):
    print("Different index labels")

# 3) Same columns?
if not df1.columns.equals(df2.columns):
    print("Different columns")

# 4) Same values?
#    Using element‑wise compare (this treats NaN==NaN as False, so you may need special handling)
equal_values = (df1 == df2)  
if not equal_values.all().all():
    # find where they differ
    diff_locations = equal_values.stack()[lambda s: ~s]
    print("Differences at:\n", diff_locations)
```

* You can wrap that up into a function to get fine‑grained diagnostics.

---

## 3. Use `pandas.testing.assert_frame_equal`

```python
from pandas.testing import assert_frame_equal

try:
    assert_frame_equal(df1, df2)
    print("DataFrames are equal")
except AssertionError as e:
    print("DataFrames differ:\n", e)
```

* Raises an exception if they differ, with a detailed message about which column, index, or value is mismatched.
* You can tweak tolerances for floating‑point (`rtol`, `atol`), ignore data‑type differences, skip index or column order, etc.

---

### Handling `NaN`s

* `df1.equals(df2)` and `assert_frame_equal` treat `NaN`==`NaN` as equal.
* A plain `(df1 == df2)` comparison will mark `NaN` vs `NaN` as `False`. To work around:

  ```python
  comp = df1.eq(df2) | (df1.isna() & df2.isna())
  comp.all().all()
  ```

---

### Summary

* **Quick check:** `df1.equals(df2)`
* **Detailed diff:** `assert_frame_equal(df1, df2)`
* **Custom logic:** compare `.shape`, `.index`, `.columns`, then element‑wise masks

Choose the one that best fits whether you just need a Boolean or a diagnostic report.
