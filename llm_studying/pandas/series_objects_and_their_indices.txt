Yes‚Äîevery `pd.Series` carries an `Index` object that holds its axis labels. You can access it via the `.index` attribute:

```python
import pandas as pd

s = pd.Series([10, 20, 30], index=["a", "b", "c"])
print(s.index)
# Index(['a', 'b', 'c'], dtype='object')

type(s.index)
# pandas.core.indexes.base.Index
```

Depending on how the Series was constructed, you might see more specific subclasses‚Äîe.g. `RangeIndex` for a default integer index, `Int64Index` for explicit integer indices, `DatetimeIndex` for datetime labels, etc.:

```python
s2 = pd.Series([1,2,3])
print(s2.index)        # RangeIndex(start=0, stop=3, step=1)
print(isinstance(s2.index, pd.RangeIndex))  # True
```

You can even manipulate it like any other Index:

* **Reset**: `s.reset_index(drop=True)` (moves the current index into a column if it‚Äôs on a DataFrame)
* **Reindex**: `s.reindex(new_labels)`
* **Rename**: `s.index = pd.Index(["x", "y", "z"])`

So yes‚Äîa Series has an `Index` object, exactly like a DataFrame has a `columns` (also an `Index`).
