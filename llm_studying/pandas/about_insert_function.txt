The `DataFrame.insert()` method lets you add a new column *at a specific position* in your DataFrame’s column order. Its signature is:

```python
DataFrame.insert(
    loc: int,
    column: Hashable,
    value,
    allow_duplicates: bool = False
)
```

* **`loc`**
  The integer position in the columns list where you want the new column to appear (0 = first column, 1 = second, …).
* **`column`**
  The name (label) for the new column.
* **`value`**
  Anything that can be aligned to the DataFrame: a scalar, list/array of the right length, or a Series.
* **`allow_duplicates`**
  Whether to allow inserting a column with a name that already exists (default is `False`).

---

### Example

```python
import pandas as pd

df = pd.DataFrame({
    'A': [1,2,3],
    'C': [7,8,9]
})

# Insert column 'B' between 'A' and 'C', with values [4,5,6]
df.insert(loc=1, column='B', value=[4,5,6])

print(df)
```

```
   A  B  C
0  1  4  7
1  2  5  8
2  3  6  9
```

* Without `insert`, if you did `df['B'] = [4,5,6]`, pandas would append `B` at the end.
* With `insert(1, 'B', ...)`, you place it as the second column.

That’s all it does—give you precise control over *where* in the column ordering your new field lands.
