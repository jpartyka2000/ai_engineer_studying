To get the **number of groups** in a grouped **Pandas DataFrame**, you can use the `ngroups` attribute after grouping the DataFrame using `groupby()`. This will return the total number of unique groups.

Here's an example:

```python
import pandas as pd

# Sample DataFrame
data = {'deal_id': [1, 1, 2, 2, 2, 3], 
        'amount_raised': [100, 200, 300, 300, 300, 400], 
        'col1': ['A', 'A', 'B', 'B', 'B', 'C']}
df = pd.DataFrame(data)

# Group by 'deal_id'
grouped = df.groupby('deal_id')

# Get the number of groups
num_groups = grouped.ngroups

print(f"Number of groups: {num_groups}")
```

### Output:
```
Number of groups: 3
```

### Explanation:
- **`grouped.ngroups`**: This returns the number of unique groups formed by the `groupby()` operation. In this example, there are 3 unique `deal_id` groups (1, 2, and 3). 

### Alternative Method:
You can also get the number of groups using `len(grouped)`:

```python
print(len(grouped))  # Output: 3
```

Both methods will give you the same result: the number of groups in the `groupby` object.
