Here are three common ways to pull out the row(s) where column `col` is maximized:

1. **Using `idxmax()` + `.loc`**

   ```python
   # returns a Series for the single row with the highest `col`
   max_row = df.loc[df['col'].idxmax()]
   ```

2. **Using `nlargest()`**

   ```python
   # returns a DataFrame of the top 1 row by `col`
   max_df = df.nlargest(1, 'col')
   ```

3. **Sorting + `head()`**

   ```python
   # sort descending by `col` and take the first
   max_df = df.sort_values('col', ascending=False).head(1)
   ```

All three will give you the row(s) corresponding to the maximum value in `col`. Choose the one whose output shape (Series vs DataFrame) best fits your downstream code.
