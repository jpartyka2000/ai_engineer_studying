The `.agg()` (or `.aggregate()`) method on a **DataFrame** is simply a way to apply one or more “aggregation” functions across either rows or columns—**even when you haven’t done a `groupby`**. It doesn’t magically group your data; it just takes the entire DataFrame (or a slice of it) and:

* **With the default `axis=0`**, passes each **column** (as a `Series`) into your function, and returns a **Series** whose index is the column names and whose values are whatever your function returned for each column.

  ```python
  # e.g. compute min and max of each column
  df.agg(['min', 'max'])
  ```

* **If you pass `axis=1`**, it passes each **row** (as a `Series`) into your function, and returns a **Series** whose index is the original DataFrame’s index and whose values are whatever your function returned for each row.

  ```python
  # turn each row into a tuple of its values
  df[['A','B']].agg(tuple, axis=1)
  # result is a Series of 2-tuples, one per row
  ```

You can also give it a list or dict of functions, in which case it returns a DataFrame with one column per function:

```python
# apply two different aggregations to each column
df.agg({ 'A': ['sum','mean'],  'B': ['min','max'] })
```

In short, **without a `groupby`**, `.agg()` is just a convenient way to run reduction or transformation functions across your DataFrame’s axes.
