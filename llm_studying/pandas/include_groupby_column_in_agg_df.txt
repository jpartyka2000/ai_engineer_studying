If you want to include the `category` column in the output along with the `mean()` and `count()` aggregations, you can reset the index after the `groupby()` operation. By default, the `groupby()` method uses the grouping column (`category` in this case) as the index. To include it as a regular column, use `.reset_index()`.

### Updated Example
```python
import pandas as pd

# Sample DataFrame
data = {
    'category': ['A', 'A', 'B', 'B', 'B', 'C', 'C', 'C', 'C'],
    'value': [10, 20, 30, 40, 50, 60, 70, 80, 90]
}
df = pd.DataFrame(data)

# Group by the 'category' column and aggregate using mean and count
result = df.groupby('category').agg(
    mean_value=('value', 'mean'),
    count=('value', 'count')
).reset_index()  # Reset the index to include 'category' as a column

# Display the result
print(result)
```

### Explanation
1. **Group By**:
   - We group the DataFrame by the `category` column using `df.groupby('category')`.

2. **Aggregation**:
   - The `.agg()` method is used to compute the `mean()` and `count()` of the `value` column.
   - `mean_value=('value', 'mean')`: Computes the mean of the `value` column.
   - `count=('value', 'count')`: Computes the count of the `value` column.

3. **Reset Index**:
   - `.reset_index()` is used to turn the `category` index into a regular column.

### Output
```
  category  mean_value  count
0        A        15.0      2
1        B        40.0      3
2        C        75.0      4
```

Now, the `category` column is included in the output as a regular column instead of an index.
