You can use the `.agg()` method to group by `internal_id` and apply different aggregation functions to different columns. For most columns, you can use `first` to take the first value of each group, and for the string columns, you can aggregate them into a list using `list`.

### Example Code:
```python
# Define the aggregation rules
agg_rules = {
    'column1': 'first',
    'column2': 'first',
    'column3': 'first',  # Add more columns as needed
    'string_column1': lambda x: list(x),  # Convert to list of strings
    'string_column2': lambda x: list(x)   # Convert to list of strings
}

# Group by 'internal_id' and apply the aggregation rules
grouped_df = df.groupby('internal_id').agg(agg_rules).reset_index()

print(grouped_df)
```

### Explanation:
1. **`agg_rules`:** Create a dictionary where the keys are the column names and the values are the aggregation functions:
   - **`'first'`:** For columns where you only need the first value of the group.
   - **`lambda x: list(x)`:** For string columns where you want to aggregate all values into a list.
2. **`groupby('internal_id').agg(agg_rules)`:** This groups the DataFrame by `internal_id` and applies the specified aggregation rules.

This will give you a `grouped_df` where:
- Columns like `column1`, `column2`, etc., will have the first value from each group.
- `string_column1` and `string_column2` will contain lists of all string values from each group.
