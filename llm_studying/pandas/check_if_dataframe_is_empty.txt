You can check if a Pandas DataFrame is empty using the `.empty` attribute:

```python
import pandas as pd

# Example DataFrames
df1 = pd.DataFrame()  # Empty DataFrame
df2 = pd.DataFrame({'A': [1, 2, 3]})  # Non-empty DataFrame

# Check if DataFrame is empty
print(df1.empty)  # Output: True
print(df2.empty)  # Output: False
```

This method returns `True` if the DataFrame has no rows or columns and `False` otherwise.
