In pandas you have a few ways to pull out a single cell, depending on whether you know its **label** or its **integer position**:

---

### 1. By label: `.loc` or `.at`

If your DataFrame’s index (row labels) and column names are meaningful strings (or other labels), use:

```python
# Assume df looks like:
#             A   B   C
# campaign1  10  20  30
# campaign2  11  21  31

# Get the value in row "campaign2", column "B":
value = df.loc['campaign2', 'B']
# or equivalently (but faster for a single scalar):
value = df.at['campaign2', 'B']
```

- **`.loc[row_label, col_label]`**: full feature, supports slicing and boolean masks  
- **`.at[row_label, col_label]`**: optimized for a single cell lookup  

---

### 2. By integer position: `.iloc` or `.iat`

If you just know the row‑number and column‑number (0‑based), use:

```python
# For the same df, to get the cell in the 2nd row (index 1), 3rd column (index 2):
value = df.iloc[1, 2]
# or faster for a scalar:
value = df.iat[1, 2]
```

- **`.iloc[row_idx, col_idx]`**: like `.loc` but for integer positions  
- **`.iat[row_idx, col_idx]`**: scalar‐only, very fast  

---

### 3. Via boolean condition + column

If you need to look up by a condition on another column:

```python
# Suppose you want the value of C for the row where A == 11:
value = df.loc[df['A'] == 11, 'C'].iloc[0]
```

This filters `df` down to matching rows, then picks column `'C'` and takes the first result.

---

Pick the method that best matches whether you’re indexing by label (`.loc`/`.at`) or by position (`.iloc`/`.iat`).
