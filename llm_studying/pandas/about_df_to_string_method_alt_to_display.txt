The `to_string()` method is a pandas DataFrame method that converts the entire DataFrame into a plain text string representation, which you then print to the console.

**Basic usage:**
```python
print(df.to_string())
```

**Key characteristics:**

1. **No truncation by default** - Unlike when you just `print(df)`, which truncates rows and columns if there are too many, `to_string()` shows everything by default.

2. **Returns a string** - It doesn't print anything itself; it just returns a string that you then pass to `print()`.

3. **Useful parameters:**
   - `index=False` - Hide the row index
   - `max_rows=50` - Limit number of rows shown
   - `max_cols=10` - Limit number of columns shown
   - `col_space=15` - Minimum width for each column
   - `justify='left'` - Text alignment ('left', 'right', 'center')

**Examples:**
```python
# Basic
print(df.to_string())

# Without index
print(df.to_string(index=False))

# Limit rows but show all columns
print(df.to_string(max_rows=20))

# Control column spacing
print(df.to_string(col_space=12, justify='right'))
```

**When it's useful:**
- Working in environments where `display()` has issues (like your PySpark case)
- Logging DataFrame contents to files
- When you want a simple, no-frills text output
- Debugging in plain terminal environments

**Downsides:**
- No formatting or styling (just plain text)
- No interactivity (can't sort, filter, or scroll like in a notebook widget)
- Can be hard to read for very wide DataFrames with many columns
