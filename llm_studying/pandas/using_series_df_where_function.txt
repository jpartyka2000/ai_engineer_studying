In pandas, the `.where()` method lets you keep values where a condition is `True` and replace the rest with something else.  It exists on both **Series** and **DataFrame**, and its signature is basically:

```python
Series.where(cond, other=…, inplace=False)
DataFrame.where(cond, other=…, inplace=False)
```

* `cond` can be a boolean mask (same shape as the object), or a callable that returns one.
* `other` is what to put in places where `cond` is `False`.  By default it’s `NaN`.
* If `inplace=True`, it modifies the object rather than returning a new one.

---

### 1) Series example

```python
import pandas as pd

s = pd.Series([1, 5, 10, 20, 50])

# Keep only values < 20, set the rest to -1
s2 = s.where(s < 20, other=-1)
print(s2)
# 0     1
# 1     5
# 2    10
# 3    20   <- note: 20 is NOT < 20, so it becomes -1
# 4    -1
# dtype: int64
```

---

### 2) DataFrame example

```python
import pandas as pd

df = pd.DataFrame({
    'age':    [15, 25, 35, 45, 55],
    'income': [30, 50, 70, 90, 110]
})

# Zero out incomes for anyone under 18
df['income_adj'] = df['income'].where(df['age'] >= 18, other=0)
print(df)
#    age  income  income_adj
# 0   15      30           0
# 1   25      50          50
# 2   35      70          70
# 3   45      90          90
# 4   55     110         110
```

You can also apply it across the whole frame:

```python
# Keep only the “valid” cells, set the rest to NaN
mask = df > 50
df2 = df.where(mask)  
print(df2)
#     age  income
# 0   NaN     NaN
# 1   NaN     NaN
# 2   35.0   70.0
# 3   45.0   90.0
# 4   55.0  110.0
```

---

### 3) Using a callable

You can pass a function that takes the entire frame/series:

```python
# Replace negative values with zero
df3 = df.where(lambda x: x >= 0, other=0)
```

---

### 4) vs. `np.where`

If you want to pick between two different columns or expressions, you can also use NumPy’s `where`:

```python
import numpy as np

# If age>=50 then flag=1 else 0
df['senior_flag'] = np.where(df['age'] >= 50, 1, 0)
```

---

**In short**:

* Use `series.where(cond, other)` to replace only the “False” entries with `other`.
* Use `df.where(cond, other)` to do it over entire frames.
* For a classic “if … else …” elementwise between two choices, `np.where(cond, val_if_true, val_if_false)` can sometimes read more clearly.
