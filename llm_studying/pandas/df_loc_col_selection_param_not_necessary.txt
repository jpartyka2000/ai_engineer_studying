No—you don’t have to.  The signature of the `.loc` indexer is effectively:

```python
df.loc[row_selector, column_selector]
```

* If you only provide the **row\_selector** (inside the brackets), pandas will return **all** columns:

  ```python
  # all columns for rows matching the index key/slice/mask
  df.loc['2025-01-02']
  df.loc[(slice(None), '2025-01-02')]   # MultiIndex example
  df.loc[mask]
  ```

* If you want to pick specific columns, pass them as the second argument:

  ```python
  # only the 'revenue' column for the matching rows
  df.loc['2025-01-02', 'revenue']

  # multiple columns:
  df.loc['2025-01-02', ['revenue','units_sold']]
  ```

Leaving out the second argument is equivalent to saying “`:`” for the columns axis—i.e. “give me everything.”
