Great question — here's the clear breakdown:

df.loc[...] returns:

What you're selecting What you get

A single row Series
Multiple rows DataFrame
A single cell Scalar value (not Series/DF)
Multiple rows + one column Series
Multiple rows + multiple cols DataFrame



---

Examples:

Assume this DataFrame:

import pandas as pd

df = pd.DataFrame({
    'Region': ['North', 'North', 'South'],
    'Store': ['A', 'B', 'A'],
    'Sales': [1000, 1500, 2000]
}).set_index(['Region', 'Store'])

🔹 Single row match:

df.loc[('South', 'A')]

✅ Returns a Series:

Sales    2000
Name: (South, A), dtype: int64

🔹 Multiple rows match:

df.loc['North']

✅ Returns a DataFrame:

Sales
Store      
A        1000
B        1500

🔹 Single cell (row+column):

df.loc[('North', 'A'), 'Sales']

✅ Returns a scalar:

1000


---

So your rule of thumb is mostly right:

> ✅ Returns a Series if exactly one row (or one column) matches, but a DataFrame if multiple rows or multiple columns are selected.
