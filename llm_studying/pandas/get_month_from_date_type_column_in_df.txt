You have two main options:

---

### 1. Convert to pandas’ datetime64 dtype and then use `.dt.month`

```python
import pandas as pd

# suppose df['date'] holds Python date objects (dtype=object)
df['date'] = pd.to_datetime(df['date'])    # now dtype=datetime64[ns]
df['month'] = df['date'].dt.month
```

After that, `df['month']` will be integers 1–12.

---

### 2. Pull `.month` directly off the Python `date` objects

If you don’t want to change dtypes, just apply a function:

```python
df['month'] = df['date'].apply(lambda d: d.month)
# or equivalently
df['month'] = df['date'].map(lambda d: d.month)
```

Each element `d` is a `datetime.date`, so `d.month` returns its month.

---

Either approach will give you a new integer column 1–12 representing the month.
