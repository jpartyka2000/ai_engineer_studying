The `.isin()` method is provided on **Pandas** objects:

* **`pd.Series`**

  ```python
  my_series.isin([…])
  ```
* **`pd.Index`** (including `DataFrame.columns` or the index itself)

  ```python
  df.index.isin([…])
  ```
* **`pd.DataFrame`** (returns a boolean DataFrame of the same shape)

  ```python
  df.isin({ 'colA': [1,2], 'colB': ['x','y'] })
  ```

It’s *not* a method on plain Python tuples, lists, or NumPy arrays (for those you’d use the top-level function `np.isin(arr, values)`).
