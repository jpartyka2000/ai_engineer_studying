You can just use the built-in `min`/`max` (or `agg`) on the datetime column. For example, assuming your dates are in `df['date_col']`:

```python
# most recent (max) and earliest (min) in one line
earliest, latest = df['date_col'].min(), df['date_col'].max()
```

Or, if you’d like a single call that returns both:

```python
df['date_col'].agg(['min','max'])
```

Both assume `df['date_col']` is already `datetime64[ns]`; if it isn’t, simply do:

```python
df['date_col'] = pd.to_datetime(df['date_col'])
```
