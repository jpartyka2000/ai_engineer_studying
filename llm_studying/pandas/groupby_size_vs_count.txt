---

🆚 size() vs count() in Pandas

Both are used after a groupby(), but they behave differently when missing (NaN) values are involved.


---

📌 .size()

Counts the total number of rows in each group, including NaNs.

Returns a Series or DataFrame with the size of each group.

Does not skip null values.


Example:

df.groupby('Group').size()


---

📌 .count()

Counts the number of non-null values in each group per column.

Skips NaN values.

Returns a Series or DataFrame depending on what you call it on.


Example:

df.groupby('Group')['Value'].count()


---

✅ Quick Comparison Table

Function Includes NaNs? Aggregates over Output shape

.size() ✅ Yes Rows Series
.count() ❌ No Non-null values in specified column(s) Series or DataFrame



---

🔍 Example DataFrame

df = pd.DataFrame({
    'Group': ['A', 'A', 'B', 'B', 'B'],
    'Value': [10, None, 5, None, 15]
})

df.groupby('Group').size()
# A    2
# B    3

df.groupby('Group')['Value'].count()
# A    1
# B    2
