Yes—when you do:

```python
df_selected = df.loc[boolean_mask]
```

pandas treats that as shorthand for:

```python
df_selected = df.loc[boolean_mask, :]
```

In other words:

* **`boolean_mask`** is a length-N array/Series of `True`/`False` indexed like `df.index`.
* **No second argument** means “take **all** columns.”

So you only need to supply the mask, and `.loc` will return exactly the rows where the mask is `True`, with every column preserved.
