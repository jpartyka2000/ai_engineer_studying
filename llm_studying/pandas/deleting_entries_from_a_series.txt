You can remove an entry by its label using either `drop()`, `pop()`, or `del`. Here are the options:

```python
import pandas as pd

# sample series
s = pd.Series([10, 20, 30], index=['a','b','c'])
print(s)
# a    10
# b    20
# c    30
# dtype: int64

# 1) Using .drop() (returns a new Series by default)
s2 = s.drop('b')
print(s2)
# a    10
# c    30
# dtype: int64

# if you really want to modify in‐place:
s.drop('b', inplace=True)


# 2) Using pop()  (removes and returns the value)
val = s.pop('b')   # val == 20
# now s no longer has index 'b'


# 3) Using del
del s['b']
# same effect as pop() but doesn’t return the value
```

If you need to drop multiple labels at once:

```python
s = s.drop(['a','c'])
```

Or, if you don’t know the label but want to filter by position or a condition, you can also do:

```python
# e.g., drop whatever is at position 1
s = s.drop(s.index[1])

# or drop by Boolean mask
s = s[s != 20]
```

Any of these will delete that one index→value pair from your Series.
