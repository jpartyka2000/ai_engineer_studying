You can do it in one go using `DataFrame.rename()` with a dict of old→new names. For example:

```python
df = df.rename(columns={
    'old_col1': 'new_col1',
    'old_col2': 'new_col2',
    'old_col3': 'new_col3'
})
```

Or in-place:

```python
df.rename(columns={
    'old_col1': 'new_col1',
    'old_col2': 'new_col2',
    'old_col3': 'new_col3'
}, inplace=True)
```

If you need to apply a pattern—say prefix every column—you can also do:

```python
df = df.rename(columns=lambda c: f"new_{c}")
```

Both approaches let you rename as many columns as you like in a single statement.
