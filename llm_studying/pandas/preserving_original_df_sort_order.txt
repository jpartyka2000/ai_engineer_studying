Pandas doesn’t have a built-in "unsort" function. Once you sort your DataFrame, the original row order is lost unless you saved it beforehand. A common workaround is to add an index column (or preserve the original index) before sorting, so you can later restore the original order. For example:

```python
# Save original order in a new column
df['original_order'] = range(len(df))

# Now sort your DataFrame
df = df.sort_values(by='duration')

# Later, unsort by sorting on the original_order column
df = df.sort_values(by='original_order').drop(columns=['original_order'])
```

If you didn’t save the original order before sorting, unfortunately, you won’t be able to recover the previous order.
