### **Enhanced Filtering Function**
```python
import pandas as pd
import operator

def filter_dataframe(df: pd.DataFrame, column: str, value, op: str = "=="):
    """Filters a dataframe based on a column, value, and comparison operator."""
    
    # Define a mapping of string operators to actual operator functions
    ops = {
        "==": operator.eq,
        "!=": operator.ne,
        ">": operator.gt,
        ">=": operator.ge,
        "<": operator.lt,
        "<=": operator.le,
    }
    
    if column not in df.columns:
        raise ValueError(f"Column '{column}' not found in DataFrame.")

    if op not in ops:
        raise ValueError(f"Unsupported operator '{op}'. Choose from {list(ops.keys())}.")
    
    return df[ops[op](df[column], value)]
```

### **Usage Example**
```python
data = {
    'Name': ['Alice', 'Bob', 'Charlie', 'David'],
    'Age': [25, 30, 35, 40],
    'City': ['NY', 'SF', 'LA', 'NY']
}

df = pd.DataFrame(data)

# Filter where Age > 30
filtered_df = filter_dataframe(df, 'Age', 30, '>')
print(filtered_df)

# Filter where City is not 'NY'
filtered_df = filter_dataframe(df, 'City', 'NY', '!=')
print(filtered_df)
```

### **Why is This Elegant?**
- **Flexible**: Supports multiple operators without complex if-else conditions.
- **Extensible**: Easy to add more operators if needed.
- **Readable & Pythonic**: Uses `operator` module for clean, efficient mapping.
