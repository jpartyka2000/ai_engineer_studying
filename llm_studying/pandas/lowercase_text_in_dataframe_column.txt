To convert all the text in the `memo` column to lowercase in a Pandas DataFrame, you can use the `.str.lower()` method:

### Example Code:
```python
# Convert all text in the 'memo' column to lowercase
df['memo'] = df['memo'].str.lower()
```

### Explanation:
- **`df['memo'].str.lower()`:** The `.str.lower()` function applies the `.lower()` method to each string in the `memo` column, converting all text to lowercase.
- **Assignment:** The result is then assigned back to the `memo` column, updating the DataFrame. 

This method efficiently converts all string data in the column to lowercase.
