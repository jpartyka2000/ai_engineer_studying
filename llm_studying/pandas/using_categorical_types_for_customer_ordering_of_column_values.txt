You can convert your column into a pandas Categorical with your desired order. For example, suppose you have a column named `'bucket'` and you want the bins ordered as follows:

```python
custom_order = ["0", "1-5", "6-10", "11-20", "21-30", ">30"]
df["bucket"] = pd.Categorical(df["bucket"], categories=custom_order, ordered=True)
```

Then, when you plot (for example, using Seaborn), the x-axis will follow that custom order:

```python
import seaborn as sns
import matplotlib.pyplot as plt

# Assume df has a column 'bucket' (ordered as set above) and a numerical column 'CLV'
plt.figure(figsize=(10,6))
ax = sns.barplot(data=df, x='bucket', y='CLV', ci=None)
plt.xlabel("Bucket")
plt.ylabel("Average CLV")
plt.title("Average CLV by Bucket")
plt.show()
```

Using `pd.Categorical` in this way forces the plotting function to respect the order you've provided in `custom_order`.
