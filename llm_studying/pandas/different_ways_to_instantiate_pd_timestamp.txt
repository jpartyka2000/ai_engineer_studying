Here are some of the most common and handy ways to create a `pd.Timestamp`:

```python
import pandas as pd
import datetime

# 1) From a date‐string (ISO, almost anything you’d pass to `to_datetime`)
ts1 = pd.Timestamp('2025-03-01')
ts2 = pd.Timestamp('2025-03-01 14:30:00')
ts3 = pd.Timestamp('2025/03/01 14:30')        # different separators

# 2) By specifying components directly
ts4 = pd.Timestamp(year=2025, month=3, day=1)
ts5 = pd.Timestamp(2025, 3, 1, 14, 30, 0)     # (year, month, day, hour, min, sec)

# 3) From a Python `datetime` or `date`
dt = datetime.datetime(2025, 3, 1, 14, 30)
ts6 = pd.Timestamp(dt)
d = datetime.date(2025, 3, 1)
ts7 = pd.Timestamp(d)

# 4) From a Unix/epoch integer or float
#    (value=seconds since 1970-01-01; you can also use 'ms', 'us', etc.)
ts8 = pd.Timestamp(1677669000, unit='s')     # same as '2023-02-28 22:30:00'
ts9 = pd.Timestamp(1677669000000, unit='ms')  # milliseconds

# 5) “Now” / current time
ts_now1 = pd.Timestamp.now()                  # current local datetime
ts_now2 = pd.Timestamp.today()                # alias for now()
ts_utc  = pd.Timestamp.utcnow()               # current UTC datetime

# 6) With time zones
ts10 = pd.Timestamp('2025-03-01 14:30', tz='UTC')
ts11 = pd.Timestamp('2025-03-01 09:30', tz='America/Denver')

# 7) From a NumPy `datetime64`
import numpy as np
arr = np.array('2025-03-01T14:30:00', dtype='datetime64[ns]')
ts12 = pd.Timestamp(arr)

# 8) Via `pd.to_datetime()` shorthand (useful in DataFrame pipelines)
ts13 = pd.to_datetime('2025-03-01')           # returns a Timestamp
```

–––  
**Which one to use?**  
- If you have a **string**, just do `pd.Timestamp(your_string)`.  
- If you’re generating programmatically, passing components (`year=…, month=…`) is crystal-clear.  
- For “right now,” use `pd.Timestamp.now()`.  
- To convert arrays or columns, `pd.to_datetime(...)` will give you a `DatetimeIndex` or `Series[datetime64]`, where each element is a `Timestamp`.
