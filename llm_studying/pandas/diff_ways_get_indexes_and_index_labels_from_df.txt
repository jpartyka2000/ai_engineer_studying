You can pull out any “index” you like—from the whole row‐index to a single level of a MultiIndex, or even a single label—simply by accessing the right attribute or slicing. Here are the most common patterns:

---

### 1. Grab the entire row Index

```python
# this gives you the Index (or MultiIndex) object for all rows
idx = df.index
```

* `idx` will be a `pd.Index` (or `pd.MultiIndex`) that you can inspect, slice, or reuse.

---

### 2. Get one level of a MultiIndex

If your DataFrame has a MultiIndex and you only want, say, the second level:

```python
# by level name
level_vals = df.index.get_level_values('level_name')

# or by position (0-based)
level0 = df.index.get_level_values(0)
```

* `level_vals` is a plain `Index` of length `len(df)` containing just that one level’s labels, aligned to each row.

---

### 3. Peek at all the unique labels of a given level

```python
# the “universe” of labels in level 0:
unique_level0 = df.index.levels[0]
```

* `df.index.levels` is a list of `Index` objects, one per level.
* Use these when you want the set of distinct keys, not one per-row.

---

### 4. Extract a single row’s index label

```python
# e.g. the very first row’s label
first_label = df.index[0]
```

* If it’s a simple Index, `first_label` is a scalar (e.g. a Timestamp or string).
* If it’s a MultiIndex, `first_label` is a tuple of (`level1_value`, `level2_value`, …).

---

### 5. Convert an Index into a Series or list

If you need it as a true `Series` (e.g. to join back to another DataFrame):

```python
idx_ser = pd.Series(df.index, name='my_index')
```

Or just a plain Python list:

```python
idx_list = df.index.tolist()
```

---

All of these return standalone objects you can store, inspect, or manipulate—and none of them modify your original DataFrame’s index.
