Yes — if you select all columns in df.loc[] (e.g., with :), the result will always be a DataFrame, even if only one row matches.

✅ Example:

import pandas as pd

df = pd.DataFrame({
    'City': ['Houston', 'Austin', 'Dallas'],
    'Population': [2300000, 1000000, 1300000]
})
df = df.set_index('City')

# Only one row matches, but all columns are selected
result = df.loc[['Houston'], :]  # <- note the double brackets

print(type(result))  # <class 'pandas.core.frame.DataFrame'>


---

⚠️ Caveat: Single brackets vs double brackets

If you use single brackets and select a scalar index label, you get:

result = df.loc['Houston', :]  # returns a Series

But if you use double brackets (even for a single label), you get:

result = df.loc[['Houston'], :]  # returns a DataFrame


---

Rule of thumb:

df.loc['label', :] → Series if 1 row

df.loc[['label'], :] → DataFrame, even if 1 row

df.loc[:, 'col'] → Series if 1 column

df.loc[:, ['col']] → DataFrame, even if 1 column
