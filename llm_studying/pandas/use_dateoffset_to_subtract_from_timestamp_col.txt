You can use `pd.DateOffset` to subtract years from a timestamp column in Pandas.

### Example:

Suppose your DataFrame looks like this:

```python
df['timestamp']
# Example: 2025-06-22 00:00:00
```

You can create two new columns like this:

```python
df['timestamp_minus_1yr'] = df['timestamp'] - pd.DateOffset(years=1)
df['timestamp_minus_2yr'] = df['timestamp'] - pd.DateOffset(years=2)
```

### Result:

If the original timestamp is `2025-06-22`, the new columns will be:

* `timestamp_minus_1yr`: `2024-06-22`
* `timestamp_minus_2yr`: `2023-06-22`

### Notes:

* This approach preserves the day and month when possible.
* `pd.DateOffset` is smart with leap years and end-of-month issues (e.g., Feb 29 becomes Feb 28 when subtracting to a non-leap year).
