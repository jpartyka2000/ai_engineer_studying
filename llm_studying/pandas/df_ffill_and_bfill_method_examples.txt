Here are plain‑text examples—no special canvas—showing both DataFrame and Series forward‑ and backward‑fill:

```python
import pandas as pd

# Example DataFrame with NaNs
df = pd.DataFrame({
    'A': [1, None, 3, None, 5],
    'B': [None, 2, None, 4, None]
})
print("Original df:")
print(df)
print()

print("df.ffill():")
print(df.ffill())
print()

print("df.bfill():")
print(df.bfill())
```

**Output:**

```
Original df:
     A    B
0  1.0  NaN
1  NaN  2.0
2  3.0  NaN
3  NaN  4.0
4  5.0  NaN

df.ffill():
     A    B
0  1.0  NaN
1  1.0  2.0
2  3.0  2.0
3  3.0  4.0
4  5.0  4.0

df.bfill():
     A    B
0  1.0  2.0
1  3.0  2.0
2  3.0  4.0
3  5.0  4.0
4  5.0  NaN
```

---

```python
# Example Series with NaNs
s = pd.Series([1, None, 3, None, 5], name='S')
print("Original s:")
print(s)
print()

print("s.ffill():")
print(s.ffill())
print()

print("s.bfill():")
print(s.bfill())
```

**Output:**

```
Original s:
0    1.0
1    NaN
2    3.0
3    NaN
4    5.0
Name: S, dtype: float64

s.ffill():
0    1.0
1    1.0
2    3.0
3    3.0
4    5.0
Name: S, dtype: float64

s.bfill():
0    1.0
1    3.0
2    3.0
3    5.0
4    5.0
Name: S, dtype: float64
```

* **`.ffill()`** (“forward fill”) propagates the last known non‑null value forward.
* **`.bfill()`** (“backward fill”) propagates the next known non‑null value backward. Both DataFrames and Series support these methods.
