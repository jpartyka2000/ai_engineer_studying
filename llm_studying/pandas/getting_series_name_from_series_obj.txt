Every `pandas.Series` carries its name (i.e. the “column name” when it came out of a DataFrame) in the `.name` attribute. So, for example:

```python
import pandas as pd

df = pd.DataFrame({
    "foo": [1,2,3],
    "bar": [4,5,6]
})

s = df["foo"]
print(s.name)   # prints "foo"
```

If you ever end up with an unnamed series (i.e. `s.name is None`), you can also forcefully set it via:

```python
s.name = "my_new_name"
```

Or, if you’ve got a one‑column DataFrame and want to grab its column name from the series form:

```python
s = df[["foo"]].iloc[:,0]
# still works
print(s.name)  # "foo"
```

In short: **`series.name`** is exactly how you retrieve (or set) a Series’ name.
