You can pass a **list** of values to `to_replace` and a single **scalar** to `value`. pandas will look for any of the listed values and swap them out for your new one. For example:

```python
import numpy as np
import pandas as pd

df = pd.DataFrame({
    'A': [1, 2, np.inf, -np.inf, 5],
    'B': ['foo', 'bar', 'baz', 'foo', 'qux']
})

# Replace both +inf and -inf with 0, all in one go:
df_clean = df.replace(to_replace=[np.inf, -np.inf], value=0)

print(df_clean)
```

Under the hood, this is equivalent to doing two separate replaces, but it’s vectorized and more concise:

```python
df_clean = df.replace([np.inf, -np.inf], 0)
```

If you want to do it **in place** (modify `df` directly), add `inplace=True`:

```python
df.replace([np.inf, -np.inf], 0, inplace=True)
```

That’s all there is to it!
