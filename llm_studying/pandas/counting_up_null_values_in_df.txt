**Quick Pandas Tip: Counting Missing Values**

That line of code has a small bug - `df.isna().sum()` returns a Series showing missing values **per column**, not the total count.

```python
# This shows missing per column (Series)
print(f"Missing values: {df.isna().sum()}")
# Output: 
# col1    2
# col2    0  
# col3    5

# To get TOTAL missing across entire DataFrame, sum twice:
print(f"Total missing values: {df.isna().sum().sum()}")
# Output: 7
```

**Other useful missing value operations:**
```python
# Percentage of missing values
print(f"Missing %: {df.isna().sum().sum() / df.size * 100:.1f}%")

# Check if any missing values exist
print(f"Has missing values: {df.isna().any().any()}")

# Count missing per row
df.isna().sum(axis=1)
```

The double `.sum()` pattern is common in pandas - first aggregates columns, second gives the grand total.
