**`Series.rank()` in pandas** assigns ranks to data points in a Series, letting you see each value’s relative position. By default, the smallest value gets rank 1, ties are assigned the **average** of their positions, and the result is a floating-point Series.

---

### Common parameters

* **`method`**
  How to handle ties (equal values):

  * `"average"` (default): assign the mean of the tied ranks.
  * `"min"`: assign the minimum rank in the group.
  * `"max"`: assign the maximum rank in the group.
  * `"first"`: rank in order they appear.
  * `"dense"`: like “min” but ranks increase by 1 between groups (no gaps).

* **`ascending`**

  * `True` (default): smallest value → rank 1.
  * `False`: largest value → rank 1 (useful for scoring).

* **`na_option`**
  How to rank `NaN` values:

  * `"keep"` (default): leave as `NaN`.
  * `"top"`: treat `NaN` as smaller than any number (rank first).
  * `"bottom"`: treat `NaN` as larger than any number (rank last).

* **`pct`**

  * `False` (default): returns integer or float ranks.
  * `True`: returns **percentile** rank (each rank divided by the number of non-NA values).

---

**Example**

```python
s = pd.Series([50, 20, 20, 80])
s.rank(method='dense', ascending=False)
# 0    2.0   (50 is 2nd highest)
# 1    3.0   (20 is tied for 3rd)
# 2    3.0
# 3    1.0   (80 is highest)
```
