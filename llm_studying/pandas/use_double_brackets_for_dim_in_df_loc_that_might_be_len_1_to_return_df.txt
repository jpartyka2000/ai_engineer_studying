Yes—that’s exactly the trick. In pandas:

* **Single bracket** on rows:

  ```python
  df.loc['2025-01-01']        # returns a Series if ‘2025-01-01’ is a single label
  ```
* **Double brackets** on rows:

  ```python
  df.loc[['2025-01-01']]      # returns a DataFrame with one row
  ```

Likewise for columns:

* **Single bracket** on columns:

  ```python
  df.loc[:, 'revenue']        # returns a Series
  ```
* **Double brackets** on columns:

  ```python
  df.loc[:, ['revenue']]      # returns a DataFrame with one column
  ```

If you want to ensure your overall `.loc` selection is always a DataFrame—regardless of whether you’re picking one label or many—just wrap that label in a list (i.e. use double brackets for that axis). For example, a single‐cell DataFrame:

```python
df2 = df.loc[['2025-01-01'], ['revenue']]
```

will always give you a DataFrame, even though you’ve selected exactly one row and one column.
