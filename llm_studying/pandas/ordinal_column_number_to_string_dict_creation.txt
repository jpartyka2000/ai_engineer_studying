Got it! If you want a dictionary that directly maps the encoded integer values (from `cat.codes`) back to the original text strings, you can use this approach:

```python
import pandas as pd

# Example data
data = {'text_column': ['apple', 'banana', 'apple', 'cherry']}
df = pd.DataFrame(data)

# Ordinal encoding
df['encoded_column'] = df['text_column'].astype('category').cat.codes

# Get the direct mapping of cat codes to text strings
mapping = dict(zip(df['encoded_column'], df['text_column']))
print(mapping)
```

### Explanation
This code uses `zip` to create a dictionary that maps each unique encoded value (from `encoded_column`) back to the original text string.

The output will look like this:

```python
{0: 'apple', 1: 'banana', 2: 'cherry'}
```

This way, `mapping` provides a direct mapping from the encoded numbers to the original text strings, as needed.
