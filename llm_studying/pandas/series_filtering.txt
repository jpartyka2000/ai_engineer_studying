To get all values in a **Pandas Series** that are equal to `True`, you can use boolean indexing. Here's how you can do it:

### Example:

```python
import pandas as pd

# Sample Series
data = pd.Series([True, False, True, False, True, False])

# Get all values that are equal to True
true_values = data[data == True]

# Show the result
print(true_values)
```

### Output:
```
0    True
2    True
4    True
dtype: bool
```

### Explanation:
- **`data == True`**: This creates a boolean mask where only the `True` values are `True` and others are `False`.
- **`data[data == True]`**: This applies the boolean mask to filter the Series and keep only the values that are `True`.

This will return a new Series containing only the `True` values from the original Series.
