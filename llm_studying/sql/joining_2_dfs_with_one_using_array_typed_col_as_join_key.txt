Here's an example using PySpark:

```python
from pyspark.sql.functions import array_contains

# Assume df_my_data and df_orig_data are your Spark DataFrames.
# We join them where the campaign_set array contains the value from first_campaign.
df_joined = df_my_data.join(
    df_orig_data,
    array_contains(df_orig_data.campaign_set, df_my_data.first_campaign),
    how="inner"
)

```

### Explanation

  This Spark SQL function returns `True` if the specified value is present in the array. In this join condition, it checks that `df_orig_data.campaign_set` contains the value from `df_my_data.first_campaign` for each row.

- **Join Type:**  
  We use an inner join (`how="inner"`) so that only rows where the join condition is met are returned.

This approach allows you to join on a condition where a scalar value (`first_campaign`) is matched if it exists in an array column (`campaign_set`) from another DataFrame.
