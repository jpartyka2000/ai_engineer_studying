To get the length of a string without leading or trailing whitespace, use `LENGTH()` or `LEN()` with `TRIM()`:

**Most databases (PostgreSQL, MySQL, SQLite, Oracle):**
```sql
SELECT LENGTH(TRIM(column_name)) AS trimmed_length
FROM your_table;
```

**SQL Server and Azure SQL:**
```sql
SELECT LEN(TRIM(column_name)) AS trimmed_length
FROM your_table;
```

**Example comparing original vs trimmed length:**
```sql
SELECT 
    column_name,
    LENGTH(column_name) AS original_length,
    LENGTH(TRIM(column_name)) AS trimmed_length,
    LENGTH(column_name) - LENGTH(TRIM(column_name)) AS whitespace_count
FROM your_table;
```

**Note:** In older versions of SQL Server (prior to SQL Server 2017), `TRIM()` doesn't exist. Use `LTRIM()` and `RTRIM()` instead:
```sql
SELECT LEN(LTRIM(RTRIM(column_name))) AS trimmed_length
FROM your_table;
```

The `TRIM()` function removes spaces from both the beginning and end of the string before calculating its length.
