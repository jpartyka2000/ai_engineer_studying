```sql
SELECT 
    CASE
        WHEN deal_stage_closed_won_stage_flag_final_rate IS NULL THEN 2
        WHEN deal_stage_closed_won_stage_flag_final_rate <= 0.02 THEN 1
        WHEN deal_stage_closed_won_stage_flag_final_rate <= 0.05 THEN 2
        WHEN deal_stage_closed_won_stage_flag_final_rate <= 1.0 THEN 3
        ELSE 4
    END AS lead_quality_bucket,
    COUNT(*) AS count
FROM pathahead.fourthrev_ecue_dev.lead_probability_results_by_row
GROUP BY 1
```

The `GROUP BY 1` groups by the first column (the CASE expression). You can't reference the alias `lead_quality_bucket` in the GROUP BY in most SQL dialects, so you either repeat the whole CASE expression or use the column position.
