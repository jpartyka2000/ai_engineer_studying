`OVER ()` defines the "window" — the set of rows the function should operate across. When the parentheses are empty, it means "no partitioning, no ordering, just use everything." So the window is the entire result set.

You can narrow the window by putting things inside the parentheses. For example, `OVER (PARTITION BY region)` would mean "only look at rows with the same region value." So if you had `SUM(COUNT(*)) OVER (PARTITION BY region)`, each row would get the total count for its region, not the grand total.

Think of it like: `OVER ()` is saying "over all rows" and `OVER (PARTITION BY x)` is saying "over all rows that share the same value of x." Empty parentheses = no restrictions = everything.
