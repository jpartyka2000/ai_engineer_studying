### **When to Use Backticks in Databricks SQL**
✅ **You can always use backticks, even for simple column names:**  

- Reserved SQL keywords (`"SELECT"`, `"FROM"`, etc.)  

Example:
```sql
SELECT `website:click_event_properties_plan_name_count`
FROM event_logs;
```
