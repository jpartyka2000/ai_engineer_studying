You can achieve this in **Databricks SQL** by:  
1. **Casting the `ARRAY<TIMESTAMP>` column to STRING**  
2. **Checking if all values are `"[]"` (an empty array in string format)**  

Here’s how you can write the SQL query:  

```sql
SELECT 
    column_name, 
    CAST(column_name AS STRING) AS column_as_string
FROM table_name
WHERE CAST(column_name AS STRING) = '[]';
```

### **If You Want to Check for All Entries Being `[]`**
To verify that **all rows** in the table have an empty array (`[]`), use `COUNT(*)`:

```sql
SELECT 
    COUNT(*) = SUM(CASE WHEN CAST(column_name AS STRING) = '[]' THEN 1 ELSE 0 END) AS all_empty
FROM table_name;
```

- This returns **TRUE (1)** if **all values** are `"[]"`, otherwise **FALSE (0)**.
