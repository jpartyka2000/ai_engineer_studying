Yes—you can get the unique combinations of more than one column by using `DISTINCT` on the column list. For example, in Spark SQL:

```sql
SELECT DISTINCT col_a, col_b
FROM   my_table;
```

That will return every `(col_a, col_b)` pair exactly once.

If you’re using the DataFrame API:

```python
# assuming `df` is your DataFrame:
df.select("col_a", "col_b").distinct()
```

Or equivalently:

```python
df.dropDuplicates(["col_a", "col_b"])
```

All three approaches will give you the distinct tuples of `(col_a, col_b)`.
