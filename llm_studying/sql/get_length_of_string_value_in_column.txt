In SQL, you can get the length of a string using the `LENGTH()` or `LEN()` function, depending on your database system:

**Most databases (PostgreSQL, MySQL, SQLite, Oracle):**
```sql
SELECT LENGTH(column_name) AS string_length
FROM your_table;
```

**SQL Server and Azure SQL:**
```sql
SELECT LEN(column_name) AS string_length
FROM your_table;
```

**Example with filtering:**
```sql
SELECT column_name, LENGTH(column_name) AS string_length
FROM your_table
WHERE LENGTH(column_name) > 10;
```

**Note:** There's a subtle difference between `LENGTH()` and `LEN()`:
- `LENGTH()` counts all characters including trailing spaces
- `LEN()` (SQL Server) excludes trailing spaces

If you need to count characters including trailing spaces in SQL Server, use:
```sql
SELECT DATALENGTH(column_name) AS byte_length
FROM your_table;
```

Or to get character length preserving spaces:
```sql
SELECT LEN(column_name + 'x') - 1 AS string_length
FROM your_table;
```
