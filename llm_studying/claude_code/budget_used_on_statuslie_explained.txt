The budget displayed in Claude Code refers to your context window usage for the current session - not a monthly allowance or project estimate.

  What it shows:
  - How many tokens your current conversation has consumed
  - Percentage of the context window used (e.g., 50,000 / 200,000 tokens = 25%)

  What it's NOT:
  - A monthly token quota
  - An estimate of tokens this project will need

  If you're on a Claude Pro subscription or API plan, there are separate rate limits and quotas, but the "budget" indicator in the Claude Code interface specifically tracks how much of the current conversation's context window you've used. When it fills up, Claude Code will automatically summarize the conversation to free up space.
