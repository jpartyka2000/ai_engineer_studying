The `claude --resume` command is one of the most useful features of Claude Code - it lets you pick up previous conversations exactly where you left off!

## How It Works

When you start Claude Code with `--resume`, it shows you an **interactive conversation picker** where you can browse and select from your recent sessions:

```bash
claude --resume
```

This displays a menu with:
- Session timestamps
- First message/topic of each conversation
- Session IDs
- Project context

Navigate with **arrow keys** and press **Enter** to resume the conversation you want.

## Quick Resume Options

**Resume most recent conversation automatically:**
```bash
claude --continue
# or shorthand
claude -c
```

**Resume a specific session by ID:**
```bash
claude --resume <session-id>
# or shorthand
claude -r <session-id>
```

**Continue with a new prompt:**
```bash
claude --continue --print "Show me our progress"
```

## What Gets Restored

When you resume a conversation, you get:
- ✅ **Full conversation history** - All previous messages and context
- ✅ **Code changes** - All file modifications and edits
- ✅ **Checkpoints** - Previous save points you can rewind to
- ✅ **Project context** - Working directory and settings
- ✅ **Tool permissions** - Previously granted approvals

## Where Conversations Are Stored

Claude Code automatically saves conversations to:
```
~/.claude/history.jsonl          # Index of all conversations
~/.claude/projects/              # Full conversation data by project
```

## Inside an Active Session

You can also switch between conversations **without exiting** Claude Code:

```bash
/resume    # Opens the conversation picker mid-session
```

## Pro Tips

### 1. View Your Conversation History

Create a custom command to see all your past conversations:

```bash
cat > ~/.claude/commands/history.md << 'EOF'
Please read my global conversation history from ~/.claude/history.jsonl 
and present it in an easy-to-scan format showing:
- Date/time
- Project name  
- First 60-80 characters of the conversation topic
- Session ID
EOF
```

Then use `/history` to see all your past sessions!

### 2. Use the VS Code Extension

If you prefer a GUI, install the **Claude Code Assist** extension (`agsoft.claude-history-viewer`) in VS Code. It gives you:
- A sidebar showing all conversations
- Search functionality
- One-click resumption
- Diff viewing

### 3. Third-Party Tool - Claude Resume

There's also a handy web-based tool called [claude-resume](https://github.com/nikbq/claude-resume):

```bash
uvx --from git+https://github.com/nikbq/claude-resume.git claude-resume
```

This opens a web interface where you can:
- Search through all chat history
- View multi-project conversations
- Copy resume commands with one click

## Important Notes

**Context Persistence:**
- Sessions persist indefinitely until you explicitly delete them
- Claude Code automatically compacts conversations near the 200K token limit
- Each project directory gets its own conversation storage

**Resume vs. Rewind:**
- **Resume** = Load a previous conversation
- **Rewind** = Roll back changes within the current session (press Esc twice or `/rewind`)

## Common Use Cases

```bash
# "Wait, how did I fix that bug last week?"
claude --resume
# [Select the conversation from last week]

# "Continue exactly where I left off"
claude --continue

# "Jump back into that API refactor project"
claude --resume <session-id>

# "Show me what we discussed about authentication"
claude --resume
# [Search for authentication-related conversation]
```

This feature is a **game-changer** because it means you never lose context. You can close your terminal, restart your computer, work on different projects, and always come back to exactly where you were!
