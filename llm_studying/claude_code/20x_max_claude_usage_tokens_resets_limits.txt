### Important Caveats

1. **Not fixed token counts** — Limits vary based on message length, conversation length, file attachments, and model used

2. **Opus burns faster** — Max 20x auto-switches from Opus → Sonnet at 50% usage to preserve capacity

3. **Shared across interfaces** — Usage is combined across claude.ai, Claude Code, and Claude mobile

4. **5-hour resets** — Your usage window resets every 5 hours, not monthly

Realistically, for Claude Code work, users report getting through substantial projects but hitting limits during intensive sessions. The recent complaints suggest heavy Claude Code usage (large codebases, many files) can burn through limits faster than chat.
