Based on my search, **Ralph** (officially called **ralph-wiggum**) is a plugin for Claude Code that enables autonomous development loops. Here's how to install it:

## Installation via Claude Code Plugin System (Recommended)

In your terminal with Claude Code, run:

```bash
/plugin install ralph-wiggum@claude-plugins-official
```

This installs the official plugin from the Claude Code plugin marketplace.

## What Ralph Does

Ralph implements an iterative loop where Claude Code works on a task, and when it tries to exit, a "Stop hook" intercepts and feeds the same prompt back, allowing Claude to keep refining until completion. It's useful for:

- Complex refactoring tasks
- Test coverage improvements  
- Multi-phase feature development
- Repetitive coding tasks

## Basic Usage

Once installed, you can start a loop like this:

```bash
/ralph-loop "Your task description" --max-iterations 20 --completion-promise "DONE"
```

**Important notes:**
- Always set `--max-iterations` to prevent infinite loops and control API costs
- Include clear completion criteria in your prompt
- The plugin requires `jq` (JSON processor) - install it first if needed

For more details on usage, commands, and best practices, check the [official plugin documentation](https://github.com/anthropics/claude-code/blob/main/plugins/ralph-wiggum/README.md).
