Yes! Claude Code is available via Homebrew, and you can upgrade it. However, there are some important things to know:

## How to Upgrade Claude Code via Homebrew

**From your regular terminal (not inside Claude Code):**
```bash
brew upgrade claude-code
```

Or to upgrade everything including Claude Code:
```bash
brew upgrade
```

## Should You Upgrade FROM WITHIN Claude Code?

**You can, but it's not recommended.** Here's why:

If you run `brew upgrade claude-code` from within a Claude Code session, you're essentially trying to update the program while it's running. This can cause issues:

- The upgrade might complete but won't take effect until you restart
- You might get odd behavior or errors
- The session could crash mid-upgrade

**Better approach:**
1. Exit Claude Code first (`/exit`)
2. Run `brew upgrade claude-code` from your regular terminal
3. Restart Claude Code

## Auto-Update Feature

**Good news:** Claude Code has **automatic updates** built in!

- Updates download and install automatically in the background
- You'll see a notification when updates are available
- Updates take effect the next time you start Claude Code
- No manual intervention needed (usually)

## Check Your Current Version

To see what version you're running:
```bash
claude --version
```

Or from within Claude Code:
```bash
/status
```

## Disable Auto-Updates (if needed)

If you want to control updates manually, add this to your settings:
```bash
# In your shell config (~/.bashrc or ~/.zshrc)
export DISABLE_AUTOUPDATER=1
```

Or in `~/.claude/settings.json`:
```json
{
  "env": {
    "DISABLE_AUTOUPDATER": "1"
  }
}
```

## Known Homebrew Update Issues

Some users have reported seeing "Update available" notifications when they're already on the latest version. This is usually a timing issue where:
- Homebrew hasn't synced the latest version yet
- Or there's a mismatch between the auto-updater and Homebrew

If you see this, try:
```bash
brew update        # Update Homebrew itself
brew upgrade claude-code
```

## Installation Methods Compared

Claude Code can be installed via:
1. **Native installer** (recommended): `curl -fsSL https://claude.ai/install.sh | bash`
2. **Homebrew**: `brew install --cask claude-code` (your method)
3. **npm**: `npm install -g @anthropic-ai/claude-code`

All methods work, but the native installer and Homebrew provide the smoothest auto-update experience.

## Bottom Line

Let the auto-updater handle it, or manually upgrade from your regular terminal (not from within Claude Code). Either way works great!
