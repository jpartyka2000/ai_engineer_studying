Yes! Claude Code **can accept screenshot images**. Here's how:

---

## On macOS

You can copy a screenshot to your clipboard using Shift + Command + Control + 4, then select an area. Paste it into Claude Code using Control + V (not Command + V).

**Key point:** Use **`Control + V`**, not `Command + V`! This trips up most Mac users.

---

## Step by Step

1. **Take a screenshot to clipboard:**
   - `Cmd + Ctrl + Shift + 4` → select area (copies to clipboard, doesn't save file)

2. **Paste into Claude Code:**
   - `Ctrl + V` (NOT `Cmd + V`)

3. **You'll see:**
   ```
   > [Image #1]
   ```
   This confirms the image was attached.

4. **Add your prompt:**
   ```
   > [Image #1]
   > Fix the layout issue shown in this screenshot
   ```

---

## Use Cases

You can provide Claude Code with image mockups of a UI and ask it to build the interface. To iterate on a design or fix a UI bug: Ask Claude Code to build something, open what it built in a browser, take a screenshot, paste the screenshot into Claude Code, and provide feedback.

Great for:
- UI mockup → code generation
- Visual bug reports
- Comparing expected vs actual output
- CSS/styling issues

---
