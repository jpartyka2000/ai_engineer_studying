Let me search Anthropic's documentation for this.Based on my search, Claude Code uses standard bash-style keyboard shortcuts for editing prompts. Here's what you can use:

**To delete a line (or parts of it):**
- **Ctrl+U** — Delete from cursor to the beginning of the line
- **Ctrl+K** — Delete from cursor to the end of the line
- **Ctrl+W** — Delete the previous word

**Helpful navigation shortcuts:**
- **Ctrl+A** — Jump to the beginning of the line
- **Ctrl+E** — Jump to the end of the line
- **Option+B** / **Option+F** — Move backward/forward one word

**Other useful ones:**
- **Double-tap Escape** — Clear the entire input
- **Up/Down arrows** — Navigate through your prompt history

So if you want to delete an entire line, the quickest way is **Ctrl+U** (if your cursor is at the end) or **Ctrl+A** followed by **Ctrl+K**.

**Note:** If Shift+Enter isn't working for new lines in multi-line prompts, you can run `/terminal-setup` and Claude Code will configure your terminal to support it.
