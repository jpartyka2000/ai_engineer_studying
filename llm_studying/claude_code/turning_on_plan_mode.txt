Press `Shift+Tab` to toggle plan mode on/off.

## What it does

In plan mode, Claude will:
- Think through the approach
- Outline a plan
- **Ask for your approval** before making any changes

It won't edit files or run commands until you confirm.

## Other ways to trigger it

```bash
# In your prompt
"Plan how to refactor this module"
"Make a plan for..."

# Or use the /plan command (if available in your version)
/plan
```

## When to use it

- Complex multi-file changes
- When you want to review the approach first
- Architectural decisions
- When you're unsure what Claude might do
