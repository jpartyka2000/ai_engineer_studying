To get the underlying model object rather than a Spark UDF function, use `mlflow.pyfunc.load_model` instead of `mlflow.pyfunc.spark_udf`. For example:

```python
loaded_model = mlflow.pyfunc.load_model(model_uri=my_model_result.best_model_uri)
```

This will load and return the original model object (a Python object) that you can call directly for predictions.
