You can grab the artifact URI for any run two ways:

1. **Within an active run**
   If you’re already inside an `mlflow.start_run()` context, just call:

   ```python
   import mlflow

   with mlflow.start_run(run_id="…") as _:
       uri = mlflow.get_artifact_uri()
       print("Artifact root URI:", uri)
   ```

   If you pass a sub-directory, e.g. `mlflow.get_artifact_uri("plots")`, you’ll get the full URI to that subfolder.

2. **From outside (or after) the run**
   Use the `MlflowClient` to fetch the run’s metadata:

   ```python
   from mlflow.tracking import MlflowClient

   client = MlflowClient()
   run = client.get_run("7969e289c8844a69b4c46b340d4ac78d")
   artifact_uri = run.info.artifact_uri
   print("Artifact root URI:", artifact_uri)
   ```

   That `artifact_uri` is the base location in which all that run’s logged artifacts live.
