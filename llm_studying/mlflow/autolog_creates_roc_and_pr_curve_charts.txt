Yes—if you enabled autologging (or are using a platform like Databricks that integrates MLflow autolog features), MLflow can automatically log and display evaluation plots like the ROC curve and precision-recall curve for classification models. 

For example, when you call:

```python
mlflow.sklearn.autolog()
```

MLflow automatically logs metrics and even visual artifacts (such as the ROC and precision-recall curves) for scikit-learn models. These plots appear in the run’s artifacts and are displayed in the MLflow UI. 

So, if you're seeing these curves without explicitly logging them, it's because autologging is capturing your model’s evaluation details automatically.
