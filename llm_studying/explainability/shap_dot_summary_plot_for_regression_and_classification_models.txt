Yes, it does make sense. SHAP summary dot plots are a versatile visualization tool that can illustrate the distribution of feature contributions for both regression and classification models. Here are some points to consider:

- **Consistent Interpretation:**  
  In both regression and classification, a SHAP dot plot shows how individual feature values influence the model’s output relative to a baseline. For regression, this typically means showing how features increase or decrease the predicted continuous value. For classification, it often shows the effect on the predicted probability for a particular class.

- **Distribution and Magnitude:**  
  The dot plot displays the distribution of SHAP values for each predictor, giving you insight into both the range and consistency of a feature’s influence, regardless of the type of target variable.

- **Class-specific Plots for Classification:**  
  In classification problems, especially with multiple classes, you might generate separate plots for each class (or focus on one class) to clearly understand the feature contributions for that particular outcome.

- **Visual Consistency:**  
  Using the same type of plot for both regression and classification can help maintain consistency in how you communicate model explanations across different types of tasks.

In summary, a SHAP summary dot plot is a valuable tool for understanding predictor influence in both regression and classification contexts, as long as you interpret the values appropriately for the task at hand.
