The dot summary plot shows individual SHAP value contributions for each instance, which provides several benefits for regression tasks:

- **Distributional Insight:** You see the full spread of how each feature affects predictions rather than just an aggregated value.
- **Directionality:** Each dot's position on the x‑axis indicates whether a feature increases or decreases the prediction.
- **Feature Value Relationships:** Colors in the dot plot often represent the actual feature values, helping you see how high and low values influence predictions.
- **Heterogeneity:** It captures variation across all data points, revealing patterns such as non-linear effects or subgroups, which a bar plot—being an average summary—can obscure.

In contrast, the summary bar plot aggregates the absolute SHAP values (usually as a mean) and loses these nuances, making the dot summary plot more informative when understanding complex relationships in regression models.
