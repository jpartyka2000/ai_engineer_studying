Yes, there are several interactive visualization libraries that can dynamically show how prediction probabilities change as a predictor value varies. These tools are particularly useful for understanding model behavior and conducting sensitivity analysis. Here are some options:

---

### **1. SHAP (SHapley Additive exPlanations)**
- **Description**: SHAP provides interactive visualizations that explain individual predictions. The `force_plot` and `dependence_plot` are particularly useful for exploring how changes in a feature value impact the prediction.
- **Key Feature**: The `shap.dependence_plot` can show how the model's output changes as a specific feature value changes, with interactions with other features highlighted.

**Interactive Example with SHAP**:
```python
import shap
import matplotlib.pyplot as plt

# Create a SHAP explainer
explainer = shap.Explainer(model, X)
shap_values = explainer(X)

# Dependence plot for a specific feature
shap.dependence_plot("sw_state", shap_values.values, X)
```

For more interactive plots, SHAP integrates well with Jupyter notebooks using `shap.initjs()` and `shap.force_plot`.

---

### **2. Partial Dependence with Dash or Streamlit**
- **Description**: Libraries like Dash and Streamlit allow you to create web-based interactive dashboards to visualize partial dependence or ICE plots with sliders to adjust feature values dynamically.

**Streamlit Example**:
```python
import streamlit as st
from sklearn.inspection import partial_dependence
import numpy as np

# Feature and model setup
feature = 'sw_state'
feature_idx = X.columns.get_loc(feature)

# Slider to adjust feature value
feature_value = st.slider(f"Adjust {feature}", float(X[feature].min()), float(X[feature].max()), float(X[feature].mean()))

# Calculate partial dependence
pd_results = partial_dependence(model, X, [feature_idx], grid_resolution=50)
x_values, y_values = pd_results['values'][0], pd_results['average'][0]

# Find closest predicted value to slider position
predicted_value = np.interp(feature_value, x_values, y_values)

# Plot
st.line_chart({"Partial Dependence": y_values, "X values": x_values})
st.write(f"Predicted value for {feature} = {feature_value}: {predicted_value}")
```

This allows for dynamic interaction with predictors.

---

### **3. Plotly for Dynamic ICE/Partial Dependence**
- **Description**: Plotly is a Python library that creates interactive visualizations. It’s great for creating dynamic ICE plots where users can hover or zoom into specific regions of interest.

**Plotly Example**:
```python
import plotly.express as px
import pandas as pd
from sklearn.inspection import partial_dependence

# Calculate partial dependence
pd_results = partial_dependence(model, X, [feature_idx], grid_resolution=50)
x_values = pd_results['values'][0]
y_values = pd_results['average'][0]

# Create a DataFrame for Plotly
pd_data = pd.DataFrame({feature: x_values, "Prediction": y_values})

# Plot with Plotly
fig = px.line(pd_data, x=feature, y="Prediction", title=f"Partial Dependence for {feature}")
fig.show()
```

---

### **4. LIME (Local Interpretable Model-agnostic Explanations)**
- **Description**: LIME provides a way to explain individual predictions by perturbing feature values and observing changes in the output.
- **Key Feature**: You can interactively explore how the model reacts to changes in the feature values.

**LIME Example**:
```python
from lime.lime_tabular import LimeTabularExplainer

# Create LIME explainer
explainer = LimeTabularExplainer(X.values, feature_names=X.columns, class_names=['Class1', 'Class2'], mode='classification')

# Explain a specific prediction
i = 0  # Row index
exp = explainer.explain_instance(X.iloc[i].values, model.predict_proba, num_features=5)

# Visualize explanation
exp.show_in_notebook()
```

| Library      | Interactivity | Integration | Use Case                      |
