## What is CUDA?

CUDA (Compute Unified Device Architecture) is NVIDIA's parallel computing platform and programming model. It provides:

1. **A way to run general-purpose code on GPUs** - GPUs were originally designed just for graphics, but CUDA lets you use them for any massively parallel computation (like matrix math in deep learning)

2. **Libraries and APIs** - Tools like cuBLAS (linear algebra), cuDNN (deep neural networks), etc. that are highly optimized for NVIDIA hardware

3. **A compiler and runtime** - The `nvcc` compiler turns CUDA code into GPU-executable binaries

## Why "development" image vs "runtime"?

NVIDIA provides different image variants:

- **`runtime`** - Contains just what's needed to *run* pre-compiled CUDA applications (smaller)
- **`devel`** - Contains the full toolkit including compilers, headers, and libraries needed to *build* CUDA applications (larger, but necessary if any Python packages need to compile CUDA extensions during installation)

The code uses `devel` because vLLM and flashinfer may need to compile custom CUDA kernels during installation.

## The practical effect

Without CUDA, frameworks like PyTorch and vLLM couldn't talk to the GPU at all. The GPU would just sit there unused. CUDA is the essential translation layer that makes GPU-accelerated machine learning possible on NVIDIA hardware.

So this base image gives you: Ubuntu 22.04 + NVIDIA drivers + CUDA 12.8 toolkit — everything needed to build and run GPU-accelerated Python applications.
