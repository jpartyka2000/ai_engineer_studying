The `Axes.text()` method is your go-to for placing arbitrary text anywhere in an axes. Here’s a quick rundown:

```python
ax.text(
    x, y,              # 1) position coordinates
    s,                 # 2) the text string
    fontdict=None,     # 3) dict of font properties (size, weight, style…)
    *,
    fontsize=None,     # 4) or keyword font styling
    color=None,
    ha=None, va=None,  # 5) horizontal & vertical alignment
    rotation=0,        # 6) angle in degrees
    alpha=None,        # 7) transparency
    transform=None,    # 8) coordinate system for x,y
    bbox=None,         # 9) draw a box around text
    **kwargs           # 10) many more (linespacing, url, zorder…)
)
```

---

### 1) **Position** (`x`, `y`)  
- By default these are in **data coordinates**.  
- To use **axes fraction** (0–1 relative to the axes), pass `transform=ax.transAxes`.

### 2) **Text** (`s`)  
- Any string (can include `\n` for newlines).

### 3) **Font styling**  
- Use `fontsize`, `fontweight`, `fontstyle`, etc., or bundle them in `fontdict={'size':12,'weight':'bold'}`.

### 4) **Alignment**  
- `ha` = `'left' | 'center' | 'right'`  
- `va` = `'bottom' | 'center' | 'top' | 'baseline'`

### 5) **Rotation & transparency**  
- `rotation=45` tilts the text.  
- `alpha=0.5` makes it semi-transparent.

### 6) **Bounding box** (`bbox`)  
- A dict like `bbox=dict(boxstyle='round', facecolor='white', alpha=0.5)`  
- Draws a styled box behind the text.

### 7) **Transforms**  
- Default: `transform=ax.transData` (data coords).  
- Common alternative: `transform=ax.transAxes` for relative (0–1) axes coords, or `fig.transFigure` for figure coords.

---

#### Minimal example

```python
import matplotlib.pyplot as plt

fig, ax = plt.subplots()
ax.plot([0,1],[0,1])
# place “Hello” at 80% along x, 20% up the axes
ax.text(
    0.8, 0.2, "Hello!",
    transform = ax.transAxes,
    ha='center', va='center',
    fontsize=12,
    color='navy',
    bbox=dict(boxstyle='round,pad=0.3', facecolor='wheat', alpha=0.7)
)
plt.show()
```

Use `ax.text()` whenever you need custom annotations—legends, data labels, side-boxes, whatever—and control exactly where and how they render.
