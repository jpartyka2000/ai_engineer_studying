 bubble chart is a type of data visualization that displays three dimensions of data using:

1. X-axis – one numeric variable


2. Y-axis – another numeric variable


3. Bubble size – a third numeric variable, often represented by the area of each circle (not radius)



Optionally, a fourth dimension can be added using:

Bubble color to represent a categorical variable.


📊 Use Cases:

Visualizing relationships between three numeric variables

Comparing financial performance across companies

Plotting population (size), GDP (x-axis), and life expectancy (y-axis) by country



---

🧪 Example (Python with Matplotlib)

import matplotlib.pyplot as plt

# Example data
companies = ['Company A', 'Company B', 'Company C', 'Company D']
revenue = [100, 150, 80, 200]      # X-axis
profit = [30, 45, 20, 50]          # Y-axis
market_cap = [1000, 1600, 600, 2200]  # Bubble size

# Create bubble chart
plt.figure(figsize=(10, 6))
plt.scatter(revenue, profit, s=market_cap, alpha=0.5)

# Add labels
for i, company in enumerate(companies):
    plt.text(revenue[i], profit[i], company, fontsize=9, ha='center')

plt.xlabel("Revenue (in millions)")
plt.ylabel("Profit (in millions)")
plt.title("Company Performance Bubble Chart")
plt.grid(True)
plt.show()

💡 Notes:

s=market_cap scales the bubble sizes.

alpha=0.5 makes the bubbles semi-transparent.

Bubble area (not radius) should correspond to data values for perceptual accuracy.
