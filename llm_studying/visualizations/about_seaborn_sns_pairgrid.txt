`PairGrid` is a Seaborn class for creating a grid of subplots that lets you plot pairwise relationships across multiple variables in a DataFrame with a high degree of customization. Here’s how it works:

1. **Initialization**

   ```python
   import seaborn as sns
   grid = sns.PairGrid(df, vars=["col1", "col2", "col3"], hue="category")
   ```

   * `df`: your DataFrame
   * `vars`: which columns to include (defaults to all numeric columns)
   * `hue`: an optional grouping variable to color-code points

2. **Mapping plots to the grid**
   After you’ve created the grid, you specify what kind of plot goes on each part of the matrix:

   ```python
   grid.map_upper(sns.scatterplot)    # upper triangle: scatter plots
   grid.map_lower(sns.kdeplot, fill=True)  # lower triangle: density plots
   grid.map_diag(sns.histplot)        # diagonal: univariate distributions
   ```

   * `map_upper` targets the upper triangle of subplots
   * `map_lower` targets the lower triangle
   * `map_diag` targets the diagonal

3. **Customization**

   * You can pass kwargs to each plot function for further styling.
   * You can also add a legend with `grid.add_legend()` if you used `hue`.

4. **When to use**

   * When you want to explore all pairwise relationships between several variables in one figure.
   * When you need more control than the convenience function `sns.pairplot` offers (which is effectively a quick wrapper around `PairGrid`).

---

**In short**, `sns.PairGrid` gives you a blank grid of axes that you fill with your choice of plots, offering granular control over how pairwise comparisons and univariate distributions are displayed.
