───────────────────────────────────────────────────────────────────
QUESTION:
───────────────────────────────────────────────────────────────────
Which of the following functions would you use to set the overall 
visual theme/style for all subsequent Seaborn plots in your notebook?

A) sns.set_palette()
B) sns.set_theme()
C) sns.configure()
D) sns.style()

───────────────────────────────────────────────────────────────────
YOUR ANSWER:      A) sns.set_palette()
CORRECT ANSWER:   B) sns.set_theme()
RESULT:           ❌ Incorrect
───────────────────────────────────────────────────────────────────

EXPLANATION:
───────────────────────────────────────────────────────────────────
- sns.set_theme() sets the overall aesthetic style including 
  background, grid, fonts, and default color palette all at once.
  
- sns.set_palette() ONLY changes the color palette — it doesn't 
  affect the background style, grid lines, or fonts.

- sns.configure() and sns.style() don't exist in Seaborn.

Example usage:
    sns.set_theme(style="darkgrid", palette="husl")

Note: sns.set_theme() replaced the older sns.set() function 
(though sns.set() still works as an alias).
