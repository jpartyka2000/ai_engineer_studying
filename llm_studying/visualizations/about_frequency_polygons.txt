A frequency polygon is a line plot that connects the midpoints of bins (or intervals) of a histogram. It's useful for showing the shape of a distribution while providing a smoother view than a traditional histogram.

Here's an example in Python using matplotlib:

```python
import numpy as np
import pandas as pd
import matplotlib.pyplot as plt

# Generate sample data
np.random.seed(42)
data = np.random.normal(loc=0, scale=1, size=1000)
df = pd.DataFrame({"data": data})

# Define bins (for example, 20 bins)
bins = np.linspace(df["data"].min(), df["data"].max(), 21)

# Get counts and bin edges using numpy.histogram
counts, bin_edges = np.histogram(df["data"], bins=bins)

# Compute the bin centers (midpoints of each bin)
bin_centers = (bin_edges[:-1] + bin_edges[1:]) / 2

# Plot the frequency polygon
plt.figure(figsize=(8, 6))
plt.plot(bin_centers, counts, marker="o", linestyle="-", color="b")
plt.xlabel("Value")
plt.ylabel("Frequency")
plt.title("Frequency Polygon")
plt.grid(True)
plt.show()
```

### Explanation

1. **Data Generation:**  
   We generate a sample dataset drawn from a normal distribution.

2. **Defining Bins:**  
   We create an array of bin edges using `np.linspace`. In this example, 20 bins are defined.

3. **Histogram Calculation:**  
   `np.histogram` computes the counts of data points in each bin and returns both the counts and the bin edges.

4. **Bin Centers:**  
   The midpoint of each bin is calculated by averaging the left and right bin edges.

5. **Plotting:**  
   We use `plt.plot()` with markers to draw the frequency polygon. The x-axis represents the bin centers and the y-axis represents the counts.

This frequency polygon provides a smoothed view of the distribution compared to a bar chart and is particularly useful when you want to visualize trends in frequency over intervals.
