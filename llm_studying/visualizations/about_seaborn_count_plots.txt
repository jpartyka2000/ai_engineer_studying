A **Seaborn count plot** is simply a bar plot that shows the number of observations in each category of a categorical variable. It’s the categorical analogue of a histogram—except instead of “bins” of a numeric variable, you get one bar per unique value.

---

### Key points

- **Purpose**  
  Quickly visualize the frequency (count) of each category in a single variable.

- **How it works**  
  Under the hood, `sns.countplot()` counts how many times each distinct value appears and then draws a bar whose height equals that count.

- **Basic usage**  
  ```python
  import seaborn as sns

  # df is your DataFrame, 'col' is a categorical column
  sns.countplot(x='col', data=df)
  ```

- **Common options**  
  - `order=`: specify the order of categories (e.g. sorted by frequency)  
  - `hue=`: add a second categorical variable to split each bar into colored segments  
  - `palette=`: choose a color palette  
  - `orient='h'`: draw horizontal bars instead of vertical  

- **When to use**  
  - Inspect class imbalance in a target variable  
  - Compare group sizes before further analysis  
  - Quick EDA on any categorical feature  

---

#### Example

```python
import seaborn as sns
import matplotlib.pyplot as plt

# Suppose df['fruit'] contains values like 'apple', 'banana', 'orange'
sns.countplot(x='fruit', data=df, order=df['fruit'].value_counts().index)
plt.title("Fruit Counts")
plt.ylabel("Number of Records")
plt.show()
```

This will draw one bar for “apple,” one for “banana,” etc., with heights proportional to how many rows in `df` had each fruit.
