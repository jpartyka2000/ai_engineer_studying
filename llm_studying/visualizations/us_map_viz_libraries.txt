Yes—there are several Python libraries that make it easy to plot a U.S. map and color or highlight individual states (or counties, metro areas, etc.). The most common ones are:

1. **Plotly (Plotly Express / graph\_objects)**

   * Built-in U.S. “choropleth” support.
   * Pass state abbreviations or FIPS codes + a value column, and it will render an interactive map where you can color–scale or set discrete colors for each region.

   ```python
   import plotly.express as px
   df = px.data.election()  # has state fips & results
   fig = px.choropleth(
       df,
       locations="state",         # two-letter state codes
       locationmode="USA-states",
       color="Maine",             # whatever column/value you like
       scope="usa",
       color_continuous_scale="Blues",
   )
   fig.show()
   ```

2. **GeoPandas + Matplotlib**

   * “Pandas for spatial data.”
   * Load a shapefile of U.S. states (e.g. from the built-in `geopandas.datasets` or Census TIGER/Line), merge your data on a state column, then call `.plot(column="your_value", cmap="OrRd")`.

   ```python
   import geopandas as gpd
   import matplotlib.pyplot as plt

   states = gpd.read_file(gpd.datasets.get_path("naturalearth_lowres"))
   us_states = states[states["iso_a2"] == "US"]
   # assume you have a DataFrame `df` with a 'state_name' and 'score' column
   merged = us_states.merge(df, left_on="name", right_on="state_name")
   merged.plot(column="score", cmap="YlGn", legend=True)
   plt.axis("off")
   plt.show()
   ```

3. **Folium**

   * Wraps Leaflet.js for interactive, zoomable web maps.
   * You can add a GeoJSON of U.S. boundaries and then a `Choropleth` layer to color states.

   ```python
   import folium
   import json

   m = folium.Map(location=[48, -102], zoom_start=3)
   with open("us-states.json") as f:
       states_geo = json.load(f)
   folium.Choropleth(
       geo_data=states_geo,
       data=df,                   # DataFrame with state & value
       columns=["state_code", "value"],
       key_on="feature.properties.STUSPS",
       fill_color="PuBu",
   ).add_to(m)
   m
   ```

4. **Bokeh**

   * Interactive Python plotting library.
   * Provides sample U.S. state polygons in `bokeh.sampledata.us_states`, which you convert into a `ColumnDataSource` and then render with `patches()`.

   ```python
   from bokeh.io import show
   from bokeh.plotting import figure
   from bokeh.sampledata.us_states import data as states

   # filter out AK & HI, extract lons/lats
   states = {code: s for code, s in states.items() if code not in ("HI","AK")}
   state_xs = [s["lons"] for s in states.values()]
   state_ys = [s["lats"] for s in states.values()]

   p = figure(title="US States", toolbar_location="left",
              plot_width=800, plot_height=500)
   p.patches(state_xs, state_ys, fill_alpha=0.7,
             line_color="white", line_width=0.5)
   show(p)
   ```

---

**Which to choose?**

* **For quick, interactive dashboards:** **Plotly** or **Folium**.
* **For publication-quality static maps:** **GeoPandas + Matplotlib**.
* **For custom interactive apps:** **Bokeh** (or frameworks built on it like Holoviews/Panel).

All of these let you “highlight” or color specific states (or counties, if you load finer‐grained shapefiles) with just a few lines of code.
