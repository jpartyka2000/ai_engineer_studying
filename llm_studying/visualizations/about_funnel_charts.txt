A **funnel chart** is a type of visualization used to show how a quantity progressively shrinks (or “funnels”) as it moves through a series of stages in a linear process. It gets its name from its typical trapezoidal shape—wide at the top, narrowing down toward the bottom—as values drop off from one step to the next.

---

### Key characteristics

1. **Sequential stages**
   Each horizontal segment represents one step in a defined process (e.g., “Website Visits” → “Sign‑ups” → “Trials” → “Paid Conversions”).

2. **Proportional width**
   The width (or area) of each segment is proportional to the metric at that stage (e.g., number of users, dollar amounts).

3. **Visual drop‑off**
   The narrowing visually highlights where the biggest drop‑offs occur, making it easy to identify bottlenecks.

---

### Common use cases

* **Sales & marketing**: track leads → opportunities → proposals → closed deals.
* **E‑commerce**: page views → product views → add‑to‑cart → checkouts → purchases.
* **Onboarding**: app installs → account creations → first action → retention.

---

### Why use one?

* **Clarity**: you immediately see which stage loses the most volume.
* **Focus**: directs attention to stages needing optimization (e.g. high abandonment).
* **Simplicity**: fewer visual elements than a multi‑bar chart—ideal for executive dashboards.

---

### Simple example

| Stage             | Count   |
| ----------------- | ------- |
| Website Visits    | 100,000 |
| Sign‑ups          | 25,000  |
| Email Confirmed   | 20,000  |
| Activated Account | 10,000  |
| First Purchase    | 2,500   |

A funnel chart would render this as five stacked bands whose widths shrink in proportion—making it immediately clear that the biggest drop is from “Activated Account” to “First Purchase.”

---

Most modern BI tools (Tableau, Power BI, Looker) and plotting libraries (e.g. Plotly, Matplotlib with custom code) offer funnel‑chart capabilities or easy ways to build them.
