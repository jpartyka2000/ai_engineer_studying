`ax.margins()` is a Matplotlib method that adds **padding (margin)** around the data limits in the plot.

---

### 🔧 Syntax:

```python
ax.margins(x=0.05, y=0.1)
```

* `x=...` → relative margin (as a fraction of data range) to add to left/right
* `y=...` → margin to add to bottom/top

For example:

```python
ax.margins(y=0.2)
```

adds 20% of the vertical data range as space both **above and below** the tallest/shortest bar.

---

### 🧠 Why use it?

* It’s useful when you don’t want to manually calculate `set_ylim()`
* It ensures no data is touching the edge of the plot
* Can prevent labels or markers from being clipped

---

### 📝 Key Notes:

* Margins are **relative to the data range**, not fixed units
* If you use `ax.set_ylim()`, then `ax.margins()` is **ignored**
* Default margins in many plot types are around `0.05` (5%)
