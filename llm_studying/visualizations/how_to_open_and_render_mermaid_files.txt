Mermaid files (`.mermaid` or `.mmd`) are just plain text files. Opening them is easy, but **rendering** them into a visual diagram requires a tool that supports Mermaid.

## Quick Options

### 1. Online Editor (Easiest)
Go to **[mermaid.live](https://mermaid.live)** and paste the file contents. You'll see the rendered diagram instantly and can export it as PNG or SVG.

### 2. VS Code
Install the **"Mermaid Preview"** or **"Markdown Preview Mermaid Support"** extension, then open the file and use the preview pane.

### 3. GitHub
Rename the file to `.md` and wrap the content in a mermaid code block:

~~~markdown
```mermaid
<your diagram code here>
```
~~~

GitHub renders it automatically when you view the file.

### 4. Obsidian / Notion
Both render Mermaid blocks natively — just paste the code into a mermaid code block.
