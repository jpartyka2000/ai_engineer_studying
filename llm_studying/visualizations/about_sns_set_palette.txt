═══════════════════════════════════════════════════════════════════
SEABORN REFERENCE — sns.set_palette()
═══════════════════════════════════════════════════════════════════

PURPOSE:
───────────────────────────────────────────────────────────────────
Sets the default color palette for ALL subsequent plots in your 
session. Only affects colors — not style, grid, fonts, or background.

═══════════════════════════════════════════════════════════════════

SYNTAX:
───────────────────────────────────────────────────────────────────
sns.set_palette(palette, n_colors=None, desat=None)

PARAMETERS:
  • palette   : Name of palette or list of colors
  • n_colors  : Number of colors in the palette (optional)
  • desat     : Desaturation factor 0-1 (optional)

═══════════════════════════════════════════════════════════════════

BUILT-IN PALETTE OPTIONS:
───────────────────────────────────────────────────────────────────
Categorical (qualitative):
  • "deep"      — default, muted colors
  • "muted"     — softer version
  • "bright"    — vivid colors
  • "pastel"    — light, soft colors
  • "dark"      — darker shades
  • "colorblind"— accessible palette

Sequential (for ordered data):
  • "Blues", "Greens", "Reds", "Purples", etc.
  • "viridis", "magma", "plasma", "cividis"

Diverging (for data with meaningful center):
  • "coolwarm", "RdBu", "RdYlGn", "BrBG"

═══════════════════════════════════════════════════════════════════

EXAMPLES:
───────────────────────────────────────────────────────────────────

# Set a named palette
sns.set_palette("pastel")

# Set with specific number of colors
sns.set_palette("husl", n_colors=8)

# Set with desaturation (0 = gray, 1 = full color)
sns.set_palette("bright", desat=0.7)

# Use a custom list of colors
my_colors = ["#FF6B6B", "#4ECDC4", "#45B7D1", "#96CEB4"]
sns.set_palette(my_colors)

# Use matplotlib colormap
sns.set_palette("tab10")

═══════════════════════════════════════════════════════════════════

TEMPORARY PALETTE WITH CONTEXT MANAGER:
───────────────────────────────────────────────────────────────────
# Only applies within the 'with' block
with sns.color_palette("husl"):
    sns.barplot(data=df, x="category", y="value")

# After the block, palette reverts to previous setting

═══════════════════════════════════════════════════════════════════

VIEW A PALETTE WITHOUT SETTING IT:
───────────────────────────────────────────────────────────────────
# Display palette as horizontal color swatches
sns.color_palette("coolwarm", as_cmap=False)
sns.palplot(sns.color_palette("Set2"))  # visual display

═══════════════════════════════════════════════════════════════════

KEY DISTINCTION:
───────────────────────────────────────────────────────────────────
sns.set_palette()  → Changes ONLY colors
sns.set_theme()    → Changes colors AND style/grid/fonts/background
