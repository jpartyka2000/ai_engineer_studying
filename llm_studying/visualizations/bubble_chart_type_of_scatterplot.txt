Yes, a bubble chart is a type of scatter plot — specifically an enhanced scatter plot where each point is represented not just by its position (x, y), but also by the size of the bubble (a third variable).

🔍 Key Differences:

Feature Scatter Plot Bubble Chart

Axes X and Y X and Y
Marker Size Fixed Variable (based on 3rd value)
Variables shown 2 3 (or 4 with color)


🧠 Think of it this way:

All bubble charts are scatter plots,

but not all scatter plots are bubble charts.


So yes — a bubble chart extends a scatter plot to show more dimensions of data in the same 2D space
