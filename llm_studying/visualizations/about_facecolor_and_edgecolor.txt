The `facecolor` parameter simply controls the **fill color** of the box (or patch) behind your text (or any artist that supports a `bbox`). For example:

```python
props = dict(
    boxstyle   = 'round,pad=0.3',
    facecolor  = 'lightyellow',  # the background fill
    edgecolor  = 'gray',         # the border line color
    alpha      = 0.8             # transparency
)
ax.text(
    1.02, 0.5, counts_str,
    transform = ax.transAxes,
    ha        = 'left',
    va        = 'center',
    bbox      = props
)
```

- **`facecolor='lightyellow'`** makes the box background a pale yellow.  
- You can use any Matplotlib color specifier here: named colors (`'salmon'`), hex codes (`'#FFCCCC'`), RGB/RGBA tuples, etc.  
- Combined with `alpha`, you can get a translucent fill so the plot grid shows through.

If you omit `facecolor`, the default is usually **white** (or no fill, depending on the artist), so specifying it lets you highlight or tone down that annotation box as you like.
