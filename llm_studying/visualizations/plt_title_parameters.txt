The plt.title() function in Matplotlib adds a title to the current axes (plot). It accepts several useful parameters to customize the title’s appearance. Here are some of the most common ones:

Basic Usage:

plt.title("My Plot Title")

Key Parameters:

Example:

plt.plot([1, 2, 3], [4, 5, 6])
plt.title("Sample Plot",
          fontsize=14,
          color='darkblue',
          loc='left',
          pad=20,
          fontdict={'weight': 'bold', 'style': 'italic'})
