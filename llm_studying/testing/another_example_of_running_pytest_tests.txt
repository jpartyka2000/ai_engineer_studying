A few options:

```bash
# Run the specific file
pytest tests/test_unit.py

# Run just the TestQuestion class
pytest tests/test_unit.py::TestQuestion

# Run a specific test
pytest tests/test_unit.py::TestQuestion::test_default_lineage

# Run all tests in the tests/ directory
pytest tests/

# Run from project root (uses testpaths from pytest.ini)
pytest
```

Since your `pytest.ini` has `testpaths = tests`, just running `pytest` from the project root will find and run everything in the `tests/` directory.
