Here’s how you can turn on pytest’s built-in live logging via the `-o` flag on the command line:

```bash
pytest \
  tests/test_end_to_end.py::test_end_to_end_tasks \
  -vv -s \
  -o log_cli=true \
  -o log_cli_level=INFO \
  --maxfail=1 --disable-warnings
```

* `-o log_cli=true`        enables pytest’s **live CLI logging**
* `-o log_cli_level=INFO`  sets the minimum level of logs to show (INFO, DEBUG, etc.)
* `-vv -s`                 gives you extra pytest verbosity and lets you see `print()` output

With this, any `logging.info(...)` or higher-level calls in your code will stream straight into your test output.
