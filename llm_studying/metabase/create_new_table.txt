To create a table (i.e., a new visualization or a new query that results in a table) in **Metabase**, follow these steps:

### 1. **Log in to Metabase**:
   - Open your Metabase instance and log in using your credentials.

### 2. **Start a New Question**:
   - Once logged in, go to the **"New"** button at the top-right corner of the Metabase homepage.
   - From the dropdown menu, select **"New Question"**.

### 3. **Select Your Data Source**:
   - Choose the database you want to use for your query. You can select from various databases that your Metabase instance is connected to (e.g., MySQL, PostgreSQL, etc.).

### 4. **Choose Query Builder or Native SQL**:
   - Metabase provides two main ways to create a table:
     - **Simple Query Builder**: If you prefer a point-and-click interface, you can select the table from the schema, filter, and summarize the data through the visual interface.
     - **Native Query (SQL)**: If you are comfortable with SQL, you can write your own SQL query to pull the specific data and generate the table.
  
   - For the **Simple Query Builder**:
     - Select the table from the list of available tables.
     - You can filter and summarize data as needed by using the filtering options.
     - Once satisfied, click on **"Visualize"** and choose **"Table"** as the visualization type.

   - For **Native Query (SQL)**:
     - Write your SQL query to define the data for the table.
     - Example:
       ```sql
       SELECT column1, column2, column3
       FROM your_table
       WHERE condition;
       ```
     - Click **"Run"** to execute the query and generate the table.

### 5. **Customize the Table**:
   - After generating the table, you can adjust settings such as column sorting, filtering, and pagination.
   - You can also format columns (e.g., dates or numbers) using the formatting options provided.

### 6. **Save the Question (Table)**:
   - Once you’re satisfied with the table, click the **"Save"** button.
   - Provide a title and, optionally, save the table to a collection so you can easily access it later.

### 7. **Use the Table in Dashboards**:
   - After saving the table, you can add it to a dashboard by clicking **"Add to Dashboard"**.

### Summary:
- **New Question** > Choose data source > Use query builder or SQL to create the table > Customize the table > Save and optionally add to a dashboard.

These steps will help you create and display a table in Metabase, based on your data source.
