Notion itself doesn’t (yet) give you a “set border color” option for images the way a design tool would. But there are a few practical work-arounds you can use, depending on how polished you want the result to look:

---

## 1. **Use a callout block as a container**

* Insert a **Callout block** (`/callout`).
* Delete the default text and icon.
* Drag or paste your image into the callout.
* Change the callout’s background color.
* The background will show around the image like a frame/border.

  > ⚠️ It’s a full background color, not just a thin border, but it’s the simplest way inside Notion.

---

## 2. **Add the border before uploading**

* Open your image in an editor (Preview on macOS, Paint/Photoshop, Canva, etc.).
* Add a colored border (or drop shadow) there.
* Save and re-upload into Notion.

  > This is the only way to get a *true* colored border of any thickness around the image itself.

---

## 3. **Use an embed with custom HTML/CSS (advanced)**

* If you need pixel-perfect styling, you can host the image on another service (like CodePen, GitHub Pages, or a personal site) with CSS that defines a border.
* Then use Notion’s **“Embed link”** option and paste the hosted URL.
* The border will appear because it’s baked into the embedded content.

---

✅ **Summary:**

* Native Notion: use a **Callout block with background color** as a pseudo-border.
* Design control: add the border in an external editor before upload.
* Advanced styling: embed an image hosted elsewhere with CSS styling.
