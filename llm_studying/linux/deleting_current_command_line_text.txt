In zsh, you can delete the current line using:

**`Ctrl+U`** - Deletes the entire line (regardless of cursor position)

Other useful line-editing shortcuts:
- **`Ctrl+K`** - Delete from cursor to end of line
- **`Ctrl+W`** - Delete word before cursor
- **`Ctrl+C`** - Cancel current command (clears line and starts fresh)

**Note**: In zsh, `Ctrl+U` deletes the *entire* line by default, which is different from bash where it only deletes from the cursor to the beginning of the line.
