On an interactive Bash prompt (using the default Readline keybindings), you can:

* **Ctrl + U** Kill (clear) from the cursor back to the start of the line.
* **Ctrl + K** Kill (clear) from the cursor to the end of the line.

To wipe the *entire* current input regardless of where your cursor is, you can combine them:

1. **Ctrl + A** Move cursor to start of line.
2. **Ctrl + K** Kill the rest of the line.

Or:

1. **Ctrl + E** Move cursor to end of line.
2. **Ctrl + U** Kill back to start.

Either sequence will leave you with a blank prompt line.
