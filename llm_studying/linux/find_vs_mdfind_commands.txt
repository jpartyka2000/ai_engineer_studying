You have a few options:

## Terminal: `find` command

```bash
find / -name "*.txt" 2>/dev/null
```

- `/` searches from the root (entire computer)
- `-name` matches the filename (case-sensitive; use `-iname` for case-insensitive)
- `2>/dev/null` suppresses permission errors

Example with wildcards:

```bash
find / -name "report*.pdf" 2>/dev/null
```

This can be slow since it traverses the entire filesystem.

## Terminal: `mdfind` (Spotlight from command line)

```bash
mdfind -name "report"
```

This uses Spotlight's index, so it's much faster. For wildcards, you can use:

```bash
mdfind "kMDItemFSName == 'report*.pdf'"
```

## Spotlight GUI

Press **Cmd + Space**, then type `name:report` to search by filename specifically. You can also open Finder, press **Cmd + F**, and set the search scope to "This Mac" with the "Name matches" filter.

---

**Recommendation**: Use `mdfind` for speed if Spotlight indexing is enabled (it usually is). Use `find` if you need to search locations Spotlight doesn't index or want more precise control over the search.
