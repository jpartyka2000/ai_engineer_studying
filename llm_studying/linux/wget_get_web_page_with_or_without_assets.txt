Yes, you can use `wget` to download a Wikipedia page as a standalone file on your Mac! Here's how:

**Basic command:**
```bash
wget https://en.wikipedia.org/wiki/Page_Name
```

**Better options for a complete, standalone file:**

1. **Download with all assets (images, CSS, etc.):**
```bash
wget --page-requisites --convert-links --adjust-extension --no-parent https://en.wikipedia.org/wiki/Page_Name
```

2. **Simpler version with key flags:**
```bash
wget -p -k -E https://en.wikipedia.org/wiki/Page_Name
```

Where:
- `-p` or `--page-requisites` downloads images, CSS, and other assets
- `-k` or `--convert-links` converts links to work locally
- `-E` or `--adjust-extension` adds .html extension if needed

**If you don't have wget installed:**

On Mac, `wget` isn't installed by default. You can install it via Homebrew:
```bash
brew install wget
```

**Alternative - use curl (pre-installed on Mac):**
```bash
curl -o page.html https://en.wikipedia.org/wiki/Page_Name
```

This saves just the HTML though, without images or other assets.

The `wget` method with the flags I mentioned will give you a proper standalone version you can open in your browser offline!
