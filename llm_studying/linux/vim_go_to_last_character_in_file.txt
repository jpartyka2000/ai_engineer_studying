In vim, to go to the last character of a file, use:

```
G$
```

Or as a single command:

```
:normal G$
```

**Breakdown:**
- `G` - Goes to the last line of the file
- `$` - Moves to the end of that line (last character)

**Alternative methods:**

1. **Using go command:**
   ```
   :9999999999999go
   ```
   This tells vim to go to byte position 9999999999999 (a number larger than any file), which will take you to the last character.

2. **Using end-of-file mark:**
   ```
   G$
   ```
   (This is the most common and straightforward)

3. **In one motion from anywhere:**
   Just type `G$` in normal mode - no colon needed.

**Note:** If you want to go to the last line but at the first non-blank character (instead of the very last character), just use `G` by itself.
