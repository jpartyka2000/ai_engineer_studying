`wc` doesn't have a recursive flag, so combine it with `find`:

```bash
# Count lines in all files recursively
find . -type f -exec wc -l {} +

# With a total at the end
find . -type f -exec cat {} + | wc -l

# Only specific file types
find . -name "*.py" -exec wc -l {} +
```

## Using shell globbing (if enabled)

```bash
# Bash (enable globstar first)
shopt -s globstar
wc -l **/*

# Only .py files
wc -l **/*.py
```

## Common `wc` flags

| Flag | Counts |
|------|--------|
| `-l` | Lines |
| `-w` | Words |
| `-c` | Bytes |
| `-m` | Characters |
