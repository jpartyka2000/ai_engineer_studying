On macOS (and most Linuxes) the easiest way is with the built-in `zip` utility. Here are a few common patterns:

1. **Zip a few files**

   ```bash
   zip archive.zip file1.txt file2.jpg file3.py
   ```

2. **Zip an entire directory (recursively)**

   ```bash
   zip -r archive.zip /path/to/folder
   ```

   The `-r` flag tells `zip` to recurse into subdirectories.

3. **Exclude certain files**

   ```bash
   zip -r archive.zip folder -x '*.DS_Store'
   ```

   The `-x` pattern will skip anything matching `*.DS_Store`.

4. **Preserve symlinks**
   By default `zip -r` will follow symlinks and store the target’s contents.  If you want to store the link itself instead, add `-y`:

   ```bash
   zip -ry archive.zip folder
   ```

5. **Alternative with `ditto`**
   macOS also ships with `ditto`, which can produce a .zip without needing to install anything:

   ```bash
   ditto -c -k --sequesterRsrc --keepParent /path/to/folder archive.zip
   ```

That’s it—pick whichever fits your use case.
