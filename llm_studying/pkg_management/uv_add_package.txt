Use `uv add`:

```bash
uv add <package-name>
```

For example:

```bash
uv add requests
```

This will add the package to your `pyproject.toml` under `[project.dependencies]` and update your `uv.lock` file. After that, `uv sync` will install it.

A few useful variations:
- `uv add requests==2.31.0` — pin a specific version
- `uv add --dev pytest` — add as a dev dependency
- `uv add --group test pytest` — add to a specific dependency group
