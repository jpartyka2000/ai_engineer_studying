Here’s a standalone guide on using `pip index versions` to discover available releases of any PyPI package:

---

## Discovering Available Versions with `pip index versions`

When you need to know which versions of a Python package have been published on PyPI, the `pip index versions` command provides a quick, built-in way to list them.

### 1. Basic Usage

```bash
pip index versions <package-name>
```

* **`<package-name>`**: Replace with the name of the library you’re interested in.
* **Output**: A descending list of released versions (stable)—from newest to oldest.

#### Example

```bash
$ pip index versions requests
Available versions:
  2.31.0
  2.30.0
  2.29.1
  2.29.0
  …  
```

### 2. When to Use It

* **Finding the latest release**: Quickly see the most recent version number before upgrading.
* **Checking historical versions**: Determine whether a specific old version exists.
* **Avoiding surprises**: Validate that a version you plan to install actually exists.

### 3. Limitations

* **Stable-only listing**: By default, it shows only final releases. Pre-releases and release candidates (e.g. `1.0.0rc1`) may not appear.
* **No metadata**: It lists version numbers only—no upload dates or changelog details.

### 4. Installing a Specific Version

Once you’ve identified the version you need, install it explicitly:

```bash
pip install <package-name>==<version>
```

#### Example

```bash
pip install openai==1.99.9
```

### 5. Checking Your Installed pip Version

Since `pip index` requires pip ≳ 21.2, ensure you have an up-to-date pip:

```bash
pip --version
```

If needed, upgrade pip:

```bash
pip install --upgrade pip
```

### 6. Viewing Pre-Release Candidates

To see pre-release (“rc”) versions, you can:

* **Browse PyPI’s Release History**: Visit the package page on pypi.org, click **Release History**, and look for entries like `1.0.0rc20`.
* **Use the PyPI JSON API**:

  ```bash
  curl -s https://pypi.org/pypi/<package-name>/json \
    | jq -r '.releases | keys[]' \
    | grep rc \
    | sort -V
  ```

---

By using `pip index versions`, you gain direct insight into what’s available on PyPI, helping you choose the right package release for your project.
