On a Mac you’ve got a few good ways to install **Terraform** — here’s the rundown:

---

## **Option 1: Homebrew (Recommended)**

If you already use [Homebrew](https://brew.sh), this is the easiest and keeps Terraform up to date.

```bash
# Make sure brew itself is up to date
brew update

# Install Terraform
brew tap hashicorp/tap
brew install hashicorp/tap/terraform

# Verify install
terraform -version
```

To upgrade later:

```bash
brew upgrade hashicorp/tap/terraform
```

---

## **Option 2: Manual Binary Install**

If you don’t want to use Homebrew:

1. Go to the [Terraform releases page](https://developer.hashicorp.com/terraform/downloads).
2. Download the `.zip` for macOS (`darwin_amd64` for Intel or `darwin_arm64` for Apple Silicon).
3. Unzip:

   ```bash
   unzip terraform_<VERSION>_darwin_amd64.zip
   ```
4. Move binary into your `$PATH`:

   ```bash
   sudo mv terraform /usr/local/bin/
   ```
5. Check:

   ```bash
   terraform -version
   ```

---

## **Option 3: Using asdf (Version Manager)**

If you manage multiple tool versions, `asdf` is handy:

```bash
brew install asdf

# Add Terraform plugin
asdf plugin-add terraform https://github.com/asdf-community/asdf-hashicorp.git

# Install Terraform version
asdf install terraform latest

# Set global or project-specific version
asdf global terraform latest
```

---

✅ **Quick tip**: On **Apple Silicon (M1/M2/M3)** Macs, Homebrew automatically fetches the correct ARM64 build, so you don’t need to worry about architecture.
