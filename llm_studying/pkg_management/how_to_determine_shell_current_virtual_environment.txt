A few ways to check:

**1. Look at your prompt**

Most shells show the active venv name in parentheses at the start of your prompt:

```
(myenv) user@machine:~/project$
```

**2. Check the `VIRTUAL_ENV` environment variable**

```bash
echo $VIRTUAL_ENV
```

This prints the full path to the active venv, or nothing if none is active.

**3. Check which Python is being used**

```bash
which python
```

If a venv is active, this will show a path inside that venv (e.g., `/home/user/project/.venv/bin/python`). If not, it'll show your system Python.
