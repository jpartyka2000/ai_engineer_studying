Editable mode (also called "development mode") lets you install a Python package so that changes to the source code take effect immediately without reinstalling.

You install a package in editable mode using:

```bash
pip install -e .
```

or for a specific path:

```bash
pip install -e /path/to/package
```

**How it works:** Instead of copying the package files into your `site-packages` directory, pip creates a link back to your source directory. This means any edits you make to the source code are reflected immediately when you import the package.

**When to use it:** It's primarily useful during development when you're actively working on a package and want to test changes without running `pip install` repeatedly.

**With pyproject.toml or setup.py:** The directory you point to needs to be a valid Python package with either a `pyproject.toml` or `setup.py` file that defines the package metadata.
