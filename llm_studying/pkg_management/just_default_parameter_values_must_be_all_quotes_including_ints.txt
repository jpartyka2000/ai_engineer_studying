In justfiles, default parameter values must be quoted strings. The bare `2` isn't valid syntax.

Change this:

```yaml
create-cue-questions input="seed_questions.jsonl" output="output.jsonl" num_questions=2 num_rounds=2:
```

To this:

```yaml
create-cue-questions input="seed_questions.jsonl" output="output.jsonl" num_questions="2" num_rounds="2":
```
