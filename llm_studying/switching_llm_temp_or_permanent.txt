Great question! Here are the ways to switch to **Opus 4.5** in Claude Code:

## Quickest Method: During Your Session

Simply type this command in Claude Code:
```bash
/model opus
```

Or use the full model name:
```bash
/model claude-opus-4-5-20251101
```

You can verify the change worked by running:
```bash
/status
```

## Other Methods

**When Starting Claude Code:**
```bash
claude --model claude-opus-4-5-20251101
```

**Set as Default (Permanent):**

Add this to your `~/.claude/settings.json`:
```json
{
  "model": "claude-opus-4-5-20251101"
}
```

Or set an environment variable in your `~/.bashrc` or `~/.zshrc`:
```bash
export ANTHROPIC_MODEL="claude-opus-4-5-20251101"
```

## Important Note About Access

⚠️ **If you're on a Pro plan**, you'll need to enable and purchase extra usage to access Opus models. Opus 4.5 is more capable but also more expensive than Sonnet, so it may have different usage limits or require additional payment.

You can check your current model anytime with `/status` and switch back to Sonnet with `/model sonnet` or Haiku with `/model haiku` if needed.
