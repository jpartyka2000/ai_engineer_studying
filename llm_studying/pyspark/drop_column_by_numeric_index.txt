In Spark you can’t drop by position directly, but you can look up the column name at that index and then drop it. For example, to drop the 0-th column:

```python
# get the 0-th column’s name
col0 = df.columns[0]

# drop it
df2 = df.drop(col0)
```

Or, if you want to drop index `i` in general:

```python
i = 0   # or any other integer
to_keep = [c for idx, c in enumerate(df.columns) if idx != i]
df2 = df.select(*to_keep)
```

Both approaches let you address columns by numeric position rather than hard-coding their name.
