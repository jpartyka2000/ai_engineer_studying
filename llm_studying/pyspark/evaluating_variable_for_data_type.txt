The code you provided contains a syntax error because **`=`** is used for assignment in Python, not for comparison. For comparison, you need to use **`==`**.

Here’s the corrected version of your code:

```python
if self.raw_outcome_win_column_type == TimestampType:
    # Your code here
```

### Explanation:
- **`=`**: Used for assignment (e.g., `a = 5` assigns the value `5` to the variable `a`).
- **`==`**: Used for comparison (e.g., `if a == 5` checks if the value of `a` is equal to `5`).

In your case, you want to compare `self.raw_outcome_win_column_type` to **`TimestampType`**, so you need to use `==`.
