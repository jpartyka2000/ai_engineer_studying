Yes, you can use the `show()` function for a single column of a DataFrame in PySpark. You simply need to select that column first using the `select()` method, and then call `show()`. Here's how you can do it:

```python
df.select("column_name").show()
```

For example, if your column is called `name`:

```python
df.select("name").show()
```

This will display the values of just the `name` column.
