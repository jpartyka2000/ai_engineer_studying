You can use PySpark's built-in function `size()` to check the length of the array. For example, if your column is named `"timestamp_array"`, you can filter out rows where the array is empty like this:

```python
from pyspark.sql.functions import size, col

df_non_empty = df.filter(size(col("timestamp_array")) > 0)
```

This will return only the rows where `"timestamp_array"` has at least one element (i.e. is not an empty list).
