**`df.first()` in PySpark**

Returns the first row of a DataFrame as a `Row` object.

**Basic usage:**

```python
df = spark.createDataFrame([
    ("Alice", 30),
    ("Bob", 25),
    ("Charlie", 35)
], ["name", "age"])

row = df.first()
print(row)
# Row(name='Alice', age=30)
```

**Accessing values from the Row:**

```python
row = df.first()

# By column name
row["name"]       # 'Alice'
row["age"]        # 30

# By index
row[0]            # 'Alice'
row[1]            # 30

# As attribute
row.name          # 'Alice'
row.age           # 30
```

**Getting a single value directly:**

```python
df.select("name").first()[0]
# 'Alice'

df.select(F.max("age")).first()[0]
# 35
```

**Important notes:**

- `df.first()` is an action (triggers execution)
- Returns `None` if the DataFrame is empty
- Equivalent to `df.head()` or `df.take(1)[0]`

**Empty DataFrame handling:**

```python
empty_df = spark.createDataFrame([], schema)
result = empty_df.first()
print(result)
# None
```

**Comparison with similar methods:**

| Method | Returns |
|--------|---------|
| `df.first()` | Single `Row` or `None` |
| `df.head()` | Single `Row` or `None` |
| `df.head(n)` | List of `Row` objects |
| `df.take(n)` | List of `Row` objects |
| `df.collect()` | List of all `Row` objects |
