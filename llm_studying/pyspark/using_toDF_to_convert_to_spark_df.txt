You can convert all column names to lowercase by using the `toDF` method along with a list comprehension. For example:

```python
df_lower = df.toDF(*[col.lower() for col in df.columns])
```

### Explanation

- **`df.columns`** produces a list of column names.
- **`[col.lower() for col in df.columns]`** creates a new list with all column names converted to lowercase.
- **`df.toDF(*new_column_names)`** returns a new DataFrame with the columns renamed according to the provided list.

This approach will give you a new DataFrame `df_lower` with all column names in lowercase.
