**Accessing PySpark Row fields:**

Given a Row object:

```python
from pyspark.sql import Row

row = Row(id=1, name="Alice", age=30)
```

---

**By attribute:**

```python
row.id      # 1
row.name    # 'Alice'
row.age     # 30
```

---

**By key (dictionary-style):**

```python
row["id"]      # 1
row["name"]    # 'Alice'
row["age"]     # 30
```

---

**By index (tuple-style):**

```python
row[0]    # 1
row[1]    # 'Alice'
row[2]    # 30
```

Index order matches the order fields were defined or the column order in the DataFrame.

---

**When to use each:**

| Method | Pros | Cons |
|--------|------|------|
| `row.name` | Clean, readable | Fails if field name has spaces or special characters |
| `row["name"]` | Works with any field name | Slightly more verbose |
| `row[0]` | Useful for iteration | Fragile if column order changes |

---

**Field names with spaces or special characters:**

```python
row = Row(**{"first name": "Alice", "user-id": 123})

# Attribute access won't work
row.first name    # SyntaxError
row.user-id       # SyntaxError

# Use key access instead
row["first name"]    # 'Alice'
row["user-id"]       # 123
```

---

**Check available fields:**

```python
row.__fields__
# ('id', 'name', 'age')
```
