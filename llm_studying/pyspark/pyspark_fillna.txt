In **PySpark**, the `fillna()` method is used to fill missing or null values in a DataFrame. This method can be applied to one or more columns and can accept either a scalar value to fill all nulls or a dictionary to specify different fill values for different columns.

### Example 1: **Filling Nulls with a Scalar Value**
To fill all null values in the entire DataFrame or in specific columns with a single value:

```python
# Sample DataFrame
data = [(1, None), (None, 'B'), (3, 'C')]
columns = ['id', 'letter']
df = spark.createDataFrame(data, columns)

# Fill all nulls in the DataFrame with a value, e.g., -1 for numeric columns and 'unknown' for string columns
df_filled = df.fillna({'id': -1, 'letter': 'unknown'})

# Show the result
df_filled.show()
```

### Output:
```
+---+-------+
| id|letter |
+---+-------+
|  1|unknown|
| -1|    B  |
|  3|    C  |
+---+-------+
```

### Example 2: **Filling Nulls in Specific Columns**
You can also use `fillna()` on specific columns of the DataFrame.

```python
# Fill null values only in the 'letter' column
df_filled = df.fillna('unknown', subset=['letter'])

df_filled.show()
```

### Example 3: **Filling Nulls for Numeric Columns**
For numeric columns, you can specify numeric values to fill nulls:

```python
# Fill nulls in numeric columns with -1
df_filled = df.fillna(-1, subset=['id'])

df_filled.show()
```

### Explanation:
- **`df.fillna(value)`**: This fills all null values in the DataFrame with the specified value.
- **`df.fillna(value, subset=[columns])`**: This fills null values in only the specified columns.
- **`fillna({'col1': value1, 'col2': value2})`**: Allows filling different values for different columns by passing a dictionary.

### Key Points:
- It works similarly to the `fillna()` function in Pandas.
- It supports filling both numeric and string columns, either with scalar values or column-specific values.
