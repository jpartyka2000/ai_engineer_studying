You cannot directly use `fillna()` to fill a column with values from another column in PySpark. The `fillna()` function only allows filling with a constant value, not with values from another column. However, you can achieve this using the `when` and `col` functions from `pyspark.sql.functions` as a one-liner.

Here's how you can do it:

```python
from pyspark.sql.functions import col, when

df = df.withColumn("name", when(col("name").isNull(), col("id")).otherwise(col("name")))
```

### Explanation
- `when(col("name").isNull(), col("id"))`: This checks if the `name` column is null and replaces it with the corresponding `id` value.
- `otherwise(col("name"))`: Keeps the original `name` if it’s not null.

This approach effectively fills in missing `name` values with the corresponding `id` values in one step.
