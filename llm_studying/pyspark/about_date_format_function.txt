In PySpark, the `date_format` function is used to convert date/timestamp values to formatted strings. Here's how to use it:

```python
from pyspark.sql.functions import date_format, col

# Basic usage
df = df.withColumn("formatted_date", date_format(col("date_column"), "yyyy-MM-dd"))
```

The function takes two arguments:
1. A date/timestamp column or expression
2. A format string that specifies the output format

Common format patterns include:
- `yyyy` - 4-digit year (2023)
- `MM` - 2-digit month (01-12)
- `dd` - 2-digit day (01-31)
- `HH` - 24-hour (00-23)
- `mm` - minutes (00-59)
- `ss` - seconds (00-59)

Examples of different formats:

```python
# Various date formats
df = df.withColumn("date_only", date_format(col("timestamp"), "yyyy-MM-dd"))
df = df.withColumn("month_year", date_format(col("timestamp"), "MMM yyyy"))
df = df.withColumn("custom_format", date_format(col("timestamp"), "dd/MM/yyyy HH:mm"))
df = df.withColumn("day_of_week", date_format(col("timestamp"), "EEEE"))
```

You can also use it in SQL expressions with `expr()`:

```python
from pyspark.sql.functions import expr

df = df.withColumn("formatted_date", expr("date_format(timestamp, 'yyyy-MM-dd')"))
```

The function handles null values by returning null, and it works within aggregations or other complex expressions as well.
