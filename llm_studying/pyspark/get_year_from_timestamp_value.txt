You can use the built‑in `year()` function from `pyspark.sql.functions`. For example, if your timestamp column is named `"event_ts"`:

```python
from pyspark.sql.functions import year, col

df = df.withColumn(
    "event_year",
    year(col("event_ts"))
)
```

That will give you a new integer column `"event_year"` containing the four‑digit year extracted from each timestamp.

---

#### Alternative: using `date_format`

If you want it as a string (e.g. `"2025"`), you can also do:

```python
from pyspark.sql.functions import date_format

df = df.withColumn(
    "event_year_str",
    date_format(col("event_ts"), "yyyy")
)
```

But for most numeric‐year use‑cases, `year()` is the simplest and most direct.
