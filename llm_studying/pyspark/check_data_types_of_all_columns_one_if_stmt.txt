To test the data types of multiple columns in a PySpark DataFrame within a single `if` condition, you can use the schema of the DataFrame and check the data types of the columns using Python's **`all()`** function for logical conditions. Here's how you can do this:

### Example Code:

Let's say you want to check if two columns (`col1` and `col2`) are both of type `StringType()`.

```python
from pyspark.sql.types import StringType, IntegerType

# Define a list of columns and their expected types
columns_to_check = ['col1', 'col2']
expected_type = StringType

# Check if all specified columns match the expected data type
if all([isinstance(df.schema[col].dataType, expected_type) for col in columns_to_check]):
    print("All columns are of type StringType")
else:
    print("One or more columns are not of the expected type")
```

### Explanation:

1. **`df.schema[col].dataType`**: Retrieves the data type of a particular column in the DataFrame.
2. **`isinstance(df.schema[col].dataType, expected_type)`**: Checks if the column's data type matches the expected type (`StringType` in this case).
3. **`all()`**: The `all()` function ensures that the condition holds for all columns. If all conditions are `True`, the entire expression evaluates to `True`.

### Example with Multiple Data Types:

If you want to check different data types for different columns, you can store both columns and types in a dictionary and check them simultaneously.

```python
# Define a dictionary with column names as keys and expected data types as values
columns_and_types = {
    'col1': StringType,
    'col2': IntegerType,
}

# Check if all specified columns match the expected data type
if all([isinstance(df.schema[col].dataType, dtype) for col, dtype in columns_and_types.items()]):
    print("All columns match their expected types")
else:
    print("One or more columns do not match the expected types")
```

This method allows you to check multiple columns for their data types efficiently in a single condition.
