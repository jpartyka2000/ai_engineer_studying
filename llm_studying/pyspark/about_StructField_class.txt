**`StructField`** is a PySpark class that defines a single field (column) within a schema. It's the building block of `StructType`, which represents the full schema of a DataFrame.

**Constructor:**

```python
StructField(name, dataType, nullable=True, metadata=None)
```

**Parameters:**

- **`name`** – column name (string)
- **`dataType`** – a PySpark data type (e.g., `StringType()`, `IntegerType()`, `ArrayType()`, or even a nested `StructType()`)
- **`nullable`** – whether the field can contain null values (default `True`)
- **`metadata`** – an optional dictionary for additional information

**Example usage:**

```python
from pyspark.sql.types import StructType, StructField, StringType, IntegerType, DoubleType

schema = StructType([
    StructField("id", IntegerType(), nullable=False),
    StructField("name", StringType(), nullable=True),
    StructField("salary", DoubleType(), nullable=True, metadata={"description": "Annual salary"})
])

df = spark.createDataFrame([(1, "Alice", 75000.0), (2, "Bob", 82000.0)], schema)
```

**Accessing `StructField` properties:**

```python
field = df.schema.fields[0]

field.name        # 'id'
field.dataType    # IntegerType()
field.nullable    # False
field.metadata    # {}
```

**Common use cases:**

- Defining explicit schemas when reading data (CSV, JSON, etc.)
- Schema validation and comparison
- Programmatically building or modifying schemas
- Adding metadata for documentation or downstream processing
