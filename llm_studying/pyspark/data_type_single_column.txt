In **PySpark**, you can get the data type of a specific column in a DataFrame using the **`schema`** property and accessing the data type of the desired column.

Here's an example:

```python
# Get the schema of the DataFrame
df.printSchema()

# Access the data type of a specific column
df.schema['column_name'].dataType
```

### Explanation:
- **`printSchema()`**: This prints the schema of the DataFrame, including column names and their data types.
- **`df.schema['column_name'].dataType`**: This directly fetches the data type of a specific column (replace `'column_name'` with your actual column name). This returns a PySpark data type object, such as `StringType()`, `IntegerType()`, `TimestampType()`, etc.

You can also check all the column data types using:

```python
for field in df.schema.fields:
    print(f"Column: {field.name}, Type: {field.dataType}")
```

This iterates through all columns and prints the name and data type of each.
