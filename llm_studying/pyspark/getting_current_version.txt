You can grab it a couple of ways from within your Databricks notebook or Python script:

1. **Via the SparkSession**

   ```python
   # In a notebook cell
   print(spark.version)
   ```

   or

   ```python
   # In pure Python
   from pyspark.sql import SparkSession
   spark = SparkSession.builder.getOrCreate()
   print(spark.version)
   ```

   This will show the Spark engine version (e.g. `3.4.1`).

2. **Via the PySpark package**

   ```python
   import pyspark
   print(pyspark.__version__)
   ```

   That tells you the PySpark client library version.

3. **Via the SparkContext**

   ```python
   print(sc.version)
   ```

   if you have a `sc: SparkContext` already in scope.

Any of those will let you confirm exactly which Spark/PySpark build your notebook or job is using.
