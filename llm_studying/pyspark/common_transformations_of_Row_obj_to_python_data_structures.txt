**Common Row transformations:**

After calling `collect()`, you get a list of Row objects. Here's how to convert them:

---

**Row to dictionary:**

```python
row = df.collect()[0]
row.asDict()
# {'id': 1, 'letter': 'a'}
```

---

**All rows to list of dictionaries:**

```python
[row.asDict() for row in df.collect()]
# [{'id': 1, 'letter': 'a'}, {'id': 2, 'letter': 'b'}]
```

---

**Access Row fields:**

```python
row = df.collect()[0]

# By attribute
row.id        # 1
row.letter    # 'a'

# By index
row[0]        # 1
row[1]        # 'a'

# By key
row["id"]     # 1
```

---

**Row to tuple:**

```python
tuple(row)
# (1, 'a')
```

---

**All rows to list of tuples:**

```python
[tuple(row) for row in df.collect()]
# [(1, 'a'), (2, 'b'), (3, 'c')]
```

---

**Single column to flat list:**

```python
[row.id for row in df.collect()]
# [1, 2, 3]

# Or using select
df.select("id").rdd.flatMap(lambda x: x).collect()
# [1, 2, 3]
```

---

**To pandas DataFrame (if it fits in memory):**

```python
df.toPandas()
```

This handles the conversion automatically.
