**`df.stat` in PySpark**

The `stat` attribute provides access to statistical functions for DataFrames.

**Available methods:**

**`df.stat.corr(col1, col2)`** – Pearson correlation between two columns:

```python
df.stat.corr("age", "salary")
# 0.85
```

**`df.stat.cov(col1, col2)`** – Covariance between two columns:

```python
df.stat.cov("age", "salary")
# 12345.67
```

**`df.stat.crosstab(col1, col2)`** – Cross-tabulation (contingency table):

```python
df.stat.crosstab("department", "gender").show()
```

```
+-------------------+---+---+
|department_gender  |  F|  M|
+-------------------+---+---+
|              Sales| 10| 15|
|        Engineering| 20| 30|
+-------------------+---+---+
```

**`df.stat.freqItems(cols, support)`** – Frequent items in columns:

```python
df.stat.freqItems(["city", "product"], 0.3).show()
# Items appearing in at least 30% of rows
```

**`df.stat.approxQuantile(col, probabilities, relativeError)`** – Approximate quantiles:

```python
df.stat.approxQuantile("salary", [0.25, 0.5, 0.75], 0.01)
# [45000.0, 62000.0, 85000.0]
```

**`df.stat.sampleBy(col, fractions, seed)`** – Stratified sampling:

```python
df.stat.sampleBy("department", {"Sales": 0.1, "Engineering": 0.2}, seed=42)
```

**Note:**

These methods return either scalar values (`corr`, `cov`), DataFrames (`crosstab`, `freqItems`, `sampleBy`), or lists (`approxQuantile`).
