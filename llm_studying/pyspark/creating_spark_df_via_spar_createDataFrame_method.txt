**Creating DataFrames with `spark.createDataFrame()`**

**Basic syntax:**

```python
df = spark.createDataFrame(data, schema)
```

**From a list of tuples:**

```python
df = spark.createDataFrame([
    (1, "Alice", 30),
    (2, "Bob", 25)
], ["id", "name", "age"])
```

**From a list of dictionaries:**

```python
df = spark.createDataFrame([
    {"id": 1, "name": "Alice", "age": 30},
    {"id": 2, "name": "Bob", "age": 25}
])
```

**From Row objects:**

```python
from pyspark.sql import Row

df = spark.createDataFrame([
    Row(id=1, name="Alice", age=30),
    Row(id=2, name="Bob", age=25)
])
```

**With explicit schema:**

```python
from pyspark.sql.types import StructType, StructField, IntegerType, StringType

schema = StructType([
    StructField("id", IntegerType(), False),
    StructField("name", StringType(), True),
    StructField("age", IntegerType(), True)
])

df = spark.createDataFrame([(1, "Alice", 30), (2, "Bob", 25)], schema)
```

**From a Pandas DataFrame:**

```python
import pandas as pd

pdf = pd.DataFrame({"id": [1, 2], "name": ["Alice", "Bob"]})
df = spark.createDataFrame(pdf)
```

**When to specify schema explicitly:**

- Need specific types (e.g., `IntegerType` instead of inferred `LongType`)
- Want non-nullable columns
- Data has nulls that make inference unreliable
- Performance (avoids inference overhead on large data)
