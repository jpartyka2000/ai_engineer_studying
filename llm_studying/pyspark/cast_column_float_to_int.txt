To convert a column to **integer type** (`int`) in PySpark, you can use the `cast()` function from `pyspark.sql.functions`. Here's how you do it:

### **Example**:

```python
from pyspark.sql import functions as F

# Assuming df is your DataFrame and 'column_name' is the name of the column you want to convert
df = df.withColumn("column_name", F.col("column_name").cast("int"))

# Show the DataFrame with the converted column
df.show()
```

### **Explanation**:
- `F.col("column_name")`: References the column you want to modify.
- `.cast("int")`: Casts the column to integer type. You can also use `"long"` for 64-bit integers if needed.

If you want to handle missing values or non-numeric values in the column before casting, you may want to use `when()` to set a default value, like so:

```python
df = df.withColumn(
    "column_name",
    F.when(F.col("column_name").isNotNull(), F.col("column_name").cast("int")).otherwise(0)
)
```

This would convert all valid numeric values to `int` and set the non-numeric values to `0`.
