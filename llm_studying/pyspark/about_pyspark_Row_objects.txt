**PySpark Row Objects**

A `Row` is a single record from a DataFrame, similar to a named tuple.

**Creating Rows:**

```python
from pyspark.sql import Row

# Positional arguments
row = Row("Alice", 30)

# Named arguments
row = Row(name="Alice", age=30)
```

**Accessing values:**

```python
row = df.first()  # Row(name='Alice', age=30)

# By name
row["name"]       # 'Alice'

# By index
row[0]            # 'Alice'

# As attribute
row.name          # 'Alice'
```

**Row as a factory:**

```python
Person = Row("name", "age")
p1 = Person("Alice", 30)
p2 = Person("Bob", 25)

df = spark.createDataFrame([p1, p2])
```

**Converting Rows:**

```python
row = df.first()

# To dictionary
row.asDict()
# {'name': 'Alice', 'age': 30}

# To tuple
tuple(row)
# ('Alice', 30)
```

**Iterating over a Row:**

```python
row = df.first()

for value in row:
    print(value)

for key in row.asDict():
    print(key, row[key])
```

**Where you encounter Rows:**

- `df.first()` – returns a Row
- `df.collect()` – returns list of Rows
- `df.take(n)` – returns list of Rows
- `df.head(n)` – returns list of Rows
- Iterating over a DataFrame

**Key characteristic:**

Rows are immutable. You cannot modify values after creation.
