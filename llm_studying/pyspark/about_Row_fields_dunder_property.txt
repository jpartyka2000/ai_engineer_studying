**Check available fields in a Row object:**

```python
row.__fields__
```

This returns a tuple of field names in the Row.

---

**Example:**

```python
from pyspark.sql import Row

row = Row(id=1, name="Alice", age=30)
row.__fields__
# ('id', 'name', 'age')
```

---

**From a DataFrame:**

```python
df = spark.createDataFrame([(1, "Alice", 30)], ["id", "name", "age"])
row = df.collect()[0]

row.__fields__
# ('id', 'name', 'age')
```

---

**Practical uses:**

**Iterate over field names and values:**

```python
for field in row.__fields__:
    print(f"{field}: {row[field]}")

# id: 1
# name: Alice
# age: 30
```

**Check if a field exists:**

```python
"name" in row.__fields__    # True
"email" in row.__fields__   # False
```

**Get number of fields:**

```python
len(row.__fields__)    # 3
```

---

**Note:**

Rows created without named fields won't have `__fields__`:

```python
row = Row(1, "Alice", 30)    # Positional, no names
row.__fields__               # None
```

Only Rows with named fields (or Rows from DataFrames) will have this attribute populated.
