In PySpark, `df.schema.fields` returns a **list of `StructField` objects** that describe each column in the DataFrame.

Each `StructField` contains:
- **`name`** – the column name (string)
- **`dataType`** – the data type (e.g., `StringType()`, `IntegerType()`)
- **`nullable`** – whether the column can contain nulls (boolean)
- **`metadata`** – optional metadata (dict)

**Example:**

```python
from pyspark.sql import SparkSession
from pyspark.sql.types import *

spark = SparkSession.builder.getOrCreate()
df = spark.createDataFrame([(1, "Alice"), (2, "Bob")], ["id", "name"])

print(df.schema.fields)
# [StructField('id', LongType(), True), StructField('name', StringType(), True)]

# Accessing individual properties:
for field in df.schema.fields:
    print(f"{field.name}: {field.dataType}")
# id: LongType()
# name: StringType()
```

This is useful for programmatically inspecting or iterating over a DataFrame's schema.
