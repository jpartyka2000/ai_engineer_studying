In PySpark, you can extract the **calendar quarter** from a timestamp column using the built-in function **`quarter()`**. This function will return the quarter number (1, 2, 3, or 4) based on the calendar year.

### Code to Extract the Quarter:

```python
from pyspark.sql import functions as F

# Assuming df is your DataFrame and 'timestamp_column' is the column with the timestamp data
df = df.withColumn('quarter', F.quarter(F.col('timestamp_column')))

# Show the updated DataFrame with the new 'quarter' column
df.show()
```

### Explanation:
- **`F.quarter()`**: This function extracts the quarter from a date or timestamp column. It assumes **calendar quarters**, where:
  - Q1 = January to March,
  - Q2 = April to June,
  - Q3 = July to September,
  - Q4 = October to December.

The new column `'quarter'` will contain the respective quarter (1, 2, 3, or 4) for each timestamp in the dataset.

### Example:
If you have a `timestamp_column` with values like:
- `2023-01-15` → Quarter 1
- `2023-04-10` → Quarter 2
- `2023-08-20` → Quarter 3

The output will correctly assign each row to the corresponding quarter.

This approach assumes that you're working with **calendar quarters**, as most businesses typically do.
