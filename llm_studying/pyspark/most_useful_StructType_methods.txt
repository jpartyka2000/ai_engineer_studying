Here's a breakdown of the most useful `StructType` methods and properties for inspecting schemas:

---

**`schema.fields`**

Returns a list of `StructField` objects representing each column.

```python
schema.fields
# [StructField('id', IntegerType(), False), StructField('name', StringType(), True)]

# Useful for iteration:
for field in schema.fields:
    print(field.name, field.dataType)
```

---

**`schema.fieldNames()`**

Returns a list of column names as strings.

```python
schema.fieldNames()
# ['id', 'name', 'age']

# Handy for quick column checks:
if "email" in schema.fieldNames():
    print("Email column exists")
```

---

**`schema["name"]`**

Returns the `StructField` for a specific column by name.

```python
schema["name"]
# StructField('name', StringType(), True)

# Access its properties:
schema["name"].dataType    # StringType()
schema["name"].nullable    # True
```

---

**`schema.simpleString()`**

Returns a compact, human-readable string representation of the schema.

```python
schema.simpleString()
# 'struct<id:int,name:string,age:int>'

# Useful for logging or quick inspection
```

---

**`schema.json()`**

Returns a JSON string representation of the schema. Useful for serialization, storage, or sharing schemas between systems.

```python
schema.json()
# '{"fields":[{"metadata":{},"name":"id","nullable":false,"type":"integer"},{"metadata":{},"name":"name","nullable":true,"type":"string"}],"type":"struct"}'

# Round-trip: recreate schema from JSON
from pyspark.sql.types import StructType
restored_schema = StructType.fromJson(json.loads(schema.json()))
```

---

**Summary table:**

| Method/Property | Returns | Use case |
|-----------------|---------|----------|
| `schema.fields` | List of `StructField` | Iterate over columns with full metadata |
| `schema.fieldNames()` | List of strings | Get column names only |
| `schema["col"]` | Single `StructField` | Inspect a specific column |
| `schema.simpleString()` | Compact string | Logging, debugging |
| `schema.json()` | JSON string | Serialization, schema storage |
