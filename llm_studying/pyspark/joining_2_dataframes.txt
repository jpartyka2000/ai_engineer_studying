You can join two Spark DataFrames using the `join()` method. For example, if you have two DataFrames `df1` and `df2` and want to join them on a common column (say `"id"`), you can do:

```python
joined_df = df1.join(df2, on="id", how="inner")
```

In this snippet:
- **`on="id"`** specifies the column to join on (you can also use a list of columns).
- **`how="inner"`** specifies the type of join. Other options include `"left"`, `"right"`, and `"outer"`.

You can also use conditions for more complex joins. For example:

```python
from pyspark.sql.functions import col

joined_df = df1.join(df2, df1["id"] == df2["user_id"], "inner")
```

This will join `df1` and `df2` where `df1.id` equals `df2.user_id` using an inner join.
