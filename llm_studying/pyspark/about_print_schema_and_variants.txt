Yes, in PySpark, you can print the schema of a DataFrame along with the data types for each column using the `.printSchema()` method. This method displays the structure of the DataFrame, including column names and their associated data types.

Here’s an example:

```python
from pyspark.sql import SparkSession

# Initialize Spark session
spark = SparkSession.builder.appName("SchemaExample").getOrCreate()

# Sample DataFrame
data = [("Alice", 25), ("Bob", 30), ("Carol", 35)]
columns = ["Name", "Age"]
df = spark.createDataFrame(data, schema=columns)

# Print schema with data types
df.printSchema()
```

Output:
```
root
 |-- Name: string (nullable = true)
 |-- Age: integer (nullable = true)
```

### Including More Details
If you want more detailed output (e.g., data type descriptions, nullable info, etc.), you can use the `.schema` attribute:

```python
# Access schema details programmatically
for field in df.schema.fields:
    print(f"Column: {field.name}, Type: {field.dataType}, Nullable: {field.nullable}")
```

This would print:
```
Column: Name, Type: StringType, Nullable: True
Column: Age, Type: IntegerType, Nullable: True
```

### Summary:
- **`df.printSchema()`**: Quick and simple for viewing schema with data types.
- **`df.schema.fields`**: More programmatic access to each column's details, including data types and nullability.
