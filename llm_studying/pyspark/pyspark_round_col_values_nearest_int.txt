To round all values in a column of a PySpark DataFrame to the nearest integer, you can use the `round()` function from `pyspark.sql.functions`. This function allows you to round numerical values to a specified number of decimal places, and setting it to `0` will round to the nearest integer.

### Here’s how to do it:

```python
from pyspark.sql.functions import round, col

# Round the values in the column 'your_column_name' to the nearest integer
df = df.withColumn("your_column_name", round(col("your_column_name"), 0))

# Display the result
df.show()
```

### Explanation
- `round(col("your_column_name"), 0)`: This rounds the values in `your_column_name` to 0 decimal places (i.e., to the nearest integer).
- `withColumn()`: This updates the column with the rounded values.

After running this code, all values in the specified column will be rounded to the nearest integer.
