In **PySpark**, the equivalent of the `.astype()` function in Pandas for casting columns to different data types is the **`cast()`** method. The `cast()` method can be applied to individual columns to change their data type. If you want to cast multiple columns, you can use the `withColumn()` function repeatedly or apply `select()` to specify the new types for each column.

### Example of `cast()` in PySpark:

#### 1. **Casting a Single Column:**
```python
from pyspark.sql import functions as F

# Sample DataFrame
data = [(1, '2023-01-01'), (2, '2023-02-01'), (3, '2023-03-01')]
columns = ['id', 'date_str']
df = spark.createDataFrame(data, columns)

# Cast 'date_str' from string to date type
df_casted = df.withColumn('date_str', F.col('date_str').cast('date'))

# Show the result
df_casted.show()
```

#### 2. **Casting Multiple Columns:**
You can use `withColumn()` multiple times to cast several columns:

```python
# Cast multiple columns ('id' to string, 'date_str' to date)
df_casted = df.withColumn('id', F.col('id').cast('string')) \
              .withColumn('date_str', F.col('date_str').cast('date'))

df_casted.show()
```

### Data Types Supported by `cast()`:
- `'string'`
- `'int'` or `'integer'`
- `'float'`
- `'double'`
- `'boolean'`
- `'date'`
- `'timestamp'`

### Summary:
- **Pandas `.astype()`** is used to change the type of a column.
- **PySpark `.cast()`** is the equivalent in PySpark and can be applied via `withColumn()` or `select()`.

This approach works similarly to `.astype()` in Pandas, allowing you to convert column data types in a distributed PySpark DataFrame.
