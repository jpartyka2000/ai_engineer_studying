To get the first **50 rows** from a PySpark DataFrame, you can use the `.limit(50)` method and then convert it to Pandas or collect the rows as a list.

### **Method 1: Using `.limit(50).show()` (Display Only)**
```python
df.limit(50).show()
```
- This will **display** the first 50 rows but **won't return** them as a list or DataFrame.

---

### **Method 2: Using `.limit(50).collect()` (Returns List of Rows)**
```python
rows = df.limit(50).collect()
for row in rows:
    print(row)
```
- `.collect()` **retrieves the rows to the driver** as a list of Row objects.

---

### **Method 3: Using `.limit(50).toPandas()` (Convert to Pandas DataFrame)**
```python
df_pandas = df.limit(50).toPandas()
print(df_pandas)
```
- Converts the first 50 rows to a **Pandas DataFrame** for easier analysis.
- **Best for small data** (avoid using it on large datasets due to memory constraints).
