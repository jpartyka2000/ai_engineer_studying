To convert the unique values of a column in PySpark into a Python list, you can follow these steps:

### Code Example:

```python
# Collect unique values into a list
unique_values_list = df.select('column_name').distinct().rdd.flatMap(lambda x: x).collect()

# Print the list
print(unique_values_list)
```

### Explanation:

1. **`select('column_name')`**: Selects the column from the DataFrame.
2. **`distinct()`**: Removes duplicates, leaving only unique values.
3. **`rdd`**: Converts the DataFrame into an RDD (Resilient Distributed Dataset).
4. **`flatMap(lambda x: x)`**: Flattens the RDD structure, turning each row into a simple value (instead of a Row object).
5. **`collect()`**: Gathers the result to the driver as a Python list.

This will give you a Python list containing all the unique values from the specified column.
