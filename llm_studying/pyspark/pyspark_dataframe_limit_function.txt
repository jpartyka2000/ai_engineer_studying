The `limit()` function in PySpark DataFrames is used to restrict the number of rows returned from a DataFrame. It allows you to specify the maximum number of rows you'd like to retrieve, which is useful when you want to inspect a sample or subset of your data without processing the entire DataFrame.

### How to Use `limit()`
Here's a basic example:
```python
df_limited = df.limit(10)
df_limited.show()
```
In this case, `df.limit(10)` will return a new DataFrame containing at most 10 rows from the original DataFrame `df`.

### Key Points
- **Does not modify the original DataFrame**: The `limit()` function returns a new DataFrame with the specified number of rows; it does not alter the original DataFrame.
- **Performance**: While `limit()` restricts the number of rows returned, the full dataset might still be processed internally before applying the limit, depending on subsequent operations or actions.
- **Common Usage**: It’s often used for quickly inspecting data or when exporting or sampling a subset of data for testing purposes.

In summary, `limit()` helps to control the number of rows in a DataFrame, making it easier to handle and view a smaller portion of the data.
