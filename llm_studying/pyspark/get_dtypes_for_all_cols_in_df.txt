You can quickly get the data types for all columns in your DataFrame using the `dtypes` attribute. This returns a list of tuples, where each tuple contains the column name and its type as a string. For example:

```python
print(df.dtypes)
```

Alternatively, if you need more detailed schema information (including nullable information and complex types), you can use:

```python
df.printSchema()
```

Both methods are useful for inspecting the types of columns in your DataFrame.
