In PySpark, you can filter columns in a DataFrame based on their datatype using the **`df.dtypes`** or **`df.schema`** attribute. The `dtypes` attribute returns a list of tuples where each tuple contains a column name and its datatype. By checking the datatype, you can filter out columns of a specific type (e.g., `TimestampType` for datetime columns).

Here’s an example of how to filter columns based on datatype, such as **datetime columns**:

### Example Code:
```python
from pyspark.sql.types import TimestampType

# Filter out columns that are of type TimestampType (datetime columns)
datetime_columns = [col for col, dtype in df.dtypes if dtype == 'timestamp']

# Show the filtered datetime columns
print(datetime_columns)

# Select only the datetime columns from the DataFrame
df_datetime_filtered = df.select(*datetime_columns)

# Show the DataFrame with only datetime columns
df_datetime_filtered.show()
```

### Explanation:
1. **`df.dtypes`**: Returns a list of tuples, where each tuple is in the format `(column_name, data_type)`.
2. **Filtering for 'timestamp'**: We filter columns where the datatype is `'timestamp'`, which corresponds to `TimestampType` in PySpark.
3. **Selecting Filtered Columns**: Use `df.select(*datetime_columns)` to select only the datetime columns from the DataFrame.

### Generalization:
You can generalize this for any other datatype by checking for different types such as `'int'`, `'string'`, or `'double'`. If you want to filter out columns by another type, replace `'timestamp'` with the relevant type in the code.

For example, to filter **integer columns**:

```python
integer_columns = [col for col, dtype in df.dtypes if dtype == 'int']
```

This will allow you to filter DataFrame columns based on their datatypes in PySpark.
