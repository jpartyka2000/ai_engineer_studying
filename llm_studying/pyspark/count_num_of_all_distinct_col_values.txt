To count the number of times each distinct value appears in a PySpark DataFrame column, you can use the `groupBy()` method followed by `count()`. This will give you the frequency of each distinct value in the specified column.

Here's how you can do it:

### Step-by-Step Example

```python
# Import necessary functions
from pyspark.sql import functions as F

# Group by the column of interest and count occurrences
value_counts_df = df.groupBy("your_column_name").count()

# Display the result
value_counts_df.show()
```

### Explanation
- `groupBy("your_column_name")`: Groups the DataFrame by the specified column, where `your_column_name` is the column for which you want to count the occurrences of each distinct value.
- `.count()`: Counts the number of rows in each group, effectively giving the frequency of each distinct value.

### Example Output
If your DataFrame column `your_column_name` contains values like ["apple", "orange", "apple", "banana"], the output will look like this:

```
+---------------+-----+
|your_column_name|count|
+---------------+-----+
|           apple|    2|
|          orange|    1|
|          banana|    1|
+---------------+-----+
```

This shows how many times each distinct value appears in the column.
