To get a list of the **data types** of all columns in a **PySpark DataFrame**, you can use the `dtypes` attribute. This returns a list of tuples where each tuple contains the column name and its associated data type.

### Example:

```python
# Sample DataFrame
data = [(1, '2023-01-01', 3.5), (2, '2023-02-01', 4.0), (3, '2023-03-01', 5.1)]
columns = ['id', 'date_str', 'value']
df = spark.createDataFrame(data, columns)

# Get the list of data types for all columns
data_types = df.dtypes

# Show the result
print(data_types)
```

### Output:
```
[('id', 'bigint'), ('date_str', 'string'), ('value', 'double')]
```

### Explanation:
- **`df.dtypes`**: This returns a list of tuples. Each tuple contains the column name as the first element and the data type as the second element.
- **Output**: The data types for the columns `id`, `date_str`, and `value` are shown as `bigint`, `string`, and `double`, respectively.

This approach allows you to quickly retrieve and review the data types of all columns in a PySpark DataFrame.
