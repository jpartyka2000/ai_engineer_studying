To get the first element from a string list (array) in each row of a PySpark DataFrame, you can use array indexing with the `getItem` function:

```python
from pyspark.sql.functions import col

# Method 1: Using getItem() function
df = df.withColumn("first_element", col("array_column").getItem(0))

# Method 2: Alternative syntax with square brackets
df = df.withColumn("first_element", col("array_column")[0])
```

Both approaches access the first element of the array (remember that arrays in PySpark are zero-indexed, just like in Python).

To handle potential null values or empty arrays, you might want to add a safety check:

```python
from pyspark.sql.functions import col, when, size, lit

df = df.withColumn(
    "first_element",
    when(size(col("array_column")) > 0, col("array_column")[0]).otherwise(lit(None))
)
```

This will return null for any rows where the array is empty, preventing "array index out of bounds" errors.
