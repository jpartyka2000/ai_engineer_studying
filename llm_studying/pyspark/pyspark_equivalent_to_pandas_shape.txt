In PySpark you don't have a single property like `.shape` in Pandas, but you can obtain the dimensions by using:

- `df.count()` for the number of rows, and  
- `len(df.columns)` for the number of columns.

For example:

```python
num_rows = df.count()
num_cols = len(df.columns)
print(f"Shape: ({num_rows}, {num_cols})")
```
