**LOESS (Locally Estimated Scatterplot Smoothing)**
LOESS (a.k.a. LOWESS) is a non-parametric method for fitting a smooth curve through noisy one-dimensional data by doing many small, local weighted regressions. It lets the data “speak for itself” without imposing a global functional form.

---

### Key Concepts

1. **Local neighborhoods**

   * For each target point $x_0$, LOESS selects the nearest fraction $\alpha$ of all data points (the “span”).
   * Points closer to $x_0$ are given higher weight, typically via a **tricube kernel**

     $$
       w(d) = (1 - |d|^3)^3,\quad |d|\le1.
     $$

2. **Weighted least squares**

   * In each neighborhood you fit a low-degree polynomial (usually linear or quadratic) by weighted least squares, using the kernel weights.

3. **Stitching together**

   * Repeat that local fit at each $x_0$ (or on a fine grid) to build a smooth curve that adapts to local trends.

4. **Tuning parameters**

   * **Span** $\alpha\in(0,1]$: fraction of data used in each local fit. Smaller → more flexible (but noisier); larger → smoother (but may miss local features).
   * **Degree**: typically 1 (linear LOESS) or 2 (quadratic).

---

### When to use LOESS

* Exploring noisy scatterplot data or time series when you have **no strong prior** on the global shape.
* You want a smooth trend estimate without committing to, say, a global polynomial of fixed degree.
* Datasets of moderate size (hundreds or low thousands of points).

---

### Python Example with `statsmodels`

```python
import numpy as np
import matplotlib.pyplot as plt
from statsmodels.nonparametric.smoothers_lowess import lowess

# 1. Generate synthetic noisy sine data
np.random.seed(0)
x = np.linspace(0, 10, 200)
y = np.sin(x) + 0.3 * np.random.randn(len(x))

# 2. Apply LOESS (frac=0.2, linear fits)
#    `frac` = 20% of points in each local fit
smoothed = lowess(endog=y, exog=x, frac=0.2, it=0)

# `smoothed` is an array of shape (200,2): [x, y_smoothed]
x_smooth, y_smooth = smoothed[:,0], smoothed[:,1]

# 3. Plot raw vs. LOESS-smoothed
plt.figure(figsize=(8,4))
plt.scatter(x, y, alpha=0.4, label='Noisy data')
plt.plot(x_smooth, y_smooth, 'r-', linewidth=2, label='LOESS (frac=0.2)')
plt.legend()
plt.title('LOESS Smoothing Example')
plt.xlabel('x')
plt.ylabel('y')
plt.show()
```

**What this does:**

* Uses 20% of the data in each local regression (`frac=0.2`).
* Fits simple linear polynomials weighted by proximity (tricube kernel).
* Returns a smooth curve that follows the sine trend but damps out noise.

---

### Pros & Cons

| Pros                               | Cons                                    |
| ---------------------------------- | --------------------------------------- |
| No global model assumption         | Computationally expensive for large $n$ |
| Captures complex local structure   | Edge behavior can be unstable           |
| Intuitive tuning via span & degree | Doesn’t extrapolate beyond data range   |

LOESS is a go-to tool for exploratory data analysis when you need a flexible, smooth fit without a predetermined functional form.
