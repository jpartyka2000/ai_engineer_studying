Based on the code, the two measures specifically designed for determining **feature uniqueness** (i.e., how distinct a feature is from other features in the dataset) are:

## 1. `redundancy_score`
This is the most direct measure of uniqueness. It calculates:

```
score = 1.0 - max_correlation_with_other_features
```

- Computes the Pearson correlation between each feature and all other features
- Takes the maximum absolute correlation
- Subtracts from 1, so **higher score = less redundant = more unique**

A feature with a high redundancy score provides information that isn't already captured by other features.

## 2. `vif_score` 
Variance Inflation Factor measures multicollinearity:

```
score = 1 / VIF = 1 - R²
```

Where R² is from regressing each feature against all other features. 

- **Higher score = lower multicollinearity = more unique**
- A feature that can be well-predicted by other features has high VIF (low score)
- A feature that's independent of others has low VIF (high score)

---

**Key difference:** 
- `redundancy_score` only considers pairwise correlations (simpler, catches linear relationships with individual features)
- `vif_score` considers how well a feature can be predicted by *all* other features combined (catches more complex redundancy patterns)

For finding unique features, you could use:
```python
results = analyzer.run_analysis(measures=[
    ("redundancy_score", 1.0),
    ("vif_score", 1.0),
])
```
