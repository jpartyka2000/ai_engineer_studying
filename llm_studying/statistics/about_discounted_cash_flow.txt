## DCF = Discounted Cash Flow

**DCF** is a valuation method that calculates what something is worth today based on its **future cash flows**, adjusted for the time value of money.

## Core Concept:

```python
# Money today > Money tomorrow (due to inflation, risk, opportunity cost)
$100 today > $100 next year

# DCF calculates: What are future cash flows worth RIGHT NOW?
Present_Value = Future_Cash_Flow / (1 + discount_rate)^years
```

## Simple Example:

```python
# You'll receive $1000 in 1 year
# Discount rate = 10% (your required return)

present_value = $1000 / (1.10)^1 = $909

# That future $1000 is worth $909 today
# Because $909 invested at 10% becomes $1000 in a year
```

## Full DCF Calculation:

```python
# Business projecting 5 years of cash flows
cash_flows = {
    'year_1': 100_000,
    'year_2': 120_000,
    'year_3': 150_000,
    'year_4': 180_000,
    'year_5': 200_000
}
discount_rate = 0.10  # 10%

# Calculate present value of each year
dcf_value = 0
for year, cash_flow in enumerate(cash_flows.values(), 1):
    pv = cash_flow / (1 + discount_rate) ** year
    dcf_value += pv
    print(f"Year {year}: ${cash_flow:,} → ${pv:,.0f}")

# Year 1: $100,000 → $90,909
# Year 2: $120,000 → $99,174
# Year 3: $150,000 → $112,697
# Year 4: $180,000 → $122,943
# Year 5: $200,000 → $124,184
# Total DCF Value: $549,907
```

### 1. **Company Valuation**

### 2. **Project Evaluation**
```python
# Should we build this new factory?
initial_investment = -$10M
annual_cash_flows = [$2M, $3M, $3M, $3M, $3M]

NPV = initial_investment + dcf(annual_cash_flows)
# If NPV > 0 → Project creates value
```


## Key Components:

1. **Cash Flow Projections** - Future money coming in
2. **Discount Rate** - Your required return (higher = riskier)
3. **Terminal Value** - Value at the end of projection period
4. **Time Period** - How far to project

## The Intuition:

```python
# Would you rather have:
option_A = "$100 today"
option_B = "$110 in one year"

# DCF helps answer this:
pv_of_option_B = 110 / 1.10 = $100  # At 10% discount rate
# They're equivalent!

# But if your discount rate is 15% (higher risk/return requirement):
pv_of_option_B = 110 / 1.15 = $95.65
# Now option A is better!
```

**Bottom line**: DCF translates future money into today's dollars, answering "What's it worth NOW?" - fundamental to all investment decisions, from buying stocks to evaluating projects.
