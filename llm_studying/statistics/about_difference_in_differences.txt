## DiD = Difference-in-Differences

**Difference-in-Differences** is a statistical technique that measures the **causal effect of a treatment** by comparing changes over time between a treatment group and control group.

## The Basic Idea:

```python
# You have two groups, two time periods
# Treatment group: gets the intervention
# Control group: doesn't get it

DiD_effect = (Treatment_After - Treatment_Before) - (Control_After - Control_Before)
#            ^^^^^ Change in treatment group ^^^^^   ^^^^ Change in control group ^^^^
```

## Simple Example:

```python
# Measuring impact of a new training program on sales

# Before training | After training
Treatment: $1000   | $1500  (Change: +$500)
Control:   $1100   | $1300  (Change: +$200)

DiD_effect = $500 - $200 = $300
# The training caused a $300 increase in sales
```

## Visual Example:

```
Sales
  ^
1500|              . Treatment (after)
1300|         ..... Control (after)
1100|    ..... Control (before)
1000| . Treatment (before)
    +------------------>
     Before   After   Time

DiD = (1500-1000) - (1300-1100) = 300
```

## Why Use DiD?

It controls for:
1. **Time trends** - Both groups might naturally improve over time
2. **Group differences** - Treatment group might always perform better
3. **Isolates the treatment effect** - What's actually due to the intervention

## Real-World Example:

```python
# Analyzing minimum wage increase impact
df = pd.DataFrame({
    'state': ['NJ', 'NJ', 'PA', 'PA'],  # NJ raised min wage, PA didn't
    'period': ['before', 'after', 'before', 'after'],
    'employment': [20.44, 21.03, 23.33, 21.17]
})

# Calculate DiD
nj_change = 21.03 - 20.44  # +0.59
pa_change = 21.17 - 23.33  # -2.16
did_effect = nj_change - pa_change  # 2.75

# Min wage increase led to 2.75 higher employment (relative to PA)
```

## Key Assumptions:
- **Parallel trends** - Groups would have changed similarly without treatment
- **No spillover** - Treatment doesn't affect control group
- **Same external shocks** - Both groups face same environment

**Bottom line**: DiD isolates the causal effect of an intervention by comparing how the treatment and control groups changed differently over time - it "differences out" both time trends and group characteristics.
