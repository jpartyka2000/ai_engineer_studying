Yes, there is a quick way to expand the width of all columns in Excel to fit the entire content, including the column names. Here's how you can do it:

### Steps:
1. **Select All Columns**: 
   - Click the triangle at the top-left corner of the worksheet (between column "A" and row "1") to select the entire sheet.
   
2. **AutoFit Column Width**: 
   - Once all columns are selected, double-click on the boundary between any two column headers (e.g., between column "A" and column "B").
   - Alternatively, after selecting all columns, right-click on any of the selected column headers and choose **"AutoFit Column Width"** from the menu.

This will automatically resize all the columns to fit the widest content in each column, including the column headers.

### Shortcut:
- You can also press **Ctrl + A** to select all columns and then double-click on any column header boundary to autofit all columns.

This method will ensure that you can see the entire column name and content in each column without manually adjusting the width of individual columns.
