**`asyncio`** and **`aiohttp`** serve complementary but distinct roles in Python’s async ecosystem:

1. **Purpose**

   * **`asyncio`** is a **core library** for writing concurrent code using the `async`/`await` syntax. It provides the **event loop**, **coroutines**, **Tasks**, **Futures**, synchronization primitives (locks, queues, events), and utilities for scheduling I/O-bound or timer-based work.
   * **`aiohttp`** is a **third-party HTTP library** built **on top of `asyncio`** that makes it easy to write **asynchronous HTTP clients** and **servers**.

2. **Functionality**

   * With `asyncio` you can orchestrate any kind of async workload: network I/O (sockets), subprocess spawning, timers, or even CPU-bound work offloaded to threads/processes.
   * With `aiohttp` you get high-level APIs for sending HTTP requests (`ClientSession`) or building web servers (`web.Application`) where each request handler can be an `async def` coroutine. Under the hood, it uses `asyncio`’s event loop and non-blocking sockets to maximize throughput.

3. **Usage**

   ```python
   import asyncio

   async def tick():
       while True:
           print("tick")
           await asyncio.sleep(1)

   asyncio.run(tick())
   ```

   Versus:

   ```python
   import aiohttp
   import asyncio

   async def fetch(url):
       async with aiohttp.ClientSession() as session:
           async with session.get(url) as resp:
               return await resp.text()

   async def main():
       html = await fetch("https://example.com")
       print(html[:100])

   asyncio.run(main())
   ```

4. **Dependencies**

   * `asyncio` is part of Python’s **standard library** (no install needed).
   * `aiohttp` must be installed separately (`pip install aiohttp`) and relies on `asyncio` under the hood.

---

**In short:**

* Use **`asyncio`** for the foundational building blocks of asynchronous programming in Python.
* Use **`aiohttp`** when you specifically need **async HTTP** client or server capabilities, leveraging `asyncio`’s concurrency under the covers.
